Falls die Mindestzykluszeit größer als die Default Bus Zykluszeit ist, entspricht die Aktualisierungszeit der Mindestzykluszeit, andernfalls entspricht die Aktualisierungszeit der Default Bus Zykluszeit.
Die minimale Aktualisierungszeit beträgt 1 ms.
Beispiel 1: Einspeisung, Servo
Default Bus Zykluszeit = KGV(250 µs, 125 µs) = 250 µs
Mindestzykluszeit = 4 * MAX(250 µs, 125 µs) = 4 * 250 µs = 1 ms
-> Aktualisierungszeit = 1 ms
Beispiel 2: Einspeisung, Servo, Vektor
Default Bus Zykluszeit = KGV(250 µs, 125 µs, 400 µs) = 2 ms
Mindestzykluszeit = 4 * MAX(250 µs, 125 µs, 400 µs) = 4 * 400 µs = 1.6 ms
-> Aktualisierungszeit = 2 ms

Vorsicht: Bei aktivierter "Steuerungshoheit von PC" ist dieser Binektoreingang unwirksam.

Hinweis: Dieses Bit dient dazu, bei Ausfall der Steuerung eine Reaktion bei den Antrieben auszulösen (F07220). Falls keine Steuerung vorhanden ist, sollte Binektoreingang p0854 = 1 gesetzt werden.
