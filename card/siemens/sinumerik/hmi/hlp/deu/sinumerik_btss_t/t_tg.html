Baustein T/TG: Werkzeugdaten: Schleifspezifische Daten

Für Schleifwerkzeuge sind spezielle Daten notwendig. Im Baustein TG sind diese speziellen Daten zusammengefasst. Die Werkzeugdaten sind über einzelne mehrzeilige Variablen adressierbar. Der Variablenzeilenindex entspricht der T-Nummer. Wird auf nicht vorhandene T-Nummern zugegriffen, wird dies negativ quittiert. Welche T-Nummern gültig sind, kann dem Baustein Werkzeugverzeichnis (TV) im zugehörigen Bereich T entnommen werden.

conntectPar Verkettungsvorschrift Schleifpara. $TC_TPG2
Spaltenindex:
2 		Zeilenindex:
Werkzeugnummer
Werkzeugnummer 		Max. Zeilenindex:
SLMAXTOOLNUMBER
Die Variable legt fest, welche WZ-Parameter von Schneide 2 (D2) und Schneide1 (D1) miteinander verkettet sein sollen.
Wird der Wert eines verketteten Parameters geändert, wird automatisch der gleiche Parameter der anderen Schneide mit dem neuen Wert überschrieben.

Wertebereich:
Sind die folgenden Bits gesetzt, werden die entsprechenden Parameter von D1 und D2 verkettet:
Bit0: Werkzeugtyp
Bit2: Geometrie Länge1
Bit3: Geometrie Länge2
Bit4: Geometrie Länge3
Bit11: Verschleiß Länge1
Bit12: Verschleiß Länge2
Bit13: Verschleiß Länge3
Bit20: Basismaß / Adaptermaß Länge1
Bit21: Basismaß / Adaptermaß Länge2
Bit22: Basismaß / Adaptermaß Länge3
Der Wert wird intern als Integer gespeichert.
Typ
TYPE_DOUBLE 		Einheit
keine Einheit 		Standardwert
 Minimalwert
 Maximalwert
Zugriffspfad HMI: /Tool/GrindingData/conntectPar[u<Werkzeugber.>, <Zeilenindex>]

Spaltenindex:

Die Länge setzt sich zusammen aus Geometrie + Verschleiß + Basis, abhängig vom WZ-Typ.

keine Einheit 		Standardwert
 Minimalwert
 Maximalwert
