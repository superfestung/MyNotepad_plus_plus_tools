Baustein T/MTUD: Multitooldaten, anwenderdefinierte Daten

Dieser Baustein enthält anwenderdefinierten Daten der Multitooldaten.
 
userData OEM-definierte Multitool-Parameter $TC_MTPCx[y] x=ParamNo y=MultitoolNo
Spaltenindex:
OEM-Multitool-Parameternummer 		Zeilenindex:
Multitoolnummer
Multitoolnummer 		Max. Zeilenindex:
32000
OEM-definierte Parameter für die Multitools für alle zulässige Datentypen, außer dem Datentyp "String".
Die OEM-Daten werden mit den MD-Einstellungen MD-Bits MD18080 $MN_MM_TOOL_MANAGEMENT_MASK, Bit0=1, Bit2=1, Bit10=1 und MD18192 $MN_MM_NUM_CC_MULTITOOL_PARAM>0 bereitgestellt.
Der Datentypen der einzelnen Parameter wird durch das MD18193 $MN_MM_TYPE_CC_MULTITOOL_PARAM festgelegt und kann aus dem File _N_COMPLETE_TOA_ACX ermittelt werden.
Die Übertragung im BTSS erfolgt immer als TYPE_DOUBLE.
Wird so auf einen OEM-definierten Multitool-Parameter lesend oder schreibend zugegriffen, der vom Datentyp "String" ist, wird ein Fehler-String zurückgegeben.
