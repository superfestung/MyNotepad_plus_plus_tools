Baustein T/TUMD: Werkzeugdaten: Magazin-Anwenderdaten

Dieser Baustein enthält die Magazin-Anwenderdaten (Double)
 
userDataDouble OEM-definierte Magazinparameter $TC_MAPCx[y] x = ParameterNo y = MagazineNo
Spaltenindex:
Magazinnummer 		Zeilenindex:
Parameternummer
Nummer des anwenderdefinierten Parameters 		Max. Zeilenindex:
numMagParams_u
Die Variable liefert OEM-definierte Parameter für die Magazine für alle zulässige Datentypen, außer dem Datentyp "String".
Die OEM-Daten werden mit den MD-Einstellungen MD-Bits $MN_MM_TOOL_MANAGEMENT_MASK, Bit0=1, Bit2=1 und MD18090 $MN_MM_NUM_CC_MAGAZINE_PARAM>0 bereitgestellt.
Der Datentyp der einzelnen Parameter wird durch das MD18091 $MN_MM_TYPE_CC_MAGAZINE_PARAM festgelegt und kann aus dem File _N_COMPLETE_TOA_ACX ermittelt werden.
Die Übertragung im BTSS erfolgt immer als TYPE_DOUBLE.
Wird so auf einen OEM-definierten Magazin-Parameter lesend oder schreibend zugegriffen, der vom Datentyp "String" ist, wird ein Fehler -String zurückgegeben.
Typ
TYPE_DOUBLE 		Einheit
keine Einheit 		Standardwert
0 		Minimalwert
 Maximalwert
Zugriffspfad HMI: /Tool/MagazineDescription/userDataDouble[u<Werkzeugber.>, c<Spaltenindex>, <Zeilenindex>]
