Baustein T/TUPD: Werkzeugdaten: Magazinplatz-Anwenderdaten

Dieser Baustein enthält die Magazinplatz-Anwenderdaten (Double)
 
userPlaceDataDouble Magazinplatz-Anwenderdaten $TC_MPPCx[y,z] x=ParamNo y=MagazineNo z=MagPlaceNo
Spaltenindex:
Magazinnummer 		Zeilenindex:
Wert berechnen
Nummer des anwenderdefinierten Parameters + numMagLocParams_u * (Nummer des Magazinplatzes - 1) 		Max. Zeilenindex:
numMagLocParams_u * magNrPlaces
Die Variable liefert OEM-definierte Parameter für die Magazinplätze für alle zulässige Datentypen, außer dem Datentyp "String".
Die OEM-Daten werden mit den MD-Einstellungen MD-Bits $MN_MM_TOOL_MANAGEMENT_MASK, Bit0=1, Bit2=1 und MD18092 $MN_MM_NUM_CC_MAGLOC_PARAM>0 bereitgestellt. )
Der Datentyp der einzelnen Parameter wird durch das MD18093 $MN_MM_TYPE_CC_MAGLOC_PARAM festgelegt und kann aus dem File _N_COMPLETE_TOA_ACX ermittelt werden.
Die Übertragung im BTSS erfolgt immer als TYPE_DOUBLE.
Wird so auf einen OEM-definierten Magazinplatz-Parameter lesend oder schreibend zugegriffen, der vom Datentyp "String" ist, wird ein Fehler-String zurückgegeben.
Typ
TYPE_DOUBLE 		Einheit
keine Einheit 		Standardwert
0 		Minimalwert
 Maximalwert
Zugriffspfad HMI: /Tool/Magazine/userPlaceDataDouble[u<Werkzeugber.>, c<Spaltenindex>, <Zeilenindex>]
