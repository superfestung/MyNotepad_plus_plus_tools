SafeUserData - Details

In der Maske "SafeUserData - Übersicht - Details" werden alle Elemente mit den aktiven Werten des angewählten "Anwender-Schnittstellen- Bausteins" angezeigt

Ist die Freigabe für das Ändern von Werten nicht vorhanden, so ist der vertikale Softkey "Ändern" ausgegraut.

Aufruf

MENU SELECT > Inbetriebnahme > Menüfortschalttaste > Safety > SafeUserData > Details

Informationen

Name

Beschreibung

Signatur

Oberhalb der Tabelle wird die aktuelle Signatur angezeigt.

Elementname

In dieser Spalte werden die Namen aller Bausteinelemente angezeigt.

Format

In dieser Spalte wird das Anzeigeformat angezeigt (B = bool, +/-D = Int/DInt).

Aktiver Wert

In dieser Spalte wird der in der PLC aktive Wert angezeigt.

Vertikale Softkeys

Name

Beschreibung

Baustein+

Sind mehrere Bausteine vorhanden, können Sie mit Baustein+ den nächsten Baustein anwählen. Gibt es nur einen Baustein, wird dieser Softkey nicht angezeigt.

Baustein-

Sind mehrere Bausteine vorhanden, können Sie mit Baustein- den vorherigen Baustein anwählen. Gibt es nur einen Baustein, wird dieser Softkey nicht angezeigt.

Baustein auswählen >

Mit diesem Softkey wird eine Auswahlbox aufgeblendet. Sie enthält eine Liste aller Bausteine. Aus dieser Liste können Sie einen Baustein auswählen. Gibt es nur einen Baustein, wird dieser Softkey nicht angezeigt.

Ändern

Mit diesem Softkey wechseln Sie zur Maske "SafeUserData - Ändern".
