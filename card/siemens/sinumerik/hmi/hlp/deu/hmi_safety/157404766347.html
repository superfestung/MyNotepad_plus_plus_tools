Testfallgruppen verwenden

Im Vorlagen-Editor von SINUMERIK Operate können Sie Testfallgruppen z. B. für spezifische Maschinen-Typen eingeben. In den jeweiligen Testtypen können Sie die Testfallgruppe(n) auswählen. Dadurch haben Sie die Möglichkeit, eine Vorlage für unterschiedliche Maschinen-Typen zu verwenden.

Testfallgruppe(n) eingeben bzw. auswählen

  1. Öffnen Sie den Vorlagen-Editor über MENU SELECT > Inbetriebnahme > Menüfortschalttaste > Safety > Abnahmetest >
    Vorlageneditor.

  2. Fügen Sie der Vorlage einen Test hinzu.

  3. Wählen Sie im Feld "Testfallgruppe(n)" bereits angelegte Gruppen aus der Liste aus.

  4. Sind noch keine Gruppen eingegeben, geben Sie im Feld "Testfallgruppe(n)" eine oder mehrere Gruppen ein.

    Die Testfallgruppe(n) können Sie dann bei den weiteren benötigten Tests über das Listenfeld zuordnen.

  5. Speichern Sie die Vorlage.

Abnahmetest - Vorlage für spezifische Testfallgruppe laden

  1. Öffnen Sie den Abnahmetest über MENU SELECT > Inbetriebnahme > Menüfortschalttaste > Safety > Abnahmetest.

  2. Laden Sie die Vorlage über den Softkey "Vorlage laden".

  3. Wählen Sie die Vorlage aus.

  4. Übernehmen Sie die Auswahl mit "OK".

    Folgender Dialog wird angezeigt.

  5. Wählen Sie aus, ob Sie die vollständige Vorlage oder eine Vorlage mit gültigen Achsinformationen verwenden möchten.

  6. Falls Sie die Vorlage mit einer bestimmten Testfallgruppe kombinieren möchten, aktivieren Sie die Option "Testfallgruppe".

    Wählen Sie anschließend die gewünschte Testfallgruppe.

  7. Bestätigen Sie die Auswahl mit "OK".
