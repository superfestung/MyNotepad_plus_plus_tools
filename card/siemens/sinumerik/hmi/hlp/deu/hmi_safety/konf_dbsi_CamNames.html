Nockenbezeichnungen/Nockentooltips

Im Fenster "Nockenbezeichnungen/Nockentooltips" können Sie Namen und Tooltips für den ausgewählten Antrieb (oder die ausgewählte Achse) definieren. Die Namen und Tooltips werden in der Maske "SI Sicherer Nocken" für den ausgewählten Antrieb (oder die ausgewählte Achse) angezeigt. Die in diesem Dialog eingegebenen Namen und Tooltips werden in einer Sprachdatei gespeichert.

Einstellungen

Wählen Sie zunächst aus dem Feld Achse/Antrieb die gewünschte Achse bzw. den Antrieb aus.

Geben Sie die gewünschten Namen bzw. Tooltips ein.

Über den Softkey "Achse/Antr. hinzufügen" können Sie eine Achse bzw. einen Antrieb hinzufügen.

Über den Softkey "Achse/Antr. löschen" können Sie eine Achse bzw. einen Antrieb inklusive der angelegten Namen und Tooltips löschen.
