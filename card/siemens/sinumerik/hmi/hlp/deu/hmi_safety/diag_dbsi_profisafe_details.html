Details PROFIsafe Telegramm

Im Fenster "Details PROFIsafe Telegramm" werden folgende Daten angezeigt:

Das Telegramm 30 überträgt folgende Nutzdaten :

  • Empfangsrichtung

    • S_STW1

  • Senderichtung

    • S_ZSW1

Das Telegramm 31 überträgt folgende Nutzdaten :

  • Empfangsrichtung

    • S_STW2

  • Senderichtung

    • S_ZSW2

Das Telegramm 901 überträgt folgende Nutzdaten :

  • Empfangsrichtung

    • S_STW2

    • S_SLS_LIMIT_A (variable SLS-Grenze)

  • Senderichtung

    • S_ZSW2

    • S_SLS_LIMIT_A_ACTIVE (aktiver SLS-Wert der Stufe 1)

    • S_CYCLE_COUNT ((Zählerwert)

    • S_XIST16 (sicherer Positionswert im 16-bit-Format)

Das Telegramm 902 überträgt folgende Nutzdaten :

  • Empfangsrichtung

    • S_STW2

    • S_SLS_LIMIT_A (variable SLS-Grenze)

  • Senderichtung

    • S_ZSW2

    • S_SLS_LIMIT_A_ACTIVE (aktiver SLS-Wert der Stufe 1)

    • S_CYCLE_COUNT ((Zählerwert)

    • S_XIST32 (sicherer Positionswert im 32-bit-Format)

Das Telegramm 903 überträgt folgende Nutzdaten :

  • Empfangsrichtung

    • S_STW2

    • S_SLS_LIMIT_A (variable SLS-Grenze)

  • Senderichtung

    • S_ZSW2

    • Z_SWC (32 Bit Safe Cam)

    • S_SLS_LIMIT_A_ACTIVE (aktiver SLS-Wert der Stufe 1)

Hinweis

Ansicht ändern

Sie können in dieser Diagnoseansicht die Signal-Informationen verschiedener Antriebe anzeigen. Zu diesem Zweck können Sie über die Softkeys "Antrieb-", "Antrieb+" oder "Antrieb auswählen" jeweils den gewünschten Antrieb einstellen.
