SafeUserData - Bausteinübersicht

In der Maske "SafeUserData - Übersicht" werden alle angelegten SafeUserData-Instanzen aufgelistet.

Hinweis

Die Sortierreihenfolge der SUD-Instanzen in der "SafeUserData - Übersicht" erfolgt anhand der im TIA-Projekt eingestellten "instanceid".

Aufruf

MENU SELECT > Inbetriebnahme > Menüfortschalttaste > Safety > SafeUserData

Informationen

Name

Beschreibung

Bausteinname

In dieser Spalte werden die "Anwender-Schnittstellen" der SafeUserData aufgelistet. Dazu können auch sog. "Nested blocks" gehören.

Nested blocks sind DB, deren Werte in einem Gesamt DB verwaltet werden. Nested blocks haben Namen mit Punkten im Namen.

Datentyp

In dieser Spalte wird der Datentyp der jeweiligen "Anwender-Schnittstelle" angezeigt.

Signatur

In dieser Spalte wird die aktuelle Signatur der SafeUserData-Instanz angezeigt.

Vertikale Softkeys

Name

Beschreibung

Export

Mit diesem Softkey wird für den Export aller SUD-Aktualwerte der Ablageort geöffnet. Sie können hier einen passenden Namen für die Exportdatei festlegen und den Export starten.

Import

Mit diesem Softkey wird für den Import aller SUD-Aktualwerte der Ablageort für Importdateien geöffnet. Hier können Sie die gewünschte Importdatei wählen und den Import starten.

Details

Mit diesem Softkey wechseln Sie zur Maske "SafeUserData - Details".
