Abnahmetests konfigurieren und durchführen

Ablauf: Abnahmetests durchführen

Der Einstellbereich "Übersicht" ist automatisch nach dem Aufruf der Abnahmetest-Funktion aktiv.

 

1.

Erfassen Sie im Einstellbereich "Übersicht" die erforderlichen Maschinendaten.

 

2.

Drücken Sie den Softkey "Erster Test".

Der Einstellbereich "Abschaltpfad" wird aktiv.

Falls Sie diesen Test nicht verwenden wollen, aktivieren Sie in diesem Einstellbereich die Option "Dieser Test ist für diese Maschine nicht erforderlich". In der gleichen Weise können Sie auch in den weiteren Einstellbereichen den jeweiligen Test abschalten.

 

3.

Drücken Sie den Softkey "Test starten".

Rechts in der Detailanzeige erscheint nun eine Auflistung der für den Test notwendigen Tätigkeiten.

 

4.

Parametrieren Sie die Kriterien für den entsprechenden Testtyp.

 

5.

Drücken Sie den Softkey "Daten erfassen".

Folgen Sie anschließend den Anweisungen am Bildschirm (z. B. Zwangsdynamisierung der Abschaltpfade aktivieren).

 

6.

In einigen Testfällen müssen Sie die Datenerfassung manuell durch Drücken des Softkeys "Fertig" beenden.

Anschließend werden in der Detailanzeige die erfassten Alarme dargestellt.

Im Feld "Erfasste Alarme" wird in den meisten Fällen jeder Alarm zweikanalig (NC/Antrieb) aufgelistet.

 

7.

Drücken Sie den Softkey "Zu den Ergebnissen".

Nach dem Abnahmetest werden die aufgetretenen Alarme aufgelistet.

Abhängig vom Testtyp kann ein Trace mit den relevanten Signalen bewertet werden.

 

8.

Legen Sie fest, welche der aufgelisteten Alarme später im Abnahmeprotokoll erscheinen sollen.
Entfernen Sie von allen Alarmen die Haken (links), die nicht im Protokoll erscheinen sollen.

 

9.

Erfassen Sie die Kriterien zur Auslösung und Bewertung des Tests.
Eine Eingabe ist hier Pflicht, um die weiteren Schritte starten zu können.

 

10.

Bewerten Sie anschließend das Testergebnis.

Prüfen Sie, ob die erwarteten Alarme eingetroffen sind. Abhängig vom Testtyp wurde ein Trace erstellt, den Sie bewerten können.

Legen Sie über die Auswahl rechts unten in der Detailanzeige fest, ob der Test (in diesem Fall: der Abschaltpfade) erfolgreich war.

Bei einer positiven Bewertung wird der Testname mit einem grünen Haken gekennzeichnet. In diesem Fall können Sie entweder einen weiteren Test im aktiven Einstellbereich durchführen oder wie im Normalfall den nächsten Test über den nächsten Einstellbereich anstoßen.

Bei einer negativen Bewertung wird der Testname mit einem roten Kreuz gekennzeichnet und Sie können den Test ggf. wiederholen.

 

11.

Korrigieren Sie bei Bedarf den Standard-Testnamen.

 

12.

Um auf die nächste Einstellbereich zu wechseln, drücken Sie auf den Softkey "Nächster Test".

Im nächsten Einstellbereich wird ein Test angelegt. In der Detailanzeige erscheint nun für diesen Testtyp eine Auflistung der notwendigen Tätigkeiten.

 

13.

Folgen Sie den Anweisungen am Bildschirm (in der Regel sind das die Schritte 3 bis 12).

Nach Durchführung alle Test der einzelnen Einstellbereiche wird am Ende der Einstellbereich "Fertigstellen" angezeigt. In diesem Einstellbereich können Sie das Abnahmeprotokoll aus den Abnahmetests erzeugen und speichern.

Abnahmeprotokoll erzeugen und speichern

Nach Abschluss aller Abnahmetests können Sie im Einstellbereich "Fertigstellen" ein Zertifikat für das Abnahmeprotokoll erstellen und speichern.

 

1.

Falls Sie in das Abnahmeprotokoll ein Zertifikat integrieren wollen, aktivieren Sie die entsprechende Option in der Detailanzeige.

 

2.

Drücken Sie den Softkey "Zertifikat erzeugen".

Die Standardablage der Dateien wird angezeigt.

 

3.

Wählen Sie hier ein Verzeichnis aus oder erstellen Sie ein neues.
Drücken Sie anschließend den Softkey "OK", um die Auswahl zu bestätigen.

Es erscheint eine Abfrage nach dem Namen des zukünftigen Zertifikats.

 

4.

Erfassen Sie hier den Namen des Zertifikats.
Drücken Sie anschließend den Softkey "OK", um die Eingabe zu bestätigen.

Das Zertifikat wird dann als RTF-Datei im ausgewählten Ablageordner abgespeichert. Durch das RTF-Format kann das Zertifikat nachträglich ergänzt werden, z. B. um ein Maschinendiagramm.
