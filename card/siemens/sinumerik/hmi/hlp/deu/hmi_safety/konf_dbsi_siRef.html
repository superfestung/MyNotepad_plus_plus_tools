Sicheres Referenzieren

Die Funktion "Sicheres Referenzieren" ermöglicht es, eine sichere Absolutposition festzulegen. Diese sichere Position wird für die folgenden Funktionen benötigt:

  • SLP (Safely-Limited Position)

  • SP (Safe Position)

  • SCA

Voraussetzung

Damit Änderungen an den Parametern dieser Safety-Funktion möglich sind, muss der Inbetriebnahme-Modus aktiviert sein. In den Optionen muss eine erweiterte Funktion eingestellt sein.

Hinweis

Die Anwenderzustimmung kann auch ohne SI-IBN-Modus gesetzt werden.

Einstellungen

In diesem Fenster können Sie folgende Einstellungen für den Antrieb vornehmen:

  • Freigabe Referenzieren über SCC (p9501.27)

    Einstellung der Freigaben für die sicheren Bewegungsüberwachungen. Über das Kontrollkästchen kann die "Freigabe Referenzieren über SIC/SCC" aktiviert werden.

  • SI Motion Anwenderzustimmung An-/Abwahl (p9726)

    Anwenderzustimmung Sichere Position

    (wird nur angezeigt wenn Funktionen mit sicherer Positionsauswertung aktiviert sind)

  • Toleranz der Positionsistwerte (p9544)

    Bei einem inkrementellen Geber werden die Istwerte nach dem Referenzieren geprüft, bei einem Absolutwertgeber beim Einschalten.

Folgende LEDs können einen Status anzeigen (blau = aktiv; grau = inaktiv):

  • Position referenziert (r9723.17)

  • Sicher referenziert (r9722.23)

Parameterliste

Jede Einstellung in dieser Dialogmaske können Sie bei Bedarf alternativ auch über eine Parameterliste vornehmen. Zur Parameterliste gelangen Sie über den Softkey ">>" und anschließend den Softkey "Parameterliste".
