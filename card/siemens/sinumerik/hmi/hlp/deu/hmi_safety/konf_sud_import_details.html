SafeUserData - Import - Details

Nach dem Importaufruf ergab eine Überprüfung, dass sich die Symbolnamen der aktiven SafeUserData-Datensätze von den Symbolnamen der Importdatei unterscheiden.

Die Unterschiede werden zur Bewertung in der Maske "SafeUserData - Import - Details" angezeigt.

Voraussetzungen

  • Der Import der SafeUserData aus einer Importdatei wurde aufgerufen.

  • Eine Überprüfung ergab Unterschiede in den Symbolnamen.

Aufruf

MENU SELECT > Inbetriebnahme > Menüfortschalttaste > Safety > SafeUserData > Import > OK (nach Meldung von Unterschieden)

Anschließend öffnet sich ein Dialog zur Bewertung der Unterschiede in den Symbolnamen.

Vertikale Softkeys

Name

Beschreibung

Abbruch

Mit diesem Softkey brechen Sie den Import der Aktualwerte ab.

Nach dem Abbruch wird die Maske "SafeUserData - Übersicht" angezeigt.

Übernehmen

Die Symbolnamen der Datensätze aus der Importdatei unterscheiden sich von den Symbolnamen des aktuellen Datensatzes.

Mit diesem Softkey übernehmen Sie die Elementwerte aller Instanzen aus der Importdatei, obwohl sich die Symbolnamen von denen des aktiven Datensatzes unterscheiden.

Eine Gegenüberstellung in der Maske "SafeUserData - Import" zeigt an, dass die Daten aus der Importdatei übernommen werden.

Verwerfen

Die Symbolnamen der Datensätze aus der Importdatei unterscheiden sich von den Symbolnamen des aktuellen Datensatzes.

Mit diesem Softkey verwerfen Sie den Import aller Elementwerte der angezeigten Instanz. Weitere Unterschiede anderer Instanzen können in weiteren Masken angezeigt werden und in gleicher Weise bewertet werden.

Eine Gegenüberstellung in der Maske "SafeUserData - Import" zeigt an, dass die Daten aus der Importdatei nicht übernommen werden.
