Maschinenmodell bearbeiten

In der Baumansicht des grafischen Editors stehen Ihnen ähnlich wie im Programmeditor Funktionen zur Verfügung, die die Erstellung und Bearbeitung eines Maschinenmodells erleichtern.

Vorgehensweise

Elemente kopieren / ausschneiden / einfügen

Wenn Sie ein Kinematik- oder Schutzbereichselement erstellen möchten, das einem bereits vorhandenen ähnlich ist, dann sparen Sie Zeit, wenn Sie das Element kopieren und an anderer Stelle einfügen. So haben Sie die Möglichkeit, z.B. auch inaktive Teilbäume in den aktiven Baum zu kopieren.

Elemente kopieren / ausschneiden und einfügen

 

1.

Positionieren Sie den Cursor auf das gewünschte Element.

 

2.

Drücken Sie den Softkey "Kopieren".

  

- ODER -

  

Drücken Sie den Softkey "Ausschneiden".

 

3.

Positionieren Sie den Cursor an die gewünschte Stelle und drücken Sie den Softkey "Einfügen".

 

4.

Drücken Sie den Softkey "OK", wenn Sie das Element als nächstes Element an die Kette anhängen wollen.

Das Element wird eingefügt und der Name wird automatisch um einen langen Unterstrich ergänzt und hochgezählt.

Hinweis:

Sie haben die Möglichkeit, den Namen des Elements im zugehörigen Eingabefenster zu ändern.

  

- ODER -

  

Drücken Sie den Softkey "Parallel", wenn Sie das Element parallel zum markierten Element einfügen wollen.

Hinweis:

Der Softkey "Parallel" steht Ihnen für Schutzbereiche nicht zur Verfügung.

Hinweis

Gleiches Element einfügen

Fügen Sie ein gleiches Element ein, so wird dieses automatisch vor das markierte Element eingefügt.

Elemente löschen

 

1.

Positionieren Sie den Cursor auf das gewünschte Element und drücken Sie auf den Softkey "Element löschen".

 

2.

Drücken Sie den Softkey "OK", um das Löschen zu bestätigen.

Das markierte Element wird entfernt.

Alle nachfolgenden Elemente werden in den Ordner "Freie Elemente" verschoben.

  

- ODER -

 

 

Wenn das markierte Element mit weiteren Elementen verknüpft ist, die ebenfalls gelöscht werden sollen, dann drücken Sie den Softkey "Mit Nachfolgern".

Hinweis

Maschinenmodell löschen

Wenn Sie den Ordner "Aktive Daten" in der Baumansicht markieren und den Softkey "Elemente löschen" drücken sowie die nachfolgende Abfrage mit "OK" bestätigen, wird das Maschinenmodell mit allen Elementen und Schutzbereichen gelöscht.

Wenn Sie STL-Dateien nicht entfernen wollen, dann brechen Sie bei der Abfrage "3D Dateien" mit "Nein" den Löschvorgang ab.

Maschinenmodell aktivieren

Nach jeder Eingabe oder Änderung eines Elementes des Kollisionsmodells wird automatisch eine Aktivierung des Kollisionsmodells durchgeführt. Wenn das Kollisionsmodell dabei fehlerhaft ist, wird eine Fehlermeldung ausgegeben und das fehlerhafte Element rosa hinterlegt. Der Softkey "Aktiviere Daten" wird aktiv und Sie können damit evtl. quittierte Alarme zum Kollisionsmodell erneut provozieren.

 

1.

Drücken Sie den Softkey "Aktiviere Daten".

Es werden alle Fehlermeldungen wieder angezeigt und Sie haben die Möglichkeit, die Fehler durch Ändern der Daten gezielt zu beheben.
