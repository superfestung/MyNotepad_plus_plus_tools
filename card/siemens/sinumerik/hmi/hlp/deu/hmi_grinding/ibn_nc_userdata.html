NC-Anwenderspeicher für Programme und Daten

Das Fenster "NC-Anwenderspeicher für Programme und Daten" ermöglicht das Anzeigen, Suchen und Ändern von gespeicherten Anwendervariablen und -daten.

Hier richten Sie die erstmalige Speichereinsteilung für wichtige Bereiche ein. Hier lassen Sie sich auf einen Blick die Speicherbelegung anzeigen, nehmen Optimierungen von Speicherzuweisungen für bestimmte Themen vor und detektieren freien Speicherplatz.

NC-Anwenderdaten

Sie erhalten eine vollständige Liste von Anwenderdaten, die Sie nach folgenden Bereichen filtern können:

  • Globale Anwenderdaten (GUD)

  • Lokale Anwenderdaten (LUD)

  • Makros

  • R-Parameter

  • Basiswerkzeuge

  • Werkzeugverwaltung

  • Frames

  • Kompensationen

  • Kurventabellen

  • Schutzbereiche

  • Kollisionsvermeidung

  • Zyklen

  • Filesystem

  • REORG

  • Interpolationspuffer

  • Abarbeiten von extern

  • Synchronaktionen

  • Kanal

  • Achse

  • DRAM

  • SRAM

Vorgehensweise

 

1.

Wählen Sie den Bedienbereich "Inbetriebnahme" an.

 

2.

Drücken Sie die Softkeys "NC" und "NC-Speicher".

 

 

Das Fenster "NC-Anwenderspeicher für Programme und Daten" wird geöffnet. Sie erhalten eine Liste aller Anwenderspeicherdaten.

 

3.

Drücken Sie den Softkey "Filter/Suchen", um die Auswahlliste "Filter" zu aktivieren.

 

4.

Wählen Sie über die Auswahlliste den gewünschten Datenbereich (z.B. Werkzeugverwaltung).

 

5.

Drücken Sie erneut auf den Softkey "Filter/Suchen", um die Auswahlliste "Suchen" zu aktivieren.

 

6.

Geben Sie den gewünschten Suchbegriff ein.

Je nach Auswahl unter "Filter" werden die in den angezeigten Anwenderspeicherdaten gefundenen Treffer aufgelistet.

 

7.

Drücken Sie den Softkey "Suche löschen", wenn Sie eine neue Suche starten wollen.
