Maschinenschutzbereiche anlegen

Ständer unter Offset

 

1.

Markieren Sie das Kinematikelement "ROOT" und drücken Sie auf den Softkey "Neues Element".

 

2.

Drücken Sie den Softkey "Kollisionselement".

 

3.

Wählen Sie im Fenster "Neues Kollisionselement" das Kollisionselement "Maschinenschutzbereich" und drücken Sie den Softkey "OK".

 

4.

Geben Sie folgende Werte ein:

  

Name:

Staender

  

Farbe:

grau

  

Detailstufe:

0

  

PLC-Bit:

-1

  

Aktivierung:

Aktiviert

 

5.

Drücken Sie den Softkey "OK", um die Werte zu speichern.
