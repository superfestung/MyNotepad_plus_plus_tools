Messungen verwalten

Im Bereich "Messungen verwalten" können Sie Ihre gespeicherten Messungen exportieren, löschen oder umbenennen.

Löschen und Exportieren sind über Schutzstufen geschützt.

 

Maschinenhersteller

Beachten Sie hierzu die Angaben des Maschinenherstellers.

Voraussetzung

  • Sie befinden sich im Fenster "Ctrl-E Analyse > Messergebnis".

  • Sie haben bereits Messungen gespeichert.

Messungen exportieren

Um Messungen für weitere Verarbeitung oder Analyse zu exportieren, gehen Sie wie folgt vor:

 

1.

Drücken Sie den Softkey "Messungen verwalten".

Das Fenster "Ctrl-E Analyse: Messungen verwalten" wird geöffnet.

 

2.

Markieren Sie die Messungen, die Sie exportieren möchten oder drücken Sie den Softkey "Alles markieren", um alle Messungen zu markieren.

 

3.

Klicken Sie auf den Softkey "Exportieren".

 

4.

Wählen Sie das Zielverzeichnis.

 

5.

Bestätigen Sie mit "OK".

Die Daten werden im Zielverzeichnis abgelegt.

Hinweis

Über den Softkey "Eigenschaften" können Sie die Messungen umbenennen.

Messungen löschen

Um Speicherplatz für neue Messungen freizugeben oder zu verhindern, dass vorhandene Messungen durch neue Messungen automatisch überschrieben werden, können Sie Messungen löschen. 

Um Messungen zu löschen, gehen Sie wie folgt vor:

 

1.

Drücken Sie den Softkey "Messungen verwalten".

Das Fenster "Ctrl-E Analyse: Messungen verwalten" wird geöffnet.

 

2.

Markieren Sie die Messungen, die Sie löschen möchten oder drücken Sie den Softkey "Alles markieren", um alle Messungen zu markieren.

 

3.

Klicken Sie auf den Softkey "Löschen".

 

4.

Bestätigen Sie die Abfrage mit "OK".

Die Messungen werden gelöscht.
