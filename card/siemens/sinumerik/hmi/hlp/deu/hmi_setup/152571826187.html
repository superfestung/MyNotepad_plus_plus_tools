Hinweis

Topologie-Regeln für SINUMERIK 828D

Der Anschluss eines externen Gebers an einer freien Schnittstelle am DRIVE-CLiQ Hub Module wird unterstützt.

Falls eine Maschine nur über Motor-Messsysteme verfügt, nehmen Sie zuerst einen DRIVE-CLiQ Hub Module in die Topologie auf und danach schließen Sie den externen Geber am DRIVE-CLiQ Hub Module an.

Der Anschluss eines externen Gebers an einer freien Schnittstelle am SINAMICS S120 Combi wird nicht unterstützt.

Weitere Einzelheiten finden Sie im Inbetriebnahmehandbuch CNC im Kapitel "Topologie-Regeln".
