HLA-Kennlinie: Ventilkennlinienparameter

Die Ventilkennlinienparameter werden in einer Liste dargestellt.

Die in der Liste angezeigten Parameter werden aus dem Antrieb ausgelesen. Geänderte Werte schreiben Sie mit der Eingabetaste in den Antrieb.

Hinweis

Änderung der Parameter nur bei Impulssperre

Ändern Sie die Parameter nicht im Antriebszustand "U" (Run), da es bei fahrender Achse zu unvorhersehbaren Effekten kommen kann.

Folgende Parameter werden aufgelistet:

  • Geschwindigkeitsgrenze positive Richtung

    Einstellung der maximalen Geschwindigkeit für die positive Richtung.

    Wert: 0,000 m/min bis 1000,000 m/min; Werkseinstellung: 1000,000 m/min

  • Geschwindigkeitsgrenze negativ wirksam

    Einstellung der maximalen Geschwindigkeit für die negative Richtung.

    Wert: -1000,000 m/min bis 0,000 m/min; Werkseinstellung: -1000,000 m/min

  • Geschwindigkeitsregler Streckenverstärkung

    Einstellung der Streckenverstärkung des Geschwindigkeitsreglers.

    Wert: 0,0 mm/Vmin bis 20000,0 mm/Vmin; Werkseinstellung: 0,0 mm/Vmin

  • Faktor Flächenanpassung positiv

    Einstellung des Faktors für die Flächenkompensation in positiver Richtung.

    Wert: 10,0 % bis 200,0 %; Werkseinstellung: 100,0 %

  • Faktor Flächenanpassung negativ

    Einstellung des Faktors für die Flächenkompensation in negativer Richtung.

    Wert: 10,0 % bis 200,0 %; Werkseinstellung: 100,0 %

  • Knickkompensation Q1 positiv Nullbereich

    Einstellung des Volumenstroms Q für Punkt 1 positiv (Nullbereich) der Knickkompensation.

    Wert: 0,01 % bis 95,00 %; Werkseinstellung: 0,01 %

  • Knickkompensation U1 positiv Nullbereich

    Einstellung der Spannung U für Punkt 1 positiv (Nullbereich) der Knickkompensation.

    Wert: 0,00 % bis 95,00 %; Werkseinstellung: 0,00 %

  • Knickkompensation Verrundung 1 positiv Nullbereich

    Einstellung der Verrundung für Punkt 1 positiv (Nullbereich) der Knickkompensation.

    Wert: 0,00 % bis 20,00 %; Werkseinstellung: 0,00 %

  • Knickkompensation Q1 negativ Nullbereich

    Einstellung des Volumenstroms Q für Punkt 1 negativ (Nullbereich) der Knickkompensation.

    Wert: 0,01 % bis 95,00 %; Werkseinstellung: 0,01 %

  • Knickkompensation U1 negativ Nullbereich

    Einstellung der Spannung U für Punkt 1 negativ (Nullbereich) der Knickkompensation.

    Wert: 0,00 % bis 95,00 %; Werkseinstellung: 0,00 %

  • Knickkompensation Verrundung 1 negativ Nullbereich

    Einstellung der Verrundung für Punkt 1 negativ (Nullbereich) der Knickkompensation.

    Wert: 0,00 % bis 20,00 %; Werkseinstellung: 0,00 %

  • Knickkompensation Q2 positiv

    Einstellung des Volumenstroms Q für Punkt 2 positiv der Knickkompensation.

    Wert: 0,20 % bis 95,00 %; Werkseinstellung: 10,00 %

  • Knickkompensation U2 positiv

    Einstellung der Spannung U für Punkt 2 positiv der Knickkompensation.

    Wert: 0,00 % bis 95,00 %; Werkseinstellung: 10,00 %

  • Knickkompensation Verrundung 2 positiv

    Einstellung der Verrundung für Punkt 2 positiv der Knickkompensation.

    Wert: 0,00 % bis 20,00 %; Werkseinstellung: 2,50 %

  • Knickkompensation Q2 negativ

    Einstellung des Volumenstroms Q für Punkt 2 negativ der Knickkompensation.

    Wert: 0,00 % bis 95,00 %; Werkseinstellung: 10,00 %

  • Knickkompensation U2 negativ

    Einstellung der Spannung U für Punkt 2 negativ der Knickkompensation.

    Wert: 0,00 % bis 95,00 %; Werkseinstellung: 10,00 %

  • Knickkompensation Verrundung 2 negativ

    Einstellung der Verrundung für Punkt 2 negativ der Knickkompensation.

  • Knickkompensation Q3 positiv Sättigung

    Einstellung des Volumenstroms Q für Punkt 3 positiv (Sättigung) der Knickkompensation.

    Wert: 0,20 % bis 100,00 %; Werkseinstellung: 100,00 %

  • Knickkompensation U3 positiv Sättigung

Weitere Aktionen:

  • Mit den Softkeys "Achse +" / "Achse -" wechseln Sie zur nächsten/vorherigen Achse.

  • Mit dem Softkey "Achse auswählen >" öffnen Sie einen Dialog, um die Achse aus einer Liste auszuwählen.

  • Mit dem Softkey "DDS auswählen" öffnen Sie einen Dialog in dem Sie den Antriebsdatensatz (DDS) auswählen, dessen Parameter in der Liste angezeigt werden sollen.

    Der ausgewählte Antriebsdatensatz gilt nicht für die Messung der Kennlinie. Die Messung wird immer mit dem, zum Zeitpunkt der Messung, im Antrieb aktiven Antriebsdatensatz durchgeführt.

    Der Antriebsdatensatz, auf den sich ein Dialog bezieht, wird im Fenstertitel ganz rechts angezeigt, z. B. DDS0.

  • Mit dem Softkey "Kennlinie kopieren" öffnen Sie einen Dialog zur Auswahl der Antriebsdatensätze, in welche die Kennlinienparameter kopiert werden sollen.

  • Mit dem Softkey "Zurück" kehren Sie zum Dialog "Ventilkennlinie messen" zurück.
