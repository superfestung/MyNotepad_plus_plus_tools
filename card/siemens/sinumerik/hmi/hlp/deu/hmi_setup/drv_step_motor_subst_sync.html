Weitere Aktionen:

  • Alternativ können Sie die Daten auch berechnen lassen. Falls die Daten nicht vorliegen, drücken Sie den Softkey "ESB Daten berechnen", dann werden die Parameter für das Ersatzschaltbild mit Hilfe des Motormodells berechnet.

  • Wählen Sie "Fertig", dann werden die Parameter berechnet. Der Fortschritt dieses Vorgangs wird in der Statuszeile ausgegeben. Danach können Sie die Änderungen nicht-flüchtig speichern (→ "OK") oder verwerfen (→ "Abbruch").
