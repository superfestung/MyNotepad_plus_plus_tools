HLA-Antriebe: Konfiguration - Geberzuordnung

In diesem Dialog ordnen Sie dem ausgewählten Antriebsobjekt (HLA Module) maximal 2 Geber zu.

Das HLA Module verfügt über folgende Schnittstellen für Geber:

  • TTL / SSI-Schnittstellen für analoge Geber

  • DRIVE-CLiQ-Anschlüsse (Sensor Module SMx)

Wenn Sie die Option "LED des Sensor Module zur Erkennung blinkend schalten" aktivieren, blinkt das ausgewählte Sensor Module abwechselnd rot-grün. Dadurch erkennen Sie das Sensor Module an der Anlage. Diese Funktion setzt voraus, dass die Geberschnittstelle die Fähigkeit hat zu blinken.

Bei der Zuordnung der über DRIVE-CLiQ angeschlossenen Geber müssen Sie folgendes beachten:

  • Wenn das Sensor Module (SMx) auf dem Antriebsobjekt bereits richtig gesteckt ist, wird das Sensor Module automatisch zugeordnet. Es können auch mehrere Sensor Module automatisch zugeordnet sein.

    Die Geber werden nach folgendem Schema automatisch zugeordnet:

    SINAMICS S120 HLA Module

     

    Achse 1

    Achse 2

    Geber 1

    X202

    --

    Geber 2

    --

    X203

    In den Zeilen Geber 1/Geber 2 sind die Bezeichnungen der DRIVE-CLiQ-Anschlüsse des Antriebsobjekts angegeben. Dieser DRIVE-CLiQ-Anschluss wird der Achse als Geber 1 oder Geber 2 zugeordnet.

  • Wenn das Sensor Module nicht auf dem zugehörigen Antriebsobjekt gesteckt ist, müssen Sie das Sensor Module manuell auswählen und zugeordnen. Die Bezeichnung des Gebers in der Auswahlliste ist nach folgendem Schema aufgebaut:

    Name und Nummer des HLA Module mit Steckplatz des Sensor Modules -

    Name und Nummer des Sensor Modules -

    Name und Nummer des Gebers

  • Nummer des Geberdatensatzes (EDS) auswählen: Bei der automatischen Gerätekonfiguration werden die zugehörigen Datensätze bereits voreingestellt. Sie können die Nummer des Geberdatensatzes ändern.

    Hinweis

    Geberzuordnung ändern

    Überprüfen Sie im Fall der geänderten Zuordnung die Topologie und die DRIVE-CLiQ-Verbindungen!
