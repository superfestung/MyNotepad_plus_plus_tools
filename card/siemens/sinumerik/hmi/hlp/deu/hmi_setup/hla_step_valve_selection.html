HLA-Antriebe: Konfiguration - Ventilauswahl

Unter "Ventilauswahl" wählen Sie die Eingabemethode der Ventildaten.

  • Bei der Option "Ventil aus Liste auswählen" wird die Auswahlliste der Ventilkonfiguration angezeigt.

    Wählen Sie das passende Ventil aus. Die Daten des Ventils werden übernommen und der Dialog "Ventildaten" wird übersprungen.

    Hinweis

    Ventilkonfigurationen eingeben oder importieren

    Wenn keine Ventilkonfigurationen eingegeben oder importiert wurden, wird eine leere Liste angezeigt. Um die Liste mit Ventilkonfigurationen zu füllen, haben Sie folgende Möglichkeiten:

    • Geben Sie Ventildaten ein.

    • Importieren Sie Ventilkonfigurationen, z. B. von SINUMERIK powerline.

      Den Import können Sie im Dialog "Maschinendaten" über "Daten verwalten" durchführen.

  • Bei der Option "Ventildaten eingeben" wird die Auswahlliste der Ventilkonfiguration nicht angezeigt. Wenn Sie das Optionskästchen "Vorlage aus Liste" aktivieren, wird die Ventilauswahlliste wieder angezeigt. Sie können dann die Ventildaten für den Dialog "Ventildaten" mit den Daten des ausgewählten Ventils vorbelegen.

Mit dem Kombinationsfeld "Suche" schränken Sie die Auswahlliste der Ventile durch einen Freitextfilter ein. Nach jedem Tastendruck während der Eingabe eines Suchtextes wird die Liste aktualisiert. Das Kombinationsfeld enthält zuvor benutzte Suchtexte.
