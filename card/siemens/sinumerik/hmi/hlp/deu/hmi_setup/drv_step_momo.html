Antriebe: Konfiguration - Motor Module

Die Kenndaten des automatisch erkannten Motor Module werden angezeigt.

Wenn Sie die Option "LED zur Erkennung blinkend schalten" aktivieren, blinkt das ausgewählte Motor Module abwechselnd rot-grün.

Optional können Sie Funktionsmodule aktivieren, die zum Teil lizenzpflichtig sind. Damit werden weitere Parameter aktiv, die zur Inbetriebnahme des jeweiligen Funktionsmoduls benötigt werden.
