HLA-Antriebe: Konfiguration - HLA Module

In diesem Dialog wird die Bestellnummer des Hydraulikmoduls angezeigt. Optionen können aktiviert werden.

  • Wenn Sie die Option "LED des HLA-Module zur Erkennung blinkend schalten" aktivieren, blinkt das ausgewählte HLA-Module abwechselnd rot-grün. Dadurch erkennen Sie das HLA-Module im Schaltschrank.

  • Im Bereich "Funktionsmodule" können Sie die angezeigten Objekte aktivieren.
