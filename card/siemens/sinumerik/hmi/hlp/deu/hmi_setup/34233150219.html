So starten Sie die automatische Gerätekonfiguration

Hinweis

Nachträgliches Stecken von SINAMICS-Komponenten

Das ordnungsgemäße Firmware-Update der projektierten SINAMICS-Komponenten ist nur dann gegeben, wenn diese im ausgeschalteten Zustand gesteckt wurden. Das nachträgliche Stecken von SINAMICS-Komponenten darf nur im ausgeschalteten Zustand erfolgen.

Firmware aktualisieren

Wenn ein Antrieb angeschlossen ist, wird die Aktualisierung der Firmware automatisch gestartet. Am jeweiligen Modul werden Sie mit einer Zustandsanzeige über den Fortschritt informiert:

  • Während die Firmware geladen wird: die LED "RDY" blinkt mit 1 Hz abwechselnd rot - grün.

  • Wenn das Laden der Firmware abgeschlossen ist: die LED "RDY" blinkt mit 2 Hz rot - grün.

  • Nach dem Aus- und Einschalten des Antriebsgerätes: die LED "RDY" leuchtet dauerhaft grün.

Automatische Gerätekonfiguration

Vorgehensweise:

  1. Nachdem das gesamte Antriebssystem hoch gelaufen ist, wird ein Dialog zur automatischen Gerätekonfiguration angezeigt: "Für das Antriebssystem (Antriebsgeräte) muss eine Gerätekonfiguration durchgeführt werden ..."

  2. Um die automatischen Gerätekonfiguration zu starten, bestätigen Sie mit "OK".

    Danach wird die Topologie des Systems automatisch ausgelesen. Dieser Vorgang kann einige Minuten dauern. Bestätigen Sie mit "OK".

  3. Danach wird ein Power-On Reset angefordert. Bestätigen Sie mit "Ja". Dieser Vorgang ist abhängig vom Ausbau des Antriebssystems und kann mehrere Minuten in Anspruch nehmen.

  4. Bestätigen Sie mit "OK", um die Inbetriebnahme fortzusetzen.

Wenn die automatische Gerätekonfiguration nicht startet, dann wurde sie bereits durchgeführt: Um einen definierten Zustand herzustellen oder um noch einmal zu starten, laden Sie "Werkseinstellungen". Danach ist wieder ein definierter Ausgangszustand hergestellt.
