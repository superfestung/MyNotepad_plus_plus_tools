HLA-Antriebe: Konfiguration - Versorgungsdaten

In diesem Dialog definieren Sie die Parameter der "Versorgungseinheit".

Für folgende Parameter der "Versorgungseinheit" können Sie Werte eingeben:

  • Hydrauliköl Elastizitätsmodul [bar]

    Dieser Parameter (p0220) beschreibt die Kompressibilität der Hydraulikflüssigkeit. Dieser Wert gibt an, welche Druckänderung nötig ist, um eine bestimmte Volumenänderung zu erreichen. Das Elastizitätsmodul bezieht sich auf ein relatives Volumen.

  • Systemdruck [bar]

    Dieser Parameter (p0221) beschreibt den Druck, den die Versorgungseinheit liefert. Der Systemdruck und die Größe der Kolbenfläche bestimmen die an der Kolbenstange verfügbare Kraft.

  • Ventil Vorsteuerdruck [bar]

    Dieser Parameter (p0222) beschreibt den Druck bei einem vorgesteuerten Ventil. Bei einem nicht vorgesteuerten Ventil muss null eingetragen werden.
