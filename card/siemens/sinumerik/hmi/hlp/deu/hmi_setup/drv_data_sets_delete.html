Antriebe: Datensatz entfernen

Vorgehensweise:

  1. Wählen Sie "Datensatz entfernen": Es wird ein Papierkorb eingeblendet.

  2. Markieren Sie das Kontrollkästchen für den Datensatz, den Sie entfernen wollen.

  3. Wählen Sie "Nächster Schritt >": Der oder die verbleibenden Datensätze werden angezeigt. Wenn Sie weniger als 8 Antriebsdatensätze konfigurieren wollen, wählen Sie in der Auswahlliste die entsprechende Anzahl aus. Dann werden die überzähligen Antriebsdatensätze entfernt.

  4. Wählen Sie "Nächster Schritt >": In der Zusammenfassung erfahren Sie, wie viele Datensätze aktuell verfügbar sind.

  5. Wählen Sie "Fertig", dann legt das System den werden die Datensätze endgültig entfernt. Der Fortschritt dieses Vorgangs wird in der Statuszeile ausgegeben.
