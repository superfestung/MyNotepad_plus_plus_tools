Messfunktionen - Übersicht

Eine Reihe von Messfunktionen ermöglicht die grafische Darstellung des Zeit- und Frequenzverhaltens von Antrieben und Regelungen an der Steuerung. Hierzu werden Testsignale mit einstellbarer Zeitspanne auf die Antriebe geschaltet.

Achtung

Verfahrbereich der Achsen

Während der Durchführung der Messungen an der Maschine ist vom Inbetriebnehmer folgendes sicherzustellen:

  • Der Not-Halt-Taster befindet sich in Reichweite.

  • Es liegen keine Hindernisse im Verfahrbereich.

Voraussetzungen zum Starten von Messfunktionen

Damit sichergestellt ist, dass keine irrtümlichen Verfahrbewegungen aufgrund von Teileprogrammen ausgeführt werden können, müssen die Messfunktionen in der Betriebsart <JOG> gestartet werden.

Achtung

Keine Überwachungsfunktionen aktiv

Während der Verfahrbewegungen im Rahmen der Messfunktionen werden keine Software-Endschalter und Arbeitsfeldbegrenzungen überwacht, da diese im Nachführbetrieb ausgeführt werden.

Der Anwender muss daher vor dem Start der Messfunktionen sicherstellen, dass die Achsen so positioniert sind, dass die im Rahmen der Messfunktionen spezifizierten Verfahrbereichsgrenzen ausreichen, um eine Kollision mit der Maschine zu verhindern.

Einheiten der Messparameter:

Die Anpassung der Testsollwerte an die jeweilige Anwendung erfolgt über Messparameter, deren Einheiten von der jeweiligen Messfunktion und Betriebsart abhängen. Alle Parameter sind mit 0 vorbelegt.

Messfunktionen starten

Messfunktionen, die eine Verfahrbewegung auslösen, werden über den spezifischen Softkey nur angewählt. Der eigentliche Start der Messfunktion und damit der Verfahrbewegung, erfolgt stets über <NC START> an der Maschinensteuertafel.

Nach dem Start der Verfahrfunktion kann das Grundbild ohne Einfluss auf die Verfahrfunktion verlassen werden. Wird das Grundbild der Messfunktion verlassen, ohne dass die Verfahrbewegung begonnen wurde, wird die Anwahl der Verfahrfunktion zurückgenommen.

Der Ablauf einer Messung gliedert sich in folgende Schritte:

  1. Verfahrbereichsüberwachung und Freigabelogik einstellen.

  2. Messtyp auswählen.

  3. Messparameter einstellen mit Softkey "Parameter".

  4. Messung durchführen mit: Softkey "Messung" → Softkey "Messung​ starten".

Abbruchkriterien

Die folgenden Ereignisse führen zum Abbruch aktiver Messfunktionen:

  • Hardware-Endschalter erreicht

  • Verfahrbereichsgrenzen überschritten

  • Not-Halt

  • Reset (BAG, Kanal)

  • NC STOP

  • Wegnahme der Reglerfreigabe

  • Wegnahme der Antriebsfreigabe

  • Wegnahme der Fahrfreigabe

  • Funktion Parken wird angewählt (im lagegeregelten Betrieb)

  • Vorschub-Override 0%

  • Spindel-Override 50%

  • Änderung der Betriebsart (JOG) oder Betriebsart JOG nicht angewählt

  • Betätigen der Verfahrtasten

  • Betätigen des Handrades

  • Alarme, die zum Achsstillstand führen
