Konfiguration der Kompensationstabellen

Vorgehensweise:

  1. Drücken Sie den Softkey "Konfiguration ändern >", um für eine ausgewählte Achse die Kompensationswerte zu editieren.

    Das Fenster "Konfiguration der Kompensationstabellen" wird geöffnet.

  2. Wählen Sie die Art der Kompensation:

    • Einfach (EEC), um eine Tabelle für die Messsystemfehlerkompensation zu konfigurieren.

    • Bidirektional (CEC), um eine Tabelle für die bidirektionale Durchhangkompensation zu konfigurieren.

    Beachten Sie die Voraussetzungen für die Inbetriebnahme.

  3. Geben Sie folgende Werte zum Erstellen der Tabelle ein: Anfangsposition, Endposition und Stützpunktabstand.

    Hinweis

    Kompensationstabelle bei SINUMERIK 828D

    Aus den Werten Anfangsposition, Endposition und Stützpunktabstand wird automatisch die Anzahl der Stützpunkte für die Kompensationstabelle berechnet.

    Die maximale Anzahl von Stützpunkten pro Kompensationstabelle für eine Achse ist bei SINUMERIK 828D fest voreingestellt.

  4. Einfach EEC: Falls mehr als ein Messsystem vorhanden ist, wählen Sie das Messsystem aus, dessen Abweichung kompensiert werden soll. Um die Kompensationstabelle für das ausgewählte Messsystem zu aktivieren, geben Sie die Anzahl der Stützpunkte in MD38000 $MA_MM_ENC_COMP_MAX_POINTS[0,1] ein.

  5. Bidirektional CEC: Wählen Sie einen Index für die jeweilige Tabelle in positiver und negativer Richtung aus. Um die Kompensationstabellen aktivieren, setzen Sie SD43100[Index] = 1.

  6. Bestätigen Sie Ihre Eingaben mit dem Softkey "Aktivieren ✓". Zum Aktivieren der Kompensationtabelle ist pro Achse ein Reset (po) notwendig. Damit werden die Maschinendaten (MD32700, MD32710) zum Aktivieren der jeweiligen Kompensationstabellen [po] und der Kompensation [cf] gesetzt.

  7. Mit Softkey "<< Zurück" kehren Sie zurück zur Anzeige der "Werte der Kompensationstabellen".

Diagnose

Bedienbereich Diagnose → "Service-Achse" → Zeile: "Kompensation Durchhang + Temperatur"
