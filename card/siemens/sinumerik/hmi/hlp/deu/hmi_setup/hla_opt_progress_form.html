HLA-Kennlinie: Messung

Beim Aufblenden des Dialogs wird zunächst nur die im Antrieb eingestellte Kennlinie angezeigt. Nach einer Messung wird die gemessene und die im Antrieb eingestellte Kennlinie angezeigt.

Voraussetzung für die Kennlinienmessung

Eine der folgenden Voraussetzungen muss vor dem Start der Messung erfüllt sein:

  • Im Dialog "Abgleich" wurden alle Abgleiche von Hand durchgeführt.

  • Im Dialog "Ventilkennlinie messen" wurde unter "Messung" die Einstellung "Kennlinienmessung mit Identifikation" ausgewählt. Bei dieser Einstellung werden die Abgleiche automatisch durchführt.

Wenn Sie die Messung ohne vorherigen Abgleich starten, erscheint ein Hinweis auf den notwendigen Abgleich. Für beide Abgleichsarten muss sich die Achse im referenzierten Zustand befinden. Wenn diese Bedingung nicht erfüllt ist, werden Sie darauf hingewiesen.

Start der Kennlinienmessung

Die Messung der Kennlinie starten Sie mit dem Softkey "Messung starten". Die aktuelle Achsposition wird in einem Balkendiagramm angezeigt. Die Messung kann mehrere Minuten dauern. Der Softkey "Messung starten" ändert seine Beschriftung in "Messung stoppen". Alle anderen Softkeys werden gesperrt.

Die Messung können Sie mit dem Softkey "Messung stoppen" stoppen. Im Diagramm wird dann keine gemessene Kennlinie dargestellt. Eine Fehlermeldung wird nicht ausgegeben.

Ergebnis der Kennlinienmessung

Hat der Antrieb die Messung abgeschlossen, stehen die Messergebnisse in den Parametern r1961[0..511] und r1962[0..511] zur Verfügung.

Nach Abschluss der Messung wird der Erfolg oder Misserfolg als Meldung ausgegeben. Die Softkeys werden wieder aktiviert.

Bei erfolgreicher Messung werden die gemessenen Daten im Diagramm als Kennlinie angezeigt. Eine automatische Optimierung der eingestellten Kennlinie wird durchgeführt. Angezeigt wird die optimierte, eingestellte Kennlinie im Dialog "Graphische Optimierung".

Bei fehlgeschlagener Messung wird eine Fehlermeldung ausgegeben. Beheben Sie das Problem und führen Sie die Messung erneut durch.

Um die Antriebssperre nach der Vermessung wieder zu lösen, müssen Sie auf der Maschinensteuertafel die RESET-Taste drücken.

Weitere Aktionen:

  • Mit dem Softkey "Messung starten" starten Sie die Messung der Ventilkennlinie. Nach dem Start der Messung ändert der Softkey "Messung starten" seine Beschriftung in "Messung stoppen". Alle anderen Softkeys werden während der Messung gesperrt. Der Messfortschritt wird mit einem Fortschrittsbalken angezeigt.

  • Mit dem Softkey "Messung stoppen" stoppen Sie die Messung der Ventilkennlinie.

  • Mit dem Softkey "Messung Parameter >" öffnen Sie den Dialog "Messparameter". Dort werden die Messparameter in einer Parameterliste dargestellt.

  • Mit dem Softkey "Zoom >" blenden Sie die Softkey-Leiste "Zoom" ein. Der Dialog "Messung" bleibt unverändert.

  • Mit dem Softkey "Abbruch" kehren Sie zum Dialog "Ventilkennlinie messen" zurück.

  • Mit dem Softkey "OK" öffnen Sie den Dialog "Graphische Optimierung".
