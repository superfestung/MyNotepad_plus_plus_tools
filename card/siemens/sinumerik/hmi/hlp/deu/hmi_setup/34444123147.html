Geberdaten - Einstellungen für Experten

Nach dem Aktivieren des Softkeys "Experte" können Sie weitere Parameter einstellen:

  • Position Warnungsbit

p435[x].3 = 1: ein

p435[x].3 = 0: aus

p435 Zehner + Einer: Position

p435.[x].2 = 1: high active

p435.[x].2 = 0: low active

  • Position Fehlerbit

p434[x].3 = 1: ein

p434 [x].3 = 0: aus

p434 Zehner + Einer: Position

p434.[x].2 = 1: high active

p434.[x].2 = 0: low active

  • Position Parity Bit

p436[x].3 = 1: ein,

p436 [x].3 = 0: aus

p436 Zehner + Einer: Position

p436.[x].2 = 1: ungerade

p436.[x].2 = 0: gerade

  • Monoflopzeit

p428[x]

Minimum: 0

Maximum: 65 535

  • Doppelübertragung

p429[x].2 = 0: ja

p429[x].2 = 1: nein

  • Füllbits

    Das Feld wird nur angezeigt, wenn Doppelübertragung eingeschaltet ist.

p449[x]

Minimum: 0

Maximum: 63

Prüfungen:

  • Wenn eine Bitposition für Warnungsbit, Fehlerbit und Parity Bit mehrfach belegt ist, wird eine Meldung mit der Aufforderung ausgegeben, die Eingaben zu überprüfen.

  • Der Wert der Füllbits wird nur überprüft, wenn Doppelübertragung aktiviert ist.

  • Der Wertebereich der Füllbits wird nicht überprüft, da diese Prüfung in der Überprüfung der Positionslänge und der maximalen Telegrammlänge enthalten ist:

    • Berechnung der Positionslänge:

      2x (Bits vor der Position+ Positionslänge + Bits nach der Position) + Füllbits ≤ 64

    • Wird der Maximalwert überschritten, wird eine die Meldung ausgegeben.

      2x (Bits vor der Position+ Positionslänge + Bits nach der Position) + Füllbits ≤ 64

  • Bits, die Zusatzinformationen mitbringen, können nur im Bereich vor oder nach der Position liegen.
