Lizenzbedarf exportieren

Wenn Sie den Lizenzbedarf ermittelt haben, können Sie die Daten über die eingerichteten externen Speicherorte exportieren.

Vorgehensweise

 

1.

Drücken Sie den Softkey "Lizenzbedarf exportieren".

Ein Fenster öffnet sich und bietet die eingerichteten Speicherorte an.

 

2.

Wählen Sie z. B. "USB", wenn Sie die Lizenzen auf einen USB-FlashDrive exportieren.

 

3.

Drücken Sie den Softkey "OK".

Die gesetzten Optionen, die nicht gesetzten aber lizenzierten Optionen, der Lizenzbedarf und der License Key werden in einer Alm-Datei gespeichert.

Zusätzlich wird eine Internetverknüpfung für den Web License Manager erstellt.
