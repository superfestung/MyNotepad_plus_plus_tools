Aufruf der Messvariante

Vorgehensweise

Das zu bearbeitende Teileprogramm bzw. ShopMill-Programm ist angelegt und Sie befinden sich im Editor.

 

1.

Drücken Sie den Softkey "Werkst. messen".

 

2.

Drücken Sie den Softkey "Abgleich Messtaster".

 

3.

Drücken Sie den Softkey "Radius an Kante".
Das Eingabefenster "Abgleich: Radius an Kante" wird geöffnet.

 

4.

Wählen Sie im Auswahlfeld Abgleichrichtungen, "2".
