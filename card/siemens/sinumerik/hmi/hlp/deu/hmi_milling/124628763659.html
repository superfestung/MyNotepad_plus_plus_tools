Daten filtern

Sie haben die Möglichkeit, die angezeigten Maschinendaten thematisch zu begrenzen und anschließend gezielt nach Zeichenfolgen zu suchen.

Wenn Sie das Filtern nutzen, werden die Einstellungen der Anzeigeoptionen deaktiviert.

Filteroptionen

  • Gruppe

    Eine Gruppe umfasst thematisch zusammengehörende Maschinen- oder Settingdaten innerhalb des angewählten Datenbereiches. Es werden dieselben Gruppen angeboten wie im Fenster "Anzeigeoptionen". Es kann jedoch nur eine Gruppe ausgewählt werden.

  • Filter

    Im Feld "Filter" geben Sie die Zeichenfolge ein, nach der Sie in der ausgewählten Gruppe suchen möchten.

    Hinweis

    Der Einsatz von Platzhaltern im Feld "Filter" wird nicht unterstützt.

    Groß-/Kleinschreibung wird ignoriert.

Durchsuchte Felder

Folgende Felder der angezeigten Maschinendaten werden nach der eingegebenen Zeichenkette durchsucht:

  • Maschinendatennummer

  • Symbolischer Name

  • Beschreibungstext

  • Bittexte

  • Enum- Texte (nur bei Antrieben)

Vorgehensweise

 

1.

Wählen Sie den Bedienbereich "Inbetriebnahme" an.

 

2.

Drücken Sie den Softkey "Masch.daten".

 

3.

Wählen Sie den gewünschten Bereich aus, indem Sie den entsprechenden Softkey drücken.

 

4.

Drücken Sie den Softkey "Filtern".

 

5.

Wählen Sie im Feld "Gruppe" den gewünschte Datenbereich.

 

6.

Geben Sie im Feld "Filter" die gewünschte Zeichenfolge ein.

Die angezeigten Daten werden nach jedem Zeichen aktualisiert.

 

7.

Um den Filter aufzuheben, drücken Sie den Softkey "Filtern" erneut.
