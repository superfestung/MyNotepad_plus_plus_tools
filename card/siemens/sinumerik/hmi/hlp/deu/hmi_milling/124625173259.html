Variablenprogrammierung

Grundsätzlich können in den Eingabefeldern der Masken statt konkreter Zahlenwerte auch Variablen oder Ausdrücke verwendet werden. Hiermit können Programme sehr flexibel erstellt werden.

Eingabe von Variablen

Beachten Sie folgende Punkte bei der Verwendung von Variablen:

  • Werte von Variablen und Ausdrücken werden nicht überprüft, da die Werte zum Zeitpunkt der Programmierung nicht bekannt sind.

  • In Feldern, die einen Text erwarten, können keine Variablen und Ausdrücke verwendet werden (z.B. Werkzeugname).

    Eine Ausnahme bildet die Funktion "Gravur", bei der Sie im Textfeld den gewünschten Text als

    "variablen Text" über eine Variable zuordnen können.

  • Auswahlfelder können generell nicht variabel programmiert werden.

Beispiele

VAR_A

VAR_A+2*VAR_B

SIN(VAR_C)
