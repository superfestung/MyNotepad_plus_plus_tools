Programm wählen

Vorgehensweise

 

1.

Wählen Sie den Bedienbereich "Programm-Manager" an.

Die Verzeichnisübersicht wird geöffnet.

 

2.

Wählen Sie den Ablageort des Programms an (z. B. "NC")

 

3.

Positionieren Sie den Cursor auf das Verzeichnis, in dem Sie ein Programm anwählen möchten.

 

4.

Drücken Sie die Taste <INPUT>.

- ODER -

 

 

Drücken Sie die Taste <Cursor rechts>.

Der Verzeichnisinhalt wird angezeigt.

 

5.

Positionieren Sie den Cursor auf das gewünschte Programm.

 

6.

Drücken Sie den Softkey "Anwahl".

Bei erfolgreicher Programmanwahl erfolgt ein automatischer Wechsel in den Bedienbereich "Maschine".
