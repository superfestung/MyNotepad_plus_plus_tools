Neue Ebene anlegen

In der Maske "Aus CAD importieren" werden gleichzeitig das 3D-Modell des Werkstücks und die vorgeschlagenen Bearbeitungsschritte angezeigt. Hier werden automatisch alle Ebenen aufgeführt, auf denen Formen erkannt wurden.

Zusätzlich können Sie im Bearbeitungsschritt "Werkstück", manuell eine Ebene hinzuzufügen, um z. B. eine Kontur in dieser Ebene zu erstellen.

Vorgehensweise

 

1.

Wählen Sie den Bearbeitungsschritt "Werkstück" aus.

 

2.

Drücken Sie den Softkey "Neue Ebene".

Das Fenster "Neue Ebene" wird geöffnet.

 

3.

Wählen Sie im Feld "Auswahl" die Fläche.

  

-ODER-

  

Selektieren Sie die gewünschte Fläche im 3D-Modell.

Die gewählten Elemente werden im 3D-Modell hervorgehoben und im Feld "Auswahl" angezeigt.

 

4.

Wählen Sie im Feld "Zuordnung" die Spindel, der Sie die Fläche zuordnen möchten.

Hinweis

Die Möglichkeit der Zuordnung zu einer Spindel gibt es nur dann, wenn die Maschine eine Gegenspindel hat.

 

5.

Bestätigen Sie die Auswahl mit "Übernehmen".

Die eingefügte Ebene erscheit in der Liste der Bearbeitungssschritte.
