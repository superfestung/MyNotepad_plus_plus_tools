Satzsuchlauf auf Positionsmuster

Sie haben die Möglichkeit, einen Satzsuchlauf auf Positionsmuster durchzuführen. Dabei legen Sie die Nummer der Startbohrung fest.

Bei ShopMill-Programmen können Sie zusätzlich noch die Technologie festlegen, mit der Sie beginnen wollen.

 

Software-Option

Für den Satzsuchlauf auf ShopMill Arbeitsschritt-Programmen benötigen Sie die Option "ShopMill/ShopTurn".

 

Maschinenhersteller

Beachten Sie hierzu die Angaben des Maschinenherstellers.

Vorgehensweise

  

Das gewünschte ShopMill-Programm bzw. G-Code-Programm befindet sich in der Satzanzeige.

 

1.

Drücken Sie den Softkey "Satzsuchl."

 

2.

Positionieren Sie den Cursor auf den Positionssatz.

 

3.

Drücken Sie den Softkey "Suchlauf starten".

Das Fenster "Suchlauf" öffnet sich.

Technologie festlegen (nur bei ShopMill-Programm)

 

4.

Wählen Sie die gewünschte Technologie und drücken Sie auf den Softkey "OK".

Im Fenster "Suchlauf" wird die ausgewählte Technologie angezeigt.

Startbohrung angeben

 

5.

Geben Sie die Nummer der Startbohrung ein und drücken Sie auf den Softkey "OK".

Die Bearbeitung des Programms beginnt mit der angegebenen Technologie (nur bei ShopMill-Programm) auf der angegebenen Startbohrung und führt sie auf allen weiteren Positionen dieses Positionsmusters sowie aller folgenden Positionsmuster aus.

 

 

Hinweis

Wenn Sie Positionen ausgeblendet haben, dann zählen für die Nummer der Startbohrung nur die eingeblendeten Positionen.

Hinweis

Automatische Technologieauswahl bei ShopMill

Die Technologie wird automatisch ausgewählt, wenn der aktuelle Datensatz gespeichert wurde, sodass ein Bezug zum aktiven Positionsmuster hergestellt werden kann.

Wenn die Technologie nicht automatisch ausgewählt wurde, müssen Sie die richtige Technologie manuell auswählen.

Hinweis

Automatische Auswahl der Startbohrung

Die Nummer der Startbohrung wird automatisch ermittelt und angezeigt, wenn der aktuelle Datensatz gespeichert wurde, sodass ein Bezug zum aktiven Positionsmuster hergestellt werden kann.

Wenn die Nummer der Startbohrung nicht automatisch ermittelt wurde, müssen Sie die Nummer der Startbohrung manuell eingeben.
