Bearbeitungsschritte übernehmen

Um die vorgeschlagenen Daten in das Programm zu übernehmen, wählen Sie den entsprechenden Bearbeitungsschritt in der Liste der Bearbeitungsschritte aus und ergänzen gegebenenfalls weitere Parameter. Der Bearbeitungsschritt wird in das Programm geschrieben und die Liste der Bearbeitungsschritte wird geschlossen.

Hinweis

Die Parametermasken der vom NX-Server erkannten Bearbeitungsschritte wie Bohren, Reiben, Ausdrehen o. dgl. enthalten im Gegensatz zu Parametermasken der gleichnamigen Zyklen den Parameter "Durchmesser". Dieser Parameter unterstützt Sie bei der richtigen Werkzeugauswahl für die jeweilige Bearbeitung.
Hinweis

Die Werte, die aus dem 3D-Modell hervorgehen, sind in den Parametermasken gelb markiert.

Eine Änderung dieser Werte wird nicht empohlen.

Voraussetzung

  • Die Liste der Bearbeitungsschritte ist geöffnet.

Vorgehensweise

 

1.

Wählen Sie im 3D-Modell das gewünschte Element per Touch- oder Mausbedienung.

Das Element wird im 3D-Modell hervorgehoben. In der Liste springt der Cursor auf den dazugehörigen Bearbeitungsschritt.

  

-ODER-

  

Navigieren Sie in der Liste mithilfe der Cursortasten zum gewünschten Bearbeitungsschritt.

Das dazugehörige Element wird im 3D-Modell farblich hervorgehoben.

 

2.

Falls mehrere, alternative Bearbeitungsschritte für das gewählte Element existieren, werden diese Schritte im Block "Alternativen" zusammengefasst.

Wählen Sie in diesem Fall die gewünschte technologische Funktion.

 

3.

Drücken Sie den Softkey "Übernehmen".

Die Parametermaske der gewählten Funktion wird geöffnet. Werte, die aus den CAD-Daten hervorgehen, werden übernommen. Alle anderen Parameter werden mit den Standardwerten vorbelegt. Die Standardwerte sind die zuletzt eingegebenen Werte der Parametermaske.

 

4.

Überprüfen Sie die Parameter und ändern Sie gegebenenfalls die Standardwerte.

 

5.

Bestätigen Sie die Angaben mit "OK".

Die Programmansicht wird geöffnet. Der neu erstellte Bearbeitungsschritt ist markiert.

 

6.

Wenn Sie weitere Bearbeitungsschritte übernehmen möchten, drücken Sie den Softkey "Aus CAD importieren" erneut.

Die Liste der Bearbeitungsschritte wird erneut geöffnet.

Hinweis

Bevor Bearbeitungsschritte auf einer Ebene übernommen werden, muss der Schritt "Schwenken Ebene" übernommen werden.

Hinweis

Schwenken nach Werkzeugwechsel (nur G-Code)

Wenn an Ihrer Maschine Schwenken eingerichtet ist, muss der Schritt Schwenken Ebene nach jedem Werkzeugwechsel eingefügt werden.

Hinweis

Bearbeitungsschritte, die Sie ins Programm übernehmen, werden mit einem grünen Haken gekennzeichnet.

Wenn anschließend der Nullpunkt für das Werkstück oder eine Ebene geändert wird, stimmen die zugehörigen Schritte nicht mehr.

Diese Schritte werden mit einem Ausrufezeichen gekennzeichnet.

Wenn Sie diese Schritte erneut übernehmen, werden die Schritte mit einem grünen Haken gekennzeichnet.
