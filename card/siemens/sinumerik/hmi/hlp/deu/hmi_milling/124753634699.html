License Key einlesen

Sie haben die Möglichkeit, den License Key per Lizenzdatei an der Steuerung einzulesen.

Vorgehensweise

 

 

Das Fenster "Lizenzierung" ist geöffnet.

License Key elektronisch einlesen

...

 

Wenn Sie über eine .zip-Datei mit passendem License Key verfügen, drücken Sie den Softkey "License Key einlesen".

Öffnen Sie den entsprechenden Speicherort (z. B. USB) mit "OK".

Die passende .zip-Datei wird automatisch erkannt und der License Key wird eingelesen.

 

 

Hinweis:

Befinden sich mehrere passende .zip-Dateien im Verzeichnis, dann werden Ihnen diese in einem nachfolgenden Auswahldialog angeboten.

Wählen Sie die gewünschte Datei aus.
