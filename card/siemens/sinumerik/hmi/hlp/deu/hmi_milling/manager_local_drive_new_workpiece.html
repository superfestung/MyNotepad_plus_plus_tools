Neues Werkstück anlegen

In einem Werkstück erzeugen Sie verschiedene Dateitypen wie Hauptprogramme, Initialisierungsdatei, Werkzeugkorrekturen.

Hinweis

Werkstückverzeichnisse

Sie haben die Möglichkeit, Werkstückverzeichnisse zu verschachteln. Beachten Sie, dass die Länge der Aufrufzeile beschränkt ist. Wenn die maximale Anzahl an Zeichen erreicht ist, werden Sie bei der Eingabe des Werkstücknamens darüber informiert.

Vorgehensweise

 

1.

Wählen Sie den Bedienbereich "Programm-Manager" an.

 

2.

Wählen Sie den gewünschten Ablageort.

 

3.

Positionieren Sie den Cursor auf den Ordner, unter dem Sie das Werkstück anlegen wollen.

 

4.

Drücken Sie den Softkey "Neu".

Das Fenster "Neues Werkstück" wird geöffnet.

 

5.

Wählen Sie bei Bedarf eine Vorlage an, wenn solche angelegt sind.

 

6.

Geben Sie den gewünschten Namen des Werkstücks ein und drücken Sie den Softkey "OK".

 

 

Ein neuer Ordner mit dem Namen des Werkstücks wird angelegt.

Der Verzeichnistyp (WPD) wird vorgegeben.

Das Fenster "Neues G-Code Programm" öffnet sich.

 

7.

Drücken Sie erneut den Softkey "OK", wenn Sie das Programm anlegen möchten.

 

 

Das Programm öffnet sich im Editor.
