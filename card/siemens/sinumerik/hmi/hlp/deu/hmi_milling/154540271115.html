Powerride für Programmbeeinflussungen einsetzen

"Powerride" ist ein zusätzlicher Dreh-/Drückschalter auf der Maschinensteuertafel. Der Powerride kombiniert den Override-Wert und den NC-Start und erleichtert das Programmeinfahren sowie die Einzelsatzbearbeitung.

 

Maschinenhersteller

Beachten Sie hierzu die Angaben des Maschinenherstellers.

Durch die Nutzung des Powerrides können Sie das Zurückdrehen des Overrides auf 0 % beim Einzelsatz und auch beim konfigurierten Halt automatisieren.

Der Override-Wert kann mithilfe der PLC-Software auf 0 % abgesenkt werden.

Für die Nutzung des Powerrides aktivieren Sie im Fenster "Konfigurierter Halt" das Kontrollkästchen "Bei Stop Override auf 0 % setzen".

Voraussetzung

  • Die Maschine befindet sich im Reset-Modus und in der Betriebsart "Automatik".

  • "Konfigurierter Halt" ist aktiviert.

Vorgehensweise

  1. Drücken Sie den Powerride-Schalter.

    Die Steuerung stoppt auf dem ersten Satzende, für das der konfigurierte Halt konfiguriert ist.

    Der PLC setzt den Override auf 0 %.

  2. Drücken Sie den Powerride-Schalter.

    Der neue Satz wird eingewechselt.

  3. Drehen Sie den Override-Schalter langsam auf.

    Die Maschine fährt los.
