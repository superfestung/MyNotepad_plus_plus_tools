Nullpunkt des Werkstücks festlegen

Wenn der Nullpunkt aus dem 3D-Modell vom Werkstücknullpunkt abweicht, haben Sie die Möglichkeit, einen neuen Bezugspunkt festzulegen.

Hinweis

Legen Sie zuerst den Nullpunkt Ebene für die relevanten Ebene fest, bevor Sie Features und Elemente mit "Übernehmen" in das Programm einfügen.

Vorgehensweise

 

1.

Wählen Sie den Bearbeitungsschritt "Werkstück" aus.

 

2.

Drücken Sie den Softkey "Nullpunkt Werkstück".

Das Fenster "Nullpunkt Werkstück" wird geöffnet.

 

3.

Wählen Sie, wie der Nullpunkt festgelegt wird.

 

4.

Wählen Sie im Feld "Auswahl" die gewünschten Bezüge.

  

-ODER-

  

Selektieren Sie die gewünschten Bezüge im 3D-Modell.

Die gewählten Elemente werden im 3D-Modell hervorgehoben und im Feld "Auswahl" angezeigt.

 

5.

Wählen Sie die Parameter "Richtung Z", "Richtung X" und "Drehung Z", um das Koordinatenkreuz auszurichten.

 

6.

Bestätigen Sie die Auswahl mit "Übernehmen".
