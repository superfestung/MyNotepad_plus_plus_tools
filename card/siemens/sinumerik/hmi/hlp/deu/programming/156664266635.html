Konturaufbereitung ausschalten (EXECUTE)

Mit EXECUTE wird die Konturaufbereitung abgeschaltet und gleichzeitig in den normalen Abarbeitungsmodus zurückgeschaltet.

Syntax

EXECUTE(<FEHLER>)

Bedeutung

EXECUTE:

Vordefinierte Prozedur zum Beenden der Konturaufbereitung

<FEHLER>:

Variable für Fehlerrückmeldung

Typ:

INT

Der Wert der Variablen zeigt an, ob die Kontur fehlerfrei aufbereitet werden konnte:

0

Fehler

1

kein Fehler

Beispiel

Programmcode
...
N30 CONTPRON(...)
N40 G1 X... Z...
...
N100 EXECUTE(...)
...
