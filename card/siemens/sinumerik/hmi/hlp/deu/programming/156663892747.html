Initialisierung von Feldvariablen (SET, REP)

Funktion

Feldvariable, die in Synchronaktionen schreibbar sind, können über die Befehle SET und REP in Synchronaktionen auch initialisiert werden.

Beispiel

Wertzuweisung an INTEGER-Anwendervariablen für Synchronaktionen im Datenbaustein SGUD (SYG_IS)

Voraussetzung: MD18661 $MM_NUM_SYNACT_GUD_INT[<x>] ≥ 5

Programmcode 
PROC MAIN 
N10 DEF INT SYG_IS[4] 
... 
WHEN TRUE DO SYG_IS[0]=REP(1,3); Wenn Bedingung erfüllt: 1. bis 3. INTEGER-Anwendervariable für Synchronaktionen im Baustein SGUD auf den Wert "1" setzen.
WHEN TRUE DO SYG_IS[3]=SET(2,4); Wenn Bedingung erfüllt: 4. und 5. INTEGER-Anwendervariable für Synchronaktionen im Baustein SGUD auf den Wert "2" bzw. "4" setzen.
...
