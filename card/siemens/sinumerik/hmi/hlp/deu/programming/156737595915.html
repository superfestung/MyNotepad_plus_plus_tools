Winkel zum Ausrichten des Werkzeugs für TRAINT berechnen (CALCTRAVAR)

Für die Drehinterpolationstransformationen TRAINT müssen Spindel und Achsen so positioniert werden, dass die Werkzeugspitze zur Drehmitte hin orientiert ist und auf y = 0 steht. Zum Bestimmen der geeigneten Winkel und Positionen sind komplexe Berechnungen erforderlich. Unterstützung bei dieser Aufgabe bietet die Funktion CALCTRAVAR.

Voraussetzung

Vor Aufruf von CALCTRAVAR muss die kinematische Kette definiert sein.

Hinweis

CALCTRAVAR berechnet die Winkel auf Grundlage der definierten kinematischen Kette. Veränderungen der kinematischen Kette nach Aufruf von CALCTRAVAR können daher dazu führen, dass die von CALCTRAVAR berechneten Werte nicht mehr korrekt sind.

Syntax

<Status> = CALCTRAVAR(<Result>, "<Trafoname>", <γ>)

Bedeutung

CALCTRAVAR(...)

Vordefinierte Funktion: Winkel zum Ausrichten des Werkzeugs für TRAINT berechnen

<Result>

2-dimensionale Ergebnisvariable

Nach Aufruf von CALCTRAVAR werden hier die für die Vorpositionierung nötigen Werte gespeichert:

  • <Result>[0]: Korrekte Position der Spindel zur Anwahl der Transformation TRAINT

  • <Result>[1]: Korrekte Frame-Drehung zur Anwahl der Transformation TRAINT

Datentyp:

Real

Hinweis:
Im Fehlerfall (<Status> ≠ 0) haben die zurückgegebenen Winkel den Wert 0.

<Trafoname>

Name des später mit TRAFOON zu aktivierenden TRAINT-Transformationsdatensatzes

Datentyp:

STRING

Wertebereich:

Alle über $NT_NAME definierte Namen von TRAINT-Transformationsdatensätzen

<γ>

Ausrichtung der Werkzeugschneide zur Kreistangente

Der Winkel γ wird für die Korrektur der Schneidenparameter mit der Funktion CUTMODK verwendet.

Datentyp:

INT

Wertebereich:

0, 180

Wert:

0

Die Schneide steht senkrecht auf der Kreistangente und ist außerhalb des Kreises (→ Außenbearbeitung).

180

Die Schneide steht ebenfalls senkrecht auf der Kreistangente, ist aber innerhalb des Kreises (→ Innenbearbeitung).

Hinweis:
Wird der Parameter nicht programmiert, wird standardmäßig der Wert 0 angenommen.

<Status>

Rückgabewert zum Funktionsstatus

Datentyp:

INT

Wertebereich:

0 ... 8

Wert:

0

kein Fehler, Aufruf war erfolgreich

1

Parameter <Name> fehlt oder ist leer

2

Parameter <γ> hat einen ungültigen Wert (darf nur 0 oder 180 sein)

3

Fehler bei der Aufbereitung der kinematischen Kette

4

Für TRAINT parametrierte Spindel nicht in der kinematischen Kette gefunden

5

Für TRAINT wurde keine Spindel parametriert

6

Parameter <Name> bezeichnet keine definierte Transformation

7

Allgemeiner Fehler

8

Kein Werkzeug programmiert
