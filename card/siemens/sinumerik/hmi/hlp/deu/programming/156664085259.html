Kontext-Variable ($P_TECCYCLE)

Funktion

Wird ein NC-Programm sowohl als Teileprogramm als auch als Technologiezyklus verwendet, können über die Systemvariable $P_TECCYCLE kontext-spezifische Programmabschnitte definiert werden:

  • $P_TECCYCLE == TRUE ⇒ Das NC-Programm wird aktuell als Technologiezyklus abgearbeitet

  • $P_TECCYCLE == FALSE ⇒ Das NC-Programm wird aktuell als Teileprogramm abgearbeitet

Anwendung

In Technologiezyklen haben Variablen- (DEF) und Makrodefinitionen (DEFINE) keine Auswirkung. Wird ein NC-Programm als Technologiezyklus abgearbeitet, das entsprechende Definitionen enthält, ist im Programmcode eine kontext-spezifische Fallunterscheidung notwendig, da die Variablen und Makros dann nicht zur Verfügung stehen.

Beispiel

Verfahrparameter über Anwendervariablen im Teileprogramm und R-Parameter im Technologiezyklus

ProgrammcodeKommentar: Verwendung im
PROC UP_1 
DEF REAL POS_X=100.0Teileprogramm
DEF REAL F_X=250.0Teileprogramm
IF $P_TECCYCLE==TRUE 
$R1=100.0Technologiezyklus
$R2=250.0Technologiezyklus
ENDIF 
IF $P_TECCYCLE==TRUE 
N100 POS[X]=$R1 FA[X]=$R2Technologiezyklus
ELSE 
N200 POS[X]=POS_X FA[X]=F_XTeileprogramm
ENDIF 
RET
