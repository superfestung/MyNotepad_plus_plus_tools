Online-Werkzeuglängenkorrektur (TOFFON, TOFFOF)

Mit der Funktion "Online-Werkzeuglängenkorrektur" ist es möglich, während einer Bearbeitung mit aktiver Orientierungstransformation (TRAORI)/Kinematischer Kette oder aktivem orientierbaren Werkzeugträger (TCARR) die effektiven Werkzeuglängen in allen drei Werkzeugrichtungen in Echtzeit zu überlagern.

Aktivierung und Deaktivierung erfolgen über die vordefinierten Prozeduren TOFFON() und TOFFOF().

Bereits beim Aktivieren kann für die angegebene Korrekturrichtung ein Offsetwert angegeben werden, der sofort herausgefahren wird.

Danach können Online-Werkzeuglängenkorrekturen über die Systemvariable $AA_TOFF[<n>] aufgeschaltet werden. Dies ist in allen drei Werkzeugrichtungen einer aktiven Orientierungstransformation/ oder eines aktiven orientierbaren Werkzeugträgers möglich. Als Index <n> werden die drei Geometrieachsnamen X, Y, Z verwendet. Es können maximal so viele Korrekturrichtungen aktiv sein, wie Geometrieachsen aktiv sind. Alle drei Korrekturrichtungen können gleichzeitig aktiv sein.

Syntax

... 
TOFFON(<CorrDir>[,<Offset>]) 
WHEN TRUE DO $AA_TOFF[<CorrDir>]=<CorrVal> 
... 
TOFFOF(<CorrDir>) 
... 

Bedeutung

TOFFON

Online-Werkzeuglängenkorrektur aktivieren

<CorrDir>

Korrekturrichtung

Werkzeugrichtung (X, Y, Z), in der die Online-Werkzeuglängenkorrektur wirksam sein soll.

<Offset>

Offsetwert (optional)

Korrekturwert, der bei der Aktivierung sofort in der angegebenen Korrekturrichtung herausgefahren werden soll.

WHEN TRUE DO ...

Synchronaktion, die startet, sobald der Satz eingewechselt wird.

$AA_TOFF[<CorrDir>]=<CorrVal>

Online-Werkzeuglängenkorrektur aufschalten

<CorrDir>

Korrekturrichtung

Werkzeugrichtung (X, Y, Z), in der der angegebene Korrekturwert interpoliert werden soll.

<CorrDir>

Korrekturwert

TOFFOF

Online-Werkzeuglängenkorrektur deaktivieren

<CorrDir>

Korrekturrichtung

Werkzeugrichtung (X, Y, Z), in der die Online-Werkzeuglängenkorrektur deaktiviert werden soll.

Nach Abwahl der Online-Werkzeuglängenkorrektur in der angegebenen Korrekturrichtung wird der Wert der entsprechenden Systemvariablen $AA_TOFF[<n>] bzw. $AA_TOFF_VAL[<n>] auf "0" zurückgesetzt. Ein Vorlaufstopp wird ausgelöst.

Hinweis

Eine wiederholte Programmierung der Anweisung TOFFON( ) mit einem neuen Offsetwert führt dazu, dass der neue Offsetwert herausgefahren wird. Der Offsetwert wird hierbei als absoluter Wert zur Variablen $AA_TOFF[<n>] addiert.

Hinweis

Vor dem Ein- bzw. Ausschalten einer Transformation muss eine überlagerte Bewegung mit TOFFOF() ausgeschaltet werden.

Beispiele

Beispiel 1 und 2: Online-Werkzeuglängenkorrektur bei aktiver Orientierungstransformation (TRAORI)

ProgrammcodeKommentar
$MC_TOFF_MODE=1; Der Korrekturwert $AA_TOFF[<n>]soll absolut verrechnet und über Reset hinaus erhalten bleiben.
$MC_TOFF_VELO[0]=1000; Geschwindigkeit für die Online-Korrekturbewegung in Werkzeugrichtung X, Y, Z
$MC_TOFF_VELO[1]=1000
$MC_TOFF_VELO[2]=1000
$MC_TOFF_ACCEL[0]=1; Beschleunigung für die Online-Korrekturbewegung in Werkzeugrichtung X, Y, Z
$MC_TOFF_ACCEL[1]=1
$MC_TOFF_ACCEL[2]=1
  
N5 DEF REAL XOFFSET 
N10 TRAORI(1); Orientierungstransformation ein
N20 TOFFON(Z); Online-Werkzeuglängenkorrektur in Werkzeugrichtung Z aktivieren.
N30 WHEN TRUE DO $AA_TOFF[Z]=10 G4 F5; Werkzeuglängenkorrektur für Werkzeugrichtung Z: 10 mm
... 
N100 XOFFSET=$AA_TOFF_VAL[X]; Positionsoffset in Werkzeugrichtung X der Variablen XOFFSET zuweisen.
N120 TOFFON(X,-XOFFSET) G4 F5; Online-Werkzeuglängenkorrektur für Werkzeugrichtung X aktivieren. ; Positionsoffset in Werkzeugrichtung X auf 0 zurückfahren.

ProgrammcodeKommentar
N10 TRAORI(1); Orientierungstransformation ein
N20 TOFFON(X); Online-Werkzeuglängenkorrektur für Werkzeugrichtung X aktivieren.
N30 WHEN TRUE DO $AA_TOFF[X]=10 G4 F5; Werkzeuglängenkorrektur für Werkzeugrichtung X: 10 mm
... 
N80 TOFFOF(X); Online-Werkzeuglängenkorrektur für Werkzeugrichtung X deaktivieren.; Positionsoffset der Werkzeugrichtung X wird gelöscht: $AA_TOFF[X]=0; Es wird keine Achse verfahren.; Zur aktuellen Position im WKS wird der Positionsoffset entsprechend der aktuellen Orientierung hinzugerechnet.

Beispiel 3: Online-Werkzeuglängenkorrektur bei aktivem orientierbaren Werkzeugträger (TCARR)

ProgrammcodeKommentar
... 
DEF REAL XOFFSET 
$MC_TOFF_MODE=1; Der Korrekturwert $AA_TOFF[<n>]soll absolut verrechnet und über Reset hinaus erhalten bleiben.
$MC_TOFF_VELO[0]=10000; Geschwindigkeit für die Online-Korrekturbewegung in Werkzeugrichtung X, Y, Z
$MC_TOFF_VELO[1]=10000
$MC_TOFF_VELO[2]=10000
$MC_TOFF_ACCEL[0]=1 ; Beschleunigung für die Online-Korrekturbewegung in Werkzeugrichtung X, Y, Z
$MC_TOFF_ACCEL[1]=1
$MC_TOFF_ACCEL[2]=1
NEWCONF 
 
; CYCLE800()
TCARR=0; Werkzeugträger deaktivieren.
$P_WPFR=CTRANS() 
$P_WPFRAME=$P_WPFR 
G0 X0 Y0 Z0 A0 C0 
M0 
 
; CYCLE800(0,"KOPF",100000,57,0,0,0,30,0,0,0,0,0,-1,100,101)
G17 
$P_WPFR=CTRANS() 
$P_WPFR=CROT(X,30) 
$P_WPFRAME=$P_WPFR 
CUT2DF TCARR=2; Werkzeugträger aktivieren.
TCOFR 
PAROT 
G0 C=$P_TCANG[3]  A=$P_TCANG[4] 
M0 
  
TOFFON(Z); Online-Werkzeuglängenkorrektur für Werkzeugrichtung Z aktivieren.
WHEN TRUE DO $AA_TOFF[Z]=10; Werkzeuglängenkorrektur für Werkzeugrichtung Z: 10 mm
M0 
G4 F5 
M0 
  
WHEN TRUE DO $AA_TOFF[Z]=0; Werkzeuglängenkorrektur für Werkzeugrichtung Z: 0 mm
M0 
G4 F5 
M0 
  
TOFFOF(Z); Online-Werkzeuglängenkorrektur für Werkzeugrichtung Z deaktivieren.
M0 
 
; CYCLE800()
TCARR=0; Werkzeugträger deaktivieren.
$P_WPFR=CTRANS() 
$P_WPFRAME=$P_WPFR 
M30
