Aufrufparameter CYCLE150

Nr.

Masken-parameter

Zyklen-parameter

Bedeutung

1

Messergeb-nisbild

S_PICT

Auswahl Ergebnisanzeige (Default = 0)

   

Werte:

EINER:

0 = Messergebnisbild AUS
1 = Messergebnisbild EIN

    

Zehner: Auswahl Anzeigemodus (Werte wie SD55613 $SCS_MEA_RESULT_DISPLAY)

1 = Messergebnisbild anzeigen - Abwahl automatisch nach 8 s
3 = Messergebnisbild anzeigen - Quittieren mit NC-Start
4 = Messergebnisbild anzeigen - nur bei den Alarmen (61303 ... 61306)

2

 

S_PROT

Auswahl Protokollieren (Default = 0)

 

Protokoll

 

Werte:

EINER: Auswahl Protokoll aus / ein / letzte Messung

0 = Protokoll AUS
1 = Protokoll EIN
2 = Protokolliere letzte Messung

 

Protokolltyp

  

ZEHNER: Auswahl Protokolltyp

0 = Standardprotokoll
1 = Anwenderprotokoll (frei definierbar)

 

Protokollformat

  

HUNDERTER: Auswahl Protokollformat

0 = Textformat
1 = Tabellenformat (für Excel Import)

 

Protokoll-daten

  

TAUSENDER: Auswahl neu schreiben oder anhängen

0 = neu
1 = anhängen

 

Protokoll

ablage

  

ZEHNTAUSENDER: Auswahl Protokollablage

0 = wie Teileprogramm
1 = Verzeichnis

3

 

S_PATH

Pfad für  Protokolldatei entsprechend Auswahl Protokollablage
(komplette Pfadangabe oder nur Dateiname, z. B.:
"//NC:/WKS.DIR/NAME.WPD oder "MESSPROTOKOLL.TXT"
