CYCLE805 - Y-Drehen

Hinweis

Für das Y-Drehen mit CYCLE805 ist die lizenzpflichtige Option "Drehbearbeitung mit der Y-Achse" erforderlich!

Syntax

CYCLE805(<_MODE>, < _TC>, < _GAMA>, <_FR_I>, <S_ZS>, <S_XS>, <_AMODE>)

Parameter

Nr.

Parameter Maske

Parameter intern

Datentyp

Bedeutung

1

 

<_MODE>

INT

Modus

An-/Abwahl

EINER:

An-/Abwahl

0 =

Y-Drehen abwählen

1 =

Y-Drehen anwählen

 

ZEHNER:

Reserviert

Nachführen

HUNDERTER:

Werkzeugspitze nachführen ja/nein (nur aktiv, wenn auf der Maschine das "Schwenken mit CYCLE800" eingerichtet ist)

0 =

Werkzeugspitze nicht nachführen

1 =

Werkzeugspitze nachführen (TRAORI)

2

TC

<_TC>

STRING[32]

Name des Schwenkdatensatzes

 

"<Name>"

Wenn mehrere Schwenkdatensätze vorhanden sind

""

(kein Name, Leerstring)

Wenn nur ein Schwenkdatensatz vorhanden ist

"0"

Abwahl Y-Drehen

4

Gamma

<_GAMA>

REAL

Winkel Gamma (Rotation um die Werkzeugachse) / Ausrichtwinkel der Schneidplatte

5

FR

<_FR_I>

REAL

Freifahren in Werkzeugrichtung (inkrementell)

6

ZS

<S_ZS>

REAL

Start/Freifahrpunkt Z

(siehe Parameter <_AMODE>, "EINER"  und "HUNDERTER")

7

XS/YR

<S_XS>

REAL

Startpunkt X / Freifahrpunkt Y

(siehe Parameter <_AMODE>, "ZEHNER"  und "HUNDERTER")

8

 

<_AMODE>

INT

Alternativmodus

EINER:

Start/Freifahrpunkt Z: Bezug der Positionsangabe

0 =

absolut

1 =

inkremental

ZEHNER:

Startpunkt X / Freifahrpunkt Y: Bezug der Positionsangabe

0 =

absolut (Durchmesser)

1 =

inkremental (Radius)

HUNDERTER:

Start/Freifahrpunkt ja/nein

0 =

nein

1 =

ja

TAUSENDER:

Arbeitsbereich (nur wirksam, wenn MD52218 Bit 17 = 1)

0 =

Arbeitsbereich 1 (WKS-Rotation um Z: -90°)

1 =

Arbeitsbereich 2 (WKS-Rotation um Z: +90°)
