Vordefinierte Funktionen

Durch den Aufruf einer vordefinierten Funktion wird die Ausführung einer vordefinierten NC-Funktion angestoßen, die im Unterschied zur vordefinierten Prozedur einen Rückgabewert liefert. Der Aufruf der vordefinierten Funktion kann als Operand im Ausdruck stehen.
