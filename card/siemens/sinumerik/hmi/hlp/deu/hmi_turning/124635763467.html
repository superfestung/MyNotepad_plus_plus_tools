Automatisiert Programmblöcke erstellen

MIt der Funktion "Automatische Blockbildung" haben Sie eine bequeme Möglichkeit, ein bereits bestehendes Programm nachträglich automatisch in gewünschte Blöcke einzuteilen.

Regeln zur Erstellung der Blöcke

In der Konfigurationsdatei seditor.ini legen Sie die Regeln fest, nach denen die Blöcke erstellt werden.

Folgende Programmschritte können Sie zur besseren Bearbeitung festlegen:

  • Im Programm benutzte Werkzeuge in Blöcke zusammenfassen

  • Zyklentypen definieren

    • Ein Hauptsatz öffnet einen neuen Block auf der obersten Ebene. Der Satz selber wird als ersten Satz in den Block eingefügt. Der Ersetzungstext aus der Konfigurationsdatei wird als Blocktext genommen.

    • Ein Untersatz wird in dem zurzeit geöffneten Block eines Hauptsatzes eingefügt. Sätze, die nicht durch die Konfigurationsdatei klassifiziert sind, werden in einem Block auf der zweiten Blockebene zusammengefasst. Dieser Block erhält den Blocktext „Freier DIN-Code“.

    • Zur Strukturierung des Programms stehen 2 Ebenen für die Blockbildung zur Verfügung.

Hinweis

Die Automatische Blockbildung lässt sich nur einmal durchführen.

Wenn Sie die Funktion ein weiteres Mal anwenden, erhalten Sie den Hinweis, dass keine Blöcke eingefügt werden können, da bereits welche vorhanden sind.

Um Blöcke in dem bereits konvertierten Programm zu erstellen, verwenden Sie den Softkey "Block bilden" des Editor.

Voraussetzung

Die Konfigurationsdatei seditor.ini ist installiert.

Vorgehensweise

 

1.

Wählen Sie den Bedienbereich "Programm-Manager" an.

 

2.

Positionieren Sie den Cursor auf dem gewünschten Hauptprogramm (*.mpf) oder auf einer Jobliste (*.job).

 

3.

Drücken Sie die Softkeys ">>" und "Automat. Blockbildung" .

Sie erhalten eine Abfrage, ob Sie automatisch Blöcke in das Programm einfügen wollen.

 

4.

Drücken Sie den Softkey "OK", um die Konvertierung zu bestätigen.
