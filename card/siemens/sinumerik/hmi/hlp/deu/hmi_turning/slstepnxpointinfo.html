Punktinformation anzeigen

Zu jedem im 3D-Modell ausgewählten Punkt können Sie Informationen anzeigen lassen.

Das ist hilfreich, um z. B. bei einer Kontur (nur X/Y-Koordinaten) die Tiefenangaben (Z) zu ermitteln. Auf diese Weise ist es möglich, diese Kontur über Bahnfräsen zu bearbeiten.

Sie können zu den folgenden Arten von Punkten Informationen anzeigen lassen:

  • Bezugspunkte von Features

  • Eck- bzw. Übergangspunkte an Kanten

Voraussetzung

  • Das gewünschte 3D-Modell ist geöffnet.

  • Die Liste der Bearbeitungsschritte ist geöffnet.

Vorgehensweise

 

1.

Drücken Sie den Softkey "Geometrieinfo".

 

2.

Drücken Sie den Softkey "Punkt".

Alle anwählbaren Punkte werden angezeigt.

Jeder Punkt besteht aus 3 Koordinatenwerten (X, Y, Z), die angezeigt werden. An Drehmaschinen wird der X-Wert zusätzlich als Durchmesserwert angezeigt.

Die angezeigten Koordinaten beziehen sich immer auf das Koordinatensystem der Ebene, aus der die Punktinfo geöffnet wurde.

  
 

3.

Klicken Sie auf den gewünschten Punkt im 3D-Modell.

Die Angaben zum angewählten Punkt werden im Fenster "Punktinfo" eingeblendet.

Sie können eine Komponente des Punkts durch Kopieren und Einfügen direkt im Programm verwenden.
Hiermit ist es z. B. möglich die Z-Komponenten einer neuen Kontur in die Bearbeitungsmaske zu übernehmen.
