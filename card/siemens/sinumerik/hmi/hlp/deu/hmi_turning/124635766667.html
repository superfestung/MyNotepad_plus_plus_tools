Konvertiertes Programm editieren

Voraussetzung

Sie haben ein Programm mit Hilfe des Softkeys "Automat. Blockbildung" in ein strukturiertes Programm konvertiert.

Vorgehensweise

 

1.

Sie haben ein konvertiertes Programm geöffnet.

Blöcke öffnen und schließen

 

2.

Drücken Sie die Softkeys ">>" und "Ansicht".

 

3.

Drücken Sie den Softkey "Blöcke aufklappen".

Es werden alle Blöcke auf der 1. Ebene geöffnet.

Um die Blöcke auf der 2. Ebene zu öffnen, drücken Sie den Softkey "Blöcke aufklappen" erneut.

 

4.

Drücken Sie den Softkey "Blöcke zuklappen".

Wenn Blöcke der 2. Ebene geöffnet sind, werden diese zugeklappt.

Um Blöcke der 1. Ebene zu schließen, drücken Sie den Softkey "Blöcke zuklappen" erneut.

Hinweis: Wenn alle Blöcke der 2. Ebene geschlossen sind, werden bereits beim ersten Betätigen des Softkeys geöffnete Blöcke der 1. Ebene zugeklappt.

Zusätzliche Blöcke manuell in zwei Ebenen anlegen

 

2.

Markieren Sie die Programmsätze, die Sie nachträglich zu einem Block zusammenfassen wollen und drücken Sie den Softkey "Block bilden".

 

3.

Geben Sie Sie im Fenster "Neuen Block bilden" eine Bezeichnung für den Block ein, ordnen Sie die Spindel zu, wählen Sie ggf. den Einfahr-Zusatzcode und den automatischen Rückzug, und drücken Sie den Softkey "Übernehmen".

 

4.

Wenn Sie innerhalb des Blocks weitere Programmsätze zu einem Block zusammenfassen wollen, klappen Sie den Block auf, z.B. mit dem Softkey "Blöcke aufklappen".

 

5.

Markieren Sie die gewünschten Sätze innerhalb des Blocks und drücken Sie den Softkey "Block bilden".

 

6.

Geben Sie im Fenster "Neuen Block bilden" die gewünschten Daten ein und drücken Sie den Softkey "Übernehmen".

Hinweis:

Wenn dem äußeren Block bereits eine Spindel zugewiesen ist, können Sie dem inneren Block keine Spindel mehr zuweisen und umgekehrt.

Einen Block erster Ebene mit übergeordneten Block erstellen

 

2.

Markieren Sie den gewünschten Block, der keinen weiteren Block beinhaltet, und drücken Sie den Softkey "Block bilden".

 

3.

Geben Sie im Fenster "Neuen Block bilden" die gewünschten Daten ein und drücken Sie den Softkey "Übernehmen".

Hinweis:

Wenn dem markierten Block bereits eine Spindel zugewiesen ist, können Sie dem neu erzeugten Block keine Spindel zuweisen.
