Positionierwinkel für Bearbeitungsbereich

Der Winkel CP hat keinen Einfluss auf die Bearbeitungsposition bezogen auf das Werkstück. Er dient ausschließlich dazu das Werkstück mit der Rundachse C so zu positionieren, dass eine Bearbeitung auf der Maschine möglich ist.
