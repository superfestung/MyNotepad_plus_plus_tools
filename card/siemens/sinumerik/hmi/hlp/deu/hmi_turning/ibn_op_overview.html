Verfügbare Operator Panels anzeigen

Im Fenster Verfügbare Bedienfelder von ... erscheint im oberen Bereich eine Liste mit allen Bedienfeldern (Operator Panels), die zur Verfügung stehen. Eine Zeile entspricht dabei einem Operator Panel und enthält folgende Informationen:

Spalte

Beschreibung

 

Laufende Nummer zur Identifikation des Bedienfeldes

Status

Status des Bedienfeldes:

: Das Bedienfeld ist aktuell am Bus verfügbar.

: Das Bedienfeld ist aktuell nicht am Bus verfügbar.

HW

Informationen zur Hardware des Bedienfeldes

Mögliche Werte sind "unbekannt" (keine Information verfügbar), NCU, PCU, PPU, alle TCUs.

Standardwert: unbekannt

Name

Name des Bedienfeldes

MCP

DIP-Schalter-Stellung für die Maschinensteuertafel (Machine Control Panel)

Die Werte sind frei wählbar von 1 bis 254 (0 bedeutet, dass die letzte aktive Maschinensteuertafel beibehalten wird, 255 bedeutet, dass keine Maschinensteuertafel verwendet wird).

Standardwert: 192

DCK

Verfügbarkeit von Direktasten (Direct Control Keys)

Aktiv: Die TCU verfügt über Direkttasten

Inaktiv: Die TCU verfügt über keine Direkttasten

TCU

Index für die TCU (Thin Client Unit)

Der Wert ist frei konfigurierbar von 0 bis 255 und dient zur Identifikation in der PLC.

EKS

DIP-Schalter-Stellung für den Chipleser für Zugriffsstufen (Electronic Key System)

Mögliche Werte reichen von 1 bis 255. Der Wert 0 bedeutet, dass kein EKS verwendet wird.

Virtuelle Tastatur

Verwendung einer virtuellen Tastatur

Mögliche Werte sind "niemals", "immer" und "auto", wobei "auto" als Standardwert gesetzt ist.

Hinweis

Die Informationen der Bedienfelder sind nur lesbar und können im Fenster OPs nicht geändert werden.

Aktualisierung der Übersicht

Beim Wechsel in das Fenster OPs wird automatisch nach verfügbaren Bedienelementen gesucht. Dies ist zu einem späteren Zeitpunkt auch über die Softkeys "Neu anzeigen" oder "OP-Daten neu laden" möglich. Letztere Aktion führt einen komplett neuen Scan durch, bei dem bereits vorhandene Bedienelemente aus dem lokalen Cache gelöscht werden. Beim Nachladen wird nur nach neuen Bedienelementen und Daten gesucht.

Der aktive DHCP-Server, auf dem sich die Daten für die Operator Panels befinden, kann eine andere NCU aber auch die lokale NCU selbst sein. Unabhängig davon wird eine Kopie der Daten auf der lokalen NCU angelegt und verwendet.

Vorgehensweise

 

1.

Wählen Sie den Bedienbereich "Inbetriebnahme" an.

 

2.

Drücken Sie den Softkey "OPs"

  

Das Fenster "OPs" wird geöffnet.

Es werden alle momentan verfügbaren Bedienelemente angezeigt.

 

3.

Drücken Sie den Softkey "Neu anzeigen", um die Übersicht zu aktualisieren.

Dabei wird nach neuen Bedienelementen und Daten gesucht.

 

4.

Drücken Sie den Softkey "OP-Daten neu laden", um eine komplett neue Suche zu starten.

Dabei werden bereits vorhandene Bedienelemente aus dem Zwischenspeicher / Puffer gelöscht.

 

5.

Drücken Sie den Softkey "TCU-Daten verteilen", um die verfügbaren Bedienelemente mit allen Teilnehmern im Anlagennetz abzugleichen.

Hinweis

Den Softkey "TCU-Daten verteilen" können Sie nur mit Verbindung zum aktiven DHCP-Server verwenden.
