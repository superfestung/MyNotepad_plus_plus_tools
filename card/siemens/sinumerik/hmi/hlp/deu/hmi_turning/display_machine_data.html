Anzeige-Maschinendaten anzeigen/ändern

Im Bedienbereich "Inbetriebnahme" greifen Sie über die Bedienoberfläche auf die Anzeige-Maschinendaten zu.

Vorgehensweise

 

1.

Wählen Sie den Bedienbereich "Inbetriebnahme" an.

 

2.

Drücken Sie den Softkey "Masch.Daten".

 

3.

Drücken Sie die Menüfortschalt-Taste.

Weitere Softkeys werden eingeblendet.

 

4.

Drücken Sie den Softkey "Anzeige MD".

Die Anzeige-Maschinendaten werden angezeigt.

 

5.

Positionieren Sie den Cursor auf den zu ändernden Eintrag und geben Sie den gewünschten Wert ein.

 

6.

Drücken Sie den Softkey "Reset (po)".

Sie erhalten eine Sicherheitsabfrage.

 

7.

Drücken Sie den Softkey "OK", um einen Warmstart durchzuführen.

 

 

-ODER -

 

 

Drücken Sie den Softkey "Abbruch", wenn Sie die Einstellungen nicht übernehmen möchten.
