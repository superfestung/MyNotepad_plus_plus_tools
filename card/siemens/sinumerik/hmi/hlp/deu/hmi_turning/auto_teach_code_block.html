Satz anwählen

Sie haben die Möglichkeit, den Unterbrechungszeiger auf die aktuelle Cursorposition zu setzen. Beim nächsten Programmstart wird die Bearbeitung an dieser Stelle fortgesetzt.

Beim Teachen können Sie auch Programmbereiche ändern, die bereits abgearbeitet sind. Dabei wird automatisch die Programmbearbeitung gesperrt.

Um das Programm fortsetzen zu können, muss ein Reset oder eine Satzanwahl erfolgen.

Vorgehensweise

 

1.

Der Teachmodus ist aktiv.

 

2.

Positionieren Sie den Cursor auf den gewünschten Programmsatz.

 

3.

Drücken Sie den Softkey "Satzanwahl".
