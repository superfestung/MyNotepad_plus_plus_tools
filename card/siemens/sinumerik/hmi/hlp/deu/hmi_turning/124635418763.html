Aufruf der Messvariante

Vorgehensweise

Das zu bearbeitende Teileprogramm ist angelegt und Sie befinden sich im Editor.

 

1.

Drücken Sie den Softkey "Werkst messen".

 

2.

Drücken Sie den Softkey "3D".

 

3.

Drücken Sie den Softkey "Kinematik".

Das Eingabefenster "Messen: Kinematik" wird geöffnet.

Anschließend können Sie über die folgenden Softkeys die folgenden Eingabefenster öffnen:

 

1. Messung

(siehe Parameter 1. bis 3. Messung)

 

2. Messung

 

3. Messung

 

Berechnen

(siehe Parameter Berechnen)
