Kinematikelemente anlegen

Offset

 

1.

Markieren Sie den Ordner "Aktive Daten" und drücken Sie auf den Softkey "Neues Element".

Sie erhalten ein Auswahlfenster mit Kinematikelementen.

 

2.

Wählen Sie das kinematische Element "Offset" und drücken Sie den Softkey "OK".

In der Baumansicht öffnet sich in der unteren Hälfte das entsprechende Eingabefenster für die Elementwerte.

 

3.

Geben Sie die Werte ein:

  

Name:

ROOT

  

Offset (X):

0

  

Offset (Y):

0

  

Offset (Z):

0

 

4.

Drücken Sie den Softkey "OK" um die Werte zu speichern.

  

-ODER-

 

 

Drücken Sie den Softkey "Abbruch" um das Eingabefenster zu schließen.
