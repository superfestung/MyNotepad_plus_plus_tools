Funktion

Mit der Funktion Interpolationsdrehen ist eine Drehbearbeitung auf einer Werkzeugmaschine mit mindestens drei Linearachsen und einer positionierfähigen Spindel möglich. Beim Interpolationsdrehen wird nicht das Werkstück für die Drehbearbeitung rotiert, sondern das Werkzeug rotiert um das Werkstück. Die Linearachsen führen die Kreisbewegung aus und die Werkzeugspindel hält die Werkzeugschneide senkrecht auf dem Kreis. Die Drehachse kann an beliebiger Werkstückposition und an 4- oder 5-Achs Maschinen, frei im Raum orientiert sein. Somit können Sie Drehoberflächen z. B. an Flanschen erzeugen, die außermittig oder sogar schräg im Raum liegen. Hiermit können Sie Rüstzeiten reduzieren.

Interpolationsdrehen ist an Fräsmaschinen und Drehmaschinen verfügbar.

 

Software-Option

Um die Funktion "Interpolations Drehen" zu nutzen, benötigen Sie die Software-Option "Interpolations Drehen".

Wenn Interpolationsdrehen frei gegeben ist, werden Drehwerkzeuge in der Werkzeugliste angeboten.

Mit dem CYCLE800 kann so geschwenkt werden, dass das Drehwerkzeug senkrecht auf der Bearbeitungsebene steht.

Mit dem CYCLE806 kann das Interpolationsdrehen eingeschaltet werden. Danach können bestimmte Drehzyklen verwendet werden.

Die Bearbeitung bei aktivem Interpolationsdrehen findet immer in der Ebene G18 statt.

Hinweis

Die Funktion "Interpolationsdrehen" ist für das Arbeiten mit "Manuelle Maschine" nicht verfügbar.

Zyklen für Interpolationsdrehen

Sie können die folgenden Standardzyklen für Interpolationsdrehen verwenden:

  • Abspanen (CYCLE951), alle Varianten

  • Abspanen (CYCLE952): Konturabspanen, Konturstechen    

  • Einstich (CYCLE930), alle Varianten

  • Freistich (CYCLE940), alle Varianten

Die gleichen Drehbearbeitungen können Sie auch unter ShopMill und ShopTurn verwenden.

Spindelpositionierung durch CYCLE806

Beim Aktivieren der Funktion Interpolationsdrehen muss die Schneide des Drehwerkzeuges im WKS auf Y = 0 stehen. Im CYCLE806 wird, wenn nötig, das WKS und die Werkzeugspindel gedreht und somit die Lage des Drehwerkzeuges angepasst. Vor dem Start der Funktion Interpolationsdrehen muss das Werkzeug auf Sicherheitsabstand zum Werkstück stehen.
