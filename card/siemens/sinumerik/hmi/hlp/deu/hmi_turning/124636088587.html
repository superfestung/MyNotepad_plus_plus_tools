Werkzeug löschen

Werkzeuge, die Sie nicht mehr verwenden, können Sie aus der Werkzeugliste entfernen, um diese übersichtlich zu halten.

Vorgehensweise

 

1.

Die Werkzeugliste ist geöffnet.

 

2.

Positionieren Sie den Cursor in der Werkzeugliste auf das Werkzeug, das Sie löschen möchten.

 

3.

Drücken Sie den Softkey "Werkzeug löschen".

Eine Sicherheitsabfrage erscheint.

 

4.

Drücken Sie den Softkey "OK", wenn Sie das angewählte Werkzeug tatsächlich löschen möchten.

 

 

Das Werkzeug wird gelöscht.

Befand sich das Werkzeug auf einem Magazinplatz, so wird entladen und anschließend gelöscht.

Mehrere Beladestellen - Werkzeug auf Magazinplatz

Haben Sie mehrere Beladestellen für ein Magazin konfiguriert, erscheint nach dem Drücken des Softkeys "Werkzeug löschen" das Fenster "Auswahl der Beladestelle".

Wählen Sie dort die gewünschte Beladestelle und drücken Sie den Softkey "OK", um das Werkzeug zu entladen und zu löschen.
