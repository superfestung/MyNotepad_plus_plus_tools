Ablauf Y-Drehen

  1. Wechseln Sie das Y-Drehwerkzeug ein.

  2. Positionieren Sie das Werkzeug vor.

  3. Starten Sie das Y-Drehen mit dem CYCLE805:

    • Das Werkzeug wird ausgerichtet.

    • Das Koordinatensystem wird entsprechend gedreht.

    • Die Startposition wird angefahren (wenn "Startpunkt = ja" ausgewählt wurde).

  4. Fahren Sie die Startposition an (wenn "Startpunkt = nein" ausgewählt wurde).

  5. Bearbeiten Sie mit den Standarddrehzyklen.

  6. Beenden Sie das Y-Drehen mit dem CYCLE805:

    • Die Freifahrposition wird angefahren (wenn "Freifahren = ja" ausgewählt wurde).

    • Das Koordinatensystem wird wieder entsprechend zurückgedreht.

    Hinweis

    In ShopTurn wird die An-/Abwahl automatisch gemacht. Die benötigten Parameter sind in den Drehmasken für Y-Drehen enthalten.

Beispiel Y-Drehen G-Code

T="Y-Drehwerkzeug"

D1 ; Schruppplatte

CYCLE805(...) ; Anwahl Y-Drehen

CYCLE951(...) ; Schruppbearbeitung

D3 ; Schlichtplatte

CYCLE805(...) ; Anwahl y-Drehen

CYCLE951(...) ; Schlichtbearbeitung

CYCLE805() ; Abwahl Y-Drehen
