Baustein C/SINF: Teileprogrammspezifische Zustandsdaten

Während der automatischen Abarbeitung eines Teileprogrammes können verschiedene Parameter die Art der Bearbeitung beeinflussen. Die aktuellen Zustandsdaten, für das angewählte Teileprogramm sind im Baustein SINF zusammengefasst. Eine Änderung der Zustandsdaten ist nur über die PLC-Nahtstelle zulässig.

singleBlockType Single Block Modus
Spaltenindex:
17 		Zeilenindex:
 Max. Zeilenindex:
Mit dieser Variable lässt sich der Single Block Modus einstellen.
o =1: Single Block Modus 1 (Hauptlauf Einzelsatz)
o =2: Single Block Modus 2 (Decodier Einzelsatz)
o =3: Single Block Modus 3 (Stopp im Zyklus)
Die Aktivierung des Einzelsatzbetriebs erfolgt über das Nahtstellensignal
DB21, ... DBX0.4 = 1

Wertebereich:
1 = Hauptlauf Einzelsatz
2 = Decodier (Vorlauf) Einzelsatz
Typ
TYPE_UWORD 		Einheit
keine Einheit 		Standardwert
 Minimalwert
 Maximalwert
Zugriffspfad HMI: /Channel/ProgramModification/singleBlockType[u<Kanalnummer>]
