OPC UA-Dialog "Modell"

Übersicht

Drücken Sie den Softkey "Modell", um zur folgenden Maske zu gelangen, und führen Sie die folgenden Vorgänge aus.

①

Im OPC UA-Dialog "Modell" werden die aktivierten CSOMs und auch die folgenden Informationen angezeigt:

  • Dateiname des/der CSOM(s)

  • Namespace(s) des/der CSOM(s)

  • Version des/der CSOM(s)

  • Importdatum

  • Aktivierungsstatus

②

Sie können ein oder mehrere neue CSOM(s) hinzufügen.

③

Sie können ein CSOM löschen.

④

Sie können ein oder mehrere CSOM(s) ändern: der Status kann aktiviert / deaktiviert werden.
