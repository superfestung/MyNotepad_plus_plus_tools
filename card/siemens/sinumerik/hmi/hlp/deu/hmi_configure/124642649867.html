Transformationen zuweisen und bearbeiten

Im Fenster Transformationen können Sie allen zur Verfügung stehenden kinematischen Elementen Transformationstypen zuweisen und parametrieren.

Vorgehensweise

  

Das Fenster "Transformationen" ist geöffnet.

Transformationen neu zuweisen

 

1.

Positionieren Sie den Cursor auf das leere kinematische Element.

 

2.

Drücken Sie den Softkey "Neu".

Das Fenster "Neue Transformation" wird geöffnet.

Folgende Transformationstypen stehen zur Verfügung:

  • dynamische Orientierungstransformation TRAORI_DYN

  • statische Orientierungstransformation (Drehen) TRAORI_STAT

  • Stirnseitentransformation TRANSMIT_K

  • Mantelflächentransformation TRACYL_K

  • Transformation "schräge Achse" TRAANG_K

 

3.

Wählen Sie den Transformationstyp.

 

4.

Wählen Sie für die Transformationstypen TRAORI_DYN und TRAORI_STAT die Anzahl der Rotationsachsen in Werkzeug- und Werkstückketten.

 

5.

Drücken Sie den Softkey "OK", um den Transformationstyp parametrieren zu können.

Hinweis

Zusätzliche Parameter für TRAORI_DYN und TRAORI_STAT

Damit die Transformationstypen parametriert werden können, muss die Anzahl der Rotationsachsen in einem zulässigen Bereich liegen.

Hinweis

Pflichteingabefelder

Einige Eingabefelder für die Parametereigenschaften werden rot hervorgehoben. Parametrieren Sie diese Eingabefelder, damit die Angaben mit dem Softkey "Ok" bestätigt werden können.

Transformationen bearbeiten

 

1.

Positionieren Sie den Cursor auf ein kinematisches Element mit zugewiesenem Transformationstyp.

 

2.

Drücken Sie die Taste <Cursor rechts> oder <INPUT>.

Das Eingabefenster für die Einstellungen öffnet sich.

 

4.

Geben Sie die gewünschten Änderungen ein und drücken Sie den Softkey "OK", um die Eingaben zu bestätigen.
