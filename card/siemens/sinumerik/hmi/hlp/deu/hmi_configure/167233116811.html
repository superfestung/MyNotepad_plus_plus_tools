Benutzergruppen löschen

Einleitung

Wenn Sie eine neue Benutzergruppe löschen möchten, können Sie den Dialog "Benutzergruppen löschen" nutzen.

Voraussetzung

  • Sie verfügen über die Rolle "Security Admin Hersteller" oder "Security Admin Anwender".

Vorgehensweise

  1. Im Bedienbereich "Inbetriebnahme" rufen Sie die Security-Regeln der Benutzerverwaltung über die Softkeys "Security > Benutzerverwaltung" auf.

  2. Klicken Sie auf den Softkey "Benutzergruppen".

  3. Selektieren Sie in der Übersichtsseite der Benutzergruppen eine Benutzergruppe, die sie löschen möchten.

  4. Klicken Sie auf "Löschen".

  5. Bestätigen Sie die Sicherheitsabfrage mit "OK". Die Benutzergruppe verschwindet aus der Liste aller Benutzergruppen.
