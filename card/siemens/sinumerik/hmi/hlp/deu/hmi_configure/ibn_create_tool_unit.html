TO-Einheit anlegen

Eine TO-Einheit ist die Zusammenfassung von Werkzeugen und Magazinen zu einer Einheit. Einer TO-Einheit können mehrere Kanäle zugeordnet sein. Die Werkzeuge und Magazine müssen innerhalb einer TO-Einheit eindeutig sein. Für eine TO-Einheit legen Sie im Fenster "TO-Einheit" die Suchparameter und den Kanal fest.

Verzeichniswerte

Sie befinden sich im Bedienbereich "Inbetriebnahme" → "NC" →"Werkzeugverwaltung" unter dem Verzeichnis "Aktive Daten" → <TO-Einheit>.

Über das Verzeichnis der TO-Einheit können Sie folgende Parameter im Fenster "TO-Einheit" einstellen:

Parameter

Bedeutung

Name

Bezeichnung, die in der Baumstruktur angezeigt wird.

Kanalnummer

Kanal, dem die TO-Einheit zugeordnet wird.

Werkzeugsuche

Suchstrategie:

  • ohne Suchstrategie

  • aktives Werkzeug

  • kürzester Weg

Leerplatzsuche

Suchstrategie:

  • ohne Suchstrategie

  • erster Platz vorwärts

  • aktueller Platz vorwärts

  • aktueller Platz rückwärts

  • letzter Platz rückwärts

  • symmetrisch aktueller Platz

  • 1:1-Tausch
