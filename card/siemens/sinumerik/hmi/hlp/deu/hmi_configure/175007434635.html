PLC-Symbol oder Adresse bearbeiten

Bei einer bearbeitbaren Trace-Konfiguration können Sie die bereits in der Signaltabelle vorhandenen PLC-Adressen oder symbolischen PLC-Namen ändern.

Vorgehensweise

  1. Um in den Bearbeitungsmodus zu wechseln, klicken Sie doppelt auf das Feld in der Spalte "Adresse / Symbol" oder wählen Sie es über die Tastatur aus und drücken "Einfg".

  2. Geben Sie eine neue PLC-Adresse (oder ein neues Symbol) ein oder ändern Sie einen Teil der vorhandenen Adresse bzw. des vorhandenen Symbols.

    Eine vorhandene PLC-Variable kann nur durch eine andere gültige PLC-Variable ersetzt werden. Falls die bearbeitete Adresse nicht gültig oder keine PLC-Variable ist, wird eine Fehlermeldung angezeigt und die Trace-Konfiguration wird nicht geändert.

Wenn die PLC-Adresse geändert wird, werden automatisch auch die Beschreibung und der Datentyp der Variablen aktualisiert. Alle Messwerte, die zu der geänderten Variablen gehören, werden gelöscht.

Hinweis

Die BTSS-Adressen anderer Variablentypen wie z. B. Servo-Variablen können nicht bearbeitet werden. Falls Sie eine solche Variable ändern möchten, löschen Sie sie aus der Konfiguration und fügen eine neue Variable hinzu.
