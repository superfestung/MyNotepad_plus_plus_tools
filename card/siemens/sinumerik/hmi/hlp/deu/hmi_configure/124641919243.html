Konfigurationsdaten für Registrierung in EUNA - Daten erstellen

Für die Registrierung Ihrer Maschine in der Datenbank EUNA (End User Notification Administration) erstellen Sie eine Konfigurationsdatei mit allen Endverbleibsdaten (Adressdaten, Logbuchdaten und Versionsdaten) der Steuerung.

Voraussetzungen

  • Die Adressdaten für den Endverbleib sind geprüft

  • Logbuch-Einträge sind angelegt

Vorgehensweise

 

1.

Wählen Sie den Bedienbereich "Diagnose".

 

2.

Drücken Sie die Softkeys "Version" und "Speichern".

Das Fenster "Versionsinformationen speichern: Ablage auswählen" wird geöffnet.

 

3.

Wählen Sie den Ablageort (z.B. USB) und drücken Sie den Softkey "OK".

Das Fenster "Versionsinformation speichern: Name" wird geöffnet.

Es wird der Name und Maschinennummer angezeigt.

 

4.

Bei Bedarf ändern Sie den Namen und geben einen Kommentar ein.

 

5.

Aktivieren Sie das Kontrollkästchen "Konfigurationsdaten (.XML)", um alle Informationen zur Maschinenidentität zu speichern.

Hinweis: Wenn Sie zusätzlich die reinen Versionsdaten in Textformat ausgeben wollen, dann aktivieren Sie das Kontrollkästchen "Versionsdaten (.txt)".

 

6.

Drücken Sie den Softkey "OK", um die gewünschten Dateien zu speichern.
