Zentrale Anwender-Domäne trennen

Voraussetzung

  • Die Benutzerverwaltung ist aktiviert.

  • Die lokale Domäne für den Hersteller ist eingerichtet. Der Security Admin für den Hersteller ist eingerichtet.

  • Die lokale Domäne für den Anwender ist eingerichtet. Der Security Admin für den Anwender ist eingerichtet.

  • Die Anwender-Domäne wurde mit einer zentralen Benutzerverwaltung verbunden.

  • Sie verfügen über die Rolle "Security Admin Anwender".

Vorgehensweise

  1. Im Bedienbereich "Inbetriebnahme" rufen Sie die Benutzerverwaltung über die Softkeys "Security > Benutzerverwaltung" auf. Die Übersichtsseite zur Benutzerverwaltung öffnet sich im Softkey "Konfiguration".

  2. Klicken Sie auf die Schaltfläche "Trennen" neben der zentralen Anwender-Domäne.

  3. Bestätigen Sie die Sicherheitsabfrage mit "OK".

Ergebnis

Sie haben die Anwender-Domäne erfolgreich von der zentralen Benutzerverwaltung getrennt.
