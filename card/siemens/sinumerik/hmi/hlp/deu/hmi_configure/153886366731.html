Displays

Für jede Display-Konfiguration können in der jeweiligen Konfigurationsdatei ein oder mehrere Displays definiert werden. Für jedes dieser Displays werden ein Name und eine Liste von Frames angegeben. Die Frames beschreiben die Aufteilung des Displays in rechteckige Bereiche, in denen Applikationen oder Menüs angezeigt werden können.

Hinweis

Die Reihenfolge der projektierten Displays ist beliebig.

Beispiel

[displays]DISPLAY001= name:=display_1, frames:="frame1, frame2, frame3"DISPLAY002= name:=display_2, frames:="frame4, frame5, frame6, frame7"…

Property

Bedeutung

name

Name des Displays. Wird z. B. zur Display-Umschaltung benötigt

frames

Enthält eine durch Komma getrennte Liste von Frames, die in diesem Display vorhanden sein sollen und damit die Aufteilung des Displays beschreiben

enableMirrorMode (true|false)

Aktiviert (true) bzw. deaktiviert (false) den Spiegelmodus für das Display

Wenn der Spiegelmodus aktiviert ist, werden bei Aufruf der Funktion setMirrorMode (true) alle Frames des Displays horizontal gespiegelt angezeigt. Andernfalls bleibt die Darstellung der Frames nach dem Aufruf der Funktion setMirrorMode (true) unverändert.

Standard ist "true".

onStartup

Enthält eine durch Semikolon getrennte Liste von Aktionen, die beim ersten Aufblenden des Displays ausgeführt werden sollen

Die möglichen Aktionen werden im weiteren Verlauf beschrieben.

onShow

Enthält eine durch Semikolon getrennte Liste von Aktionen, die bei jedem Aufblenden des Displays ausgeführt werden sollen

Die möglichen Aktionen werden im weiteren Verlauf beschrieben.
