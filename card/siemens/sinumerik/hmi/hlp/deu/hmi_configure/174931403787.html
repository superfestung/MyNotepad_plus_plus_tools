Maschinenrollen, Benutzergruppen und Benutzer

Die SINUMERIK Benutzerverwaltung unterscheidet drei Arten von Gruppen:

Maschinenrollen

  • Maschinenrollen bilden eine Gruppe zum Aggregieren von Rollen auf Maschinenebene über mehrere Applikationen der Maschine.

  • Nur der Security Admin Hersteller kann Maschinenrollen anlegen, verändern oder löschen.

  • Für jede Maschinenrolle kann durch den Hersteller entschieden werden, ob diese vom Anwender vergeben werden kann.

  • Maschinenrollen haben den Vorteil, dass der Hersteller oft einen besseren Überblick über alle Bestandteile eines SINUMERIK Systems und deren implementierter Rechtevergabe hat. Der Anwender muss sich dann mit der Rechtevergabe der Einzelkomponenten nicht mehr auseinandersetzen. Dies übernimmt der Hersteller für ihn.

  • Maschinenrollen vererben ihre Funktionsrechte untereinander.
    Ein Benutzer erbt von den Maschinenrollen, zu denen er gehört, den AccessLevel. Wenn der Benutzer zu mehreren Maschinenrollen gehört, bekommt er den höchsten enthaltenen AccessLevel. Ein hoher AccessLevel beinhaltet immer auch die niedrigeren AccessLevels.
    Zusätzlich kann eine Maschinenrolle auch die Rolle des Security Admins beinhalten und dieses Rolle an die enthaltenen Benutzer weitergeben. Die Rolle "Security Admin Hersteller" können nur Benutzer haben, die der Domäne "Hersteller" angehören; die Rolle "Security Admin Anwender" können nur Benutzer haben, die der Domäne "Anwender" angehören.. Es kann deswegen keinen Benutzer mit beiden Security Admin-Rollen geben.

  • Wenn die Hersteller-Domäne mit einer zentralen Benutzerverwaltung verbunden ist, dann werden Maschinenrollen (inklusive des Security Admin Hersteller) automatisch auf bestehende Benutzergruppen auf dem UMC-Server zugeordnet.

  • Wenn die Anwender-Domäne mit einer zentralen Benutzerverwaltung verbunden ist, wird nur die Rolle Security Admin Anwender automatisch zugewiesen.

Benutzergruppen

  • Eine Benutzergruppe ist eine Gruppe zur organisatorischen Zusammenfassung von Benutzern.

  • Es gibt lokale und zentrale Benutzergruppen.

  • Den Benutzergruppen können in SINUMERIK Operate Maschinenrollen zugeordnet werden.

Benutzer

Ein Benutzer kann dann wiederum Mitglied von einer oder mehreren Benutzergruppe(n) werden. Über das Login des Benutzers kann die Benutzervewaltung jederzeit zuordnen, welche Rechte / Rollen welcher Benutzer hat. Es sind dann nur diejenigen Funktionen für den Benutzer zugänglich, die vorher in der Maschinenrolle über die Benutzerverwaltung definiert wurden.
