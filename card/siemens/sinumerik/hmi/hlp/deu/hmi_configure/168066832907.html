Übersicht Benutzergruppen

Benutzer werden in Benutzergruppen zusammengefasst. Diese Benutzergruppen können Sie Maschinenrollen und den damit verbundenen SINUMERIK-Zugriffsrechten zuordnen.

Hinweis

Bearbeitung nur Domänen-spezifisch möglich

Beachten Sie, dass die Bearbeitung der Benutzergruppen immer Domänen-spezifisch ist:

  • Anmelden als Security Admin Hersteller : Sie können Benutzergruppen der Domäne Hersteller bearbeiten.

  • Anmelden als Security Admin Anwender: Sie können die Benutzergruppen der Domäne Anwender bearbeiten.

Voraussetzung

  • Sie verfügen über die Rolle "Security Admin Hersteller" oder "Security Admin Anwender".

Liste der Benutzergruppen

Benutzergruppen

In der Liste der Benutzergruppen sehen Sie alle definierten Benutzergruppen der Benutzerverwaltung für die aktive Domäne:

  • Lokale Benutzergruppen

  • Zentrale Benutzergruppen

Es werden Name und Beschreibung der Benutzergruppen aufgeführt, sowie die Maschinenrolle, der die jeweilige Benutzergruppe zugeordnet ist.

Über Schalflächen haben Sie die Möglichkeit folgende Aktionen zu starten:

  • Lokale Benutzergruppe anlegen

  • Zentrale Benutzergruppe anlegen

  • Benutzergruppe ändern

  • Benutzergruppe löschen
