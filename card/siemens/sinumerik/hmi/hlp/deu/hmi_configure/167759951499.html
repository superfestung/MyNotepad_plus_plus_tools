Benutzer löschen

Einleitung

Wenn Sie einen Benutzer löschen möchten, können Sie den Dialog "Benutzer löschen" nutzen.

Voraussetzung

  • Sie verfügen über die Rolle "Security Admin Hersteller" oder "Security Admin Anwender".

Vorgehensweise

  1. Im Bedienbereich "Inbetriebnahme" rufen Sie die Benutzerverwaltung über die Softkeys "Security > Benutzerverwaltung" auf.

  2. Klicken Sie auf den Softkey "Benutzer".

  3. Selektieren Sie in der Übersichtsseite der Benutzer einen Benutzer, den sie löschen möchten.

  4. Klicken Sie auf "Löschen".

  5. Bestätigen Sie die Sicherheitsabfrage mit "OK". Der Benutzer verschwindet aus der Liste aller Benutzer.

Ergebnis

Der Benutzer ist aus der Liste der Benutzer gelöscht.
