Benutzergruppen ändern

Einleitung

Wenn Sie eine neue Benutzergruppe ändern möchten, können Sie den Dialog "Benutzergruppen ändern" nutzen.

Voraussetzung

  • Sie verfügen über die Rolle "Security Admin Hersteller" oder "Security Admin Anwender".

Vorgehensweise

  1. Im Bedienbereich "Inbetriebnahme" rufen Sie die Benutzerverwaltung über die Softkeys "Security > Benutzerverwaltung" auf.

  2. Klicken Sie auf den Softkey "Benutzergruppen". Klicken Sie auf der Übersichtsseite auf "Ändern", um eine bestehende Benutzergruppe zu ändern.

  3. Führen Sie ihre Änderungen bezüglich Name, Beschreibung, Zuweisung zu Maschinenrollen oder Zuweisen von Benutzern durch. Wenn Sie eine zentrale Benutzergruppe ändern, können Sie über "Auswählen" den Namen für die zentrale Benutzergruppe vom zentralen UMC-Server ändern.

  4. Klicken Sie auf "OK", um die Änderungen zu übernehmen und den Dialog zu schließen.
