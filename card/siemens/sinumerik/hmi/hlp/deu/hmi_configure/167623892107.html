Benutzer abmelden

Einleitung

Wenn Sie sich als Benutzer abmelden möchten oder den Benutzer wechseln möchten, können Sie den Abmeldedialog nutzen.

Voraussetzung

  • Sie sind als Benutzer im System angemeldet.

Vorgehensweise

  1. Klicken Sie auf die Anmelden-Schaltfläche rechts oben in SINUMERIK Operate und wählen Sie im Kontextmenü "Abmelden". Der Abmeldedialog öffnet sich.
    Alternativ können Sie den Shortcut <Ctrl-I> benutzen, um den Abmelde-Dialog zu öffnen.


    Anmelden-Schaltfläche

  2. Bestätigen Sie die Sicherheitsabfrage mit "OK".

Ergebnis

Sie sind als Benutzer in SINUMERIK Operate abgemeldet. Im Moment des Abmeldens fallen Sie als Benutzer auf die Rechte "Zugriffsstufe 7 / Schlüsselschalter 0" zurück. Im Moment des Abmeldens springen Sie ins Maschinen Grundbild zurück.

Sie können sich nun erneut anmelden oder den Benutzer wechseln.
