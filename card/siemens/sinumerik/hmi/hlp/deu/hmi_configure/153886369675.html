Frames

Ein Frame ist ein Container für Applikationen oder Menüs und beschreibt in erster Linie die gewünschte Position und Größe der Applikation und Menüs in Pixel. Zusätzlich gibt man hier noch die Applikation bzw. das Menü an, die/das nach dem Start des Display Managers in dem Frame angezeigt werden soll. Die Applikationen, die den Frames zugeordnet werden können, werden in der Datei systemconfiguration.ini angegeben.

Beispiel

[frames]FRAME001= name:=frame1, x:=0, y:=0, width:=460, height:=970, app:=mydialogFRAME002= name:=frame2, x:=460, y:=0, width:=1460, height:=1080, app:=OPERATEFRAME003= name:=frame3, x:=0, y:=970, width:=460, height:=110, menu:=mymenuFRAME004= name:=frame4, x:=0, y:=0, width:=1280, height:=970, app:=mydialog2, runableApps:="mydialog, mydialog2"FRAME005= name:=frame5, x:=0, y:=970, width:=1280, height:=110, menu:=menu2FRAME006= name:=frame6, x:=1280, y:=0, width:=640, height:=480, app:=OPERATEFRAME007= name:=frame7, x:=1280, y:=480, width:=640, height:=600, app:=mydialog3…

Wenn SINUMRIK Operate in einem Frame angezeigt werden soll, wird als Applikation der Namen "OPERATE" angegeben. Zu beachten ist, dass SINUMERIK Operate nur in bestimmten Größen verwendet werden kann. Die aktuell unterstützten Größen (in Pixel) sind:

  • 640x480

  • 760x505

  • 800x480

  • 800x600

  • 1024x768

  • 1024x1014

  • 1080x810

  • 1280x768

  • 1280x800

  • 1280x1024

  • 1300x768

  • 1366x768

  • 1440x1080

  • 1454x1080

  • 1854x1014

  • 1854x1080

  • 1920x1080

Property

Bedeutung

name

Name des Frames. Wird z. B. in der Frameliste der Displays verwendet

x, y, width, height

Absolute Position (x, y), Breite und Höhe des Frames in Pixel

app

Applikation, die beim ersten Aufblenden des Frames in diesem Frame angezeigt wird.

runableApps

Liste von Applikationen, die im Falle eines Applikationstauschs in diesem Frame angezeigt werden dürfen.

menu

Menü, das im Frame angezeigt wird.

Wird ignoriert, wenn "app" gleichzeitig verwendet wird
