Variablen einfügen

Im Fenster "Fahrtenschreiber-Einstellung" können Sie bis zu 50 Variablen einfügen, die Sie protokollieren oder beobachten möchten.

Sie wählen verfügbare Variablen aus der Auswahlliste oder fügen Sie manuell in die Liste ein.

Variablen aus einer Variablenliste einfügen

 

1.

Sie befinden sich im Fenster "Fahrtenschreiber-Einstellung".

 

2.

Drücken Sie den Softkey "Variablenauswahl".

Das Fenster "Aktionsprotokoll-Variablen für die kontinuierliche Überwachung" öffnet sich.

 

3.

Drücken Sie den Softkey "Variable einfügen", um eine Variable aus einer Liste aller vorhandenen Variablen einzufügen.

Das Fenster "Variable auswählen" öffnet sich.

 

4.

Wählen Sie die gewünschte Variable aus der Liste.

 

5.

Klicken Sie auf den Softkey "Hinzufügen".

Das Fenster "Variablen-Attribute" öffnet sich.

 

6.

Übernehmen Sie oder verändern Sie bei Bedarf die angezeigten Eigenschaften, z. B. die Achszuordnung.

 

7.

Um die Variablenanzeige einzugrenzen, wählen Sie in der Kopfzeile die jeweilige Kategorie aus, z. B. "Systemvariablen".

-ODER-

Drücken Sie den Softkey "Filter/Suchen", um über das Auswahlfeld "Filter" die Anzeige der Variablen einzugrenzen und/oder über das Eingabefeld "Suchen" die gewünschte Variable auszuwählen.

 

8.

Um die Suchanfrage zu löschen, drücken Sie den Softkey "Suche löschen".

 

9.

Drücken Sie den Softkey "OK", um Änderungen zu bestätigen.

- ODER -

Drücken Sie den Softkey "Abbruch", um Änderungen zu verwerfen.

Variablen manuell einfügen

 

1.

Sie befinden sich im Fenster "Fahrtenschreiber-Einstellung".

 

2.

Drücken Sie den Softkey "Variablenauswahl".

Das Fenster "Variablen für laufende Überwachung" öffnet sich.

 

3.

Positionieren Sie den Cursor in die Spalte "Variable" und geben Sie die gewünschte Variable ein.

 

4.

Wählen Sie das unterstütze Format der Variablen in der Spalte "Format" aus der Auswahlliste.

 

5.

Geben Sie bei Bedarf einen beschreibenden Text in der Spalte "Kommentar" ein, um z. B. die Variablenkonfiguration zu erläutern.

Die Kommentare werden im Fenster "Fahrtenschreiber Protokolldatei" in Echtzeit dargestellt.

Wenn Sie einen Kommentar ändern, fließen die Änderungen in die Datei "actual_action.com" für alle Einträge ein, die der jeweiligen Variablen entsprechen.

Wenn eine Variable doppelt in der Liste enthalten ist, wird der zuletzt erfasste Kommentar für beide Variablen übernommen.

 

6.

Drücken Sie den Softkey "OK", um Änderungen oder das Löschen zu bestätigen.

- ODER -

Drücken Sie den Softkey "Abbruch", um Änderungen zu verwerfen.
