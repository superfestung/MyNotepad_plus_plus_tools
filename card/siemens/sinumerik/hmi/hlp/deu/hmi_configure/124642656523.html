Transformationen kopieren/einfügen oder löschen

Wenn Sie eine Transformation erstellen möchten, die einer bereits vorhandenen ähnlich ist, dann sparen Sie sich Zeit, wenn Sie die bestehenden Eigenschaften kopieren und an anderer Stelle einfügen.

Vorgehensweise

  

Das Fenster "Transformationen" ist geöffnet.

Elemente kopieren und einfügen

 

1.

Positionieren Sie den Cursor auf das gewünschte Element.

 

2.

Drücken Sie den Softkey "Kopieren".

 

3.

Positionieren Sie den Cursor an die gewünschte Stelle und drücken Sie den Softkey "Einfügen".

Hinweis:

Sie haben die Möglichkeit, den Namen der Transformation im zugehörigen Eingabefenster zu ändern.

 

4.

Drücken Sie den Softkey "OK", wenn Sie die Eigenschaften der Transformation übertragen wollen.

Die Transformation wird eingefügt.

Hinweis

Eine Transformationskopie kann nur auf ein leeres Element übertragen werden.

Elemente löschen

 

1.

Positionieren Sie den Cursor auf das gewünschte Element und drücken Sie den Softkey "Löschen".

 

2.

Drücken Sie den Softkey "OK", um das Löschen zu bestätigen.

Die Transformation für das markierte Element wird entfernt.
