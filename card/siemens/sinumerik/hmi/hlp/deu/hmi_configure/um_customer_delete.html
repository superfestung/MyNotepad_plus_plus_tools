Lokale Anwender-Domäne löschen

Einleitung

Wenn Sie eine lokale Anwender-Domäne wieder löschen möchten, können Sie den Dialog "Anwender-Domäne löschen" nutzen.

Achtung

Datenverlust

Bitte beachten Sie, dass Sie unwiderruflich alle konfigurierten Richtlinien, Maschinenrollen, Benutzergruppen und Benutzer der Domäne verlieren, wenn Sie die Domäne löschen.

Wenn die lokale Anwender-Domäne mit einer zentralen Benutzerverwaltung verbunden ist, wird diese Verbindung beim Löschen der Domäne automatisch getrennt.

Überlegen Sie deswegen sorgfältig, ob Sie die Domäne wirklich löschen möchten oder ob Sie die Daten im Security-Archiv gesichert haben.

Voraussetzung

  • Die Benutzerverwaltung ist aktiviert.

  • Eine lokale Domäne für den Hersteller ist definiert und der Security Admin für den Hersteller ist eingerichtet.

  • Eine lokale Domäne für den Anwender ist definiert und der Security Admin für den Anwender ist eingerichtet.

  • Sie verfügen über die Rolle "Security Admin Hersteller".

Vorgehensweise

Lokale Anwender-Domäne löschen

  1. Im Bedienbereich "Inbetriebnahme" rufen Sie die Benutzerverwaltung über die Softkeys "Security > Benutzerverwaltung" auf. Die Übersichtsseite zur Benutzerverwaltung öffnet sich im Softkey "Konfiguration".

  2. Klicken Sie auf die Schaltfläche "Anwender-Domäne löschen".

  3. Bestätigen Sie die Sicherheitsabfrage mit "OK".

Ergebnis

Die lokale Anwender-Domäne ist gelöscht. Alle Benutzer und Einstellungen der Domäne wurden ebenfalls gelöscht, es sei denn, Sie haben vorab die Daten im Security-Archiv gesichert.
