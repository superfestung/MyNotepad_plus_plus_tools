Allgemeines Vorgehen

Ablauf

  • Legen Sie eine Trace-Sitzungdatei folgendermaßen an:

    • Wählen Sie Variable aus und legen Sie deren Darstellung fest

    • Stellen Sie das Grafikfenster ein, sowie die Speichermodalität

    • Stellen Sie die Triggereigenschaft ein, um Beginn bzw. Ende einer Aufzeichnung festzulegen

  • Speichern Sie die Aufzeichnung und deren eingestellten Eigenschaften

  • Analysieren Sie das Ergebnis, indem Sie Einstellungen für die Darstellung vornehmen, um weitere Einzelheiten näher zu betrachten (z. B. Zoomen, Scrollen, Ausschnitt erfassen usw.)
