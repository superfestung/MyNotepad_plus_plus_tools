Benutzer ändern

Einleitung

Wenn Sie bei einem Benutzer den Benutzernamen, das Passwort, den Klarnamen oder die Zuordnung zu einer Benutzergruppe ändern möchten, können Sie den Dialog "Benutzer ändern" nutzen.

Hinweis

Benutzer-Passwort vergessen?

Auch wenn ein Benutzer sein Passwort vergessen hat, kann der Security Admin in diesem Dialog das alte Passwort durch ein neues Passwort ersetzen.

Voraussetzung

  • Sie verfügen über die Rolle "Security Admin Hersteller" oder "Security Admin Anwender".

Vorgehensweise

  1. Im Bedienbereich "Inbetriebnahme" rufen Sie die Benutzerverwaltung über die Softkeys "Security > Benutzerverwaltung" auf.

  2. Klicken Sie auf den Softkey "Benutzer". Klicken Sie auf der Übersichtsseite auf "Ändern", um einen Benutzer zu ändern.

  3. Führen Sie ihre Änderungen bezüglich Benutzername, Namen, Passwort, Zuordnung zu Maschinenrollen bzw. Benutzergruppen oder Sperr-Status durch. Klicken Sie auf "OK", um die Änderungen zu übernehmen und den Dialog zu schließen.

Ergebnis

Die Änderungen sind in der Liste der Benutzer sichtbar.
