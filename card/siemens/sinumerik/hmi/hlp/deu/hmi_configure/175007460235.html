Trace-Sitzungsdatei laden

Sie haben die Möglichkeit, bereits gespeicherte Sitzungsdateien zu laden, um sie neu zu starten oder sich die Trace-Aufzeichnung von anderen Systemen anzeigen zu lassen.

Einstellungen

Bedeutung

Variablen, Einstellungen

Sie können eine bereits angelegte Trace-Sitzung neu starten.

Variablen, Einstellungen und aufgezeichnete Werte

Sie können sich das Ergebnis einer Trace-Sitzung anschauen, Variable ändern und ggf. neu starten.

Voraussetzung

Es ist eine Sitzungsdatei von Trace-Aufzeichnungen im XML-Format aus folgenden Systemen vorhanden:

  • SINUMERIK Operate

  • Analyze MyMachine /SINUMERIK Tuning Center

Hinweis

Eine Trace-Sitzung, die in einer anderen Anwendung erzeugt und im Kompatibilitätsmodus geladen wurde, ist nicht editierbar.

Wenn die Trace-Konfiguration nicht zur Steuerung passt, ist ein Trace-Start nicht möglich. Passen Sie die Trace-Konfiguration an die Steuerung an um die Aufzeichnung danach neu starten zu können.

Vorgehensweise

  1. Die Funktion "Trace" ist angewählt.

  2. Drücken Sie den Softkey "Trace laden".

    Das Fenster "Trace laden: Bitte Datei auswählen" öffnet sich.

  3. Wählen Sie die gewünschte Sitzungsdatei aus dem entsprechenden Verzeichnis, z. B. "Trace":

    Standardpfad: .../user/sinumerik/hmi/data/trace/

    -oder-

    Drücken Sie den Softkey "Suchen" um den Dialog "Datei suchen" zu nutzen.

  4. Wenn Sie eine Sitzungsatei gewählt haben, drücken Sie den Softkey "OK.

    Das Fenster "Trace laden: Session..." wird geöffnet.

    Hinweis

    Wenn die geladene Datei keine Messdaten enthält, wird die Sitzung direkt geöffnet.

  5. Markieren Sie das Kontrollkästchen "Variablen, Einstellungen", wenn Sie die Trace-Sitzung gleich starten möchten.

    -oder-

    Markieren Sie das Kontrollkästchen "Variablen, Einstellungen und aufgezeichnete Werte", wenn Sie sich die Trace-Sitzung anzeigen lassen möchten.

  6. Drücken Sie den Softkey "OK".

    Die Daten der gewählten Sitzung werden geladen und im Fenster "Trace" angezeigt.
