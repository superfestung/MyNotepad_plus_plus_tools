Trace-Aufzeichnung starten

Vorgehensweise

 

1.

Sie haben die Funktion "Trace" angewählt.

 

2.

Erstellen Sie eine neue Trace-Konfiguration und wählen Sie für jede Variable die Parameter aus.

Sie können auch eine vorhandene Konfiguration laden.

 

3.

Klicken Sie auf das Symbol "Trace starten" in der Symbolleiste.

Wenn die Aufzeichnung erfolgreich gestartet wurde, wird die Aufzeichnungszeit im Trace-Fenster unten angezeigt.

Bei Fehlern zeigt die Anwendung eine Pop-up-Fehlermeldung an.

 

4.

Sie können auf das Ende der Aufzeichnung warten oder in der Symbolleiste auf das Symbol "Trace stoppen" klicken.

Die aufgezeichneten Daten werden im Trace-Diagramm angezeigt.

 

 

Hinweis

Die aufgezeichneten Daten werden nicht gespeichert, wenn Sie die Trace-Aufzeichnung eines Antriebs mit dem Symbol "Trace abbrechen" in der Symbolleiste stoppen.

Für Trace starten und stoppen können Sie auch die entsprechenden Softkeys von SINUMERIK Operate nutzen.

Nach dem Start einer Antriebstrace-Aufzeichnung umfasst die in der Statuszeile angezeigte Aufzeichnungsdauer nicht nur die Aufzeichnung selbst, sondern auch die Zeit, die für das Herunterladen der aufgezeichneten Daten von der Steuerung erforderlich ist.
