Passwort ändern

Einleitung

Wenn Sie ihr persönliches Benutzer-Passwort ändern möchten, können Sie den Dialog "Passwort ändern" nutzen.

Voraussetzung

  • Sie verfügen über eine beliebige Benutzerrolle in der Benutzerverwaltung und sind im System mit Benutzernamen und Passwort angemeldet.

Vorgehensweise

  1. Klicken Sie auf Anmeldung-Schaltfläche rechts oben in SINUMERIK Operate und wählen Sie im Kontextmenü "Passwort ändern". 


    Passwort ändern

  2. Geben Sie zunächst ihr aktuelles Passwort zur Autorisierung ein.

  3. Vergeben Sie dann ein neues Passwort. Beachten Sie die im Dialog vorgegebenen Passwortregeln.

  4. Wiederholen Sie das neue Passwort.

  5. Bestätigen Sie den Dialog mit "OK".

Ergebnis

Das persönliche Passwort des Benutzers ist nun geändert. Der Benutzer kann sich ab sofort mit dem neuen Passwort einloggen.
