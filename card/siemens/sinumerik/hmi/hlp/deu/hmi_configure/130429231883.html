Übersicht

Damit Ihre Maschine vom Siemens-Support besser unterstützt wird, müssen Sie die Maschine registrieren. Die Registrierung ist für Service und Support-Zusatzleistungen relevant.

Sie haben die folgenden Möglichkeiten, Ihre Maschine zu registrieren:

  • Mithilfe eines maschinenspezifischen QR-Codes

  • Manuell über das Registrierungsportal mit der URL "www.siemens.com/sinumerik/register"

Registrierung über den QR-Code

Der QR-Code wird nach dem Hochlauf des SINUMERIK Operate angezeigt.

Zum Scannen des QR-Codes verwenden Sie die Kamera-App auf Ihrem mobilen Gerät. Nach dem Scannen des QR-Codes öffnet sich die URL des Registrierungsportals in Ihrem Browser.

Nach dem Eintragen der für die Registrierung erforderlichen Daten wird im Registrierungsportal ein Code generiert.

Diesen Code geben Sie anschließend in SINUMERIK Operate für die Registrierung ein.
