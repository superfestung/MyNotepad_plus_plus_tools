Maschinenrollen löschen

Einleitung

Wenn Sie eine neue Maschinenrolle löschen möchten, können Sie den Dialog "Maschinenrollen löschen" nutzen.

Voraussetzung

  • Sie verfügen über die Rolle "Security Admin Hersteller".

Achtung

Mögliche Auswirkungen auf die Domäne Anwender

Bedenken Sie, bevor Sie eine Maschinenrolle löschen, dass das Löschen von geteilten Maschinenrollen (d. h. Maschinenrollen, die sowohl von Hersteller als auch vom Anwender benutzt werden können) auch Auswirkungen auf die Domäne Anwender haben.

  • Wägen Sie deswegen sorgfältig ab, ob Sie Maschinenrollen wirklich löschen möchten.

Vorgehensweise

  1. Im Bedienbereich "Inbetriebnahme" rufen Sie die Benutzerverwaltung über die Softkeys "Security > Benutzerverwaltung" auf.

  2. Klicken Sie auf den Softkey "Maschinenrollen".

  3. Selektieren Sie in der Übersichtsseite der Maschinenrollen eine Maschinenrolle, die sie löschen möchten.

  4. Klicken Sie auf "Löschen".

  5. Bestätigen Sie die Sicherheitsabfrage mit "OK". Die Maschinenrolle verschwindet aus der Liste aller Maschinenrollen.

Ergebnis

Die ausgewählte Maschinenrolle ist gelöscht.
