Zeitserver verwenden

Damit die Steuerung immer automatisch über korrekte Zeitinformationen verfügt, können Sie Zeitserver mit der Steuerung verbinden. Die Einstellungen nehmen Sie über das Fenster "Datum und Uhrzeit einstellen" vor.

Voraussetzung

Um Zeitserver verwenden zu können, benötigen Sie mindestens die Zugriffsstufe 3 (Anwender).

Verbindungszustand prüfen

Das Symbol rechts im Feld "Name / IP-Adresse" zeigt den Zustand der Serververbindung.

Symbol

Bedeutung

 

Die Einstellungen wurden gerade konfiguriert, aber vom System noch nicht verarbeitet. Die Verbindung zum Zeitserver wird hergestellt.

 

Es besteht eine aktive Verbindung mit dem eingestellten Zeitserver. Das System synchronisiert erfolgreich Zeitangaben. Die Zeitvorgabe erfolgt durch Zeitserver.

 

Synchronisation mit dem eingestellten Zeitserver ist nicht aktiv. Die Zeitvorgabe erfolgt (per Default) durch PLC (Keine Verbindung zum Zeitserver).

Hinweis

Das System prüft zyklisch die Zeitserververbindung. Wenn eine Änderung vorgenommen wird, kann es einige Sekunden dauern, bis die Einstellung wirksam wird.

Vorgehensweise

 

1.

Wählen Sie den Bedienbereich "Inbetriebnahme" an.

 

2.

Drücken Sie den Softkey "HMI".

 

3.

Drücken Sie den Softkey "Datum Uhrzeit".

Das Fenster "Datum und Uhrzeit einstellen" wird geöffnet.

 

4.

Bei SINUMERIK Operate in der NCU:

Aktivieren Sie das Kontrollkästchen "Zeitserver verwenden (UTC Zeit)".

Bei SINUMERIK Operate auf IPC/PC:

Aktivieren Sie das Kontrollkästchen "Zeitserver für NCU / PU-Zeit verwenden (UTC Zeit)".

Das Feld "Name / IP-Adresse" wird angezeigt.

 

5.

Bei SINUMERIK Operate in der NCU:

Geben Sie im Feld "Name / IP-Adresse" den DNS-Namen oder die IP-Adresse des Zeitservers ein.

Sie können bis zu drei Zeitserver getrennt durch Leerzeichen eingeben.

Bei SINUMERIK Operate auf IPC/PC:

Wählen Sie im Feld "Name / IP-Adresse" eine Optionsmöglichkeit.

Für IPC, falls vorhanden:

  • pool.ntp.org - Standard-Zeitserver

  • ETH1 - IP-Adresse Firmennetz

  • ETH2 - IP-Adresse Anlagennetz

Sie können auch einen eigenen Zeitserver (DNS-Name oder IP-Adresse) eingeben.

Für IPC: Einen der lokal vorhandenen Ethernet-Adapter

 

6.

Drücken Sie den Softkey "Übernehmen".

Wenn eine Verbindung zu einem Zeitserver besteht, werden die Einstellungen mit dem System sofort synchronisiert. Danach wird zyklisch geprüft, ob die Einstellungen wirken. Das Fenster "Datum und Uhrzeit einstellen" bleibt geöffnet.

Der Softkey "Übernehmen" ist nur aktiv, wenn Sie Änderungen vorgenommen haben.

 

7.

Bestätigen Sie die Eingabe mit dem Softkey "OK".

Die Zeitservereinstellungen und die aktuellen Zeitangaben werden von der Steuerung übernommen.

Hinweis

Wenn das Kontrollkästchen "Zeitserver verwenden (UTC Zeit)" deaktiviert wird, müssen Sie das Datum und die Uhrzeit manuell einstellen.
