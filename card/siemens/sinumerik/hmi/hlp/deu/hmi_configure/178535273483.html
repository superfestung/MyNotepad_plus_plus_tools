SD-Karte verschlüsseln / entschlüsseln

Einleitung

Sie können Teile der SD-Karte der NCU verschlüsseln, um die darauf gespeicherten Daten (OEM- und User-Daten) vor unbefugtem Zugriff und Manipulation zu schützen.

Wenn Sie die SD-Kartenverschlüsselung aktiviert haben, können die Daten nur mit dem Gerät gelesen werden, mit dem sie verschlüsselt wurden. So können Sie sicherstellen, dass sensible Daten, die Sie speichern, nicht in die falschen Hände geraten, falls die SD-Karte verloren geht oder gestohlen wird.

Bei der SINUMERIK SD-Karten-Verschlüsselung handelt es sich um eine partielle Verschlüsselung, bei der die wichtigsten Ordner der SD-Karte verschlüsselt werden.

Folgede Ordner werden verschlüsselt:

  • Ordner /oem

  • Ordner /user

Voraussetzung

Hinweis

Diese Funktion bezieht sich nur auf NCU-only-Systeme der aktiven NCU.
Hinweis

Passwort für Security-Archive / Security-Wiederherstellung wurde bereits gesetzt.

Das Passwort für Security-Archive / Security-Wiederherstellung ist Voraussetzung für das Verschlüsseln der SD-Karte. Ohne die Definition dieses Passworts ist die Funktion nicht bedienbar.

  • Wenn Sie auf die Daten der verschlüsselte SD-Karte von einer anderen NCU/PU zugreifen möchten, als die NCU/PU, welche die Daten verschlüsselt hat, müssen Sie sich im Boot-Menü durch die Eingabe des Passworts für Security-Archive / Security-Wiederherstellung authentifizieren.
Hinweis

Einstellung nicht Teil des Security Archivs

Beachten Sie, dass die Einstellung der SD-Kartenverschlüsselung nicht im Security Archiv gespeichert wird und kann nur manuell aktiviert/deaktiviert werden.

  • Mit aktiver Benutzerverwaltung: Sie verfügen über die Rolle "Security Admin" oder die Zugriffsstufe 2 (Service).

    ODER

  • Ohne aktive Benutzerverwaltung: Sie verfügen mindestens über die Zugriffsstufe 3 (Anwender).

  • Sie verfügen über ausreichend freien Speicher auf der SD-Karte. Der freie Speicher sollte 1,2 Mal größer sein, als der belegte Speicher der beiden Ordner /oem und /user. Der freie Speicherplatz wird während des Aktivierungs-/Deaktivierungsprozesses benötigt, danach kann er wieder verwendet werden.

SD-Karte verschlüsseln

  1. Im Bedienbereich "Inbetriebnahme" rufen Sie die SD-Karten-Verschlüsselung über die Softkeys "Security > HMI-Security" auf. Die Übersichtsseite für alle HMI-Security-Einstellngen öffnet sich.

  2. Klicken Sie im Bereich "SD-Karten-Verschlüsselung" auf "Aktivieren". Eine Sicherheitsabfrage zum Neustart der NCU öffnet sich. Der Neustart wird längere Zeit in Anspruch nehmen.

  3. Bestätigen Sie die Sicherheitsabfrage mit "Ok".

Die NCU startet neu und die SD-Karte wird verschlüsselt.

SD-Karte entschlüsseln

  1. Im Bedienbereich "Inbetriebnahme" rufen Sie die SD-Karten-Verschlüsselung über die Softkeys "Security > HMI-Security" auf. Die Übersichtsseite für alle HMI-Security-Einstellngen öffnet sich.

  2. Klicken Sie im Bereich "SD-Karten-Verschlüsselung" auf "Deaktivieren". Eine Sicherheitsabfrage zum Neustart der NCU öffnet sich. Der Neustart wird längere Zeit in Anspruch nehmen.

  3. Bestätigen Sie die Sicherheitsabfrage mit "Ok".

Die NCU startet neu und die SD-Karte wird entschlüsselt.
