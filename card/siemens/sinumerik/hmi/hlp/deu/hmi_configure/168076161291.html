Übersicht Benutzer

Sie benötigen Benutzer für einen personalisierte Rechtevergabe. Benutzer werden in Benutzergruppen zusammengefasst. Diese Benutzergruppen können Sie Maschinenrollen und den damit verbundenen SINUMERIK-Zugriffsrechten zuordnen.

Nur explizit definierte Benutzer dürfen sich bei aktiver Benutzerverwaltung in SINUMERIK Operate anmelden.

Hinweis

Bearbeitung nur Domänen-spezifisch möglich

Beachten Sie, dass die Bearbeitung der Benutzer immer Domänen-spezifisch ist:

  • Anmeldung als Security Admin Hersteller : Sie können Benutzer der Domäne Hersteller bearbeiten.

  • Anmeldung als Security Admin Anwender: Sie können die Benutzer der Domäne Anwender bearbeiten.

Voraussetzung

  • Sie verfügen über die Rolle "Security Admin Hersteller" oder "Security Admin Anwender".

Liste der Benutzer

Benutzer

In der Liste der Benutzer sehen Sie alle definierten Benutzer der Benutzerverwaltung für die aktive Domäne.

Es werden Benutzername und Name aufgeführt, sowie die 1-n Benutzergruppen, der die jeweiligen Benutzer zugeordnet sind. Außerdem wird angezeigt, ob ein Benutzer vom Security Admin gesperrt ist. Ein gesperrter Benutzer kann sich nicht im System anmelden.
