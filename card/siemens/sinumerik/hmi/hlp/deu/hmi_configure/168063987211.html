Übersicht Maschinenrollen

Maschinenrollen sind dafür gedacht, Rollen von unterschiedlichen Teilen des Systems (NCK, PLC, SINAMICS, OA-Applikationen, WinCC Unified, etc.) auf Maschinenebene zu einer maschinenglobalen Rolle zusammenzufassen. Einer Maschinenrolle können Sie eine SINUMERIK-Zugriffstufe (Rolle) zuordnen.

Voraussetzung

  • Sie verfügen über die Rolle "Security Admin Hersteller".

Liste der Maschinenrollen

Maschinenrollen

In der Liste der Maschinenrollen sehen Sie alle definierten Maschinenrollen der Benutzerverwaltung für die Hersteller-Domäne.

Es werden Name und Beschreibung der Maschinenrolle aufgeführt, sowie die zugeordente SINUMERIK-Zugriffsstufe.

Es sind bereits einige Standard-Maschinenrollen vordefiniert. Diese können Sie verwenden und/oder ändern:

  • Maschinenrolle Hersteller

  • Maschinenrolle Service

  • Maschinenrolle Anwender

  • Maschinenrolle Schlüsselschalter 0-3

  • Security Admin Hersteller

Außerdem wird über das Optionskästchen "darf vom Endkunden verwendet werden" angezeigt, welche Maschinenrollen auch für den Anwender zur Benutzung freigegeben sind. Nur wenn der Haken gesetzt ist, kann der Anwender diese Rolle vergeben. Somit kann sich der Hersteller entscheiden, welche Rechte der Anwender auf der Maschine benutzen kann.
