Zertifikat anzeigen

Einleitung

Wenn Sie sich Details zum gewählten Zertifikat anzeigen lassen möchten, nutzen Sie den Dialog "Zertifikat anzeigen".

Voraussetzung

  • Mit aktiver Benutzerverwaltung: Sie verfügen über die Rolle "Security Admin" oder die Zugriffsstufe 3 (Anwender).

ODER

  • Ohne aktive Benutzerverwaltung: Sie verfügen mindestens über die Zugriffsstufe 3 (Anwender).

Vorgehensweise

  1. Im Bedienbereich "Inbetriebnahme" rufen Sie den Zertifikatsspeicher über die Softkeys "Security > Zertifikatsspeicher" auf.

  2. Selektieren Sie ein Zertifikat, das im Zertifikatsspeicher gespeichert ist, und klicken Sie auf "Zertifikat anzeigen". Die Detail-Anzeige zum Zertifikat öffnet sich. Im oberen Bereich der Anzeige sehen Sie folgende Daten:

    • Die Metadaten des Zertifikats, die beim Importieren vergeben wurden.

    • Den Status des Zertifikats: sie sehen ob das Zertifikat gültig ist oder ob die Gültigkeit des Zertfikats bereits abgelaufen ist.

    • Der Zertifikatspfad zeigt die Hierarchie der Zertifikatskette an. Wenn es sich um ein Endentitätszertifikat handelt, wird der Common Name des Zertifikats angezeigt. Wenn der Common Name im Zertifikat nicht gefunden wird oder wenn eine Hierarchie der Zertifikatskette leer ist, wird als Text "certificate" angezeigt. Sie können in der Hierarchie auch eine höhere Hierarchieebene wählen.

    Im unteren Bereich der Anzeige sehen Sie die internen Daten, die aus dem Zertifikat selber ausgelesen wurden. Diese internen Daten sind nicht editierbar.

Mit einem Klick auf "Schließen" können Sie die Detail-Anzeige zum Zertifikat wieder schließen.

Referenz

Im Dialog "Zertifikat anzeigen" können Sie folgende Aktion ausführen.

  • Zertfikat löschen. Die Aktion "Zertifikat löschen" entfernt das Zertifikat/die Zertifikatskettendatei.
