Trace-Ereignisse

Wenn eine Trace-Konfiguration aus einer Sitzungsdatei geladen wird, werden auch die Trace-Ereignisse aus der Datei geladen.

Wenn in dieser Anwendung eine neue Trace-Konfiguration erstellt wird, werden die Variablen mit den folgenden Standardeinstellungen nachverfolgt:

  • Servo-Variablen: Lageregeltakt (Servo-Takt)

  • NC-Variablen: Interpolationszyklus (IPO-Zyklus)

  • PLC-Variablen auf dem System SINUMERIK ONE: PLC-Zyklusvariable mit OB1

  • PLC-Variablen auf dem System SINUMERIK 828D: Interpolationszyklus (IPO-Zyklus)
