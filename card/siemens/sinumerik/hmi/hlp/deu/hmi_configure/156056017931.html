SMB-Sicherheitseinstellungen

SMB ist ein Netzwerkprotokoll für Datei-, Druck- und andere Dienste. Es wird von der NCU verwendet, um auf Dateien auf Windows-Freigaben zuzugreifen. Sie können einige Optionen des SMB-Protokolls konfigurieren

  • SMB V1 in der automatischen Aushandlung zulassen Wenn diese Option aktiviert ist, verwendet die NCU möglicherweise die veraltete/unsichere Protokollversion V1

  • "SMB-Verschlüsselung verwenden" ermöglicht die Verschlüsselung der übertragenen Dateien, unterstützt seit V3

Sie haben die Möglichkeit, die security-relevanten SMB-Protokolleinstellungen über die Bedienoberfläche von SINUMERIK ONE auf der Steuerung einzustellen.

Voraussetzung

  • Mit aktiver Benutzerverwaltung: Sie verfügen über die Rolle "Security Admin" oder die Zugriffsstufe 2 (Service).
    ODER

  • Ohne aktive Benutzerverwaltung: Sie verfügen mindestens über die Zugriffsstufe 3 (Anwender).

Vorgehen

  1. Wechseln Sie im Bedienbereich "Inbetriebnahme" die Anzeige "Security-Einstellungen" über
    "MENU SELECT > Inbetriebnahme > Security > HMI-Security".

  2. Klicken Sie unter "SMB auf NCU" auf den Softkey "Ändern" und wählen Sie die erforderlichen Security-Einstellungen für das SMB-Protokoll:

    • SMB V1 in Auto-Negotiation zulassen (SMB V1 in der Aushandlung der Protokollversion zulassen)

    • SMB-Verschlüsselung verwenden (nur ab SMB V3)

  3. Bestätigen Sie mit "OK".

  4. Starten Sie das System neu.

    Die SMB-Security-Einstellungen sind aktiv.
