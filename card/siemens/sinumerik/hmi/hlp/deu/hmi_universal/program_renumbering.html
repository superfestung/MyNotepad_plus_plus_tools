Programm neu nummerieren

Sie haben die Möglichkeit, die Satz-Nummerierung des im Editor geöffneten Programms nachträglich zu ändern.

Voraussetzung

Das Programm ist im Editor geöffnet.

Vorgehensweise

 

1.

Drücken Sie den Softkey ">>".

Eine neue vertikale Softkey-Leiste wird eingeblendet.

 

2.

Drücken Sie den Softkey "Neu nummerieren".

Das Fenster "Neu nummerieren" wird geöffnet.

 

3.

Geben Sie die Werte für die erste Satznummer sowie für die Schrittweite der Satznummer ein.

 

4.

Drücken Sie den Softkey "OK".

Das Programm wird neu durchnummeriert.

Hinweis

  • Wenn Sie nur einen Abschnitt neu nummerieren möchten, markieren Sie vor dem Funktionsaufruf die Programmsätze, deren Satznummerierung Sie bearbeiten möchten.

  • Wenn Sie für die Schrittweite den Wert "0" eingeben, werden alle vorhandenen Satznummern aus dem Programm bzw. aus dem markierten Bereich gelöscht.
