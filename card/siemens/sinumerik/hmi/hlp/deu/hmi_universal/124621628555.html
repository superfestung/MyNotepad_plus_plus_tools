Nullpunktverschiebung löschen

Sie haben die Möglichkeit, die Nullpunktverschiebungen zu löschen. Dabei werden die eingetragenen Werte zurückgesetzt.

Vorgehensweise

 

1.

Wählen Sie den Bedienbereich "Parameter" an.

 

2.

Drücken Sie den Softkey "Nullp.versch.".

...

3.

Drücken Sie die Softkeys "Übersicht", Basis" oder "G54...G599".

 
 

4.

Drücken Sie den Softkey "Details".

 

5.

Positionieren Sie den Cursor auf die Nullpunktverschiebung, die Sie löschen möchten.

 

6.

Drücken Sie den Softkey "NPV löschen".

Sie erhalten eine Sicherheitsabfrage, ob Sie wirkich die Nullpunktverschiebung löschen wollen.

 

7.

Drücken Sie den Softkey "OK", um den Löschvorgang zu bestätigen.
