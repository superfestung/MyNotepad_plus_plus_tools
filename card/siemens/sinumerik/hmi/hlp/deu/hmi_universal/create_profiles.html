Energiesparprofile anlegen und bearbeiten

Im Inbetriebnahmemodus haben Sie die Möglichkeit, folgende Änderungen in der Liste der Energiesparprofile vorzunehmen:

  • Neue Energiesparprofile mit Vorwarnzeit und Aktivierungszeitraum einrichten

  • Zustandskonditionen, Vorwarnzeit und Aktivierungszeitraum ändern

  • Energiesparprofile löschen

  • Extern erstellte Energiesparprofile ins System importieren

  • Energiesparprofile auf ein externes Medium sichern

    Hinweis

    Maximale Anzahl an Energiesparprofilen

    Es können maximal 8 Energiesparprofile angelegt werden.

    Wenn bereits die maximale Anzahl an Profilen vorhanden ist, steht der Softkey "Neu" nicht mehr zur Verfügung. Um ein neues Energiesparprofil anzulegen, entfernen Sie ein Profil über den Softkey "Löschen" oder verändern Sie ein bestehendes Profil über den Softkey "Ändern".
    Hinweis

    Zugriffsstufe der Energiesparprofile für Anwender

    Mit dem Maschinendatum 51071 $MNS_ ACTIVATE_CTRL_E legen Sie die Zugriffstufe für den Softkey "Energiesparprofile" im Bedienbereich "Parameter" fest (standardmäßig Schutzstufe 1; muss vom Hersteller explizit frei geschaltet werden).

    Hinweis

    Bezeichnungen von angelegten Energiesparprofilen

    Die Namen von neu angelegten Energiesparprofilen sind nur an der Steuerung sichtbar, an der Sie erstellt wurden.

Energiebedingungen

Folgende Kriterien stehen zur Konfiguration eines Energiesparprofils zur Verfügung:

Zustandskonditionen

Bedeutung

Keine Taste auf der Bedientafel gedrückt

Zur Aktivierung des Energiesparprofils darf keine Taste der Bedientafel gedrückt sein.

Keine Taste auf Maschinensteuertafel gedrückt

Zur Aktivierung des Energiesparprofils darf keine Taste der Maschinensteuertafel gedrückt sein.

Kein Bildwechsel erfolgt

Zur Aktivierung des Energiesparprofils darf kein Bildwechsel stattfinden.

Keine Kommunikation mit externem Gerät (USB, Ethernet, V24) aktiv

Zur Aktivierung des Energiesparprofils darf z. B. kein Datenaustausch mit einem externen Gerät stattfinden.

NC-Kanal/Kanäle im Reset

Zur Aktivierung des Energiesparprofils müssen sich NC-Kanal/Kanäle im Reset-Zustand befinden.

Leitrechnersignal verbietet

Zur Aktivierung des Energiesparprofils darf kein Signal vom Leitrechner eingehen.

PLC-Anwendersignal verbietet

Zur Aktivierung des Energiesparprofils darf kein Signal vom PLC-Anwenderprogramm eingehen.

Der OEM kann weitere Zustandskonditionen auf dieses Signal vereinen und unterschiedlich über die Profile wirken lassen.

Aktivierung des Energiesparprofils nach

Hier geben Sie die Zeitspanne ein, nach der das Energiesparprofil aktiv wird.

Vorwarnung nach

Hier geben Sie ein, wann die Anzeige der Vorwarnung beginnt.

Hinweis:

Tragen Sie in die Felder "Aktivierung des Energiesparprofils nach" und "Vorwarnung nach" den gleichen Wert ein, so erscheint keine Meldung mit der Angabe der Restzeit bis zur Akivierung des Energiesparprofils.

Hinweis

Schutzstufen für Bearbeitung der Energiesparprofile

Die Schutzstufe, die benötigt wird, um Energiesparprofile einzurichten oder zu bearbeiten, legen Sie im Maschinendatum 51072 $MNS_ACCESS_EDIT CTRL_E fest (standardmäßig Schutzstufe 2 (Service)).

Vorgehensweise

 

1.

Wählen Sie den Bedienbereich "Inbetriebnahme" an.

 

2.

Drücken Sie die Softkeys "HMI" und "Ctrl-Energy".

Das Fenster "SINUMERIK Ctrl-Energy: Definition der Energiesparprofile" wird geöffnet.

Energieprofile neu anlegen / ändern

 

3.

Positionieren Sie den Cursor auf die Zeile, auf der Sie ein neues Energiesparprofil anlegen wollen und drücken Sie den Softkey "Neu".

Das Fenster "SINUMERIK Ctrl-Energy: Definition der Energiesparprofile" wird geöffnet.

Geben Sie in das Feld "Bezeichnung" den gewünschten Namen des neuen Energiesparprofils ein. Aktivieren Sie die gewünschten Zustandskonditionen. Legen Sie die Zeiten zur Anzeige der Vorwarngrenze und der Aktivierung des Energiesparprofils fest.

 

 

- ODER -

 

 

Positionieren Sie den Cursor auf das Energiesparprofil, das Sie bearbeiten wollen, und drücken Sie den Softkey "Ändern".

Das Fenster "SINUMERIK Ctrl-Energy: Definition der Energiesparprofile" wird geöffnet.

Geben Sie im Eingabefeld "Bezeichnung" den geänderten Namen des Energiesparprofils ein. Selektieren bzw. deselektieren Sie die Zustandskonditionen, die für das geänderte Energiesparprofil wirksam sein sollen, und ändern Sie bei Bedarf die Zeitangaben.

 

4.

Drücken Sie auf den Softkey "OK".

Die neuen bzw. geänderten Energieprofile werden aktiviert.

Die geänderten Texte (def_conditions<lang>.ts) werden ins Verzeichnis \oem\sinumerik\hmi\lng gespeichert.

Energieprofile importieren

 

5.

Drücken Sie den Softkey "Profile importieren.

Das Fenster "Energiesparprofile importieren" wird eingeblendet.

Wählen Sie die gewünschte XML-Datei und drücken Sie den Softkey "OK". Nach einer Nachfrage werden die Energieprofile in die PLC gespielt.

Die zugehörigen Textdateien (def_conditions<lang>.ts) werden im Verzeichnis \oem\sinumerik\hmi\lng abgelegt.

Hinweis:

Nach dem Import von Energiesparprofilen ist ein Neustart erforderlich, um die Profile wirksam zu machen.

Energieprofile exportieren

 

6.

Drücken Sie den Softkey "Profile exportieren".

Das Fenster "Energiesparprofile exportieren: Ablage auswählen" wird eingeblendet.

 

7.

Wählen Sie den gewünschten Ablageort und drücken Sie den Softkey "OK".

Die sprachunabhängigen Zustandsdaten werden im XML-Format abgelegt, die zugehörigen Texte in einer .ts-Datei. Sie haben die Möglichkeit für die XML-Datei einen Namen zu wählen (Vorbelegung: def_conditions.xml).

Die Textdateien werden unter einem festen Namen abgespeichert. Es werden immer alle definierten Energiesparprofile exportiert.

Hinweis:

Beachten Sie bitte, dass auch die von Siemens mitgelieferten Energiesparprofile gelöscht werden können. Es empfiehlt sich in jedem Fall, die Sparprofile zu exportieren und zu sichern.

Energiesparprofile löschen

 

8.

Wählen Sie das Energiesparprofil, das Sie aus der Liste löschen wollen, und drücken Sie den Softkey "Löschen".

 

9.

Bestätigen Sie die Nachfrage mit "OK", um das Profil zu entfernen.
