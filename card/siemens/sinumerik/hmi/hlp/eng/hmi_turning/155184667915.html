Overview of zero points

You must appropriately set the zero point for the 3D model so that the positions of the models match the zero point of the workpiece. For the main spindle and counterspindle, the zero point can be set to machining step "Workpiece".

Separate zero points can be defined for the main spindle and counterspindle. The main spindle coordinate system is always clockwise.

For turning machines with counterspindle, you define the transition between main spindle and counterspindle machining separately for inside and outside machining operations.

If swiveling is set up at your machine, you have the option of defining a dedicated zero point for each plane.

Workpiece zero point position

You have the option of defining the zero point of the workpiece using elements of the 3D model:

  • Unchanged

    The selection "unchanged" takes the workpiece zero point from the 3D model.

    If the zero point for the counterspindle is set to "unchanged", then it corresponds to the zero point of the main spindle.

  • Turned part

  • Point of intersection, 3 surfaces

  • Point of intersection, 2 edges

  • About feature

    The zero point is defined at the center point of the selected model.

Position zero point plane

You have the option of defining the zero point of the plane using elements of the 3D model:

  • The same as the workpiece

    With selection "the same as the workpiece", the zero point of the plane remains unchanged, i.e. as defined in the workpiece.
    This selection is suitable for machining operations performed at the base plane.

  • Point of intersection, 3 surfaces

  • Point of intersection, 2 edges

  • About feature

    The zero point is defined at the center point of the selected model.

Orientation of the coordinate system

The Z axis is aligned vertically with respect to a freely selectable surface. The direction of the X axis is defined using an edge. The direction of the X axis can be adapted using parameter "Rotation Z". For instance, the direction of the X axis can be inverted by applying an offset of 180°.

Point of rotation for swiveling

In addition to the zero point of the plane, you define the point of rotation for swiveling.

The point of rotation does not influence the machining, but is used to provide a better understanding of the values in the "Swivel plane" screen form.

Here you choose between the following settings:

  • The same as the workpiece

    When selecting "the same as workpiece" then the point of rotation corresponds to the workpiece zero point.

  • Point of intersection, 3 surfaces

  • Point of intersection, 2 edges

An additional offset of the coordinates can be entered in the parameters XD, YD and ZD.
