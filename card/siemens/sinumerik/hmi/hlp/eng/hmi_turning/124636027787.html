Tool types

A number of tool types are available when you create a new tool. The tool type determines which geometry data is required and how it will be computed.

In the vertical softkey bar you switch between the following tool groups:

  • Favorites

  • Milling cutter

  • Drill

  • Turning tools

  • Special tool

Software option

To be able to display the Y turning tools, you need the "Turning with the Y axis" option.

Note

The inclined plunge cutters, inclined cutting tools and inclined threading tools are only available on turning machines with a B axis.
