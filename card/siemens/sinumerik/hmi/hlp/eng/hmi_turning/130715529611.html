Import from CAD

To import models, select a CAD file.

The selected file is uploaded to the NX server and analyzed. The results are converted into potential machining steps in SINUMERIK Operate, and displayed in the list of machining steps.

Requirement

  • The connection to the NX server has been set up.

  • The CAD file can be accessed via the Program Manager.

  • A G code or ShopMill/ShopTurn program has been created, and opened in the program editor.

Procedure

 

1.

Press the "Various" and "Import from CAD" softkeys.

Window "Import from CAD" opens.

  
 

2.

Navigate to the archive location, and position the cursor on the CAD file required.

 

3.

Click "OK" to confirm the selection.

The list of machining steps opens.

The 3D model is displayed in the left-hand part of the window.
The recognized models as well as alternative machining steps are listed in the right-hand part.

Note

When opening a CAD file with the "Import from CAD" function, a dialog is displayed with the certificate verification. Check the keys displayed and verify the match using the "Trust" softkey.

Change 3D model view

You can change the 3D model view for optimal viewing in the following ways:

  • Enlarge and reduce model view

    Press the <+> or <-> key if you wish to enlarge or reduce the 3D model.

    The 3D model is enlarged or reduced from the center.

  • Rotate model

    Move the model with the right mouse button pressed to rotate it.

  • Move model

    Move the model with the left mouse button pressed to move it.
