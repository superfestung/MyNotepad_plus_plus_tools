Reading in the license key

You can read the license key into the control using the license file.

Procedure

 

 

The "Licensing" window is open.

Read in license key electronically

...

 

If you have the appropriate license key in the form of an .zip file, then press the "Read-in license key" softkey.

Open the appropriate storage location (e.g. USB) with "OK".

The matching .zip file is detected automatically and the license key is read in.

 

 

Note:

If there are multiple .zip files in the directory, they will be offered to you in the form of a selection dialog.

Select the desired file.
