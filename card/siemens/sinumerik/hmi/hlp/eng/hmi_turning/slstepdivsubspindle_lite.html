ZV (abs) - only for work offset. write "yes"

Z value of the work offset.

mm

ZV (inc)

Workpiece zero is offset in Z direction (the sign is also evaluated)

The parameter is used to ensure that the correct display is shown in the simulation. It has no influence on machining itself.

The workpiece is re-clamped when switching between the main spindle and counterspindle. The new work offset defines the position for machining at the machine. However, the simulation must know by which amount the work offset has shifted with respect to the workpiece, so that both sides of the machining can be displayed.
