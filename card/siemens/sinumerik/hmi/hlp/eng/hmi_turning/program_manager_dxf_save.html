Saving the DXF file

You can save DXF files that you have reduced and edited.

Requirement

The DXF file is open in the editor.

Procedure

 

1.

Reduce file according to your requirements and/or select the working areas.

- OR -

 
 

2.

Press the "Back" and ">>" softkeys.

 
 

3.

Press the "Save DXF" softkey.

 

4.

Enter the required name in the "Save DXF Data" window and press "OK".

The "Save As" window opens.

 

5.

Select the required storage location.

 

6.

If required, press the "New directory" softkey, enter the required name in the "New Directory" window and press the "OK" softkey to create a directory.

 

7.

Press the "OK" softkey.
