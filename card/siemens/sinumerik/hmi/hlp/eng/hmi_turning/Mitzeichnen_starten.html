Starting simultaneous recording

Procedure

 

1.

Load a program in the "AUTO" mode.

 

2.

Press the "Prog. ctrl." softkey and activate the checkboxes "PRT no axis movement" and "DRY run feedrate".

The program is executed without axis movement. The programmed feedrate is replaced by a dry run feedrate.

 

3.

Press the "Sim. rec." softkey.

 

4.

Press the <CYCLE START> key.

The program execution is displayed graphically on the screen.

 

5.

Press the "Sim. rec." softkey again to stop the recording.
