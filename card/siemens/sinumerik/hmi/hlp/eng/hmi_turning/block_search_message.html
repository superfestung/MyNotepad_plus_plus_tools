Continuing program from search target

Press the "CYCLE START" key twice to continue the program from the desired position.

  • The first CYCLE START outputs the auxiliary functions collected during the search. The program is then in the Stop state.

  • Before the second CYCLE START, you can use the "Overstore" function to create states that are required, but not yet available, for the further program execution.

    If the set position is not to be approached automatically after the program start, you can also traverse the tool manually from the current position to the set position by changing to JOG mode for the REPOS function
