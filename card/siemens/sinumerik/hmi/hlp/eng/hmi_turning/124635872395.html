Setting the zero offset

Directly select the work offset in the "Parameter" operating area in the work offset list.

 

Machine manufacturer

Please refer to the machine manufacturer's specifications.

Procedure

 

1.

"Manual Machine" is active.

 

2.

Select the "Parameter" operating area.

 

2.

Press the "Work offset" and "G54...599" softkeys.

 

 

The "Work Offset G54...599" window opens.

 

3.

Position the cursor on the desired work offset and press the softkey "WO selection"

 

 

Now return to the basic screen and enter the selected work offset in the "Work offset" field.
