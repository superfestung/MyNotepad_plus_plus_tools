Thread advance (inc)

The starting point for the thread is the reference point (X0, Z0) brought forward by the thread advance W. The thread advance can be used if you wish to begin the individual cuts slightly earlier in order to also produce a precise start of thread.

Thread run-in (inc)

The thread run-in can be used if you cannot approach the thread from the side and instead have to insert the tool into the material (e.g. lubrication groove on a shaft).

Thread run-in = thread run-out (inc)
