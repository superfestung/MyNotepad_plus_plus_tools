Drilling depth

  • Shank (drilling depth in relation to the shank)

    The drill is inserted into the workpiece until the drill shank reaches the value programmed for Z1. The angle entered in the tool list is taken into account.

  • Tip (drilling depth in relation to the tip)

    The drill is inserted into the workpiece until the drill tip reaches the value programmed for Z1.

Note: If it is not possible to define an angle for the drill in the tool management, it will not be possible to select tip or shank (always tip, 0 field).
