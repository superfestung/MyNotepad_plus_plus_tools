Position circle function - HOLES2

Function

You can program holes on a pitch circle with defined radius with the "Position pitch circle" cycle. The basic angle of rotation (α0) for the 1st position is relative to the X axis. The control moves by the specified angle as a function of the total number of holes. The angle it calculates is identical for all positions.

The tool can approach the next position along a linear or circular path.
