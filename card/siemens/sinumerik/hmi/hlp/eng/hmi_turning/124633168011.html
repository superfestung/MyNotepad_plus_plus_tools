Grooving residual material function - CYCLE952

Function

The "Grooving residual material" function is used when you want to machine the material that remained after grooving along the contour.

During grooving ShopTurn, the cycle automatically detects any residual material and generates an updated blank contour. For a G code program, the function must have been previously selected. Material that remains as part of the finishing allowance is not residual material. The "Grooving residual material" function allows you to remove unwanted material with a suitable tool.

 

Software option

To machine residual material, you require the "Residual material detection and machining" option.
