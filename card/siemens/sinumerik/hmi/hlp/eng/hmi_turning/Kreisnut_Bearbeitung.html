Machining - SLOT2

Machining type

You can select the machining mode for milling the circumferential groove as follows:

  • Roughing

    • Plane-by-plane

      During roughing, the individual planes of the groove are machined one after the other from the center point of the semicircle at the end of the groove until depth Z1 is reached.

    • Helical

      Starting from the center point of the semicircle, milling is carried out on an inclined (helical) path from the reference point of the circumferential slot (top) down to the slot depth once along the slot edge.

  • Finishing

    • Plane-by-plane

      In "Finishing" mode, the edge is always machined first until depth Z1 is reached. The groove edge is approached on the quadrant that joins the radius. In the last infeed, the base is finished from the center point of the semicircle to the end of the groove.

    • Helical

      When finishing, the tool is moved helically downward at the outer wall of the circumferential slot

  • Edge finishing

    • Plane-by-plane

      Edge finishing is performed in the same way as finishing, except that the last infeed (finish base) is omitted.

    • Helical

      Edge finishing behaves in the same way as finishing.

  • Chamfering

    Chamfering involves edge breaking at the upper edge of the circumferential groove.

    Geometries when chamfering inside contours

    Note

    During chamfering, the end mill behaves like a centering tool with a 90° tip angle.

    Note

    The following error messages can occur when chamfering inside contours:

    • Safety clearance in the program header too large
      This error message appears when chamfering would, in principle, be possible with the parameters entered for FS and ZFS, but the safety clearance then could not be maintained.

    • Immersion depth too large
      This error message appears when chamfering would be possible through the reduction of the immersion depth ZFS.

    • Tool diameter too large
      This error message appears when the tool would already damage the edges during insertion. In this case, the chamfer FS must be reduced.

    Note

    Helical machining

    The tool must be suitable for this machining. In case of doubt, a tool with "face tooth cutting across center" must be used.
