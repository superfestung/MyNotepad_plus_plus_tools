General procedure

Proceed as follows to accept data from a CAD file into a G code program or a ShopTurn program using function "Import from CAD":

  • Create and open a program

  • Define a blank (optional)

  • Import a 3D file

  • Define the workpiece zero

  • Dimensions of the blank are directly transferred as blank step into the program (optional)

  • Define a zero point plane for each plane (only for swiveling machines)

  • Select a machining step from the alternatives, add any missing parameters and insert into the program using "Apply".

  • Make supplements/additions to the program (e.g. tool change, positioning blocks, add blank)

Note

All user entries made are automatically backed up.
