Defining the zero point of the plane

For each machining plane, you can define your own reference point.

Note

First set the zero point plane for the relevant plane before inserting features and elements into the program with "Accept".

Procedure

 

1.

Select machining step "Swivel plane".

 

2.

Press softkey "Work offset plane".

The "Work offset plane" window opens.

 

3.

Select how the point of rotation for swiveling is defined.

 

4.

In the "Select" fields, select the required surfaces or edges.

  

- OR -

  

Select the required references in the 3D model.

The selected elements are highlighted in the 3D model and displayed in field "Selection".

 

5.

If the point of rotation cannot be directly selected, enter an additional offset of the coordinates in parameters X0, Y0 and Z0.

 

6.

Select how the zero point of the plane is defined, and select the required references.

 

7.

To align the coordinate origin, select parameters "Direction X" and "Rotation Z".

The orientation of the Z axis is defined by the current plane.

 

8.

Confirm your selection with "Accept".

The set values for the zero point plane can be viewed under X1, Y1 and Z1.
