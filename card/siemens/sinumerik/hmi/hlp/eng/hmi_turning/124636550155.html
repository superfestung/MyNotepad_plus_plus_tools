Deleting a user view

Procedure

 

1.

Select the "Start-up" operating area.

 

2.

press the "Machine data" softkey.

 

3.

Press the "User views" softkey.

 

4.

Press the "Manage view" softkey.

 

5.

Select the desired view from the selection list.

 

6.

Press the "Delete view" softkey.

or

 

With the final confirmation prompt you will be requested to confirm with either "Yes" or cancel with "No".

The "User views" dialog is then redisplayed.

Just like before, as an alternative, it is possible to delete a user view under "System data" under the following path:  ../user/sinumerik/hmi/template/user_views

Note

User views that are part of the system cannot be deleted.
