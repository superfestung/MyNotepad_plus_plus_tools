Program structure

G_code programs can always be freely programmed. The most important commands that are included in the rule:

  • Set a machining plane

  • Call a tool (T and D)

  • Call a work offset

  • Technology values such as feedrate (F), feedrate type (G94, G95,....), speed and direction of rotation of the spindle (S and M)

  • Positions and calls, technology functions (cycles)

  • End of program

For G code programs, before calling cycles, a tool must be selected and the required technology values F, S programmed.

A blank can be specified for simulation.
