Function - CYCLE72

Function

You can mill along any programmed contour with the "Path milling" cycle. The function operates with cutter radius compensation. You can machine in either direction, i.e. in the direction of the programmed contour or in the opposite direction.

For machining in the opposite direction, contours must not consist of more than 170 contour elements (incl. chamfers/radii). Special aspects (except for feed values) of free G code input are ignored during path milling in the opposite direction to the contour.

The following machining types are available for path milling:

  • Plane-by-plane

  • Helical

Note

Activating G40

We recommend that you activate G40 before the cycle call.

Clamping the spindle

For ShopTurn, the "Clamp spindle" function can be set up by the machine manufacturer.
