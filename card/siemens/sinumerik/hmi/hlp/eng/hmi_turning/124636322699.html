New ShopTurn program

In the part program and workpiece directories, you can create ShopTurn programs and then subsequently generate the machining steps for them.

Procedure

 

1.

Select the "Program manager" operating area.

 

2.

Select the desired storage location and position the cursor on the folder in which you would like to store the program.

 

3.

Press the "New" softkey.

 

4.

Press the "ShopTurn” softkey.

The "New Step Sequence Program" window opens.

The "ShopTurn" type is specified.

 

5th

Enter the desired program name and press the "OK" softkey.
