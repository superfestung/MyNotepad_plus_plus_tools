Function - CYCLE85

Function

With the "Reaming" cycle, the tool is inserted in the workpiece with the programmed spindle speed and the feedrate programmed at F.

If Z1 has been reached and the dwell time expired, the reamer is retracted at the programmed retraction feedrate to the retraction plane.

Clamping the spindle

For ShopTurn, the "Clamp spindle" function can be set up by the machine manufacturer.

 

Machine manufacturer

Please refer to the machine manufacturer's specifications.
