Measuring drills - Special applications

Measure drill - special applications

The tool probe has been calibrated with G18 active as is usual for turning tools.

Function

If drills are used on lathes with a length compensation as for milling machines (channel-specific SD42950: $SC_TOOL_LENGTH_TYPE=0), then also a drill can be measured in this application.

Length L1 is always calculated in the 3rd axis (tool offset axis) of the actual plane G17 to G19. This also characterizes the position of the tool.

G17: L1 in Z axis (corresponds to axial position)

G18: L1 in Y axis (no turning machine application)

G19: L1 in X axis (corresponds to radial position)

Conditions

Length L1 is determined if the following conditions are satisfied:

  • the active tool is of type 2xy (drill)

  • Channel-specific SD42950: $SC_TOOL_LENGTH_TYPE=0

  • G17 or G19 are active and
