Machining - CYCLE72

Machining type

You can select the machining mode (roughing, finishing, or chamfer) for path milling. If you want to "rough" and then "finish", you have to call the machining cycle twice (Block 1 = roughing, Block 2 = finishing). The programmed parameters are retained when the cycle is called for the second time.

It is also possible to choose between machining the contour with a cutter radius offset or traversing on the center-point path.
