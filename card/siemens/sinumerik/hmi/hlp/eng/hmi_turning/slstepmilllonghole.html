Machining type

  • Plane-by-plane

    The tool is inserted to infeed depth in the pocket center.

    Note: This setting can be used only if the cutter can cut across center.

  • Oscillating
    Insert with oscillation along center axis of longitudinal slot:

    The cutter center point oscillates along a linear path until it reaches the depth infeed. When the depth has been reached, the path is traversed again without depth infeed in order to eliminate the inclined insertion path.
