Accepting a position row

Procedure

  

You are in the "Position row" input screen.

Open a DXF file

 

1.

Press the "Import from DXF" softkey.

  

Select the storage location and place the cursor on the relevant DXF file.

 

2.

Select the storage location and place the cursor on the relevant DXF file.

  

You can, for example, use the search function to search directly for a DXF file in comprehensive folders and directories.

 

3.

Press the "OK" softkey.

The CAD drawing opens and can be edited for selection of the drilling positions.

The cursor takes the form of a cross.

Specifying a reference point

 

4.

Press the "Select element" softkey.

Navigate the orange selection symbol by repeatedly pressing the softkey for the desired reference point and press the "Accept element" softkey.

 

Specifying clearance with end point

 

5.

Press the "Select element" softkey.

Navigate the orange selection symbol by repeatedly pressing the softkey for the desired end point and press the "Accept element" softkey.

  

Transferring the position pattern to the cycle and the program

 

6.

Press the "OK" softkey.

You return to the "Position row" input screen.

 

7.

Press the "Accept drilling points" softkey.

The selected position pattern is transferred to the program.

Resetting actions

  

If you want to discard actions, press the "Undo" softkey.

Operation with mouse and keyboard

In addition to operation using softkeys, you can also operate the functions with the keyboard and with the mouse.
