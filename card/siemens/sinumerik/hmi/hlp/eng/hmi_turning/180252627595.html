Creating a new plane

The 3D model of the workpiece and the suggested machining steps are displayed in the "Import from CAD" screen form. All planes, on which models were identified, are automatically listed.

In machining step "Workpiece", you can also manually add a plane, for example to create a contour in this plane.

Procedure

 

1.

Select machining step "Workpiece".

 

2.

Press softkey "New plane".

Window "New plane" opens.

 

3.

In field "Selection" select the surface.

  

- OR -

  

Select the required surface in the 3D model.

The selected elements are highlighted in the 3D model and displayed in field "Selection".

 

4.

In field "Assignment" select the spindle to which you wish to assign the surface.

Note

The option of assigning to a spindle is only available if the machine has a counterspindle.

 

5.

Confirm the selection with "Accept".

The inserted plane appears in the list of machining steps.
