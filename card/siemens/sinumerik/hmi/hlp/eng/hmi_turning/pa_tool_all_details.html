Displaying all tool details

All parameters of the selected tool are displayed as list in the "Tool parameters - internal data" window.

The parameters are displayed, sorted according to the following criteria:

  • Tool data

  • Cutting edge data

  • Monitoring data

Note

Authorized personnel

In this specific window, only qualified service personnel should make changes.

Protection level

You require access level "Manufacturer" (protection level 1) in order to edit the parameters in the detail window.

 

Machine manufacturer

Please observe the information provided by the machine manufacturer.

Procedure

...

1.

The tool list, the wear list, the OEM tool list or the magazine is open.

 

2.

Position the cursor to the desired tool.

 

3.

If you are in the tool list or in the magazine, press the ">>" and "Details" softkeys.

 

 

- OR -

 

 

If you are in the wear list or OEM tool list, press the "Details" softkey.

 

4.

Press the "Internal data" softkey.

The "Tool parameters - internal data" window opens.

All of the tool parameters are displayed in the list.

 

5.

Use the scroll bar to display all the parameters in the list.

  

- OR -

...

 

Press the "Tool data", "Cutting edge data" or "Monitoring data" softkey to jump directly to the required parameters in the list.
