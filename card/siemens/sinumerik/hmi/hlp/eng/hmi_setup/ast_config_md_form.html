Measurement result: Machine data

Machine data to tune the speed controller / position controller:

The table provides an overview of which machine data and parameters have been changed by the tuning.

  • The "Original" column contains the values before the tuning.

  • The "Tuned" column contains the values that have been entered by the automatic servo tuning or the user.

⇒ Further actions:

  • Softkeys "Position controller" and "Speed controller" to toggle between the two views.

  • Softkey "<< Back" to change back to the parameter view.
