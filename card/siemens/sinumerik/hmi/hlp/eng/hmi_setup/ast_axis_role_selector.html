Path interpolation: Gantry axis grouping

This softkey is available under the following conditions:

  • The tuned axes belong to a gantry axis grouping.

  • A tuning result for the path interpolation is already available.

In this dialog, select the view of the axis-optimal or path-optimal tuning results. The following columns are displayed:

  • User tuned (parameter values that are entered by the commissioning engineer).

  • Auto tuned (parameter values that are suggested by the automatic servo tuning).

  • Original (parameter values that are currently active and valid in the drive).

⇒ Further actions:

  • Softkeys "Position controller" and "Speed controller" to toggle between the two views.

  • Softkey "Bode diagrams" to switch to the graphic diagram view.

  • Softkey ">>" to toggle the softkey bar.

  • Softkey "Select axis" to toggle between the views of the axis-optimal and path-optimal results.

  • Softkey "Filter details" to display the Bode diagram of the selected filter and change parameters. This softkey is only active if a filter is selected.

  • Softkey "Close" to leave the dialog.
