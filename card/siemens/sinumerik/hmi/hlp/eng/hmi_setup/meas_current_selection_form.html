Current control loop: Selecting the measurement type

The current control loop must be measured:

  • For diagnostic purposes in the case of a fault.

  • If no standard data is available for the motor/power unit combination being used (third-party motor).

For the following measuring types, the current setpoint filter is not involved:

This measuring type is recommended for vertical axes.

Additional measuring types:

The speed controller is deactivated when measuring the current control loop. This is the reason why a vertical axis can drop.

Torque-generating current setpoint (input)

Notice

Safety measures for hanging/suspended axes

For hanging/suspended axes, measuring the current control loop requires that users apply special safety measures, for instance:

  • Perform the measurement using external weight compensation.

  • Perform the measurement without external weight compensation, e.g. by safely and reliably clamping/locking the drive.

Further actions:

  • Softkey "Axis +"; "Axis -" or "Select axis >" to select the axis that is to be measured.

  • Softkey "Load meas. >" to load the saved measuring data of an axis (xml file).

  • Softkey "Clear history" to delete any measurement data that already exists for this axis.

  • Softkey "Measure" to start the measurement in the "Measurement" dialog (with graphic display).

  • Softkey "Meas. parameters >" to change to the parameter view.
