Jerk filter - Function

Function description

High axial jerk values are combined with highly damping filters for movements outside the workpiece or for roughing. This significantly reduces the processing time. In order to achieve an optimum effect in the case of different machining operations, a FIR low-pass is parameterized depending on the dynamic response mode. For an FIR low-pass filter, appropriate values for the frequency and tolerance are entered for each specific dynamic mode. The window type in MD32409 $MA_JERK_FIR_WINDOW = 1 (Hamming) is preset.

If, when processing with DYNROUGH, minimum processing time and, for DYNPREC, high machining accuracy are important, these requirements can be met with a FIR low-pass.

⇒ FIR low-pass: Robust setting for all of the axes in the channel

For the active dynamic mode in the part program, the same values are calculated for all of the axes in the channel and written to the machine data. The calculation leads to an optimal result if the dynamic response attributes of the axes are similar. The CALCFIR instruction is also newly programmed after each changeover of the dynamic mode to explicitly activate the calculation for the FIR low-pass filter.

⇒ FIR low-pass ADV : Axis-specific setting

For the active dynamic mode in the part program, the values are calculated axis-specifically for the axes in the channel and written to the machine data. This filter is suitable for axes whose dynamic response attributes considerably differ from one another. The CALCFIR instruction is also newly programmed after each changeover of the dynamic mode to explicitly activate the calculation for the FIR low-pass ADV filter.

Active dynamic response mode

The active dynamic mode of the G code group 59 is read in again during power-up, during an operator panel reset, and at the end of a part program by evaluating MD20150 $MC_GCODE_RESET_VALUE[58] and MD20152 $MC_GCODE_RESET_MODE[58].

The following machine data is implicitly recalculated and activated depending on the dynamic mode during power-up, during an operator panel reset, and at the end of a part program:

MD32407 $MA_JERK_FIR_FREQ

FIR low-pass - cutoff frequency

MD32408 $MA_JERK_FIR_ORDER

FIR low-pass - filter order

You can find additional information on CALCFIR in the NC Programming Programming Manual.

Additional information on the input of jerk filters and the switchover of position setpoint filter chains is provided in the Axes and Spindles Function Manual.
