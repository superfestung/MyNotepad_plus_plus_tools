Hydraulic linear drives: Configuration - supply data

Define the parameters of the "Supply unit" in this dialog.

You can enter values for the following "Supply unit" parameters:

  • Hydraulic fluid modulus of elasticity [bar]

    This parameter (p0220) describes the compressibility of the hydraulic fluid. This value specifies which pressure change is required to achieve a certain volume change. The modulus of elasticity refers to a relative volume.

  • System pressure [bar]

    This parameter (p0221) describes the pressure delivered by the supply unit. The system pressure and the size of the piston area determine the force available at the piston rod.

  • Valve precontrol pressure [bar]

    This parameter (p0222) describes the pressure for a precontrolled valve. Zero must be entered for a valve that is not precontrolled.
