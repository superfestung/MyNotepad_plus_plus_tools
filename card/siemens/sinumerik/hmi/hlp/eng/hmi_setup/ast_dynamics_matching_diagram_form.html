Path interpolation: Checking the diagram view

"Check path interpolation" dialog for the diagram view.

Further actions:

  • Softkey "Show result >" to display the tuning result of an individual axis.

  • Softkey "Parameter" to change from the diagram view into the parameter view.

  • Softkey "Fault diagrams" to display the deviations of the axes to one another.

  • Softkey "Diagram >" to start diagram editing (cursor, zoom).

  • Softkey "Display legend >" to display legends for the diagrams. You can also hide curves and display further signal charts.

  • Softkey "Tuning log" to display the window in which the tuning log is output.

  • Softkeys "Cancel" and "Accept" to reject or accept measured values.
