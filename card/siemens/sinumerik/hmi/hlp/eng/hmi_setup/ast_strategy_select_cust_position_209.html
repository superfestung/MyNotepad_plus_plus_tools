Position: User-defined strategy (209)

Strategy 203 "Kv without speed setpoint filter and maximize FFW" is the recommended strategy.

Strategy 209 includes the tuning of the position controller gain (MD32200) without speed setpoint filter. In this case, feedforward control is also parameterized.

If feedforward control is not desirable, for example because other axes have not been set up for feedforward control, then strategy 207 (FFWOF) can be selected. When selecting a predefined strategy (203 or 207) the tuning objective (rugged control response, normal control responsible or fast control response) can be selected. If other adaptations are required, then the user can select a user-defined strategy (209) or "Adapt strategy". When using "Adapt strategy", the predefined values are transferred into the user-defined strategy and can be edited.

For large machines or axes with low natural frequencies, we recommend that the position controller is tuned using speed setpoint filters. This is selected by activating the "Methods for maximum Kv" checkbox.

DSC active:

Dynamic stiffness control (DSC) is automatically used and activated by the automatic servo tuning. It is recommended as it facilitates higher gain factors. Depending on the tuning objective (rugged/standard/fast), the following are specified for the gain factor:

  • Kv reduction

  • Kv upper limit

Kv reduction:

With this factor (= 0: 100% reserve, = 1: no reserve) you define as to whether the maximum possible gain factor should be set. The gain factor is limited by the amplitude response of the controller. It is not permissible that the position controller overshoots. Servo gain factor = 1: The position controller just manages not to overshoot. The gain value determined on the basis of the amplitude response is taken over with any reserve.

In order to make the position controller more robust with respect to mechanical changes, depending on the tuning objective (robust/moderate/maximum responsiveness), it is preassigned with: 0.4/0.6/0.8

Kv upper limit:

Even if, from a pure theoretical perspective, higher values for the Kv factor would be possible, the Kv factor is additionally limited using this absolute numerical value. This also achieves the appropriate level of ruggedness. The Kv factor must be entered in the m/mm/min unit.

Feedforward control mode:

In addition to deselecting feedforward control (= "none"), speed feedforward control and torque feedforward control can be selected. If telegram 136 is selected for communication between the drive and NC, then torque feedforward control is set. This permits a minimum following error, and therefore the highest contour precision. All axes interpolating along a path must have the same feedforward control mode.

Methods for maximum Kv:

If the position control gain (Kv factor) is limited as a result of mechanical natural frequencies, the following options are available to dampen any oscillation:

  • Standard: Maximizing the Kv factor with the actual speed controller setting.

  • PT1 speed setpoint filter 2: Use the second speed setpoint filter as low pass filter to smooth any mechanical oscillation. By using filter 2, speed feedforward control setpoints are also smoothed.

  • Damping tuned speed controller: The speed controller is post-tuned with the objective of achieving the highest possible damping with the lowest mechanical natural frequency.

  • PT2 speed setpoint filter 1: The same as for "PT1 speed setpoint filter 2", however, the speed setpoint filter 1 is only in the closed position control loop when DSC is active. As a consequence only the position controller output is smoothed.

Speed setpoint filter:

If it is already known in advance that setpoint filters will be required to stabilize the position control loop, then these can be preset and reserved here. In this case, a pre-assignment is not made when selecting the strategy. To activate the required filters, they must be reserved by activating the "Res" checkbox. The filter parameters can then be set.

⇒ Further actions:

  • Softkey "Adapt strategy" to adapt the pre-assigned values of a predefined strategy. The user-defined strategy (209) is assigned the values of the predefined strategy. All parameters can be set in the "User-defined strategy". The tuning objective "Moderate responsiveness" is the presetting for the "User-defined strategy".

  • Softkey "Meas. parameters >" to adapt the parameters for a measurement. This softkey is only active when "User-defined strategy" is selected.

  • Softkeys "Axis" and "Speed" to toggle between axis and speed controller when selecting a strategy.

  • Softkeys "Cancel" and "OK" to cancel or accept the new settings.
