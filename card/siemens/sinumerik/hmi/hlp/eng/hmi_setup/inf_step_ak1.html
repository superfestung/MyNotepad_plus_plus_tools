Infeed: Line Module Overview

The data on the line filter, the Voltage Sensing Module (VSM) and the line voltage is displayed here in the infeed drive object overview following the commissioning.

The device connection voltage entered in the "Power data" dialog is displayed: This is the basis for monitoring the line voltage (p0281 - p0283) for the alarm threshold and shutdown threshold.

Further actions:

  • Use the "Change" softkey to make the settings on the drive object displayed.

  • Use the "Power data" softkey to branch the "Configuration – Power data" dialog out directly.

    This softkey is only active if you have put the infeed into operation and have not momentarily saved the data.
