HLA characteristic: Error messages

If you enter invalid or inconsistent values while tuning the valve characteristic, the "Error messages" dialog is displayed. The errors that have occurred are listed.

The following error messages can be output:

  • Gradient positive knee range ≤ 0

    The Q value of the positive zero range is greater than or equal to the Q value of the positive knee range.

  • Gradient positive saturation range ≤ 0

    The Q value of the positive knee range is greater than or equal to the Q value of the positive saturation range.

  • Positive saturation range reverses

    The characteristic does not go into the saturation in the positive saturation range.

  • Positive zero range overlaps knee range

    The positive zero range and the positive knee range including rounding overlap.

  • Positive knee range overlaps saturation range

    The positive knee range and the positive saturation range including rounding overlap.

  • Gradient negative knee range ≤ 0

    The Q value of the negative zero range is less than or equal to the Q value of the negative knee range.

  • Gradient negative saturation range ≤ 0

    The Q value of the negative knee range is less than or equal to the Q value of the negative saturation range.

  • Negative saturation range reverses

    The characteristic does not go into the saturation in the negative saturation range.

  • Negative zero range overlaps knee range

    The negative zero range and the negative knee range including rounding overlap.

  • Negative knee range overlaps saturation range

    The negative knee range and the negative saturation range including rounding overlap.

Further actions:

  • "Back" softkey to return to the "Tuning" dialog. You can rectify the cause of the error there.
