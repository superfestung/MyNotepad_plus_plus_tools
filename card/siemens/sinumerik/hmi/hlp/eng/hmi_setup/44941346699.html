Measuring functions - overview

A range of measuring functions allows the time and frequency response of drives and controls to be displayed in graphical form at the control. For this purpose, test signals with an adjustable interval are connected to the drives.

Notice

Traversing range of axes

When performing measurements at the machine, the commissioning engineer must ensure the following:

  • The Emergency Stop button is always within reach.

  • No obstacles are in the traversing range.

Requirements for starting measuring functions

To ensure that no erroneous traversing movements due to part programs can be carried out, the measuring functions have to be started in <JOG> mode.

Notice

No monitoring functions active

When traversing movements are carried out within the framework of measuring functions, no software limit switches and working area limitations are monitored, since these are carried out in follow-up mode.

Prior to starting traversing movements, the user must therefore ensure that the axes are positioned such that the traversing limits specified within the framework of the measuring functions are sufficient to prevent collision with the machine.

Measurement parameter units:

The test setpoints are adapted to the application in question using measurement parameters, whose units depend on the particular measuring function and operating mode. The default setting for all parameters is 0.

Starting the measuring functions

Measuring functions initiating a traversing movement are only selected using the specific softkey. The actual start of the measuring function and therefore traversing movement is always carried out with <NC-START> at the machine control panel.

Once the traversing function has been started, the main screen can be exited without any affect on the traversing movement. If the main screen of the measuring function is quitted without the traversing movement being initiated, the selection of the traversing function is canceled.

The measurement sequence is divided into the following steps:

  1. Setting the traversing range monitoring and enable logic.

  2. Select the measurement type.

  3. Set the measurement parameters using the "Parameters" softkey.

  4. Perform the measurement with: Softkeys "Measurement" → "Start measurement".

Criteria for canceling the operation

The following events will cancel active measuring functions:

  • Hardware limit switch reached

  • Traversing range limits exceeded

  • Emergency stop

  • Reset (mode group, channel)

  • NC STOP

  • No controller enabling command

  • Canceling drive enable

  • Canceling traversing enable

  • Selection of parking (in position-controlled operation)

  • Feed override = 0%

  • Spindle override = 50%

  • Change in operating mode (JOG) or operating mode JOG not selected

  • Actuation of traversing keys

  • Actuation of handwheel

  • Alarms leading to axis shutdown
