Drives: Configuration - Additional motor data

Here you configure additional motor data:

  • Motor holding brake configuration:

    If a connected brake is recognized during the device configuration, then the system automatically activates the brake control and displays "Brake control as sequence control" as the default setting (terminals on the Motor Module).

    Motor holding brake always open: For diagnostic purposes the brake opens permanently.

    If no brake is detected, "No motor brake available" is activated:

  • "Extended brake control" function module:

    The extended brake control function also activates parameters in order to carry out more complex brake control operations, e.g. approaching the brake. You can find more details on this in the function diagrams in the SINAMICS S120 List Manual /LH1/ - in the Chapter "Function Modules".

  • Brake control diagnostics evaluation:

    For an AC drive with "Safe Brake Relay," the "Safe Brake Control" function requires that the type of the brake control be set in parameter p1278, to "Brake control with diagnostic evaluation" (p1278 = 0). This parameter is automatically set for booksize components.

    BICO terminals: r0899.12 "Open holding brake" must be connected to a terminal on the Control Unit, e.g. to terminal DO14: p0744 = BICO 3:899.12

    Brake control after sequence control, connection via BICO: Enables the control of an external brake which is not connected via the terminals on the Motor Module. There is no monitoring of the brake here. The control occurs via SINAMICS.

Overvoltage protection

Please note the advice regarding the overvoltage protection for high-speed Siemens motors:

  • If you have connected a Voltage Protection Module (VPM) then activate the checkbox.

  • If no VPM is connected then the speed is limited in order to prevent the drive from being destroyed.

Bearing version

The "Bearing version" selection is only offered when this motor is available with different bearing versions.

You can set the bearing type here: Thus, the speed is limited to the maximum speed of the bearing in order to prevent mechanical destruction. When "No specification" is selected, the maximum speed of the bearing is not monitored.

Motor temperature sensor

Depending on the drive parameter assignment, only specific temperature sensor types are supported.

As parts of the actual drive parameter assignment are only calculated when actuating the "Finish" softkey at the end of the drive commissioning wizard, it may occur that in the current dialog some temperature sensor types are marked with the following note: "*) Motor temperature sensor type is not supported by the actual drive parameter assignment". This marking disappears once the drive has been commissioned if the selected motor temperature sensor is supported by the drive parameter assignment. Select the motor temperature sensor type according to the motor data sheet if it is marked with *) in the selection list.

Technological application (p0500)

The "Technological application" selection list is only displayed when selecting a third-party synchronous rotary motor. Here you can set whether it is a spindle drive or feed drive, so that the limits and standardizations are calculated accordingly.
