Spindle tuning - Measuring and tuning

Selection of the gear stage for tuning:

  • Select the gear stage.

  • Start the measuring process with the "Tune >" soft key.

  • Repeat this process for all gear stages.

Note

Checking the mechanical system

Before starting the measurement, ensure that the mechanical gear stage that is associated with the selected gear stage is in place.

⇒ Additional actions:

  • Press the ">> Back" softkey to switch to the "Spindle tuning" dialog.

  • Press the "Optimization >" softkey to start optimization.

  • Press the "Cancel" softkey to exit the dialog.

  • Press the "Next step >" softkey to select the parameters for tapping.
