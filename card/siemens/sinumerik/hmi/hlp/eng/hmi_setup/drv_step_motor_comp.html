Drives: Configuration - Motor assignment

Requirement: If present a free motor component can be allocated to the SERVO or a new motor component can be generated for the SERVO.

The dialog "motor assignment" is only displayed in the following cases:

  1. No motor component is assigned to the current SERVO.

  2. There are free motor components available with which the already assigned motor components can be replaced.
