Drives: Configuration - control type/setpoints

Here you set the control type and the PROFIBUS telegram type as well as the number of DDS drive data sets.

  • Default: The control type, PROFIBUS telegram and the number of DDS are generally pre-assigned correctly.

  • Control type:

    If no motor encoder is assigned to the SERVO then "speed control (without encoder)" is automatically set as the control type.

    If a motor encoder is assigned to the SERVO then "speed control (with encoder)" is automatically set as the control type.

  • Telegram type: The default setting is telegram 136: In the event of a change in the telegram type the PLC configuration must also be adapted.

  • Data set: The default number for the DDS is 1. If a drive data set switchover is required, several DDS can be set up if s necessary. Points to bear in mind: A higher number of DDS slows down the SINAMICS system boot up. Further DDS and MDS can also be set up subsequently in the Data set wizard.
