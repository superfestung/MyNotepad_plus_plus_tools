Drives: Configuration - summary

All data with which the drive object has been configured is displayed in the summary.

Further actions:

  • "< Previous step" softkey to be able to return to previous dialog.

  • "Finish" softkey to terminate the commissioning of this drive object.

    In case of changes, a query to save is output: Confirm with "OK" to save the data to a non-volatile memory after switching off and switching on. The data is saved to the CompactFlash card and backed up to a commissioning archive with drive data.

  • Text file: for saving the displayed summary (optional)
