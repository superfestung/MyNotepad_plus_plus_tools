Entering a new jerk filter - FIR low-pass ADV

FIR low-pass  ADV : Axis-specific setting

For the active dynamic mode in the part program, the values are calculated axis-specifically for the axes in the channel and written to the machine data. This filter is suitable for axes whose dynamic response attributes considerably differ from one another. The CALCFIR instruction is also newly programmed after each changeover of the dynamic mode to explicitly activate the calculation for the FIR low-pass ADV filter.

Precondition:

  • MD32890 $MA_DESVAL_DELAY_ENABLE = 1 The phase filter setpoint is active.

  • The maximum axial jerk during path motion in MD32431 $MA_MAX_AX_JERK lies in the range of 5 ... 500 m/s3 for the selected linear axis; in the range of 1.8 ... 180 rev/s3 for a rotary axis. If a value is outside this range, alarm 21900 is pending.

Entering a new jerk filter:

Depending on the dynamic mode, enter the following value for the selected axis:

  • Frequency [Hz]

  • Tolerance for linear axes [mm/inch]

  • Tolerance for rotary axes [°degrees]

The following axis machine data is set active with the "Accept" softkey:

MD32400 $MA_AX_JERK_ENABLE = 1

Axial jerk limitation

MD32405 $MA_CALCFIR_FREQ

FIR low-pass - frequency [Hz]

MD32406 $MA_CALCFIR_TOL

FIR low-pass - Tolerance [mm / inch / °degrees]

MD32404 $MA_CALCFIR_SELECT = 3 or 4

FIR low-pass - Calculation for the selected filter ladder

MD32895[i] $MA_DESVAL_DELAY_TIME

Axial phase filter setpoint: Time constant

Note

Only one variant of the type "FIR low-pass" is permitted in a position setpoint filter chain.

⇒ Additional actions:

  • Use the "Change" softkey to adjust the parameters of the jerk filter.

  • Use "Reset (po)" to activate the filter type in the following axis machine data:

    • MD32402 $MA_AX_JERK_MODE = x5 in the 1st position setpoint filter ladder

    • MD32402 $MA_AX_JERK_MODE = 5x in the 2nd position setpoint filter ladder

  • The "Cancel" softkey discards the entries.

  • Use the "Accept" softkey to activate the entries for the jerk filter.
