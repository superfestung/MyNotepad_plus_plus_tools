This is how you start the automatic device configuration

Note

Subsequently inserting SINAMICS components

The firmware of the configured SINAMICS components is only correctly updated if the components were inserted in the switched-off state. Subsequently inserting SINAMICS components must only be performed in the switched-off state.

Updating firmware

When a drive is connected, the update of the firmware is started automatically. On the appropriate module, you will be informed of the progress in the form of a status display:

  • While the firmware is loading: The "RDY" LED flashes at 1 Hz alternately red - green.

  • When the loading of the firmware is completed: The "RDY" LED flashes red - green at 2 Hz.

  • After switching off and switching on the drive device: The "RDY" LED is continuously green.

Automatic device configuration

Procedure:

  1. After the complete drive system has powered-up, a dialog box for automatic device configuration is displayed: "A device configuration must be carried out for the drive system (drive units) ... "

  2. Select "OK" in order to start the automatic device configuration.

    The system topology is then automatically read-out. This operation can take several minutes. Confirm with "OK".

  3. A power on reset is then requested. Confirm with "Yes". This procedure depends on the expansion stage of the drive system and can take several minutes.

  4. Confirm with "OK" to continue the commissioning.

If the automatic device configuration does not start then it has already been implemented: To establish a defined state or to start once more, please load "Factory settings". After that, a defined output state has again been established.
