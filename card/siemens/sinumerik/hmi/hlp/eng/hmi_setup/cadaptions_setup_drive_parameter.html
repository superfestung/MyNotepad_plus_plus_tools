Adaptations - Configuring the drive

Procedure:

  1. Select a drive parameter for the adaptation:

    [1] Speed controller P gain (Kp)

    [2] Speed controller integral time inverse value (1/Tn)

    [3] Current setpoint filter 1 denominator/numerator frequency

    [4] Current setpoint filter 2 denominator/numerator frequency

    [5] Speed setpoint filter 1 denominator/numerator frequency

    [6] Speed controller reference model natural frequency

  2. To select the following drive parameters, you must enable another function modules (⇒ constraints):

    [7] APC/AVS/APC-ECO derivative-action time (Tv)

    [8] APC filter 2.1 denominator/numerator frequency

  3. The values for "Adaption of lower/upper limit value" are permanently in effect and are preset to the maximum range of 0...3000%.

  4. To enable the adaptation for the drive dataset (DDS), enter the corresponding number.

Note

Unique parameter assignment

If a drive adaptation factor of 1...4 is assigned as output variable to an input variable with a parameter [1]...[8], this assignment will be retained for another input variable.

Example:

The output variable "Drive adaptation factor 1" with the parameter [1] Speed controller P gain (Kp) is assigned to the input variable "Inertia". Thus when adapting the input variable "Position" the parameter [1] Speed controller P gain (Kp) is also assigned to the drive adaptation factor 1.

⇒ Additional actions:

  • Press the "Cancel" softkey to cancel the inputs.

  • Press the "Accept" softkey to save the data in the drive.

Constraints

The following settings are checked for the adaptation of drive parameters:

  • The same telegram (146, 148 or 149) is configured in the NC and in the drive.

  • The "Controller parameter adaptation" function module is enabled for this drive.

The function module is enabled under Setup → Drive → Dialog "Configuration - Motor Module".

  • A SINUMERIK license for the option "Advanced Position Control ECO" (6FC5800-0_M12-0Yx0) is required to use the "Active Vibration Suppression (AVS/APC-ECO)" function module with SINUMERIK. This allows the following control parameter to be adapted:

    [7] APC/AVS/APC-ECO derivative-action time (Tv)

  • A SINUMERIK license for the option "Advanced Position Control" (6FC5800-0_M13-0Yx0) is required to use the "Advanced Positioning Control (APC)" function module with SINUMERIK. This allows the following control parameters to be adapted:

    [7] APC/AVS/APC-ECO derivative-action time (Tv)

    [8] APC filter 2.1 denominator/numerator frequency
