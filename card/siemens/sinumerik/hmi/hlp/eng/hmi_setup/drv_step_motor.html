Drives: Motor configuration

A distinction is made between the following types of motor configuration:

  • Automatically detected motor with DRIVE-CliQ

  • Select a standard Siemens motor from the list:

    First select the motor type. In order to restrict the list for selecting the motors, enter text in the "Search" field which works as a filter. The character * is used as a wildcard. The text works for all column contents in the list except for units and connection type.

  • Enter motor data for third-party motor:

    • with pre-assignment from the list of SIMOTICS motors as a template

    • without pre-assignment of the motor data
