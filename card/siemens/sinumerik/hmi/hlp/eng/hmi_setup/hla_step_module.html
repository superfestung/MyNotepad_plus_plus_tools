Hydraulic linear drives: Configuration - HLA module

The order number of the hydraulic module is displayed in this dialog. Options can be activated.

  • If you activate the option "LED of the HLA module to flash for recognition", the selected HLA module will flash alternately red-green. In this way, you can identify the HLA module in the control cabinet.

  • You can activate the displayed objects in the "Function modules" area.
