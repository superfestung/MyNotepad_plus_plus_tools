Drives: Adding a data set

Precondition: The relevant drive must have been commissioned in order to edit data sets.

Procedure:

  1. Select the required drive with "Drives +/-" and then select the "Data sets" softkey.

  2. Select "Add data set". Enter the number of data sets in the selected input field and confirm with the <Input> key. The defined data set combination can be found in the following table.

  3. Then select the data which should be used to create and fill in the additional data sets. The graphic display helps you with this.

  4. Select "Next step >" to copy MDSx. The data set wizard will guide you through the next steps. An overview of the created data sets is provided in the summary.

  5. Select "Finish" to copy the drive data sets. The progress for this process is provided in the status bar.

  6. You can then save the non-transient changes (→ "OK") or cancel them (→ "Cancel").
