Hydraulic linear drives: Reference value / function module overview

The reference variables and activated function modules are displayed in the reference variable and function module overview.

Note the following:

  • Create additional data sets with the "Data sets" softkey. The reference variables are not recalculated when creating additional drive data sets. The reference variables are blocked via parameter p0573.

  • A license is required for certain function modules in some cases (→ "Licenses" → "All options").
