Drives: Configuration - reference values

The reference values are calculated once at the first commissioning of the drive. After this any recalculation is blocked via the p0573 parameter. When creating additional DDS/MDS, for example for star/delta changeover, the reference values are not therefore recalculated.

The reference values also act as limits. If this area is not sufficient then these must be adapted manually, for example when adding an additional motor or when operating a feed motor which is far above the nominal speed.

The reference values act as a standardization between the NC and drive for the actual value acquisition, the setpoint value provision as well as for the measurement functions. All relative values provided e.g. with the Trace refer to these reference values.

Note

If work is being performed with different DDS with different motor data then the reference values remain the same as these are not switched over with the DDS. The resulting conversion factor must be taken into consideration (e.g., when recording a trace).
