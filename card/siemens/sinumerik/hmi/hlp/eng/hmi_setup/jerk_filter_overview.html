Active jerk filter - overview

All active jerk filters are displayed in the channel in the tabular overview.

A jerk filter is active and displayed if the following is true: MD32400 $MA_AX_JERK_ENABLE = 1

Support for commissioning with SINUMERIK Operate is provided for the following jerk filters:

  • FIR low-pass (dynamic mode) → MD32402 $MA_AX_JERK_MODE = 5

  • FIR low-pass ADV (dynamic mode) → MD32402 $MA_AX_JERK_MODE = 5

  • Double moving averaging → MD32402 $MA_AX_JERK_MODE = 4

  • Moving averaging → MD32402 $MA_AX_JERK_MODE = 2

⇒ Additional actions:

  • Press the "Channel +/-" softkey to switch channels.

  • Press the "New >" softkey to define a new jerk filter.

  • Press the "Change >" softkey to adjust the filter parameters of the selected jerk filter.

  • Press the "Delete" softkey to delete the selected jerk filter.

  • Press the "<< Back" softkey to exit this dialog.
