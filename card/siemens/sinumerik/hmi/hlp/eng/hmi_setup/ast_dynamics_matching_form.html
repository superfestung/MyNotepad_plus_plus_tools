Path interpolation: Overview of axes

Overview of the axes involved in the path interpolation:

  • Prior to tuning:

    Checkboxes to select the axes that should be started for a tuning of the path interpolation. The selected axes are tuned so that the same following error results at the same velocity.

  • Following tuning:

    Marking (green/yellow) of the axes for which tuning of the path interpolation has already been carried out.

⇒ Further actions:

  • Softkey "Edit & tune" to start the automatic tuning of the path interpolation of the selected axes.

  • Softkey "Select strategy" to select the tuning strategy for the path interpolation.

  • Softkey "Display path data >" to display the machine data changed for the interpolation.

  • Softkey "Load path data >" to load a file with the interpolation data to the display.

  • Softkey "Tuning log" to display the window in which the tuning log is logged.

  • Softkey ">> Back" to switch over to the vertical softkey menu.
