Current control loop: Start measurement

Dialog to start tuning.

Further actions:

  • Softkey "Start meas." to start the measurement.

  • Softkey "Meas. parameters >" to return to the "Measurement parameters" dialog.

  • Softkey "Time series plots" to switch to the Bode diagram view.

  • Softkey "Diagram >" to start editing the diagram (legend, cursor).

  • Softkey "Show plot legend >" to display legends for the current diagrams.

  • Softkey "Files >" to save the measurement results in a file, load them from a file or "<< Back" to the "Measurement parameters" dialog.

  • Softkey "Exit" to cancel the measurement.
