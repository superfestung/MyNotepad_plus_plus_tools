Example: Adapting friction compensation with adaptive characteristics

Introduction

This example describes how the friction compensation of a linear axis can be reduced depending on the position and direction of rotation of the rotary axis.

Precondition

Set the following axis machine data to adapt the friction compensation value for an axis:

MD32490 $MA_FRICT_COMP_MODE = 3 or 4

MD32500 $MA_FRICT_COMP_ENABLE = 1

Example

Procedure:

  1. In dialog "Adaptations", select softkey "New >", to enter a new adaptation.

  2. Select a mode, for example 1, for "Permanent" active.

  3. As input axis select a rotary axis, and as output axis , a linear axis.

  4. For the input axis, select "Position" as the input variable and for the output axis select "Friction compensation value plus" as the output dimension.

  5. Enter the values according to the following characteristic:

     

    Input value [°]

    Output value [%]

    1

    179.980

    100.0

    2

    179.990

    10.0

    3

    180.000

    1.0

    4

    181.000

    1.0

    5

    181.010

    10.0

    6

    181.020

    100.0

  6. Complete the entries by pressing softkey "Accept".
