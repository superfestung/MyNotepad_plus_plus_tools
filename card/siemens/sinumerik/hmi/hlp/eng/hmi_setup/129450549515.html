Tips for jerk filters

When entering a new jerk filter, the following parameters are already preset:

  • Filter type: FIR low-pass ADV

  • Axis: The first axis for which no FIR low-pass ADV is parameterized is suggested.

  • Filter ladder: 1

The filter type and the axis can be edited:

  • If a new filter type is selected, the axis and the filter ladder remain unchanged.

  • If a different axis for which no FIR low-pass has yet been defined is selected, the presets for the filter type and filter ladder apply.

  • If a different axis for which an FIR low-pass has already been defined is selected, this filter type is no longer available for selection. The position setpoint Filter ladder 2 is set.

Further information regarding the switchover of position setpoint filter ladders can be found in the Function Manual "Axes and Spindles".

Note

Rule

Only one variant of the type "FIR low-pass" is permitted in a position setpoint filter chain.
