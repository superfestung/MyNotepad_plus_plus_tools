Selecting an axis

The automatic servo tuning is used for semiautomatic and full automatic tuning of the speed controller, position controller and speed/torque feedforward control of rotating servomotors as well as direct linear drives and torque motors. The speed controller, position controller and the control of synchronous and induction spindles as well as a gantry axis grouping and master-slave axes are also tuned.

The following filters are automatically set:

  • Current setpoint filter

  • Speed setpoint filter

The measurements of the speed-controlled system and possibly the mechanical frequency response are used as basis for the controller calculations. If a second encoder is being used, the position controller is always calculated as if the second encoder would always be used for the position control.

  • You can cancel the tuning run at any time by pressing the "Cancel" softkey. The original data is then restored (from the restoration point).

  • A measurement can be canceled at any time using the "Cancel" softkey or from the machine control panel by initiating an NC reset as well as override = 0.

The currently selected drive data set (DDS) or axis parameter set is always tuned.

Various default settings can be selected to enter the tuning target, i.e. the required controller dynamic response. Robust/moderate/maximum responsiveness. This selection results in the default setting of the following values:

  • Amplitude margin

  • Phase margin

  • Maximum bandwidth

Based on these values, the P gain and the integral time of the speed controller can be calculated automatically.

The user can also directly define the controller dynamic response. The "User-defined strategy (108)" should be used for this purpose. It is also possible to preset which controller and which filter are to be tuned. The start dialog of the automatic servo tuning displays an axis overview with all of the existing axes.

⇒ Additional actions:

  • "Tune >" softkey to directly start the tuning with a predefined strategy.

  • "Select strategy >" softkey to select the strategy to tune the controller for this axis.

  • "Show axis >" softkey to directly display or change the results and measurements of axes of this machine which have already been tuned with the automatic servo tuning. This softkey is only active if a measurement has already been performed and measured values exist.

  • "Load axis data >" softkey to load and change saved axis tuning data. Data associated with another machine can also be opened and post-tuned.

  • "Tuning log" softkey to display the individual steps in a separate window.

  • "Options >" softkey to adapt the settings regarding the tuning sequence for all axes.

  • "Interpolation path >" softkey to align the dynamic response of the axes selected for interpolation path optimal. Prerequisite is that the axes have first been individually tuned, and have been set to achieve the optimum axis response (without taking into account other path axes).
