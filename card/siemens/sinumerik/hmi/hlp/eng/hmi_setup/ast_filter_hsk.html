A maximum of 10 current setpoint filters can be defined if all of the options are available.

⇒ Additional actions:

  • Softkey "Drive +/-" or "Select drive", to select a drive.

  • Softkey "Filter group >", to select a filter group and DDS:

    • Current setpoint filter

    • Speed setpoint filter

    • Speed actual value filter

  • Softkey "<<", to switch over the vertical softkey bar and edit the filter and diagram.

  • Softkey "Save drive object", to permanently save the filter parameters in the drive.

  • Softkey "Undo", to undo the filter definitions, i.e. the values are copied from the drive into the control system and the two versions have identical values.

  • Softkey "Accept", to write new or modified filter definitions from the control system into the drive (SINAMICS S120​) so that the two versions have identical values.
