Compensation table values

Instead of transferring the compensation data to the NC via the part program or ini file, you can manually enter the compensation values using this dialog for the particular axis or import these.

 

Software option

You require the following option in order to use this function:

"Bidirectional compensation" (6FC5800-0_M54-0Yx0)

A dialog opens in the "Commissioning" → softkey "NC" → softkey "Leadscrew error" operating area:

  • Without compensation table: No compensation tables have been configured – or compensation tables that had been created, were deleted.

  • With compensation table: If a compensation table has already been configured, the table is displayed and it is shown as to whether a compensation is active/inactive.

  • Simple EEC: If there is more than one measuring system, the measuring system can be selected from the selection box to the left above the table.

Procedure:

  • Select the axis in which the compensation should be active. Carefully note the preconditions for commissioning.

  • Softkey "EEC" and softkey "CEC": The softkeys for switchover are only displayed, if, for an axis, compensation tables are configured for simple and for bidirectional compensation.

  • Select softkey "Configuration >" to change to dialog "Configuring compensation tables".

Alternatively, compensation values can also be imported from a Renishaw RTL, RTA or STY file.

  • To do this, in the dialog root screen extend the vertical softkey bar to include softkey ">>" and select softkey "Import data".

  • Select where the file should be saved and the required Renishaw file.

  • Select the NC axis in which the compensation values should be active.

  • Simple EEC: If more than one measuring system exists, then select the measuring system.

  • Press softkey "Activate ✓" to transfer the compensation values for the selected axis to the NC. Activation with reset (po) is required once for each axis in order to create file NC_CE.INI or AX_EEC.INI; after this, the values can be changed without reset (po). This means that the machine data (MD32700, MD32710) to activate the relevant compensation tables and the compensation are set.

Entering compensation values:

  • Select softkey "Change values >" to enable the table cells to be edited. Enter the required compensation value for each interpolation point.

Diagnostics

Operating area Diagnostics → "Service axis" → Line: "Compensation sag + temperature"
