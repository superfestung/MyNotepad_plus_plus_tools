Friction compensation with torque injection pulse

Parameterizing machine data

The following axis machine data must be parameterized in order to use an acceleration-dependent torque injection pulse also for friction compensation:

Designation

Unit

Machine data

Acceleration-dependent amplitude of the torque injection pulse

---

MD32576 $MA_FRICT_TORQUE_STEP

Delay time of the torque injection pulse

[s]

MD32577 $MA_FRICT_T_PULSE_DELAY_TIME

Ramp-up time of the torque injection pulse

[s]

MD32578 $MA_FRICT_T_PULSE_SMOOTH_TIME

Weighting factor for the amplitude of the torque injection pulse

---

MD32588 $MA_FRICT_T_STEP[0...9]

Enter a factor between -1.0 and 1.0 in MD32588 $MA_FRICT_T_STEP[0...9] for each acceleration value in order to weight the amplitude of the torque injection pulse.
