HLA characteristic: Measure characteristic

You can measure, parameterize and tune the valve characteristic of the hydraulic control valve on this page.

The following parameters are displayed:

  • Status (on the right above the "bar diagram")

    Values displayed for the selected axis:

    • Active, with red, yellow or green flashing background.

    • Inactive, with red, yellow or green background.

    Background color:

    • Red: The axis is not referenced.

    • Blue: The travel path has not been determined, only for "Characteristic measurement with identification".

    • Yellow: The piston adjustment has not been performed.

    • Green: The drive is ready.

  • Bar diagram

    The bar diagram represents the traversing range of the axis which is used to measure the valve characteristic.

    The total length of the bar represents the cylinder stroke and the position displayed in the bar, the piston position.

    Key to the meaning of the bar color:

    • Blue: The travel path has not been determined.

    • Yellow: The piston adjustment has not been performed.

    • Green: The piston adjustment has been performed and the axis is ready.

    The bar positions are converted to absolute positions from the relative piston position of the drive and the cylinder stroke. The absolute position corresponds to the piston position. The left (minimum) position corresponds to the absolute position minus the piston position. The right (maximum) position corresponds to the absolute position plus the cylinder stroke minus the piston position.

  • Absolute position [mm]

    The absolute position of the axis is displayed.

  • Axis type

    The type of the axis is displayed, e.g. linear.

  • Drive number

    The number of the axis is displayed.

  • Drive name

    The name of the axis is displayed.

  • Drive test travel enable

    The drive test travel enable can be performed by the PLC with or without active monitoring of the axis:

    • Without PLC

    • With PLC

  • Monitoring

    The possible settings are:

    • Active

      The upper and lower limits are monitored. The measurement is stopped when a limit is violated.

    • Not active

      The upper and lower limits are not monitored.

  • Upper limit / lower limit

    Enter the upper limit and the lower limit for the axis position.

  • Measurement

    The possible settings are:

    • Characteristic measurement

      Select this setting when the axis has already been adjusted.

    • Characteristic measurement with identification

      Select this setting if the axis is to be automatically fully adjusted during the measurement (see "Configuration Adjustment" dialog). Note the requirements specified in the "Configuration Adjustment" dialog. The entire traversing path must also be free for the measurement. Further information on these actions (control direction correction, valve offset correction, piston adjustment, etc.) that are performed during this measurement can be found at parameter p1959 in the SINAMICS S120/S150 List Manual.

Further actions:

  • "Axis +" / "Axis -" softkeys to change to the next/previous axis.

  • "Select axis >" softkey to open a dialog in order to select the axis from a list.

  • "Load meas. >" softkey to open a file selection dialog to load a measurement. The file can contain characteristic data saved automatically or saved manually.

  • Valve parameter" softkey to open the "Valve Characteristic Parameters" dialog. The valve characteristic parameters are displayed there in a list.

  • S"Adjust >" softkey to open the "Configuration Adjustment" dialog. You can adjust the position of the piston, the pressure and the valve offset there.

  • "Measure >" softkey to open the "Measurement" dialog. You can start a measurement of the valve characteristic there. After the measurement, the measured characteristic and the characteristic set in the drive are displayed in a diagram.

  • "Meas. parameters >" softkey to open the "Measurement Parameters" dialog. The measurement parameters are displayed there in a list.
