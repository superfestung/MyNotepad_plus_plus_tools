Hydraulic linear drives: Configuration - valve selection

You select the method of entering valve data under "Valve selection".

  • For the "Select valve from list" option, the selection list for valve configuration is displayed.

    Select the appropriate valve. The valve data is accepted, and the "Valve data" dialog is skipped.

    Note

    Enter or import valve configurations

    If no valve configurations have been entered or imported, an empty list is displayed. You can fill the list with valve configurations in the following ways:

    • Enter valve data.

    • Import valve configurations, e.g. from SINUMERIK powerline.

      You can perform the import in the "Machine data" dialog via "Manage data".

  • For the "Enter valve data" option, the selection list for valve configuration is not displayed. The valve selection list is displayed again if you activate the "Template from list" checkbox. You can reassign the valve data for the "Valve data" dialog with the data of the selected file.

Using the "Search" combo box, you can restrict the drop-down list of valves using a free text filter. While entering a search text, the list is updated with each keystroke. The combo box contains previously used search texts.
