Drive units - topology

In the "Topology" window you can see the DRIVE-CLiQ wiring of the components for the selected drive unit. The connected component connections are arranged directly next to one another and shown in a table. This means that you can follow the wiring of every DRIVE-CLiQ line starting from the Control Unit or NX expansion module (e.g. up to an encoder).

The wiring of the drive objects "from/to" is displayed below the table.

⇒ Additional actions:

  • Softkeys "Drive device +", "Drive device -" can only be used if the system has more than one drive device.

  • Press the "Change" softkey in order to carry out the following actions on the drive object selected:

    • Enabling/disabling drive object

    • Delete the component

    • Delete a drive object

  • "Configure drive unit" softkey, in order to subsequently set the following values for the selected drive device or drive system back to the default values (= factory setting):

    • Drive object names

    • Telegram types

    • BICO relay for terminal wiring

    • Second encoder data set

  • "Add component" softkey is only active if a new component is inserted: The component is added to the target topology with this softkey.

  • "Display options" softkey to configure the default display:

    • Comparison between target and actual states

      Black: Target and actual states match.

      Gray: Non DRIVE-CLiQ components such as encoders or motors without DRIVE-CLiQ interface (e.g. POS 1) are always displayed in gray in the "Actual/target topology comparison" view.

      Red: Target and actual states do not match, e.g. if a connection in the actual topology is missing in the target topology.

    • All components are displayed when the filter is "off" (also components without DRIVE-CLiQ).

    • Selection or deselection of columns: Show or hide columns.
