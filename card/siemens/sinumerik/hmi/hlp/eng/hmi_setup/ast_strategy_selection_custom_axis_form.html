Axis: Selecting a strategy

Here, you can select which controllers are to be tuned for this axis, and whether the control loop should be checked using an appropriate measurement. From the options listed below, a selectable combination of measuring/tuning/checking is obtained:

Speed controller:

Measuring

Tuning/not tuning

Checking/not checking

Position controller:

Measuring

Tuning/not tuning

Checking/not checking

Measurements must be performed for tuning as well as for checking:

  • For tuning: Speed-controlled system (and mechanical frequency response)

    The control loop can also be calculated based on these measurements and the selected controller parameters.

  • For checking: Reference frequency response to setpoint input (closed speed controller and/or closed position controller)

    For the selected strategy, the steps to be undertaken are listed in a table with the relevant checkbox. The preliminary measurement to determine the excitation can always be selected or deselected. The preliminary measurement is always recommended when first commissioning the axis or for unknown mass relationships (load/motor). The measurement parameters for the following measurement are automatically determined after the preliminary measurement. User-defined measurement parameters are then overwritten.

After selecting the strategy for the axis as a whole, additional details can be preset for the individual controllers (speed and position). If a user-defined strategy was not selected, then these controllers are already set with values corresponding to the tuning objective.

⇒ Further actions:

  • Softkey "Adapt strategy" to accept the settings of a predefined strategy as user-defined. When selecting the user-defined strategy, this softkey is not active.

  • Softkey "Meas. parameters >" to adapt the measurement parameters for a measurement. The measurement parameters for a predefined strategy can be adapted before or after the respective measurement. With the definition of a predefined strategy, the measurement parameters can already be set before the measurement for all measurements. Select the measurements in succession and press the "Meas. parameters >" softkey.

  • Softkeys "Speed" and "Position" to toggle between speed controller and position controller when selecting a strategy.

  • Softkeys "Cancel" and "OK" to cancel or accept the new settings.
