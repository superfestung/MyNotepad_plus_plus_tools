Preconditions for commissioning

Ball screw production tolerances lead to larger or smaller dimensional deviations (sag and angularity error compensation, abbreviated: CEC). These are complemented by dimensional deviations used by the measuring system as well as its installation tolerances on the machine (lead screw error and measuring system error compensation, abbreviated: EEC) and additional machine-dependent error sources.

Because these dimensional deviations directly affect the accuracy of workpiece machining, they must be compensated for by position-dependent compensation values. The compensation values are derived from measured error curves and entered in the CNC in the form of compensation tables during commissioning.

Measuring system error compensation (EEC)

The following axis machine data are relevant to the compensation for measuring system errors:

MD32700 $MA_ENC_COMP_ENABLE[0,1]

to activate the measuring system error compensation.

MD38000 $MA_MM_ENC_COMP_MAX_POINTS[0,1]

to define the number of interpolation points.

Index 0 is valid for encoder 1; index 1 is valid for encoder 2.

Sag and angularity error compensation (CEC)

This description is valid for: "Direction-dependent lead screw error compensation using option "bidirectional compensation" – and the compensation tables available in SINUMERIK Operate.

Precondition:

 

Software option

In order to use this function a license is required for the following option:

"Bidirectional compensation" (6FC5800-0_M54-0Yx0)

The following machine data are relevant to compensate direction-dependent sag and angularity errors:

General machine data:

MD18342[n] $MN_MM_CEC_MAX_POINTS

to define the total number of interpolation points for memory reservation.

Note

Compensation table for SINUMERIK 828D

For SINUMERIK 828D, the maximum number of interpolation points per compensation table for one axis is fixed.

Axis machine data:

MD32710 $MA_CEC_ENABLE

to activate sag compensation for this axis.

General setting data:

SD41300[n] $SN_CEC_TABLE_ENABLE

to include the compensation table [n] in the calculation.

More information

The option "Sag compensation, multi-dimensional" (6FC5800-0_M55-0Yx0) can be parameterized via machine data and system variables: ⇒ "Compensations" chapter in the Monitoring and Compensating Function Manual.

MD32720 $MA_CEC_MAX_SUM

to limit the maximum effective compensation value (sum).

MD32730 $MA_CEC_MAX_VELO

to limit the maximum change in the axis velocity in MD32000 $MA_MAX_AX_VELO.
