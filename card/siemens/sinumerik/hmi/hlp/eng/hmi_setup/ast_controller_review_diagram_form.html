Measuring result: Bode diagram

The Bode diagram initially shows the calculated, closed control loop. The values from the "User tuned" column are displayed. When checking the diagram, the values proposed by the automatic servo tuning can either be accepted or adapted by the commissioning engineer.

⇒ Further actions:

  • Softkey "Zoom >" to specifically show areas separately for the X axis and Y axis.

  • Softkey "Represent scaling" to set the value range.

  • Softkey "Cursors >" to use cursor A and cursor B in the diagram as position display.

  • Softkey "Signal configuration" to configure signals that should be displayed.

  • Softkey "Display legends" to hide the legends.

  • Softkey "Manual parameters" to change to the parameter view.

  • Softkey "<< Back" to toggle the softkey bar.
