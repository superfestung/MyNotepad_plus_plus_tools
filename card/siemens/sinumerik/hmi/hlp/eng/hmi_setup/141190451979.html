Installing and commissioning

Benefits

SINAMICS Technology Extensions are configurable functions or technologies that are used to extend SINAMICS firmware of the drive to create a customized SINAMICS S120 converter to address a specific application. The extensions allow high-performance application-specific solutions to be created for various areas: for example, for test stands, storage and retrieval machines, transport systems based on linear motors and multi-winding motors in mechanical engineering.

SINAMICS Technology Extensions are subsequently installed as software packages. SINUMERIK Operate supports you during installation and commissioning.

Requirements

The workflow functions exclusively for SINAMICS drives.

To install and commission a SINAMICS Technology Extension access level "Service" or "Manufacturer" is required.

Commissioning dialogs are provided in operating area "Commissioning" → Softkey "Drive system" → Softkey "Technology Extensions".

 

The online help for "Technology Extensions" at SINUMERIK Operate supports you when commissioning the system.

Workflow

Step 1: Install <name>.tgz either from the USB storage medium or network drive.

Step 2: Activate Technology Extension at the drive object (servo or infeed): This includes the automatic restart of the SINAMICS drive.

Step 3: Activate the option under softkey "Licensing" as some Technology Extensions are options that require a license.
