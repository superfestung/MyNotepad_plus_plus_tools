Drive units - PROFIBUS

Configuration via PROFIBUS or PROFINET is displayed in this dialog.

Further actions:

  • The "Drive Unit +", "Drive unit -" or "Select drive unit... " softkeys are only displayed if more than one drive unit exists in the system.

  • With the "Change" softkey you can activate the editing function: Then you can change the telegram type or assign the SERVO to another axis (↑ up" or "↓ down").

    The telegrams displayed depend on the settings in the PLC project and must match this, otherwise error messages appear.

  • Softkey "Display options" to configure the default display.

    You can select or deselect the following columns:

    • NC axis

    • NC telegram type

    • NC PROFIsafe address

    • I/O addresses

    • Slot   [number]

    • Slot type   [input or output]

    • Length   [byte]

    • SINAMICS drive object

    • SINAMICS telegram type

    • SINAMICS PROFIsafe address
