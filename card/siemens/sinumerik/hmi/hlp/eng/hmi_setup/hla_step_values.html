Hydraulic linear drives: Configuration - reference variables

In this dialog you define reference variables for velocity, pressure, force, power, angle and acceleration. The reference variables are calculated once at the first commissioning of the drive.

In order to prevent reference values being calculated, activate the "Inhibit automatic reference value calculation" option.

The reference variables act as a standardization between the NC and drive for the actual value acquisition, the setpoint value provision as well as for the measurement functions. All of the quantities specified as relative value refer to these reference quantities. The reference variables additionally act as limits. If this range is not sufficient, you must adapt the reference variables manually.

The reference variables are not dependent on the parameter set. You can define as to whether reference variables should be automatically preassigned using parameter p0340.0.

p0340 automatic parameter calculation

Bit 0 = 1 preassign reference values

Bit 4 = 1 preassign limit values

Note

If you are working with several DDS with different valve characteristics, the reference variables remain the same as these are not switched over with the DDS. The resulting conversion factor must be taken into consideration, e.g. when recording a trace.

Save the reference variables with the "OK" softkey.
