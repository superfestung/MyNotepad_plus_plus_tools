Commissioning cogging torque compensation

Benefits

Cogging torque compensation is a drive function which compensates for the cogging torques of synchronous motors and thus ensures a better radial eccentricity. Cogging torques are a property of the specific motor. Therefore, each motor requires a separate compensation table. The compensation values are automatically determined during the learning run.

Cogging torque compensation is suitable for:

  • Synchronous motors

  • Type SERVO drive objects

Requirement

 

Software option

In order to use this function a license is required for the following option:

"Cogging torque compensation" (6FC5800-0_D50-0Yx0)

Call the function at the control in the commissioning operating area: Softkey "NC" → Menu forward key → Softkey "Cogging torque compensation". The softkey is not shown until the option has been set.

 

The online help for cogging torque compensation at SINUMERIK Operate supports you during parameterization.

Commissioning procedure

Procedure:

  • Setting option requiring a license

  • Activate the function module for the respective axes in the drive commissioning wizard.

  • Configure the learning run.

  • Test and start automatically generated part program.

  • Starting the learning run.

  • Activate and save the compensation values for the selected drive data sets (DDS).

  • Display and check compensation values.
