Hydraulic linear drives: Configuration - summary

All the configuration data of the drive object is displayed in the summary.

Further actions:

  • Softkey "< Previous step" to be able to return to the previous dialog.

  • Softkey "Finish" to terminate the commissioning of this drive object.

    In case of changes, a query to save is output: Confirm with "OK" to save the data to a non-volatile memory after switching off and switching on. The data is saved to the CompactFlash card and backed up to a commissioning archive with drive data.

  • Option "Save text in the file ..."

    In order to save the summary of the configuration in a file, activate this option.
