Encoder data - Encoder type absolute, SSI protocol & HTL/TTL

For an absolute encoder with SSI protocol the following settings must be configured:

  • Encoder type

  • Power supply

  • Track setting

  • Inversion

  • Resolution

  • Measuring system protocol

  • Expert

Note

"Error messages" softkey

If invalid values or value combinations are selected then this is indicated by the following responses from the system:

  • The invalid options are marked with a red background and a brief message text is displayed.

  • The "Error messages" softkey is available under which the whole message text of the pending error is displayed.

  • The parameter dialog can only be exited using the "Cancel" softkey. The "OK" softkey for saving the parameter only becomes active if all faults have been eliminated.

Track settings

The following parameters are set in this section:

Default:

p405[x].2 = 0: No track monitoring

Level

p405 [x].1=0: HTL

p405 [x].1=1: TTL

Signal

p405 [x].0=0: unipolar

p405 [x].0=1: bipolar
