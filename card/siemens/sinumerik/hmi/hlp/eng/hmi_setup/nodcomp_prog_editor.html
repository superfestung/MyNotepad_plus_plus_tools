Using the program editor

A parts program with a cycle call is generated for every selected compensation relationship.

Procedure:

  1. In order to adapt the parts program, select "Change". An entry can be made in the field when it is orange.

    • Insert a line break with <SHIFT> + <INPUT>.

    • Acknowledge with "OK" to exit the editing.

    • To reject the entry, select "Cancel".

  2. To exit the program editor and start the parts program, press the "OK" softkey.
