Drive units - configuration

In the "Configuration" window you can obtain a list of the connected components for the selected drive unit. You can identify the assignment of the components to the drive objects, e.g. the assignment of the motor module, motors and encoders to the drive objects and their assignment to the NC machine axes (after the axis assignment).

Further actions:

  • The softkeys "Drive Unit +", "Drive unit -" or "Select drive unit... " are only displayed if more than one drive unit exists in the system.

  • "Change" softkey in order to carry out the following changes to the configuration:

    • Change names of drive object and components

    • Change comparison level between the actual topology and setpoint topology

      The comparison level (high/medium/low/minimum) is used only for diagnostic purposes in order to pinpoint an error, e.g. when swapping two components; not when replacement parts are used.

      Note

      This change is only recommended if there is already a current backup of the drive data available.

  • Press the softkey "Load firmware" to update the firmware of the selected component.

  • "Details" softkey to display the technical data of the selected component.

  • "Sort" softkey to sort the components in accordance with the following criteria:

    • Drive object

    • Component number

    • Wiring

    • Axis number

  • "Display options" softkey to configure the default display:

    • Topology comparison between setpoint and actual states

      Black: Setpoint and actual states match.

      Gray: Existing connection in the setpoint topology which is missing in the actual topology.

      Red: Setpoint and actual states do not match, e.g. if a connection in the actual topology is missing in the setpoint topology.

    • All components are displayed when the filter is "off" (even components without DRIVE-CLiQ).

    • Selection or deselection of columns: Show or hide columns.
