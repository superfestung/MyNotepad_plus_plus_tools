Drive devices - interconnections

Each drive unit contains a large number of input and output variables as well as internal control variables. BICO technology (binector connector technology) allows the drive to be adapted to a wide variety of conditions.

Digital and analog signals, which can be interconnected as required by means of BICO parameters, are identified by the prefix BI, BO, CI, or CO in their parameter name. These parameters are identified accordingly in the parameter list or in the function diagrams:

  • Binectors (digital)

    BI: Binector input, BO: Binector output

  • Connectors (analog)

    CI: Connector input, CO: Connector output

To interconnect two signals, a BICO input parameter (signal sink) must be assigned to the desired BICO output parameter (signal source).

⇒ Further actions:

  • "Search" softkey: The dialog for finding BICO parameters opens.

  • You can toggle between displaying binectors and connectors using the "Display connectors"/"Display binectors" softkey.

  • You can toggle between displaying inputs and outputs using the "Display inputs"/"Display outputs" softkey.

  • "Display options" softkey to configure the parameters display.
