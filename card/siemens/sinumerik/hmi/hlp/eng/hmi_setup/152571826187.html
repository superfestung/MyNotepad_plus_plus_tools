Note regarding the S120 Combi

Note

Topology rules für SINUMERIK 828D

It is possible to connect an external encoder to a free interface on the DRIVE-CLiQ Hub Module.

If a machine only has motor measuring systems, first include a DRIVE-CLiQ Hub Module in the topology and then connect the external encoder to the DRIVE-CLiQ Hub Module.

It is not possible to connect an external encoder to a free interface on the SINAMICS S120 Combi.

You can find more details in the "Topology rules" chapter of the CNC Commissioning Manual.
