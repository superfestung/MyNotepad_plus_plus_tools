Axis: Set options

JOG mode: Meaning of the options

  • Perform all measurements of each measurement series with the first NC start:

    All repeats (e.g. motion in the positive and negative axis direction) within a measurement series are started automatically.

  • Start measurements automatically:

    The start screen of each measurement series is skipped and the measurement process is directly started with the default measurement parameters.

  • Automatic confirmation of the measurement:

    The measurement result is not displayed. This is only recommended when a fully automatic tuning is required. The measurement cannot be repeated and improved through a variation of the measurement parameters.

  • Automatic acknowledgement of the controller data:

    The "Controller data overview" display is skipped. The controller data determined by the algorithm is directly activated. The parameters determined by the algorithm can no longer be controlled or changed during the tuning.

  • Automatic data activation using the operator panel reset:

    The "Operator panel reset" signal is generated by the algorithm. If this option is deactivated, the "Operator panel reset" is requested via a message.

  • Automatically backup drive boot files (ACX format):

    After completing the tuning of a machine axis, the drive data is automatically saved on the CompactFlash card in the ACX format. If this option is deactivated, a query is made.

  • Allow strategy selection during tuning:

    The dialog to select the tuning strategies for the speed controller and position controller is displayed. This option is not recommended for a fully automatic tuning.

  • Perform preliminary measurement to determine the excitation:

    A preceding, additional measurement is performed to more precisely determine the measurement parameters for each measurement series. This is recommended when measuring directly driven machine axes for the first time. This option is only relevant for cases in which a predefined tuning strategy has been selected. You can select this option for the user-defined strategy.

  • Switch off gantry consistency check:

    The checking of the controller parameters in the gantry axis grouping is switched off with this option. Axes in a gantry axis grouping are first tuned in succession. On completion of the tuning, a check is made as to whether the controller parameters of the individual axes in the gantry axis grouping are identical. If the dynamic response varies too much, a message is output.

  • Suppress all warnings for measurements

    No warnings are output during the measurement. Activation of this option is only recommended for experienced commissioning engineers and is only required when the automatic servo tuning is to be performed without queries. Prerequisite is that the automatic servo tuning has been tested previously (and, if required, the strategy selection and measurement parameters have been adapted).

AUTO mode:

 

Software option

In order to use this function a license is required for the following option: "AST call by part program" (6FC5800-0_S10-0Yx0)

This option is not available for SINUMERIK 828D.

With the option "Assign this HMI for "Call AST from the part program"", you select SINUMERIK Operate to perform the automatic servo tuning in the AUTO mode:

  • If this option is selected, the current SINUMERIK Operate performs the automatic servo tuning.

  • If this option is not selected, a different SINUMERIK Operate performs the automatic servo tuning.

AUTO mode: General sequence

The machine manufacturer stores "session files" on the system CompactFlash card, which the end user calls in the part program. The session files are XML files in which the respective axes that are to be retuned, are set up and stored with the associated tuning strategy for the application. The following settings must be made by the machine manufacturer so that the cycle call for the automatic servo tuning from the part program functions at the end user:

  • The instance of SINUMERIK Operate on which the cycles are executed has been selected.

  • The XML files with the appropriate tuning settings for the respective axis, which are required by the cycles, are available.

Further actions:

  • "Clear history" softkey to delete already existing tuning data.

  • "Export data" softkey to export the tuning files.

  • "Cancel" and "OK" softkeys to respectively cancel and accept the new settings.
