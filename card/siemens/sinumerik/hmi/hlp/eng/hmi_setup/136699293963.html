Commissioning a jerk filter

Benefits

To achieve high surface quality, the excitation of mechanical oscillations on the machine is kept to a minimum. When milling freeform surfaces, it can be advantageous to smooth the setpoint characteristics of the machine axes by means of a jerk filter. The parameterization of an axial jerk filter depending on the processing technology (dynamic mode) has the following effect:

  • Damping of machine vibrations

  • Shorter processing time

Precondition

 

Software option

In order to use a FIR low-pass jerk filter, a license is required for the following option:

"Top Speed Plus" (6FC5800-0_S62-0Yx0)

This option includes the use of the FIR low-pass filter type and the switchover from position setpoint characteristic filter chains when the axes are at a standstill.

After the option is set, the "Jerk filter" softkey is displayed in the operating area "Commissioning" → Softkey "NC" → Menu forward key.
