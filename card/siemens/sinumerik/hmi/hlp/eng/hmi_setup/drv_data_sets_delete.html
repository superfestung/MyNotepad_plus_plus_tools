Drives: Remove data set

Procedure:

  1. Select "Remove data set": A recycle bin is displayed.

  2. Select the check box for the data set that you wish to remove.

  3. Select "Next step >": The remaining data set(s) are displayed. If you want to configure less than 8 drive data sets then select the corresponding number from the dropdown list. The redundant drive data sets are then removed.

  4. Select "Next step >": In the summary you can see how many data sets are currently available.

  5. Select "Finish", and the system creates the (missing words) the data records are finally removed. The progress for this process is provided in the status bar.
