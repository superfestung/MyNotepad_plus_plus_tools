Path interpolation: Checking the parameter view

"Check path interpolation" dialog with switchover between "Path optimal" and "Axis optimal".

"Controller data overview" after completion of the path interpolation tuning or loading of the tuning data from the xml file.

⇒ Further actions:

  • Softkey "Show result >" to display or edit the controller data of the selected axis after the automatic servo tuning.

  • Softkey "Bode diagram" to display all position controllers of the path interpolation one above the other in a Bode diagram.

  • Softkey ">>" (switchover menu) to switch over the vertical softkey bar.

  • Softkey "Machine data >" to display the relevant drive parameters.

  • Softkey "Tuning log" to display the window in which the tuning log is output.

  • Softkeys "Cancel" and "Accept" to reject or accept measured values.
