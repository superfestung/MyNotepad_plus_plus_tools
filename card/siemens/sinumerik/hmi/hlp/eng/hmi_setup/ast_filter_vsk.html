Filter (softkey "Filter details")

The softkey "Filter details" is only active if one of the following filters is selected.

  • Current setpoint filter

  • Speed setpoint filter

  • Actual speed value filter

The "Filter" dialog shows the filter parameters found during the tuning: The settings made here only become active if the new, manual settings from the speed controller are accepted. All filters which are either of the "Low pass (PT2)" or "Second order" type, were calculated for the tuning.

Note

Difference between "Filter details" and "Active filters"

This dialog should not be confused with the currently effective settings in the "Active filters" dialog.

⇒ Further actions:

  • Softkey "Copy filter" to copy a selected filter.

  • Softkey "Insert filter" to overwrite the currently selected filter with the values from the copied filter.

  • Softkey "Diagram >" to start diagram editing (cursor, zoom) for this filter, e.g. "Scaling", "Zoom" and "Display legend".

  • Softkeys "Cancel" and "OK" to cancel or confirm filter settings.
