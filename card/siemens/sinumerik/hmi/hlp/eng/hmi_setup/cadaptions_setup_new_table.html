Adaptations - enter new adaptation

In order to enter a new adaptation table:

  1. Select a mode

  2. Select an input variable: Moment of inertia; position or velocity

  3. Select an output variable: The available output variables are defined according to the input variables.

  4. Enter the size of the adaptation table; at least 2 (= default setting) for a simple adaptation characteristic.

  5. Enter input and output values

Rules for entering adaptation tables:

The input and output values of an adaptation table are only valid and written to the machine data with the "Accept" softkey if the adaptation table fulfills the following criteria:

  • Output values less than 100 % are not permissible for the output variable "acceleration".

  • In a simple adaptation table, one of the two output values is permitted to be zero.

  • In a complex adaptation table, the first and last output value is permitted to be zero.

  • The input values must be entered into a complex adaptation table in ascending order.

  • The increase of output values between two points along the characteristic is limited to a factor of 30. Insert additional intermediate points along the characteristic to achieve an overall factor higher than 30 (Example: Adapting friction compensation with adaptive characteristics).

⇒ Additional actions:

  • Use the "Drive >" softkey to configure a drive adaptation.

  • Press the "Cancel" softkey to reject the adaptation table.

  • Press the "Accept" softkey to save the adaptation table.
