Adding an external encoder

Note

Connecting several external encoders

Proceed step-by-step to connect more than one encoder:

  • Connect an encoder and commission the encoder.

  • In the next step, connect an additional encoder.

Note that it is only possible to commission the encoder if the axes are not in closed-loop control.

Procedure:

  1. Before connecting the encoder, create a commissioning archive with NC and drive data.

  2. After connecting the encoder to a DRIVE-CLiQ interface, drive alarm 201416 is output.

  3. In operating area, select "Commissioning" → Softkey "Drive system" → Softkey "External encoder": The associated Control Unit and the component number of the encoder are automatically determined and entered. A drop-down list is displayed if the external encoder can be assigned to several drives or several axes.

  4. For absolute encoders, the encoder data are automatically determined. Enter the grid spacing and the fine resolution if you are using an incremental encoder. Optionally, for both encoder types, you can invert the position actual value in the drive.

  5. The encoder is commissioned in the following steps: This procedure can take several minutes and is completed with the prompt to Reset (po).

  6. Confirm with "OK". More encoder data can be display with the "Details" softkey.
