Encoder data - settings for experts

After activating the "Expert" softkey you can set additional parameters:

  • Alarm bit position

p435[x].3 = 1: on

p435[x].3 = 0: off

p435 Tens + Ones: Position

p435.[x].2 = 1: high active

p435.[x].2 = 0: low active

  • Error bit position

p434[x].3 = 1: on

p434 [x].3 = 0: off

p434 Tens + Ones: Position

p434.[x].2 = 1: high active

p434.[x].2 = 0: low active

  • Parity bit position

p436[x].3 = 1: on,

p436 [x].3 = 0: off

p436 Tens + Ones: Position

p436.[x].2 = 1: uneven

p436.[x].2 = 0: even

  • Monoflop time

p428[x]

Minimum: 0

Maximum: 65 535

  • Double transmission

p429[x].2 = 0: yes

p429[x].2 = 1: no 

  • Filler bits

    The field is only displayed if double transmission is switched on.

p449[x]

Minimum: 0

Maximum: 63

Checks:

  • If a bit position is assigned multiple times for alarm bit, error bit and parity pit then a message is received that the input data needs to be reviewed.

  • The value of the filler bits is only checked if double transmission is activated.

  • The value range of the filler bits is not checked as this test is contained in the check on the position length and maximum message frame length:

    • Calculation of the position length:

      2x (bits before the position+ position length + bits after the position) + filler bits ≤ 64

    • If the maximum value is exceeded then a message is displayed.

      2x (bits before the position+ position length + bits after the position) + filler bits ≤ 64

  • Bits that provide additional information can only be in the area before or after the position.
