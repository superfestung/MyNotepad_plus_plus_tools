SINAMICS Technology Extensions - Overview

The installed and activated Technology Extensions are displayed in the overview. Table content can be sorted according to the following criteria:

  • Technology packages

  • Activated on the drive object

⇒ Additional actions:

  • Using softkey "Add", select a Technology Extension for installation from a USB storage medium or network drive. After a Reset (po) and the control has run-up, select the drive object for which the Technology Extension should be activated.

  • Using softkey "Remove", delete a Technology Extension under the precondition that the Technology Extension is no longer activated for any drive object.

  • Using softkey "Activate/deactivate >", activate all or selected SINAMICS Technology Extensions.
