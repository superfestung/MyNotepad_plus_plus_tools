Measurement result: Position controller data overview

The result of tuning the position controller is displayed here:

  • "User tuned" column displays the tuned values which the user can change. These values can also be changed in the Bode diagram display via "Diagram" → "Manual parameters". The values are written to the drive parameters by pressing the "Accept" softkey.

  • "Auto tuned" column displays the new tuned values of the automatic servo tuning.

  • "Active" column displays the currently active drive parameters in the drive.

Servo gain factor

MD32200

Feedforward control mode: Torque feedforward control is recommended.

MD32620

  • Equivalent time constant current control loop for feedforward control

    The calculated time constant of the subordinate controller (total time including dead times and current setpoint filter).

MD32800

  • Equivalent time constant, current control loop MD32800

    As the time constants are already taken into account in the SINUMERIK, the value in MD32800 is added to the internal time constants. As a consequence, this value can differ from the line above. This time constant defines the following error. A higher following error is obtained if this is selected too high. If this is selected to be too low, the axis overshoots when positioning.

Speed feedforward control balancing time constant [ms]

This value is effective in addition to the time constants set in MD32800. p1429 only acts on the setpoints of the speed feedforward control, and can be used to finely adapt the speed feedforward control values when torque feedforward control is active.

p1429

Moment of inertia for torque feedforward control [kgm²]

Here, the total moment of inertia of this axis is displayed, converted over to the motor side [kgm²​]. This value is used to calculate the feedforward control torque. The calculated value can be changed.

MD32650

Dynamic Stiffness Control (DSC) active:

DSC is always recommended to improve the position controller dynamic response, therefore DSC is automatically activated. In the drive, the encoder selection for DSC (p1192) and the factor of the encoder adaptation (p1193) are not changed.

MD32640

The current setpoint filters are displayed according to the same criteria.

⇒ Further actions:

  • Softkey "Speed controller" is only active if a complete tuning run has already been performed, or after selecting "Show result"/"Load axis data".

  • Softkey "Bode diagrams" to switch over from the parameter view to the frequency response display.

  • Softkey ">>" to toggle the softkey bar.

    • Softkey "Retune" to recalculate the controller data with modified tuning objective or (to recalculate) with reserved filters. No new measurements have to be performed for this. This operation can be repeated as many times as necessary.

    • Softkey "Return to tuned" to copy values from the "Auto tuned" column to the "User tuned" column if user-tuned changes are not required.

    • Softkey "Return to original" to copy values from the "Active" column to the "User tuned" column and to check the current effective settings on the Bode diagram.

    • Softkey "<<" to toggle the softkey bar.

    • Softkey "Save in file" to create an XML file with the controller data shown here (parameter and Bode diagram) and the previous measurements. The file also contains the data and diagrams of the speed controller. For instance, this file can also be saved to a USB storage medium. Using these methods, the controller data and measurements generated here can be saved and transferred and opened on another SINUMERIK Operate. The saved xml file can also be used for "Offline tuning", as it includes the axis measurements. It is also possible to tune the controller without having a connection to the machine. If you want to load this file to another machine, you have the option of loading the measured values to the NC or to the drive.

    • Softkey "Generate report" to generate an RTF document with tables and Bode diagrams.

  • Softkey "Machine data" to display the list of active and tuned controller data.

  • Softkey "Filter details" to display the Bode diagram of the selected filter and change parameters.

    This softkey is only active if a speed setpoint filter is selected.

  • Softkey "Accept" to write the values from the "User tuned" column to the drive.
