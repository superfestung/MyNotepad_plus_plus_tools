Configuring compensation tables

Procedure:

  1. Press the "Change configuration" softkey to edit the compensation values for a selected axis.

    The "Configuration of the compensation tables" window is opened.

  2. Select the Type of compensation:

    • Simple (EEC) to configure a table for the measuring system error compensation.

    • Bidirectional (CEC) to configure a table for the bidirectional sag compensation compensation.

    Carefully note the preconditions for commissioning.

  3. Enter the following values to create the table: Initial position, End position and Distance between interpolation points.

    Note

    Compensation table for SINUMERIK 828D

    The number of interpolation points for the compensation table is automatically calculated from the Initial position, End position and Distance between interpolation points.

    For SINUMERIK 828D, the maximum number of interpolation points per compensation table for one axis is fixed.

  4. Simple EEC: If more than one measuring system exists, then select that measuring system whose deviation is to be compensated. To activate the compensation table for the selected measuring system, enter the number of interpolation points in MD38000 $MA_MM_ENC_COMP_MAX_POINTS[0,1].

  5. Bidirectional CEC: Select an index for the particular table in the positive and negative directions. Set SD43100[Index] = 1 to activate the compensation tables.

  6. Confirm your entries using the "Activate ✓" softkey. A reset (po) is required for each axis to activate the compensation table. This means that the machine data (MD32700, MD32710) to activate the relevant compensation tables [po] and the compensation [cf] are set.

  7. Using softkey "<< Back", you return to the display of the "Values from the compensation tables".

Diagnostics

Operating area Diagnostics → "Service axis" → Line: "Compensation sag + temperature"
