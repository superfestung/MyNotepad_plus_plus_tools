Cogging torque compensation - part program

A part program is created from the entries for the Learning run.

Procedure:

  1. In order to adapt the parts program, select "Change". An entry can be made in the field when it is orange.

    • Insert a line break with <SHIFT> + <INPUT>.

    • Acknowledge with "OK" to exit the editing.

    • To reject the entry, select "Cancel".

  2. To exit the program editor and start the part program, acknowledge with "OK".

  3. Press the NC Start key to start program execution.
