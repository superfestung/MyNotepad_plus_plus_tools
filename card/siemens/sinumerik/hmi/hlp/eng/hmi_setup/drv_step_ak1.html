Drives: Motor Module overview

In the overview of the SERVO drive object all the data is displayed which was capable of being ascertained with the automatic device configuration.

Further actions:

  • The "Drive +", "Drive -" or "Select drive" softkey selects the drive that you wish to configure next.

  • The "Change" softkey makes subsequent changes and other settings on the displayed drive object, e.g. assign the encoders. For motors with DRIVE-CLiQ all the information is ascertained with the automatic device configuration. Commissioning is not required. Commissioning is required for motors without DRIVE-CLiQ, so-called "third-party motors". The motor data is entered via the "Change" softkey.

  • The "Reference values" softkey is active if the "Reference value overview" is selected with the <PAGE DOWN> key.

  • The "Encoder data" softkey configures assigned encoders subsequently.

  • The "Axis assignment" softkey assigns an axis to a drive.

  • The "Select MDS" softkey is active if more than one MDS has been set up. This can be done via the "Data sets" softkey. Open the drop-down list for direct selection with the "Select MDS ... " softkey. Configure the motor associated with the selected motor data set via the "Change" softkey.
