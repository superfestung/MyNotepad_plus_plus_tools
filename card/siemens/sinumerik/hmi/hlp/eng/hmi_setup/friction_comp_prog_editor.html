Operating the program editor

Program extensions are only possible before or after the cycle call. For each selected axis, a dedicated cycle call is created; these are called and executed one after the other.

Procedure:

  1. In order to adapt the parts program, select "Change". An entry can be made in the field when it is orange.

    • Set a line break with <SHIFT> + <ENTER>.

    • Acknowledge with "OK" to exit the editing.

    • To reject the entry, select "Cancel".

  2. To exit the program editor and start the parts program, acknowledge with "OK".
