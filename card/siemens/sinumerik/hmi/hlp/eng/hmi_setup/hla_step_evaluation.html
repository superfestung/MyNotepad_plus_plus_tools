Hydraulic linear drives: Configuration - calculations

This dialog is only displayed in the change run if previously data was changed in the dialogs.

The values to preassign the parameters are calculated from the valve, cylinder and system data.

You can activate the following default values:

  • Reference values

  • Controlled system gain and natural frequencies

  • Characteristic data

  • Controller values

  • Limit values

The table shows a list of the parameters, which can be influenced by the calculation as a result of the activated checkbox. This list has the same properties as the expert list for the drive parameters. The values cannot be changed.
