Connecting a new component

If you connect a new component (e.g. SMC20) via DRIVE-CLiQ to the drive system, the drive detects the change in the topology and the new component is configured. It is then possible to assign the component to a drive object using the drive wizard.

Notice

Connecting a new component

Only connect components when the drive unit is switched off.
Notice

Deactivate Safety Integrated temporarily

If axes still have Safety Integrated activated, a message is displayed to inform you that Safety Integrated must be deactivated in order to configure the probe.

Following configuration of the probe, Safety Integrated must be activated once again.

Add component

Procedure:

  1. Switch the system on again. During run-up, it will be detected that an older firmware version is present. An update is initiated automatically and the firmware is loaded from the CompactFlash card to this DRIVE-CLiQ component of the drive system.

    Note

    Updating of the firmware could take several minutes. The flashing LED on the component indicates that the process is underway.

  2. The following message is issued in SINUMERIK Operate upon completion of the firmware update:

    Control Unit

    Progress

    Power-up status

    CU_I_3.3:1

     ☐☐☐☐ 100%

    [800] Initialization completed.

  3. Select "Commissioning" → "Drive system" → "Drive unit" → "Topology" in the operating area: New components which have not yet been commissioned are highlighted in red in the table and are designated a component number > 200.

  4. Select the "Add component" softkey and confirm the prompt "Accept component" with "OK".

  5. A power-on reset is required following configuration of the device. Confirm the request with "OK" to add the new component to the actual topology.

Result: The component is included in the topology, although it is not yet assigned to any drive.

Assigning an encoder to a drive

Procedure:

  1. To assign an encoder to a drive, press the menu back key and then the "Drives" softkey.

  2. Select a drive and then press the "Modify" softkey. Navigate to the "Configuration - Encoder assignment" dialog using the "Next step" softkey.

  3. Select the probe for configuration from the list as Encoder 1, Encoder 2, or Encoder 3 and follow the subsequent configuration steps.

  4. Upon completion, close the configuration with "Finish" and confirm the prompt "Save as non-volatile" with "Yes".

Result: The probe is assigned to a drive and may be selected for the drive trace.
