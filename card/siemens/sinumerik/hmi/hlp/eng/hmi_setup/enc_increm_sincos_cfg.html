Encoder data - Encoder type incremental, sin/cos

For an incremental encoder the following settings must be configured:

  • Encoder type

  • Inversion

  • External zero mark

  • Resolution

  • Zero marks

  • Synchronization

Note

"Error messages" softkey

If invalid values or value combinations are selected, then this is indicated by the following responses from the system:

  • The invalid options are marked with a red background and a brief message text is displayed.

  • The "Error messages" softkey is available under which the whole message text of the pending error is displayed.

  • The parameter dialog can only be exited using the "Cancel" softkey. The "OK" softkey for saving the parameter only becomes active if all faults have been eliminated.

Synchronization for linear and rotary encoders

The synchronization operator controls are only displayed for:

  • Incremental encoders with encoder number 1 (motor encoder)

  • Third-party motors

  • Linear and torque motors as of SINAMICS Version 4

The following parameters are set in this section:

Coarse synchronization:

 

  • None

No information is available on the pole position.

(Factory setting)

  • Pole position identification

p1982 [y] = 0: Pole position identification off

p1982 [y] = 1: Pole position identification for the commutation

p1982 [y] = 2: Pole position identification for plausibility check

  • Hall sensors

p404 [x].6 = 0: Hall sensors deactivated

p404 [x].6 = 1: Hall sensors activated

  • C/D track

p404 [x].5 = 0: C/D track deactivated

p404 [x].5 = 1: C/D track activated

  • Pole position plausibility check

A plausibility check of the pole position is performed after every encoder reset.

Fine synchronization:

 

  • None

p404 [x].15 = 0: deactivated

p404 [x].15 = 1: activated

  • Selected zero mark

p430[x].24 = 0: not selected

p430[x].24 = 1: selected
