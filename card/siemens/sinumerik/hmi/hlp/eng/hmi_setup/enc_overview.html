Requirements

Access level: Service or manufacturer

Linear encoders are permissible types (incremental or absolute). The linear scale does not belong to the SINAMICS target topology.

To add an encoder, the machine configuration is checked for the following conditions:

MD30200 $MA_NUM_ENCS < 2

Number of encoders

p0140

Encoder Data Sets (EDS) number

  • The following applies for telegram type 1x6: p0140 ≤ 1

  • The following applies for telegram type 1x8: p0140 ≤ 2

Refer to the following Table for permissible telegrams

Exceptions are: Telegrams 139 and 149 for Weiss spindles equipped with only one encoder.
