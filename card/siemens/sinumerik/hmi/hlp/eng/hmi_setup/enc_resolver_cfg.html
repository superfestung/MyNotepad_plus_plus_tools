Encoder data - Encoder type Resolver

For a rotary encoder with resolvers the following settings must be configured:

  • Encoder type

  • Inversion

  • Resolution (rotary encoder only)

    The number of pole pairs acts on the following parameters:

    p408 [x] Rotary encoder pulse no: Minimum: 1; Maximum: 32

    Note

    Field-oriented control of synchronous motors requires a clear reference between the pole position and encoder position. This reference must also be observed for measuring gears, therefore the ratio of the number of pole pairs to the encoder revolutions must be whole numbers and must be ≥ 1.

    Example: Number of pole pairs 17, measuring gears 4.25 , ratio = 4
