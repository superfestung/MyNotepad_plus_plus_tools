Overview of drive system

In the "Drive system overview" dialog you will see a tabular display of all drive objects that belong to the drive system and are connected via DRIVE-CLiQ connections:

Before the first commissioning only the connected drive units are listed.

  • The automatic device configuration immediately starts for "Factory setting >". The data of the components from the list are read out and displayed.

  • In order to start the automatic device configuration later, press the "Drive unit" softkey.

  • If not all drive components are connected yet, press "Cancel".

After the first commissioning all drive objects that have been found are displayed and all horizontal softkeys are active.

⇒ Further actions:

  • Softkey "Hardware replacement >", to activate a component that has been replaced: This softkey is only visible if components with DQI encoders are being used.

  • Softkey "Activate drives" to open a tabular overview of the drive state. All of the drive objects, which are connected via DRIVE-CLiQ, are displayed.

  • "Factory setting >" softkey, to restore a defined status.

    In order to load the factory settings for an individual component first select the corresponding component:

    • Current drive object: SERVO (Motor Module)

    • Drive unit: CU (integrated in the NCU) or NX

    • Drive system: all drive components.

  • Softkey "Reset (po) >", in order to initiate a power on reset.

  • "Identify via LED" softkey to identify the drive object selected in the control cabinet.
