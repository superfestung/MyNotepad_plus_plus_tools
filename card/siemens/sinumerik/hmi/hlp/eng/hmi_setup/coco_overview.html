Cogging torque compensation - overview

Every axis, for which function module Cogging torque compensation was activated when commissioning, is displayed in the overview:

  • Cogging torque compensation is not activated: -

  • Cogging torque compensation is activated: Number of the DDS.

⇒ Additional actions:

  • Use the "Learning run >" softkey to determine the compensation values for the drive.

  • Use softkey "Activate in DDS" to activate or deactivate the cogging torque compensation in the drive in the selected drive data sets.

  • You use the "Diagnostics >" softkey to check the compensation values after the learning run.
