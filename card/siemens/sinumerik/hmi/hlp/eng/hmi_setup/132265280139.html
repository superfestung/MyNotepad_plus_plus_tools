Spindle tuning sequence

Benefits

The automatic spindle tuning function supports you in the following scenarios:

  • Controller tuning in the various modes: axis mode and spindle mode.

    The speed controller, position controller, and feedforward control are optimized.

  • Calibration of the following errors for tapping by means of path tuning

  • Calibration of the following errors for axis mode by means of path tuning

The progress of spindle tuning is clearly visible to the machine manufacturer:

  • The tuning strategy set for the spindle.

  • The suitable default setting of the tuning strategy for the selected use case.

  • The mode that must still be optimized and the mode that is already optimized.

  • The gear stages that must still be optimized and the ones that are already optimized.

Preconditions

Only one channel is supported: MD20000 $MC_CHAN_NAME

Only one gear stage is supported: MD35090 $MA_NUM_GEAR_STEPS = 1

The gear change is disabled: MD35010 $MA_GEAR_STEP_CHANGE_ENABLE = 0H

Axis mode is only supported for the first gear stage: MD35014 $MA_GEAR_STEP_USED_IN_AXISMODE = 1

Note

Tuning sequence

Only tune the spindle after completing the tuning of all of the other axes.

At least one optimized geometry axis is necessary to optimize the spindle for the tapping.

Spindle tuning sequence

In operating area "Setup" → "Tuning/Test" → "Automatic servo tuning: Axis selection", select the spindle for tuning:

①

Axis selection

The "Automatic servo tuning: Axis selection" dialog shows the current status of the axes/spindles:

  • Select the spindle.

  • Start the spindle tuning with the softkey "Tune >".

②

Parameter

  • Select the spindle type: e.g. milling spindle, rotary table.

  • Select the spindle mode.

When selecting tapping, in the last step the spindle can be optimized for tapping.

③

Spindle tuning

Tuning of the first gear stage of the spindle.

The tuning result for the axis mode is identical to the tuning result of the spindle mode.

④

Tapping

If the spindle is used for tapping, the spindle mode and parameter set 2 are automatically selected. The geometry axis used for tapping also switches to parameter set 2 for the tuning procedure.

The respectively correct parameter set for the spindle and for the geometry axis are selected for path interpolation.

The tuning status of the parameter sets is displayed in the last step of tuning:

not optimized

axis-optimized

axis-optimized (for spindles only)

path-optimized

Tuning strategy

Different tuning strategies are applied, depending on the operating range of the spindle:

  • Spindle for milling machines:

    Tuning aggressiveness = 0.3

    KP is lower and TN is higher than with a linear axis.

  • Spindles for turning machines or grinding machines:

    Tuning aggressiveness = 0.2

    KP is lower and TN is higher than with a milling spindle.

  • The spindle is used as a rotary table.

    Tuning aggressiveness = 0.5

    KP and TN behave identically to a linear axis.

  • If the preset strategy does not meet the requirements, select a different strategy or "Custom strategy".

  • The same tuning strategy that was selected for the path axes is selected for the axis mode.

  • The position controller is always optimized for tapping.

Result

The optimized parameters for the axis are displayed in the "Controller review" dialog.

The optimized parameters of the path interpolation are displayed in the "Interpolation path review" dialog.
