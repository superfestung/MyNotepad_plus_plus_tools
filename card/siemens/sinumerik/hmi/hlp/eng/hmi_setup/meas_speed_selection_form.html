Speed control loop: Selecting the measurement type

The response characteristics for the motor measurement system are analyzed when measuring the speed control loop. Various measurement parameters are available depending on the basic measurement setting that has been selected.

Further actions:

  • Softkey "Axis +"; "Axis -" or "Select axis >" to select the axis that is to be measured.

  • Softkey "Load meas. >" to load the saved measurement data of an axis (xml file or measurement report).

  • Softkey "Clear history" to delete any measurement data that already exists for this axis.

  • Softkey "Measure" to start the measurement in the "Measurement" dialog (with graphic display).

  • Softkey "Measurement parameters >" to switch over from the parameter view to the machine data view – and vice versa.
