Adaptations - Overview

In this dialog, all adaptations are shown in a tabular overview.

The dynamic response and control parameters can be adjusted as functions of the following input variables:

  • Moment of inertia

  • Position

  • Velocity

Inputting an adaptation writes the following machine data:

Mode

MD16501 $MN_CADAPT_MODE

Input axis

MD16504 $MN_CADAPT_INPUT_AX

Input variable

MD16502 $MN_CADAPT_INPUT = 1 (default setting)

Input value

MD16506 $MN_CADAPT_INPUT_VALUE_1

MD16507 $MN_CADAPT_INPUT_VALUE_2

Output axis

MD16505 $MN_CADAPT_OUTPUT_AX

Output variable

MD16503 $MN_CADAPT_OUTPUT

Output value

MD16508 $MN_CADAPT_OUTPUT_VALUE_1

MD16509 $MN_CADAPT_OUTPUT_VALUE_2

⇒ Additional actions:

  • Press the "New >" softkey to enter a new adaptation table.

  • Press the "Change >" softkey to change values or the number of table rows.

  • Press the "Filter >" softkey to filter the overview by individual columns.

  • Press the "Diagnostics" softkey to display the currently effective values.

  • Press the "Delete" softkey to delete the selected adaptation table.
