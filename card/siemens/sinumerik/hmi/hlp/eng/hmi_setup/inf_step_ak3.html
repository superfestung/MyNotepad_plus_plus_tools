Infeed: Reference variable overview

The data on the reference variables and operating mode which was calculated automatically is displayed here in the infeed drive object overview.

The reference frequency is always set to 55 Hz; the infeed determines the actual line frequency internally (theoretical value; see also: Description p2000).

The operating state is displayed: If the line voltage > 415 V, the infeed goes to uncontrolled operation. The infeed to the uncontrolled operation can be switched manually via p3400 Bit0 = 1.

Further actions:

  • With the "Change" softkey you can make other settings on the drive object displayed.
