Entering a new jerk filter - Averaging

Enter the time constant for the smoothing effect for the selected axis.

The following machine data is set active with the "Accept" softkey:

MD32400 $MA_AX_JERK_ENABLE = 1

Axial jerk limitation

MD32410 $MA_AX_JERK_TIME

Axial jerk limitation: Time constant [s]

Use the "Change" softkey to adjust the parameters of the jerk filter.

The filter type is activated in the following axis machine data using a "Reset (po)":

MD32402 $MA_AX_JERK_MODE = x2 in the 1st position setpoint filter ladder

MD32402 $MA_AX_JERK_MODE = 2x in the 2nd position setpoint filter ladder

⇒ Additional actions:

  • Press the "Cancel" softkey to cancel the inputs.

  • Use the "Accept" softkey to accept the entries for the jerk filter.
