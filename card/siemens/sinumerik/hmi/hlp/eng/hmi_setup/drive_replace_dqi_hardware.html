Drive system - replacing the hardware

When a part has to be replaced, for a motor with DQI encoder (type: 1FK7, 1FT7), only the DQI encoder is replaced.

  • "Save DQI data" softkey, to save data from the DQI encoder to the system CompactFlash card (data upload).

  • "Load DQI data" softkey, to load the data from the system CompactFlash Card to the DQI encoder (data download).

The following Control Unit parameters are involved:

  • p4690: Spare part component number

  • p4691: Save/import spare part data

Notice

Backing up data

After successfully deleting or loading, you must switch off and switch on the drive component in order to save data to the non-volatile drive memory.
