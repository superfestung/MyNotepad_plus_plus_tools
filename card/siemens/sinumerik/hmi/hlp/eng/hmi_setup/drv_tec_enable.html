SINAMICS Technology Extensions - Activation

Precondition:

The axis associated with the drive object must not be in closed-loop control. This information is displayed under "Service axis" in the "Diagnostics" operating area.

Select the relevant drive unit and drive object in order to activate or deactivate the selected Technology Extension.

The drive is then automatically restarted in order to activate the parameters.

⇒ Additional actions:

  • Using softkey "Select all", activate the selected TEC on all drive objects of the drive device.

  • Using softkey "Deselect all", deactivate the selected TEC on all drive objects of the drive device.

  • Using softkey "Cancel", you can reject all entries and changes.

  • You can accept all entries and changes using softkey "OK".
