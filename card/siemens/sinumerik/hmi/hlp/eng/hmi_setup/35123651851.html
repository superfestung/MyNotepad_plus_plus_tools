Circularity test: Optimization with the position setpoint filter

The axial jerk limitation MD32400 $MA_AX_JERK_ENABLE is set via a time constant and is always active.

Machine data for setpoint filter:

  • MD32402 $MA_AX_JERK_MODE = Type 2 is recommended, type 1 is preselected as default value for compatibility reasons. Parameterizing a pure band-stop filter is expressly not recommended.

  • MD32402 $MA_AX_JERK_MODE (filter type) and MD32410 $MA_AX_JERK_TIME > 0 is effective only if MD32400 $MA_AX_JERK_ENABLE = 1 is set.
