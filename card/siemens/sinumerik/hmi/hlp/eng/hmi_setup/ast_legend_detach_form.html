Description of the signals:

  • Closed control loop without feedforward control, calculated: The position control loop is calculated based on the measurements of the speed-controlled system and mechanical frequency response including controller parameters and filter.

  • Open control loop: Calculated open position control loop, including speed setpoint filter, scaled with 1/(servo gain factor):

  • Controlled system: Controlled system of the position control loop without speed setpoint filter, scaled with 1/(Kv factor):

  • Mechanical frequency response: Transfer response from the motor encoder up to the load encoder (2nd encoder)

  • Coherence, mechanical frequency response: Shows the coherence in addition to the Bode diagram.

  • Closed speed control loop, calculated:

  • Closed control loop with feedforward control, calculated: The feedforward controller generates a higher bandwidth of the setpoints; as a consequence, generally the amplitude response has higher values. If required, this can be smoothed/filtered using speed setpoint filter 2.

  • n filter: Speed setpoint filter (transfer function of both filters):

  • Disturbance frequency response: Input signal: Torque; output signal: Load position = compliance:

  • Disturbance frequency response torque: Input signal: Torque; output signal: Torque setpoint after the current setpoint filter:

  • Diagrams calculated from "Manual" column: Shows the calculated frequency responses, which were calculated with the values from the "Manual" parameter column. The user can adapt these. The frequency response can also be updated.

  • Active diagrams: Shows the calculated frequency responses which were calculated with the values from the parameter column ("Manual"/"Original"). If tuning has been performed previously with the automatic servo tuning and no parameters have been changed manually, these are the currently effective parameters in the NC and in the drive.

In addition to the graphic display of the Bode diagram, additional details for evaluating the measurement can be displayed.

⇒ Further actions:

  • Softkey "Zoom >" to specifically show areas separately for the X axis and Y axis.

  • Softkey "Represent scaling" to set the value range.

  • Softkey "Cursors >" to use cursor A and cursor B in the diagram as position display.

  • Softkey "Signal configuration" to configure signals that should be displayed.

  • Softkey "Display legend" to hide and display the legends for the displayed signals.

  • Softkey "Manual parameters" to change to the parameter view.

  • Softkey "<< Back" to toggle the softkey bar.
