Position control loop: Selecting the measurement type

This measuring function basically analyzes the response to the active position measuring system. If the function is activated for a spindle without a position measuring system, an alarm is output. Depending on the measured variable selected, various measurement parameter lists are displayed.

The following measuring functions are available for measuring the position control loop:

Measuring type

Measuring variables (signals, input/output)

Reference frequency response

Position setpoint / --

Setpoint step change

Input: Position setpoint

Output:

  • System deviation

  • Following error

  • Actual speed value

Setpoint ramp

Mechanical frequency response

 

⇒ Further actions:

  • Softkey "Axis +"; "Axis -" or "Select axis >" to select the axis that is to be measured.

  • Softkey "Load meas. >" to load the saved measuring data of an axis (xml file or measurement report).

  • Softkey "Clear history" to delete any measurement data that already exists for this axis.

  • Softkey "Measure" to start the measurement in the "Measurement" dialog (with graphic display).

  • Softkey "Measurement parameters >" to change to the parameter view.
