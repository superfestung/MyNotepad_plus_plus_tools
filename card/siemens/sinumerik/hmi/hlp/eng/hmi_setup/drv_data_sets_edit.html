Drives: Modify data set

In order to modify data sets the following requirements must be met:

  • The "Modify data set" softkey works only on EDS encoder data sets.

  • The softkey can only be operated if more than 1 encoder data set is assigned to an encoder.

Procedure:

  1. Select "Modify data set". The encoder data sets are activated.

  2. Select a different encoder data set for the encoder from the list.

  3. Select "OK" to adopt the new parameters into the data sets:

    The progress for this process is provided in the status bar.
