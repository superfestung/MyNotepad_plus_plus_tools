Initialization of array variables (SET, REP)

Function

Array variables that can be written in synchronous actions can also be initialized using the SET and REP commands in synchronous actions.

Example

Value assignment to INTEGER user variables for synchronous actions in the data block SGUD (SYG_IS)

Precondition: MD18661 $MM_NUM_SYNACT_GUD_INT[<x>] ≥ 5

Program code 
PROC MAIN 
N10 DEF INT SYG_IS[4] 
... 
WHEN TRUE DO SYG_IS[0]=REP(1,3); If condition fulfilled: 1st to 3rd INTEGER user variable for synchronous actions in the SGUD block to be set to the value "1".
WHEN TRUE DO SYG_IS[3]=SET(2,4); If condition fulfilled: 4th and 5th INTEGER user variable for synchronous actions in the SGUD block to be set to the value "2" or "4".
...
