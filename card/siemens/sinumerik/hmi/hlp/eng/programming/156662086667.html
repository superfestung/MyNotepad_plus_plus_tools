Commands to modify the translation component

The absolute or additive translation component can be modified using the following commands:

Translatory component

Commands

Absolute (coarse offset)

  • TRANS

  • G58

  • CTRANS

  • CFINE

  • $P_PFRAME[X,TR]

Additive (fine offset)

  • ATRANS

  • G59

  • CTRANS

  • CFINE

  • $P_PFRAME[X,FI]
