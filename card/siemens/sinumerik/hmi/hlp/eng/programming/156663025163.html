Overview of transformation types

Transformations adapt the control to different machine kinematics.

Activating / deactivating

A transformation is activated by a function call in the NC program. Its parameters can be used to define both the orientation of the tool in space and the orientation movements of the rotary axes.

→ Chapter "Deselecting a transformation (TRAFOOF)"

Orientation transformation

In the case of orientation transformations (three, four and five-axis transformations; TRAORI), the programmed position data always refer to the tip of the tool, which is tracked orthogonally to the machining surface in space. The Cartesian coordinates are converted from the basic coordinate system to the machine coordinate system and relate to the geometry axes. These describe the operating point. For the optimum machining of surfaces configured in space in the working area of the machine, machine tools require other axes in addition to the three linear axes X, Y and Z. The additional axes describe the orientation of the tool in space and are called orientation axes in subsequent sections. They are available as axes of rotation on four machine types with different kinematics:

  1. Two-axis swivel head

    e.g. cardanic tool head with a rotary axis parallel to a linear axis with fixed tool table

  2. Two-axis rotary table

    e.g. fixed swivel head with tool table rotating about two axes

  3. Single-axis swivel head and single-axis rotary table

    e.g. a rotatable swivel head with a rotated tool with the tool table rotating about an axis

  4. Two-axis swivel head and single-axis rotary table

    e.g. with a tool table that can rotate about an axis and a rotatable swivel head with a tool rotating about itself

3- and 4-axis transformations are special types of 5-axis transformation and are programmed in the same way as 5-axis transformations.

The functional scope of generic 3/4/5/6-axis transformation is suitable both for transformations for orthogonal rotary axes and for the cardanic milling head and, like all other orientation transformations, can also be activated for these four machine types with TRAORI. In the generic 5/6-axis transformation, the tool orientation has an additional third degree of freedom, in which the tool can be rotated around its own axis in space relative to the tool direction.

→ Chapter "Orientation transformation"

Initial tool orientation setting regardless of kinematics

ORIRESET

If an orientation transformation is active using TRAORI, then ORIRESET can be used to specify the initial settings of up to 3 orientation axes with the optional parameters A, B, C. The programmed parameters are assigned to the rotary axes according to the order of the orientation axes determined by the transformation.

Note

The sequence of the orientation axes depends on the configured machine kinematics. For CA head kinematics, the C axis is the first orientation axis and the A axis the second orientation axis. For AC table kinematics, the A axis is the first orientation axis and the C axis the second orientation axis.

More information:

See Function Manual Transformations for more information on the different machine kinematics.

Programming ORIRESET(A, B, C) results in the orientation axes moving in linear and synchronous motion from their current position to the specified initial setting position.

→ Chapter "Variants of orientation programming and initial setting (ORIRESET)"

Kinematic transformation

In the case of kinematic transformations (TRANSMIT, TRACYL, TRAANG), positions can be programmed in the Cartesian coordinate system. The control system transforms the programmed traversing movements of the Cartesian coordinate system to the traversing movements of the real machine axes.

TRANSMIT and TRACYL

For milling on turning machines, the following machining types can be activated for the agreed transformation:

  • Face machining in turning clamp with TRANSMIT

    → Chapter "Activating face end transformation (TRANSMIT)"

  • Machining of arbitrarily running grooves on cylindrical bodies with TRACYL

    → Chapter "Activate cylinder surface transformation (TRACYL)"

TRAANG

If the option of setting the infeed axis for inclined infeed is required, e.g. for grinding technology, TRAANG can be used to program a configurable angle for the transformation declared.

→ Chapter "Activating an oblique angle transformation with programmable angle (TRAANG)".

→ Chapter "Oblique plunge-cutting on grinding machines (G5, G7)".

Cartesian PTP travel

Kinematic transformation also includes "Cartesian PTP travel", for which up to 8 different articulated joint positions can be programmed. Although the positions are programmed in a Cartesian coordinate system, the movement of the machine occurs in the machine coordinates.

→ Chapter "Cartesian PTP travel".

Concatenated transformation

Two transformations can be switched one after the other. For the second transformation chained here, the motion parts for the axes are taken from the first transformation.

The first transformation can be:

  • Orientation transformation TRAORI

  • Polar transformation TRANSMIT

  • Cylinder transformation TRACYL

  • Inclined axis transformation TRAANG

The second transformation must be Inclined axis TRAANG.

→ Chapter "Activate concatenated transformation (TRACON)".
