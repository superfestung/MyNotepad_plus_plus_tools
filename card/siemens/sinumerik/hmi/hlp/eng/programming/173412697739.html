Changing over the position controller parameter set (SCPARA): More information

More information

Enable

Parameter set changeover must be enabled for the axis:

MD35590 $MA_PARAMSET_CHANGE_ENABLE[<Ax>] = 1 or 2

Changeover response

The changeover response depends on whether the gain factor of the closed loop position control (KV factor) changes between the old and the new parameter set and whether a changeover time is set for interpolation of the position controller gain (MD35592 $MA_PARAMSET_CHANGE_TIME).

Reading the parameter set number

The number of the selected parameter set (specified parameter set) can be read via the system variable $AA_SCPAR.

Lock

A parameter set changeover can also be requested via the NC/PLC interface. To avoid changeover conflicts, the PLC user program can lock the parameter set changeover via SCPARA. While the lock is active, parameter set changeovers requested via SCPARA are rejected without an error message.
