Online tool length compensation (TOFFON, TOFFOF)

With the "Online tool length compensation" function, it is possible to override the effective tool lengths in all three tool directions in real time during machining with active orientation transformation (TRAORI)/kinematic chain or active orientable tool carrier (TCARR).

Predefined procedures TOFFON() and TOFFOF() are used for activation and deactivation.

Already during activation an offset value can be specified for the specified direction of compensation and this is immediately traversed.

Online tool length compensations can be applied via the $AA_TOFF[<n>] system variable. This is possible in all three tool directions of an active orientation transformation/or an active orientable tool carrier. The three geometry axis names X, Y, Z are used as index <n>. The number of active compensation directions must not exceed the number of active geometry axes. All three offset directions can be active at the same time.

Syntax

... 
TOFFON(<CorrDir>[,<Offset>]) 
WHEN TRUE DO $AA_TOFF[<CorrDir>]=<CorrVal> 
... 
TOFFOF(<CorrDir>) 
... 

Meaning

TOFFON

Activate online tool length compensation

<CorrDir>

Correction direction

Tool direction (X, Y, Z), in which the online tool length compensation should be active.

<Offset>

Offset value (optional)

Offset value to be traversed immediately in the specified offset direction when activated.

WHEN TRUE DO ...

Synchronous action that starts as soon as the block is changed in.

$AA_TOFF[<CorrDir>]=<CorrVal>

Apply online tool length compensation

<CorrDir>

Correction direction

Tool direction (X, Y, Z) in which the specified offset value is to be interpolated.

<CorrDir>

Correction value

TOFFOF

Deactivate online tool length compensation

<CorrDir>

Correction direction

Tool direction (X, Y, Z) in which the online tool length compensation should be deactivated.

After deselecting online tool length compensation in the specified compensation direction, the value of the corresponding system variable $AA_TOFF[<n>] or $AA_TOFF_VAL[<n>] is reset to "0". A preprocessing stop is triggered.

Note

Repeated programming of the instruction TOFFON( ) with a new offset value causes the new offset value to be traversed. The offset value is added to the variable $AA_TOFF[<n>] as an absolute value.

Note

Before switching on or switching off a transformation, an overlaid motion must be deactivated out using TOFFOF().

Examples

Example 1 and 2: Online tool length compensation with active orientation transformation (TRAORI)

Program codeComment
$MC_TOFF_MODE=1; The offset value $AA_TOFF[<n>] is to be absolutely calculated and retained beyond reset.
$MC_TOFF_VELO[0]=1000; Velocity for online offset motion in tool direction X, Y, Z
$MC_TOFF_VELO[1]=1000
$MC_TOFF_VELO[2]=1000
$MC_TOFF_ACCEL[0]=1; Acceleration for the online offset motion in tool direction X, Y, Z
$MC_TOFF_ACCEL[1]=1
$MC_TOFF_ACCEL[2]=1
  
N5 DEF REAL XOFFSET 
N10 TRAORI(1); Orientation transformation ON
N20 TOFFON(Z); Activate online tool length compensation in tool direction Z.
N30 WHEN TRUE DO $AA_TOFF[Z]=10 G4 F5; Tool length compensation for tool direction Z: 10 mm
... 
N100 XOFFSET=$AA_TOFF_VAL[X]; Assign the position offset in tool direction X to the variable XOFFSET.
N120 TOFFON(X,-XOFFSET) G4 F5; Deactivate online tool length compensation for tool direction X. ; Return the position offset in tool direction X to 0.

Program codeComment
N10 TRAORI(1); Orientation transformation ON
N20 TOFFON(X); Deactivate online tool length compensation for tool direction X.
N30 WHEN TRUE DO $AA_TOFF[X]=10 G4 F5; Tool length compensation for tool direction X: 10 mm
... 
N80 TOFFOF(X); Deactivate online tool length compensation for tool direction X.; Position offset of tool direction X is deleted: $AA_TOFF[X]=0; No axis is traversed.; The position offset is added to the actual position in the WCS corresponding to the actual orientation.

Example 3: Online tool length compensation with active orientable tool carrier (TCARR)

Program codeComment
... 
DEF REAL XOFFSET 
$MC_TOFF_MODE=1; The offset value $AA_TOFF[<n>] is to be absolutely calculated and retained beyond reset.
$MC_TOFF_VELO[0]=10000; Velocity for online offset motion in tool direction X, Y, Z
$MC_TOFF_VELO[1]=10000
$MC_TOFF_VELO[2]=10000
$MC_TOFF_ACCEL[0]=1 ; Acceleration for the online offset motion in tool direction X, Y, Z
$MC_TOFF_ACCEL[1]=1
$MC_TOFF_ACCEL[2]=1
NEWCONF 
 
; CYCLE800()
TCARR=0; Deactivate tool carrier.
$P_WPFR=CTRANS() 
$P_WPFRAME=$P_WPFR 
G0 X0 Y0 Z0 A0 C0 
M0 
 
; CYCLE800(0,"HEAD",100000,57,0,0,0,30,0,0,0,0,0,-1,100,101)
G17 
$P_WPFR=CTRANS() 
$P_WPFR=CROT(X,30) 
$P_WPFRAME=$P_WPFR 
CUT2DF TCARR=2; Activate tool carrier.
TCOFR 
PAROT 
G0 C=$P_TCANG[3]  A=$P_TCANG[4] 
M0 
  
TOFFON(Z); Activate online tool length compensation for tool direction Z.
WHEN TRUE DO $AA_TOFF[Z]=10; Tool length compensation for tool direction Z: 10 mm
M0 
G4 F5 
M0 
  
WHEN TRUE DO $AA_TOFF[Z]=0; Tool length compensation for tool direction Z: 0 mm
M0 
G4 F5 
M0 
  
TOFFOF(Z); Deactivate online tool length compensation for tool direction Z.
M0 
 
; CYCLE800()
TCARR=0; Deactivate tool carrier.
$P_WPFR=CTRANS() 
$P_WPFRAME=$P_WPFR 
M30
