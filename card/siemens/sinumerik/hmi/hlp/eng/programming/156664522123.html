<_GMODE>

INT

Geometrical mode (evaluation of programmed geometrical data)

Reserved

TENS:

Reserved

Compatibility mode

Normal machining

THOUSANDS:

0 =

Compatibility mode

1 =

Dimensioning via center

2 =

Dimensioning of corner point, pocket position +LENG/+WID

3 =

Dimensioning of corner point, pocket position -LENG/+WID

Dimensioning of corner point, pocket position +LENG/-WID

Dimensioning of corner point, pocket position -LENG/-WID

Complete machining/remachining

Compatibility mode (process <_AP1>, <_AP2> and <_AD> as before)

1 =

Complete machining

2 =

Post machining
