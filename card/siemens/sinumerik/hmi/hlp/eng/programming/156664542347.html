CYCLE61 - Face milling

Syntax

CYCLE61(<_RTP>, <_RFP>, <_SDIS>, <_DP>, <_PA>, <_PO>, <_LENG>, <_WID>, <_MID>, <_MIDA>, <_FALD>, <_FFP1>, <_VARI>, <_LIM>, <_DMODE>, <_AMODE>)

Parameters

No.

Parameter mask

Parameter internal

Meaning

Z0

REAL

Reference point of tool axis, height of blank (abs)

SC

REAL

Safety clearance (to be added to reference point, enter without sign)

Z1

<_DP>

REAL

X0

Y0

REAL

Y1

<_WID>

REAL

Corner point 2 in 2nd axis (abs/inc), see <_AMODE>

9

DZ

<_MID>

REAL

Maximum depth infeed (inc; enter without sign)

10

DXY

<_MIDA>

REAL

UZ

<_FALD>

REAL

Finishing allowance, depth

12

F

<_FFP1>

REAL

Machining feedrate

<_VARI>

INT

Machining type

UNITS:

Machining

1 =

Roughing

2 =

Finishing

TENS:

Machining direction

1 =

parallel to the 1st axis, one direction

2 =

parallel to the 2nd axis, one direction

3 =

parallel to the 1st axis, changing direction

4 =

parallel to the 2nd axis, changing direction

14

 

<_LIM>

INT

Limits

UNITS:

Limit 1st axis minus

0 =

No

1 =

Yes

TENS:

Limit 1st axis plus

0 =

No

1 =

Yes

HUNDREDS:

Limit 2nd axis minus

0 =

No

1 =

Yes

THOUSANDS:

Limit 2nd axis plus

0 =

No

1 =

Yes

15

 

<_DMODE>

INT

Display mode

UNITS:

Machining plane G17/18/19

0 =

Compatibility, the plane effective before the cycle call remains active

1 =

G17 (only active in the cycle)

2 =

G18 (only active in the cycle)

3 =

G19 (only active in the cycle)

16

 

<_AMODE>

INT

Alternative mode

1 =

Incremental

Units for plane infeed (<_MIDA>)

mm

1 =

% of tool diameter

HUNDREDS:

Reserved

THOUSANDS:

Length of surface

0 =

Incremental

1 =

Absolute

TEN THOUSANDS:

Width of surface

Incremental

1 =
