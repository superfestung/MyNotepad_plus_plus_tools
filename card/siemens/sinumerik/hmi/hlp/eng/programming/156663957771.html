Reading and writing of system variables

The system variables of the NC are listed in the "System Variables" Parameter Manual with their respective properties. System variables that can be read or written in the action part of synchronous actions are marked with an "X" in the corresponding line (Read or Write) of the "SA" (synchronous action) column.

Note

System variables used in synchronous actions are implicitly read and written synchronous to the main run.
