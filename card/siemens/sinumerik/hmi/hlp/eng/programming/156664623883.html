CYCLE806 - Interpolation turning

Note

Interpolation turning with CYCLE806 is an option that requires a license.

Syntax

CYCLE806(<_MODE>, <S_XC>, <S_YC>, <_STA>, <_SV1>, <_SV2>, <_SDIR>, <_AMODE>, <_DMODE>, <S_TRA>, <S_XS>, <S_ZS>)
