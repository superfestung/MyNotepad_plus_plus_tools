Synchronization

Differences in the channels occur as a result of different levels of tool wear, different work offsets and different compensations. This results in different tool radius compensations and therefore different tool center point paths with different dynamic responses.

Synchronization is internally realized by slowing down the faster channel so that the channels remain in synchronism with one another within a specific tolerance and do not asynchronously drift apart. In addition to the differences already listed, the tolerance that can be maintained mainly depends on the dynamic response values of the channels and/or axes and the part program.

Note

Tolerances

As a result of differences in the channels (work offsets, tool wear, compensations), different tolerances are required for synchronization.

Constraints

A combination with the following functions is not permissible:

  • Transformations

  • Swivel data sets and swiveling

  • Online tool offset

  • Overlaid motion

  • Couplings

  • Intelligent Load Control (ILC)

  • Intelligent dynamic response adaptation (IDC)

  • Balanced spindle acceleration

  • Measuring functions

  • External collision avoidance

  • Rapid traverse with active tool radius compensation

  • Variable programming in the part program
