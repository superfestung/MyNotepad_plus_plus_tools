Reading of the active maximum acceleration

The currently active maximum acceleration value of an axis in single axis interpolation can be read with the system variable $AA_ACC[<Ax>]. To ensure that the acceleration correction active for the axis is considered, it must be read in the correct context:

Acceleration override

Reading of $AA_ACC[<Ax>]

Content of $AA_ACC[<Ax>]

Result

Active in the NC program

In the NC program

Corrected maximum acceleration value

+

In synchronized action

Maximum acceleration value according to parameterization

-

Active in synchronized action

In the NC program

Maximum acceleration value according to parameterization

-

In synchronized action

Corrected maximum acceleration value

+

Example:

Program codeComment
... 
N100 ACC[X]=1 
N110 STOPRE 
N120 R0=$AA_ACC[X] ; R0 shows the corrected maximum acceleration value (1% of the parameterized maximum acceleration), which is currently active in the NC program for movements of the X axis
N130 DO $R1=$AA_ACC[X]; R1 shows the unchanged maximum acceleration value that is currently active in synchronized actions for movements of the X axis.
