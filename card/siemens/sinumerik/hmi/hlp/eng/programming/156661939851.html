Example: Application of the second gear-stage data block

The switching thresholds of the second gear-stage data set for the maximum and minimum speed are evaluated for G331/G332 and when programming an S value for the active leading spindle. Automatic M40 gear-stage change must be active. The gear stage as determined in the manner described above is compared with the active gear stage. If they are found to be different, then the gearbox stage is changed.

Program codeComment
N05 M40 S500; Programmed spindle speed: 500 rpm
... 
N50 G331 S800; Leading spindle: Gearbox stage 2 is selected
N55 SPOS=0; Position the spindle
N60 G331 Z-10 K5; Tapping; Spindle acceleration from second gearbox stage data block 2
