Programmable frames (overview)

Frames are programmed using commands of G function group 3. These commands, subsequently called frame instructions, are either active in absolute terms or additive:

Absolute frame instruction

 

Additive frame instruction

  • Deletes all previously programmed frames.

 

  • Attached to existing frames.

  • Reference point is the zero point of the current workpiece coordinate system set using G54 ... G57, G505 ... G599 (settable zero point system, SZS).

 

  • The reference point is the zero point of the currently set workpiece coordinate system or last programmed using a frame instruction.

Example: TRANS

 

Example: ATRANS

    

Additional properties

  • Frame instructions must be programmed in a separate NC block.

  • Frame instructions can be used individually or combined as required.

  • Frame instructions are executed in the programmed sequence.

  • Additive instructions are frequently used in subprograms. The basic instructions defined in the main program are not lost after the end of the subprogram if the subprogram has been programmed with the SAVE attribute.

Meaning

TRANS/ATRANS

Workpiece coordinate system offset in the direction of the specified geometry axis or axes

More information:

→ Chapter "Programmable work offset (TRANS, ATRANS)"

ROT/AROT

WCS rotation by:

  • linking individual rotations around the specified geometry axis or axes

    or

  • rotation through angle RPL=... in the current machining plane (G17/G18/G19)

More information:

→ Chapter "Programmable rotation (ROT, AROT, RPL)"

ROTS/AROTS:

Workpiece coordinate system rotation by means of the specification of solid angles

The orientation of a plane in space is defined unambiguously by specifying two solid angles. Therefore, up to two solid angles may be programmed:

ROTS/AROTS X... Y... / Z... X... / Y... Z...

More information:

→ Chapter "Programmable rotation with solid angles (ROTS, AROTS, CROTS)"

CROTS

CROTS works in the same way as ROTS but refers to the valid frame in the database.

More information:

→ Chapter "Programmable rotation with solid angles (ROTS, AROTS, CROTS)"

SCALE/ASCALE

Scaling in the direction of the specified geometry axis or axes to increase/reduce the size of a contour

More information:

→ Chapter "Programmable scaling (SCALE, ASCALE)"

MIRROR/AMIRROR

Workpiece coordinate system mirroring by means of mirroring (direction change) the specified geometry axis

More information:

→ Chapter "Programmable mirroring (MIRROR, AMIRROR)"
