Changing over the position controller parameter set (SCPARA)

With the NC command SCPARA, it is possible to change over the parameter set used for the closed loop position control of an axis/spindle from the NC program or a synchronized action in order to adapt the closed loop position control quickly to changed properties of the machine during operation.

Syntax

SCPARA[<Ax>]=<n>

Meaning

SCPARA

Changing over the position controller parameter set

<Ax>

Axis identifier (channel axis)

Type:

AXIS

<n>

Parameter set number

Type:

INT

Range of values:

1 ... 6

Example

Program codeComment
...
N110 SCPARA[X]=3; Changing over to the 3rd position controller parameter set for axis X.
...
