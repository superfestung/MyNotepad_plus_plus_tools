Behavior for tools with relevant cutting edge position

For tools with the relevant cutting edge position (turning and grinding tools, tool types 400 - 599), then the tool length always refers to the cutting edge reference point. The necessary offsets when activating tool radius compensation are completely executed in preprocessing.

Intersection calculation

The straight line between the tool edge center points at the block start and block end is used to calculate intersection points with the approach and retraction block. The difference between the tool edge reference point and the tool edge center point is superimposed on this movement. For approach and/or retraction with KONT, the movement is superimposed in the linear subblock of the approach or retraction movement. Therefore, the geometrical conditions for tools with or without relevant cutting edge position are identical.

Constraints

Tool change

In circle blocks and in motion blocks containing rational polynomials with a denominator degree > 4, it is not permitted to change a tool with active tool radius compensation in cases where the distance between the tool edge center point and the tool edge reference point changes. With other types of interpolation, contrary to the previous state, it is now possible to change when a transformation is active (e.g. TRANSMIT).

3D circumferential milling

For tool radius compensation with variable tool orientation, the transformation from the tool edge reference point to the tool edge center point can no longer be performed using a simple work offset. Tools with a relevant tool cutting edge are therefore not permitted for 3D peripheral milling (alarm)

Note

The subject is irrelevant with respect to face milling as only defined tool types without relevant cutting edge position are permitted for this operation.

Behavior for tools without permitted tool type

Tools with a type, which has not been explicitly approved, are treated as a ball end mill with the specified radius. The specification of a cutting edge position is ignored.
