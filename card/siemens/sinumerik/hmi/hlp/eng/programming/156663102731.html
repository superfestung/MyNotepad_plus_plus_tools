Programming of the orientation axes (ORIAXES, ORIAXESFR, ORIVECT, ORIANGLE, ORIEULER, ORIRPY, ORIRPY2, ORIVIRT1, ORIVIRT2)

The "Orientation axes" function describes the orientation of the tool in space and is achieved by programming the offset for the rotary axes. An additional, third degree of freedom can be achieved by also rotating the tool about itself. In this case, the tool is oriented in space via a third rotary axis for which 6-axis transformation is required. The rotation of the tool about itself is defined using the THETA angle of rotation in accordance with the type of interpolation of the rotation vectors (see "Rotations of the tool orientation (ORIROTA, ORIROTR, ORIROTT, ORIROTC, THETA)").

The axis identifiers A2, B2 and C2 are used to program the orientation axes.

Syntax

N... ORIAXES/ORIAXESFR/ORIVECT; Linear or large-circle interpolation
N... G1 X Y Z A B C 
  
N... ORIPLANE; Orientation interpolation of the plane; See note below
  
N... ORIMKSN... ORIWKS; Rotation in the machine coordinate system; Rotation in the workpiece coordinate system; See note below
  
N... ORIANGLE; Orientation interpolation in virtual axis angles
  
N... ORIEULER/ORIRPY/ORIRPY2: Orientation angle Euler/RPY angle
N... G1 X Y Z A2= B2= C2=; Angle programming of virtual axes
  
N... ORIVIRT1/ORIVIRT2; Virtual orientation axes def. 1/2
N... G1 X Y Z A3= B3= C3=; Direction vector programming

Note

ORIPLANE

Other rotary axis offsets of the orientation axes can be programmed for orientation changes along the peripheral surface of a taper in space (see "Orientation programming along the peripheral surface of a taper (ORIPLANE, ORICONCW, ORICONCCW, ORICONTO, ORICONIO)").

ORIWKS / ORIMKS

See "Reference of the orientation axes (ORIWKS, ORIMKS):"

Meaning

ORIAXES

Linear interpolation of the orientation axes via the shortest path

ORIAXESFR

Linear interpolation of the orientation axes to the programmed positions without consideration of the shortest path

ORIVECT

Large-circle interpolation (identical to ORIPLANE)

A= B= C=

Programming the machine axis position

ORIANGLE

Orientation programming in virtual axis angles

ORIEULER

Orientation programming via Euler angle

ORIRPY

Orientation programming via RPY angle

The rotation sequence is XYZ and:

  • A2 is the angle of rotation around X

  • B2 is the angle of rotation around Y

  • C2 is the angle of rotation around Z

ORIRPY2

Orientation programming via RPY angle

The rotation sequence is ZYX and:

  • C2 is the angle of rotation around Z

  • B2 is the angle of rotation around Y

  • A2 is the angle of rotation around X

A2= B2= C2=

Angle programming of virtual axes

ORIVIRT1/ORIVIRT2

Orientation programming using virtual orientation axes

Definition 1:
Definition according to MD21120 $MC_ORIAX_TURN_TAB_1

Definition 2:
Definition according to MD21130 $MC_ORIAX_TURN_TAB_2

A3= B3= C3=

Direction vector programming of direction axis
