Not for motor type: -Scaling: -Expert list: 1
Min Max Factory setting
0 2 0
Description: Sets the processing mode for PROFIdrive STW1.10 "master control by PLC".
Generally, control world 1 is received with the first receive word (PZD1) (this is in conformance to the PROFIdrive profile). The behavior of STW1.10 = 0 corresponds to that of the PROFIdrive profile. For other applications that deviate from this, the behavior can be adapted using this particular parameter.
