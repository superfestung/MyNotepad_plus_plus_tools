Block T/TD: Tool data: General data

In addition to the tool offset data other tool characteristics are stored for managing the tools. The module TD contains the general data of the tools. The tool characteristics can be addressed via individual multiple-line variables. The variable line index corresponds to the T-number. If non-existent T-numbers are accessed, the request is acknowledged negatively. The module Tool directory (TV) in the associated T area shows which T-numbers are valid.

toolMon Type of tool monitoring $TC_TP9
Column index:
9 		Line index:
Tool number
Tool number 		Max. line index:
SLMAXTOOLNUMBER
If a monitoring mode is activated for the tool, the current monitoring parameters are evaluated and, if necessary, the tool status is set to "disabled" or "prewarning limit reached".
An existing tool disable is not cancelled.
This variable is bit-coded to activate various monitoring modes simultaneously.

Value Range:
o 0: no tool monitoring
o 1: tool life monitoring
o 2: no. of workpieces monitoring
o 4: monitoring of edge wear parameters using wear limit
o 8: monitoring of total offset parameters using wear limit
Type
TYPE_UWORD 		Unit

0 		Minimum Value
 Maximum Value
Access Path HMI: /Tool/Data/toolMon[u<Tool Area>, <Line Index>]
