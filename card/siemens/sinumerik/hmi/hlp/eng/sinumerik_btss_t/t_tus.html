Block T/TUS: Tool data: user monitoring data

This block contains the user monitoring data of the tool data
 
userData OEM parameters for cutting edge monitoring $TC_MOPCx[y,z] x=ParamNo,y=T-Number,z=Edge
Column index:
Tool number 		Line index:
Calculate value
Number of the user-defined parameter + (number of the cutting edge -1) * numCuttEdgeParams_tus		 Max. line index:
numCuttEdgeParams_tus * maxnumCuttEdges_Tool
The variable returns the OEM-defined parameters for cutting edge monitoring.
The data are made available by the MD settings MD bits $MN_MM_TOOL_MANAGEMENT_MASK, bit1=1 (TMMO) and bit2=1 and MD18098 $MN_MM_NUM_CC_MON_PARAM>0.
The data type of the individual parameters is specified by the MD18099 $MN_MM_TYPE_CC_MON_PARAM, and can be determined from the file _N_COMPLETE_TOA_ACX.
The transfer in the OPI is always as TYPE_DOUBLE.
