Block T/TOET: Edge-related coarse total offsets, transformed setup offsets

One set of edge-related transformed total offsets exists for each tool edge and operating location.
This module corresponds totally to module T / TOE.
 
edgeECData Transformed location-dep. setup offsets "coarse"
Column index:
Tool number 		Line index:
Calculate value
((EdgeNo-1) * (maxnumEdgeSC * numParams_SC)) + ((EdgeSC - 1)* numParams_SC) + ParameterNo 		Max. line index:
numParams_SC * maxnumEdgeSC * maxnumCuttEdges_Tool
Transformed, location-dependent setup offsets (coarse) are entered before machining, and taken into account in the total offset (fine). The location-dependent offsets relate to the geometric data of a cutting edge.
DL: Location-dependent offsets related to the particular operating location.
The function is activated by machine data MD18108 $MN_MM_NUM_SUMCORR, MD 10110 $MN_MM_MAX_SUMCORR_PER_CUTTEDGE, MD 18112 $MN_MM_KIND_OF_SUMCORR, bit4 and MD 18080 $MN_MM_TOOL_MANAGEMENT_MASK, bit8.
The individual setup offset parameter sets are differentiated by the number ranges of system variable $TC_ECP:
o x = 13-21, activatable with DL=1
o x = 23-31, activatable with DL=2
o x = 33-41, activatable with DL=3
o x = 43-51, activatable with DL=4
o x = 53-61, activatable with DL=5
o x = 63-71, activatable with DL=6
The setup offsets are activated in addition to the total offsets if MD18112 $MN_MM_KIND_OF_SUMCORR, bit4=1 is set.
