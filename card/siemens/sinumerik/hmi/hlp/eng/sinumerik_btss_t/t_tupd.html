Block T/TUPD: Tool data: user magatine place data

This block contains the user magazine location data (double)
 
userPlaceDataDouble Magazine place user data $TC_MPPCx[y,z] x=ParamNo y=MagazineNo z=MagPlaceNo
Column index:
Magazine number 		Line index:
Calculate value
Number of the user-defined parameter + numMagLocParams_u * (number of the magazine location - 1) 		Max. line index:
numMagLocParams_u * magNrPlaces
The variable returns the OEM-defined parameters for the magazine locations for all permissible data types, except the data type "String".
The OEM data are made available with the MD settings MD bits $MN_MM_TOOL_MANAGEMENT_MASK, bit0=1, bit2=1, and MD18092 $MN_MM_NUM_CC_MAGLOC_PARAM>0.
The data type of the individual parameters is specified by MD18093 $MN_MM_TYPE_CC_MAGLOC_PARAM, and can be determined from the file _N_COMPLETE_TOA_ACX.
The transfer in the OPI is always as TYPE_DOUBLE.
If there is read or write access to an OEM-defined magazine location parameter of the data type "String", an error string is returned.
