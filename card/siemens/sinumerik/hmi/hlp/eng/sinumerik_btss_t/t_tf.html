Block T/TF: Parametrizing, return parameters of _N_TMGETT, _N_TSEARC

This module is used for parameterizing as well as for the return parameters of PI services _N_TMGETT and _N_TSEARC. Access to this module must be T area specific and exclusive. It is up to the clients to guarantee this by using the semaphore mechanism (PI service _N_MMCSEM) with the function number for _N_TMSEARCH.
With _N_TMGETT, NO parameterizing elements (input parameters) are relevant; the only relevant one is the result parameter "resultToolNr".

resultCuttingEdgeNrUsed D numbers of cutting edges since last count $A_USEDD
Column index:
5 		Line index:
Calculate value
((i-th tool carrier-1) * line2 of column3 (resultNrOfCutEdgesUsed)) + consecutive number of the cutting edge used 		Max. line index:
Zeile 1 * Zeile 2 von resultNrOfCutEdgesUsed
D numbers of the cutting edges used since the last workpiece count, that have been used on the tool holder previously defined by resultNrOfCutEdgesUsed. Various D offsets for a tool mean that there are multiple entries of the tool. Consequently, a T number can be present more than once.
The variables resultToolNrUsed and resultCuttingEdgeNrUsed form a pair with the same line index consisting of the internal tool number and the cutting edge number for a tool used since the last cutting edge count.

Value Range:
o >0: D number (can also be contained multiple times if D offsets of various tools were in use).
o 0: No more cutting edges have been used since the last workpiece count.
o -2: The addressed i-th tool holder is greater than the MD 18075 $MN_MM_NUM_TOOLHOLDERS
o If tool monitoring (TMMO) is not active, an error code is returned for a missing variable.
Type
TYPE_DWORD 		Unit
No unit 		Default Value
0 		Minimum Value
0 		Maximum Value
32000
Access Path HMI: /Tool/Find/resultCuttingEdgeNrUsed[u<Tool Area>, <Line Index>]
 
resultNrOfCutEdgesUsed Number of cutting edges since last workpiece count $A_USEDND
Column index:
3 		Line index:
Index see descr.
Meaning of the index: See description 		Max. line index:
2*max.Anz. der Distanzbez. zw.Mag. und WZ-Haltern + 2 = 66
Number of different cutting edges used on the tool holder "Tool holder" since the last count, including the cutting edge currently in use. Each tool used is included at least once.
o Line 1: Number of possible tool holders
o Line 2: Number of possible cutting edges.
o Line i+1: Number of the i-th tool holder
o Line i+3: Number of cutting edges used on i-th tool holder since the last count. This corresponds to $A_USEDND.
The various lines are used to filter the cutting edges displayed.

Value Range:
o For line 2: -2: TMMO is not active
o For line (i*2)+2: 0: No more cutting edges have been used since the last count.
o For line (i*2)+2: -2: TMMO is not active
