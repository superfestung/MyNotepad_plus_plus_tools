Block T/TDC: Tool parameters of the Siemens application

Tool parameters of the Siemens application
 
toolDataChangeInfo Change service for tool data
Column index:
Number of the ring buffer entry 		Line index:
TDC parameter
TDC parameter: Line 1-7 (see description) 		Max. line index:
7
The service works for Siemens applications. The change service outputs changes to tool data. The ring buffer of the tool data change data is organized in columns.
Each column contains a change notice
The size of the ring buffer is defined in MD28450 $MC_ MM_TOOL_DATA_CHG_BUFF_SIZE (= max. column index)

Value Range:
o Line 1: Column of the last entry in the ring buffer
o Line 2: Number of overruns of the ring buffer
o Line 3: Block number of the changed date
o Line 4: Parameter number of the changed date
o Line 5: 1st index of the parameter of the changed date
o Line 6: 2nd index of the parameter of the changed date
o Line 7: 3rd index of the parameter of the changed date (Only with "ECP", "SCP", distance relationship and new CE number of the cutting edge)
Type
TYPE_UWORD 		Unit
No unit 		Default Value
 Minimum Value
 Maximum Value
