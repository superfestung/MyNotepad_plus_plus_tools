Block T/TOST: Edge-related location-dependent fine total offsets, transformed

One set of edge-related transformed total offsets exists for each tool edge and operating location.
This module corresponds totally to module T / TOS.
 
edgeSCData Transformed total offset - fine
Column index:
Tool number 		Line index:

Transformed total offsets (fine) - location-dependent offsets comprise the error values that make up the deviation of the workpiece from the setpoint dimension.
The parameters of the location-dependent offsets relate to the geometric data of a cutting edge. DL: Location-dependent offsets related to the particular operating location.
The function is activated by machine data MD18108 $MN_MM_NUM_SUMCORR, MD 10110 $MN_MM_MAX_SUMCORR_PER_CUTTEDGE and MD 18080 $MN_MM_TOOL_MANAGEMENT_MASK, bit8.
The individual total offset parameter sets are differentiated by the number ranges of
system variable $TC_SCP:
o x = 13-21, activatable with DL=1
o x = 23-31, activatable with DL=2
o x = 33-41, activatable with DL=3
o x = 43-51, activatable with DL=4
o x = 53-61, activatable with DL=5
o x = 63-71, activatable with DL=6
