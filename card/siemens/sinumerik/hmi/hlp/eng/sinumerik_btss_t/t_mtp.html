Block T/MTP: Multitool data, place data

This block contains the location data of the multitool data.
 
mtPlaceData Multitool place data diverse, siehe Variablenbeschreibung
Column index:
Multitool number 		Line index:
Multitool place data
(MtLocNo-1) * numMultiToolPlaceParams + ParamNo 		Max. line index:
numMultiToolPlaceParams * maxNumPlacesPerMultitool
Defines the multitool location data and location distances:
o P1: Location distance length ($TC_MTPPL)
o P2: Location distance angle ($TC_MTPPA)
o P3: Location type (read access only) ($TC_MTPP2)
o P4: Location status (bit array) ($TC_MTPP4)
Bit 0=1: Disabled
Bit 0=0: Disabled
Bit 1=1: Free to hold a tool
Bit 1=0: Occupied
o P5: Tool number T of the tool at this location ($TC_MTPP6)
o P6: Number of the adapter at this location ($TC_MTPP7)
Type
TYPE_DOUBLE 		Unit

 Minimum Value
 Maximum Value
