Block T/MTV: Multitool data: directory

The MTV data block can be used for the following purposes:

1. Display of all multitools. The most important multitool information is collected in the MTV block. Existing multitools are sorted without gaps by ascending multitool number (=tool number). This means that variables defined as 1-dimensional arrays in this block contain all multitool information without gaps. The line index used to address a specific array has no connection with the multitool number, but is simply a sequential number. Inserting/deleting multitools dynamically changes the contents of a line.

2. Targeted access to the multitool data in the MTD, MTP, MTUD, MTAD, MTUP and MTAP blocks. Which multitools are actually defined should be determined from the MTV block before accessing an element in the stated blocks.

multitoolKindOfDist Type of distance coding for multitool locations
Column index:
6 		Line index:
Multitool number
Serial Multitool number 		Max. line index:
numMultiTools
The location distances (KindOfDistance) within the multitool can be defined in various ways:
o 1 = Distance coding with multitool location number
o 2 = Distance coding with length specifications
o 3 = Distance coding with angle specifications
The distance coding can only be changed for as long as no MT locations have been generated.

Value Range:
Value range:
o 0: No multitool, or magazine management (TMMG) is not active
o 1: Multitool with location coding
o 2: Multitool with length coding
o 3: Multitool with angle coding
