Block T/TAMD: Application-specific magazine data (double)

The TAMD block contains application-specific information about the available tool magazines.
The application-specific magazine data are reserved for SIEMENS applications.
 
siemDataDouble Siemens application magazine data $TC_MAPCSx[y]

numMagParams_tam
Magazine parameters for Siemens applications for all permissible data types, except the data type "String".
The Siemens application data are made available with the MD settings MD bits MD18080 $MN_MM_TOOL_MANAGEMENT_MASK, bit0=1, bit2=1 and MD18200 $MN_MM_NUM_CCS_MAGAZINE_PARAM>0.
The data type of the individual parameters is specified by MD18201 $MN_MM_TYPE_CCS_MAGAZINE_PARAM, and can be determined from the file _N_COMPLETE_TOA_ACX.
The transfer in the OPI is always as TYPE_DOUBLE.
If there is a read or write access to a magazine parameter of data type "String" for Siemens applications, an error string is returned.
