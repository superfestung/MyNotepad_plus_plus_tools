Block T/TOE: Edge-related coarse total offsets, setup offsets

One set of edge-related coarse total offsets, setup offsets, exists for each tool edge and operating location.
This module corresponds totally to module T / TOS, edge-related location-dependent fine total offsets.
 
edgeECData Location-dependent setup offsets "coarse" $TC_ECPx[t,d]
Column index:
Tool number 		Line index:
Calculate value
((EdgeNo-1) * (maxnumEdgeSC * numParams_SC)) + ((EdgeSC - 1)* numParams_SC) + ParameterNo 		Max. line index:
numParams_SC * maxnumEdgeSC * maxnumCuttEdges_Tool
Location-dependent (coarse) setup offsets are entered before machining, and taken into account in the total offset (fine). The location-dependent offsets relate to the geometric data of a cutting edge.
DL: Location-dependent offsets related to the particular operating location.
The function is activated by machine data MD18108 $MN_MM_NUM_SUMCORR, MD10110 $MN_MM_MAX_SUMCORR_PER_CUTTEDGE, MD18112 $MN_MM_KIND_OF_SUMCORR, bit4 and MD18080 $MN_MM_TOOL_MANAGEMENT_MASK, bit8.
The individual setup offset parameter sets are differentiated by the number ranges of system variable $TC_ECP:
o $TC_ECP13-$TC_ECP21, $TC_ECP1_RADIUS3, $TC_ECP1_LEN6 activatable with DL=1
o $TC_ECP23-$TC_ECP31, $TC_ECP2_RADIUS3, $TC_ECP2_LEN6 activatable with DL=2
o $TC_ECP33-$TC_ECP41, $TC_ECP3_RADIUS3, $TC_ECP3_LEN6 activatable with DL=3
o $TC_ECP43-$TC_ECP51, $TC_ECP4_RADIUS3, $TC_ECP4_LEN6 activatable with DL=4
o $TC_ECP53-$TC_ECP61, $TC_ECP5_RADIUS3, $TC_ECP5_LEN6 activatable with DL=5
o $TC_ECP63-$TC_ECP71, $TC_ECP6_RADIUS3, $TC_ECP6_LEN6 activatable with DL=6
The setup offsets are activated in addition to the total offsets if MD18112 $MN_MM_KIND_OF_SUMCORR, bit4=1 is set.
