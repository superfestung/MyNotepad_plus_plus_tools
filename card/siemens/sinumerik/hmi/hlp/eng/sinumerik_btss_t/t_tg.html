Block T/TG: Tool data: Grinding-specific data

Special tool data are required for grinding tools. These data are contained in the module TG. They can be addressed via several multiple-row variables. The row index corresponds to the T number. If a non-existent T-number is addressed negative acknowledgment is returned. The module tool directory (TV) in the associated area T shows which T-numbers are valid.

conntectPar Chaining rule for grinding parameters $TC_TPG2
Column index:
2 		Line index:
Tool number
Tool number 		Max. line index:
SLMAXTOOLNUMBER
The variable specifies which tool parameters of cutting edge 2 (D2) and cutting edge 1 (D1) are to be chained.
If the value of a chained parameter is changed, the same parameters of the other cutting edge are automatically overwritten with the new value.

Value Range:
If the following bits are set, the corresponding parameters of D1 and D2 are chained:
Bit0: tool type
Bit2: geometry length1
Bit3: geometry length2
Bit4: geometry length3
Bit11: wear length1
Bit12: wear length2
Bit13: wear length3
Bit20: base dimension/adapter dimension length1
Bit21: base dimension/adapter dimension length2
Bit22: base dimension/adapter dimension length3
The value is stored internally as an integer.
Type
TYPE_DOUBLE 		Unit
No unit 		Default Value
 Minimum Value
 Maximum Value
Access Path HMI: /Tool/GrindingData/conntectPar[u<Tool Area>, <Line Index>]

paramNrCCV Compensation parameters for SUG $TC_TPG9
Column index:
9 		Line index:
Tool number
Tool number 		Max. line index:
SLMAXTOOLNUMBER
Compensation parameters for the function SUG ("constant perimeter speed of grinding wheel"). These parameters define which compensation value is to be used for SUG, tool monitoring and centerless grinding. The value always refers to cutting edge D1.

Value Range:
Value range:
o 3: Length 1
o 4: Length 2
o 5: Length 3
o 6: Radius
The length consists of geometry + wear + basis, depending on the tool type.
Type
TYPE_DOUBLE 		Unit
No unit 		Default Value
 Minimum Value
 Maximum Value
Access Path HMI: /Tool/GrindingData/paramNrCCV[u<Tool Area>, <Line Index>]
