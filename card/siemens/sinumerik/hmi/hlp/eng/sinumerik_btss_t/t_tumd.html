Block T/TUMD: Tool data: user magazine data

This block contains the user magazine data (double)
 
userDataDouble OEM-defined magazine parameters $TC_MAPCx[y] x = ParameterNo y = MagazineNo
Column index:
Magazine number 		Line index:
Parameter number
Number of the user-defined parameter 		Max. line index:
numMagParams_u
The variable returns the OEM-defined parameters for the magazines for all permissible data types, except the data type "String".
The OEM data are made available with the MD settings MD bits $MN_MM_TOOL_MANAGEMENT_MASK, bit0=1, bit2=1 and MD18090 $MN_MM_NUM_CC_MAGAZINE_PARAM>0.
The data type of the individual parameters is specified by MD18091 $MN_MM_TYPE_CC_MAGAZINE_PARAM, and can be determined from the file _N_COMPLETE_TOA_ACX.
The transfer in the OPI is always as TYPE_DOUBLE.
If there is read or write access to an OEM-defined magazine parameter of the data type "String", an error string is returned.
