Block T/MTUP: Multitool place user data

This block contains the user multitool location data.
 
userPlaceData OEM-defined multitool location parameters $TC_MTPPCx[y,z] x=ParamNo y=MtNo z=MtPlaceNo
Column index:
Multitool number 		Line index:
Calculate value
(MtLocNo-1)*numMultiToolPlaceParams_mtup+ParamNo 		Max. line index:
numMultiToolPlaceParams_mtup * maxNumPlacesPerMultitool
OEM-defined data for the multitool locations for all permissible data types, except the data type "String".
The OEM data are made available with the MD settings MD bits MD18080 $MN_MM_TOOL_MANAGEMENT_MASK, bit0=1, bit2=1, bit10=1 and MD18194 $MN_MM_NUM_CC_MTLOC_PARAM>0.
The data type of the individual parameters is specified by MD18195 $MN_MM_TYPE_CC_MTLOC_PARAM, and can be determined from the file _N_COMPLETE_TOA_ACX.
The transfer in the OPI is always as TYPE_DOUBLE.
