multitoolMyPlace Owner magazine location of the tool
Column index:
6 		Line index:
Multitool number
Multitool number 		Max. line index:
32000
The variable returns the magazine location from which the tool was loaded (owner magazine location).
A magazine location becomes the owner magazine location if the tool is loaded on a location of magazine location type 1 ($TC_MPP1=1).
For fixed-location coded tools in buffers, the number states the location onto which the tool is loaded back into the magazine.

Value Range:
o 0: Number of the owner magazine location
o 0: Meaning depends on further variables:
o $A_TOOLMLN >0: Manual tool
o $A_TOOLMLN=0: Error is output, e.g. tool is not loaded

 Minimum Value

multitoolStateL Multitool status $TC_MTP8
Column index:
7 		Line index:
Multitool number
Multitool number 		Max. line index:
32000
The status of a multitool is defined by bits 0 - 17.
Detailed information can be found in the "Tool Management" Function Manual.

Value Range:
Value range:
o Value 0: Not enabled
o Bit 0: Active multitool
o Bit 1: Enabled
o Bit 2: Disabled
o Bit 3: Measured
o Bit 4: Prewarning limit reached
o Bit 5: Multitool is being changed
o Bit 6: Fixed-location-coded
o Bit 7: Multitool was in use
o Bit 8: Multitool in buffer magazine with transport order
o Bit 9: Ignore disabled status of the multitool
o Bit 10: Multitool must be unloaded
o Bit 11: Multitool must be loaded
o Bit 12: Master tool
o Bit 13: Reserved
o Bit 14: Marked for 1:1 replacement
o Bit 15: Manual tool
o Bit 16: Multitool is disabled if a tool in the multitool is disabled
o Bit 17: Multitool is at a disabled magazine location
Type
TYPE_UDWORD 		Unit
No unit 		Default Value
0 		Minimum Value
 Maximum Value
Access Path HMI: /Tool/MT/multitoolStateL[u<Tool Area>, <Line Index>]
