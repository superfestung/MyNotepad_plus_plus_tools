Siemens application parameters for multitools for all permissible data types, except the data type "String".
Siemens application data are made available with the MD settings MD bits MD18080 $MN_MM_TOOL_MANAGEMENT_MASK, bit0=1, bit2=1, bit10=1 and MD18196 $MN_MM_NUM_CCS_MULTITOOL_PARAM>0.
Data types of the individual parameters are specified by MD18197 $MN_MM_TYPE_CCS_MULTITOOL_PARAM, and can be determined from the file _N_COMPLETE_TOA_ACX.
The transfer in the OPI is always as TYPE_DOUBLE.
If there is a read or write access to a Siemens application parameter for multitools of the data type "String", then an error string is returned.
