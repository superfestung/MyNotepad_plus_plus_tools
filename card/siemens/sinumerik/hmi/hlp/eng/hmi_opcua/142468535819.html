Deleting OPC UA model

Procedure

  1. To delete one customer specific object model, press the softkey "Delete Model".

    Softkey Delete Model

  2. Select the model you want to delete from the displayed list and press softkey "OK".

    OPC UA Models Delete

  3. A pop-up screen is displayed asking to confirm the delete action:

    Delete Model popup

    Press softkey "OK" to delete the model.

    Press softkey "Cancel" to return to the "OPC UA Models" screen.
