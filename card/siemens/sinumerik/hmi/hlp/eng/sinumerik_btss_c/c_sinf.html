Block C/SINF: Part-program-specific status data

During automatic execution of a part program different parameters can influence the type of machining. The current status data for the selected part program are combined in module SINF. The status data must only be changed via the PLC. interface.

singleBlockType Single block mode
Column index:
17 		Line index:
 Max. line index:
Single block mode can be set with this variable.
o =1: Single block mode 1 (main run, single block)
o =2: Single block mode 2 (decoder, single block)
o =3: Single block mode 3 (stop in the cycle)
Single block mode is activated by the interface signal
DB21, ... DBX0.4 = 1
