Block C/FG: Channel-specific frames for grinding applications

These occur only if $MC_MM_NUM_G_FRAMES > 0 and $MN_MM_NUM_GLOBAL_G_FRAMES = 0, otherwise all grinding frames are configured globally for the NCU.

The following frame indices are possible:
0: GRAME1
1: GRAME2
2: GRAME3
3: GRAME4
...
n: GRAMEn
...
99: GRAME100

The maximum frame index is: $MC_MM_NUM_G_FRAMES - 1

The PI service SETUFR has to be called to activate the grinding frames.
