punchActive Punching or nibbling active
Column index:
1 		Line index:
 Max. line index:
1
Identification of punching or nibbling active. The part program turns off/on punching and nibbling with 'SPOF', 'SON' and 'PON'.
Rapid punching and nibbling are turned on/off with 'SONS' and 'PONS'.

The variable 'punchActive' specified the present state.

Value Range:
0 = inactive
1 = punching active
2 = nibbling active
3 = rapid punching active
4 = rapid nibbling active
Type
TYPE_UWORD 		Unit
No unit 		Default Value
 Minimum Value
 Maximum Value
Access Path HMI: /Channel/Nibbling/punchActive[u<Channel numb>]
