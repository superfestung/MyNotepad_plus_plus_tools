Block C/WAL: Working area limitation

This block contains the working area limitation data.
 
waCSCoordSys Coordinate system for working area limitation $P_WORKAREA_CS_COORD_SYSTEM
Column index:
1 		Line index:
Channel axis index + working area limitation group * numMachAxes		 Max. line index:
numMachAxes * $MC_MM_NUM_WORKAREA_CS_GROUPS
Coordinate system for working area limitation

Value Range:
Identifier for the coordinate system in which the working area limitation is to apply.
The following are valid:
0: Working area limitation in the WCS
3: Working area limitation in the SZS
Special feature of line addressing: Any channel axis index can be selected as the channel axis index.
Within a working area limitation group, the values are identical.
Type
TYPE_UWORD 		Unit
No unit 		Default Value
0 		Minimum Value
0 		Maximum Value
3
