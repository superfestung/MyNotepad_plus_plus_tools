Block C/FU: Channel-specific settable frames

This only applies if $MC_MM_NUM_USER_FRAMES > 0 and $MN_MM_NUM_GLOBAL_USER_FRAMES = 0, otherwise all settable frames have an NCU-global configuration.

The following frame indices are possible:
0 = G500
1 = G54
2 = G55
3 = G56
4 = G57
5 = G505
6 = G506
:
n = G5n
:
99 = G599

The maximum frame index is: $MC_MM_NUM_USER_FRAMES - 1

The PI service SETUFR has to be called in order to activate the settable frames.
