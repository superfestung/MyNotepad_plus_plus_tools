Block C/VSYN: Channel-specific user variables for synchronous actions

This module contains channel-specific user variables for synchronous actions
 
acFifoN FIFO variable for synchronous actions $AC_FIFOx[y] , x = FIFONo (1-10) y = ParameterNo
Column index:
10 		Line index:
Parameter
See description below 		Max. line index:
MD $MC_MM_LEN_AC_FIFO+6
FIFO variable for synchronous actions (Note: SYNACT only)
The number of columns depends on the number of FIFOs

Line index:
1=2: access to the first element read in (corresponds to: $AC_FIFOx[1])
2: dto. (corresponds to: $AC_FIFOx[1])
3: access to the last element read in (corresponds to: $AC_FIFOx[2])
4: sum of all FIFO elements (corresponds to: $AC_FIFOx[3])
5: number of elements available in FIFO (corresponds to: $AC_FIFOx[4])
6: current write index in relation to start of FIFO (corresponds to: $AC_FIFOx[5])
7 etc: FIFO contents (corresponds to: $AC_FIFOx[6-])

Note: $AC_FIFOx[0] cannot be read via OPI, as this access changes the contents of the FIFO.
Type
TYPE_DOUBLE 		Unit
No unit 		Default Value
 Minimum Value
 Maximum Value
Access Path HMI: /Channel/SelectedFunctionData/acFifoN[u<Channel numb>, <Line Index>]
