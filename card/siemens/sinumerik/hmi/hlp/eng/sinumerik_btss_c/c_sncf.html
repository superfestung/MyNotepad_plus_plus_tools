Block C/SNCF: Active G functions

All G functions are organized in G groups. Only one function of each G group can be active at a time.
The module SNCF consists of a single variable that is organized as an array. The row index corresponds to the G group number.
 
ncFkt Active G function
Column index:
1 		Line index:
G-GroupNo
G group number 		Max. line index:
numGCodeGroups
Active G-function of relevant group
G <No>.
If there is no function active within the corresponding G-group, the variable returns an empty string "\0".
Type
TYPE_STRING 		Unit
No unit 		Default Value
 Minimum Value
 Maximum Value
Access Path HMI: /Channel/SelectedFunctions/ncFkt[u<Channel numb>, <Line Index>]
