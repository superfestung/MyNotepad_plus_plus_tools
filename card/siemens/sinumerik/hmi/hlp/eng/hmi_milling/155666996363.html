Sequence for Y turning

  1. Load the Y turning tool.

  2. Preposition the tool.

  3. Start Y turning with the CYCLE805:

    • The tool is aligned.

    • The coordinate system is rotated accordingly.

    • The starting position is approached (if "Starting point = yes" was selected).

  4. Approach the starting position (if "Starting point = no" was selected).

  5. Machine using the standard turning cycles.

  6. Start Y turning with CYCLE805:

    • The retraction position is approached (if "Retract = yes" was selected).

    • The coordinate system is rotated back again accordingly.

    Note

    In ShopTurn selection/deselection is realized automatically. The parameters required are included in the turning screen forms for Y turning.

Example Y turning G code

T="Y turning tool"

D1 ; roughing plate

CYCLE805(...) ; select Y turning

CYCLE951(...) ; roughing

D3 ; finishing plate

CYCLE805(...) ; select Y turning

CYCLE951(...) ; finishing

CYCLE805() ; deselect Y turning

M30
