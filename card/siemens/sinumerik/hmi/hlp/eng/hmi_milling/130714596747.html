Accepting the machining steps

To accept the suggested data in the program, select the appropriate machining step in the list of machining steps, and if necessary, add additional parameters. The machining step is written to the program and the list of machining steps is closed.

Note

The parameter screen forms of the machining steps recognized by the NX server, such as drilling, reaming, boring, etc., contain the parameter "Diameter" in contrast to the parameter screen forms of the cycles with the same name. This parameter supports you in the correct tool selection for the respective machining operation.
Note

The values resulting from the 3D model are marked in yellow in the parameter screen forms.

Changing these values is not recommended.

Requirement

  • The list of machining steps is open.

Procedure

 

1.

In the 3D model, select the required element, using either touch or mouse.

The element is highlighted in the 3D model. The cursor jumps to the associated machining step in the list.

  

- OR -

  

Using the cursor keys, navigate in the list to the required machining step.

The element is highlighted in color in the 3D model.

 

2.

If several, alternative machining steps exist for the selected element, then these steps are listed in the "Alternatives" block.

In this case, select the required technological function.

 

3.

Press the "Accept" softkey.

The parameter screen form of the selected function opens. The values that came from the CAD data are accepted. All other parameters are preassigned with default values. The default values are the values last entered into the parameter screen form.

 

4.

Check the parameters, and if necessary, change the default values.

 

5.

Confirm the data with "OK".

The program view opens. The newly created machining step is marked.

 

6.

If you wish to accept additional machining steps, then press the "Import from CAD" softkey.

The list of machining steps opens again.

Note

Before accepting machining steps on a plane, step "Swivel plane" must be accepted.

Note

Swiveling after a tool change (only G code)

If swiveling is set-up at your machine, then step "Swivel plane" must be inserted after each tool change.

Note

Machine steps, which you transfer into the program, are marked with a green checkmark.

If the zero point for the workpiece or a plane is subsequently changed, the associated steps are no longer correct.

These steps are marked using an exclamation mark.

If you again accept these steps, then the steps are marked using a green checkmark.
