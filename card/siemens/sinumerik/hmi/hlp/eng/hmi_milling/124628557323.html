File and directory names

The following rules are to be observed when assigning names to files and directories:

  • All letters are permissible (with the exception of umlauts, special characters, language-specific special characters, Asian or Cyrillic characters)

  • All digits

  • Underscores (_).

  • The name can be a maximum of 24 characters long

Note

To avoid problems with Windows applications, do not use the following terms as program names or directory titles:

  • CON, PRN, AUX, NUL

  • COM1, COM2, COM3, COM4, COM5, COM6, COM7, COM8, COM9

  • LPT1, LPT2, LPT3, LPT4, LPT5, LPT6, LPT7, LPT8, LPT9

Please note that these terms, including those with extensions (e.g. LPT1.MPF, CON.INI) can lead to problems if they are transferred to a Windows environment by copying, archiving or uploading, for example.
