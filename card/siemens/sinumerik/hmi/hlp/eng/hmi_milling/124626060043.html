Function - LONGHOLE

Function

In contrast to the groove, the width of the elongated hole is determined by the tool diameter.

Internally in the cycle, an optimum traversing path of the tool is determined, ruling out unnecessary idle passes. If several depth infeeds are required to machine an elongated hole, the infeed is carried out alternately at the end points. The path to be traversed in the plane along the longitudinal axis of the elongated hole changes its direction after each infeed. The cycle searches for the shortest path when changing to the next elongated hole.

Note

The cycle requires a milling cutter with a "face tooth cutting over center" (DIN 844).
