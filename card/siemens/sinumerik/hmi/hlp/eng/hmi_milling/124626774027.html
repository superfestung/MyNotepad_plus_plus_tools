Grooving residual material function - CYCLE952

Function

The "Grooving residual material" function is used when you want to machine the material that remained after grooving along the contour.

For a G code program, first select the "Grooving residual material" function. Material that remains as part of the finishing allowance is not residual material. The "Grooving residual material" function allows you to remove unwanted material with a suitable tool.

The "Grooving residual material" function is a software option.
