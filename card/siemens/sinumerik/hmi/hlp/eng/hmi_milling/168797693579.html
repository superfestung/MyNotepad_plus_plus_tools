Displaying distance information

You can define the following distances in the 3D model:

  • Point - Point

  • Point - Edge

  • Point - Edge

  • Edge - Edge

  • Edge - Plane

  • Plane - Plane

Determining the distances is helpful, for example to measure wall thicknesses or clearances.

The distance is displayed in absolute terms (3D) and parallel to the axis for the active plane.

Requirement

  • The required 3D model is open.

  • The list of machining steps is open.

Procedure

 

1.

Press the "Geometry info" softkey.

 

2.

Press the "Offset" softkey.

  
 

3.

Select the desired entry in the selection list under "Offset".

The data for distance measurement are displayed.
