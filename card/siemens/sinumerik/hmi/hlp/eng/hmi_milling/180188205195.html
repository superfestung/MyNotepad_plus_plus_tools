Displaying angle information

You can define the following angle in the 3D model:

  • Edge - edge: Angle between two edges

This is helpful, for example to determine a chamfer angle

You can select all straight edges. If an intersection edge is involved, then only the unique values of the intersection point are displayed.

Requirement

  • The required 3D model is open.

  • The list of machining steps is open.

Procedure

 

1.

Press the "Geometry info" softkey.

 

2.

Press softkey "Angle".

  
 

3.

Click on the edges in the 3D model for which you want to determine the angle.

The following angle values are displayed:

  • αXY: Projection of the 3D angle to the current XY plane

  • βXY: Supplementary angle to 180°

  • α3D: Angle in space

  • β3D: Supplementary angle to 180°

Note

If αXY and βXY or α3D and β3D are added, then 180° is obtained.

The point of intersection of the elements is also displayed.
