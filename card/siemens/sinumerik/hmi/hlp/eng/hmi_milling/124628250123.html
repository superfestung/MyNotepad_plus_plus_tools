Traversing axes

For preparatory actions and simple traversing movements, input the parameters directly into the "Manual Machine" input fields of the basic screen.

Tool selection

 

1.

"Manual Machine" is active.

Tool selection

 
 

2.

Select the desired tool in "T".

 

3.

Enter the feedrate (F) and the spindle speed (S).

 

4.

Select the direction of spindle rotation (e.g. clockwise direction of rotation):

 

 

 

 

 

- OR -

 

 

Set the direction of rotation via the machine control panel.

 

5.

Press the <CYCLE START> key.

 

 

The spindle starts immediately after the tool is selected.

Note:

Please refer to the machine manufacturer's specifications.
