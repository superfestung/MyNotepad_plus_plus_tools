Block search for position pattern

It is possible performing a block search for the position pattern. You can define the number of the starting hole.

With ShopMill programs you can also define the technology with which you want to start.

 

Software option

You require the "ShopMill/ShopTurn" option for the block search for ShopMill machining step programs.

 

Machine manufacturer

Please observe the information provided by the machine manufacturer.

Procedure

  

The required ShopMill program or G code program is in the block display.

 

1.

Press the "Block search" softkey.

 

2.

Position the cursor to the position block.

 

3.

Press the "Start search" softkey.

The "Block search" window opens.

Define technology (only with ShopMill programs)

 

4.

Select the required technology and press the "OK" softkey

The selected technology is displayed in the "Search" window.

Specify starting hole

 

5.

Enter the number of the start hole and press the "OK" softkey.

Program processing starts with the specified technology (only for ShopMill programs) at the specified starting hole, and continues to all additional positions of this position pattern and all of the following position patterns.

 

 

Note

If you have hidden certain positions, then only the displayed positions are applicable for the starting hole number.

Note

Automatic technology selection with ShopMill

The technology is automatically selected when the current data set has been saved so that a reference to the active position pattern can be established.

If the technology is not selected automatically, you must select the correct technology manually.

Note

Automatic selection of start hole

The number of the start hole is automatically determined and displayed when the current data set has been saved so that a reference to the active position pattern can be established.

If the number of the start hole was not determined automatically, you must enter the number of the start hole manually.
