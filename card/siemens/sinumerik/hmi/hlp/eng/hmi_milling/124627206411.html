Polar coordinates

If a workpiece has been dimensioned from a central point (pole) with radius and angles, you will find it helpful to program these as polar coordinates.

You can program straight lines and circles as polar coordinates.

Defining a pole

You must define the pole before you can program a straight line or circle in polar coordinates. This pole acts as the reference point of the polar coordinate system.

The angle for the first line or circle then needs to be programmed in absolute coordinates. You can program the angles for any additional straight lines and circles as either absolute or incremental coordinates.

Procedure

 

1.

The ShopMill program to be edited has been created and you are in the editor.

 

2.

Press the menu forward key and the "Polar" softkey.



 

3.

Press the "Pole" softkey.
