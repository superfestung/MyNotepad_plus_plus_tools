Open a DXF file

Procedure

 

1.

Select the "Program Manager" operating area.

 

2.

Choose the desired storage location and position the cursor on the DFX file that you want to display.

 

3.

Press the "Open" softkey.

The selected CAD drawing will be displayed with all its layers, i.e. with all graphic levels.

 

4.

Press the "Close" softkey to close the CAD drawing and to return to the Program Manager.
