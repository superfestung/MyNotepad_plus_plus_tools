Multiple selection in the tool management lists

Multiple selection is available in the tool management lists.

You select several tools, e.g. to delete them from the list, load them, unload them, or move them to other magazine locations.

For multiple selection, you use either the "Select" softkey, the touch or mouse controls, or the cursor keys.

You can use multiple selection for the following tasks:

  • Deleting a tool

  • Loading and unloading a tool

  • Relocating a tool

  • Unloading or deleting a tool on the code carrier

  • Equipping a multitool with tools

  • Removing tools from a multitool

  • Deleting a multitool

  • Loading and unloading a multitool

  • Relocating a multitool
