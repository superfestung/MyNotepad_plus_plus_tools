Function - CYCLE78

Function

You can use a drill and thread milling cutter to manufacture an internal thread with a specific depth and pitch in one operation. This means that you can use the same tool for drilling and thread milling, a change of tool is superfluous.

The thread can be machined as a right- or left-hand thread.
