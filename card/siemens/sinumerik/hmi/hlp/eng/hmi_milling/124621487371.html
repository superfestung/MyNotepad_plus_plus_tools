Special "tilde" character

If the shift key for letters and special characters is pressed, the keyboard assignment changes to the special characters.

①

<Tilde>

In the Editor or in alphanumeric input fields, the special character <Tilde> is entered with the <Tilde> key. In numerical input fields, the <Tilde> key changes the sign of a number between plus and minus.
