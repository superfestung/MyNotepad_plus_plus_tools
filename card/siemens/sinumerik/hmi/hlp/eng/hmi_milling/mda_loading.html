Loading an MDA program from the Program Manager

Procedure

 

1.

Select the "Machine" operating area.

 

2.

Press the <MDA> key.

The MDI editor opens.

 

3.

Press the "Load MDI" softkey.

The "Load into MDI" window opens. It shows you a view of the program manager.

 

4.

Position the cursor to the corresponding storage location, press the "Search" softkey and enter the required search term in the search dialog if you wish to search for a specific file.

Note: The place holders "*" (replaces any character string) and "?" (replaces any character) make it easier for you to perform a search.

 

5.

Select the program that you would like to edit or execute in the MDI window.

 

6.

Press the "OK" softkey.

The window closes and the program is ready for operation.
