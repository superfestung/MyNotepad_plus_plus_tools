Filtering data

You have the option of limiting the machine data displayed to specific topics and then to specifically search for character strings.

The settings of the display options are deactivated if you use the filter function.

Filter options

  • Group

    A group encompasses machine or setting data within the selected data areas that are associated with a particular topic. The same groups are listed as in the "Display options" window. However, only one group can be selected.

  • Filter

    Enter the character string in the "Filter" field that you wish to search for in the selected group.

    Note

    The system does not support the use of place holders in the "Filter" field.

    No distinction is made between uppercase and lowercase type.

Searched fields

A search is made in the following fields of the machine data displayed for the character string that was entered:

  • Machine data number

  • Symbolic name

  • Descriptive text

  • Bit texts

  • Enum texts (only for drives)

Procedure

 

1.

Select the "Start-up" operating area.

 

2.

Press the "Mach. data" softkey.

 

3.

Select the required area by pressing the relevant softkey.

 

4.

Press the "Filter" softkey.

 

5.

Select the required data area in the "Group" field.

 

6.

Enter the required character string in the "Filter" field.

The displayed data are updated after each character.

 

7.

Press the "Filter" softkey again to deactivate the filter.
