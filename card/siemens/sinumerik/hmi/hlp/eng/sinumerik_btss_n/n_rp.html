Block N/RP: Arithmetic parameters

Arithmetic parameters are specific, predefined variables which are addressed with the address R and subsequent number. The contents and meaning of arithmetic parameters is specified by person programming the part program. The number of parameters can be set in the machine data 18156 (MM_NUM_R_PARAM_NCK).
 
RG Global R variables $RG[x] x = ParameterNo
