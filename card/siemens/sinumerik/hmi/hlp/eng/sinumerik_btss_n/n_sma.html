Block N/SMA: State data: Machine axes

All state data that are dependent on machine movement and are defined specifically for machine axes (geometry and special axes) are combined in module SMA. Supplementary information is to be found in module SEMA. The individual variables are defined as fields where the line index is the number of the machine axis (assigned to the current channel). The variable "name" in module SMA with the line index in question identifies the axis.
The assignment of the line indices in modules SMA and SEMA is identical.
