- With interpolating axes or axis/spindle couplings, maintaining the contour or coupling during the braking phase is not guaranteed.
- The braking ramp may be ineffective or not maintained if the drive used operates its own braking ramp sequence logic (e.g. OFF3 ramp for SS1 and SS2 in SINAMICS).
- With SS2E, the braking ramp must also be activated via the axis-specific NC/PLC interface signal DB31, ... DBX34.3.
- With coupled axes, the braking time must be parameterized in each axis of this group.
Setting recommendation
A value should be set so that a braking ramp is obtained based on drive parameter p1135 (OFF3 return time). (See also setting instructions in the Function Manual).

It influences the channel-wide IPO response distribution of Safety Integrated:

1 = For internal stops, the axes and machining spindles interpolating with the axis in question are also influenced via the triggered safety alarms.
Other axes/spindles in the channel, however, continue without disturbance. In the case of external stops (without an alarm) all other axes/spindles are not influenced by the safety axis/spindle stop. If the other axes/spindles stop together with the safety axis/spindle in certain machining situations, the user must implement this at his own responsibility using the PLC or synchronous action operations.
