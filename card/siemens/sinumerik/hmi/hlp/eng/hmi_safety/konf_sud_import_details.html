SafeUserData - Import - Details

After the input call, a check indicated that the symbol names of the active SafeUserData data sets differ from the symbol names of the import file.

For evaluation, the differences are displayed in screen form "SafeUserData - Import - Details".

Requirements

  • The import of SafeUserData from an import file was called.

  • A check indicated differences in the symbol names.

Call

MENU SELECT > Commissioning > Menu forward key > Safety > SafeUserData > Import > OK (after notification of differences)

A dialog to evaluate the differences in the symbol names then opens.

Vertical softkeys

Name

Description

Cancel

Using this softkey, you cancel the import of the actual values.

After cancellation, screen form "SafeUserData - Overview" is displayed.

Accept

The symbol names of the data sets from the import file differ from the symbol names of the current data set.

Using this softkey, you accept the elements of all instances from the import file, although symbol names differ from those of the active data set.

A juxtaposition in screen form "SafeUserData - Import" indicates that the data were accepted from the import file.

Reject

The symbol names of the data sets from the import file differ from the symbol names of the current data set.

Using this softkey, you reject the import of all element values of the displayed instance. Additional differences of other instances can be displayed in additional screen forms and evaluated in the same way.

A juxtaposition in screen form "SafeUserData - Import" indicates that the data were not accepted from the import file.
