Details of the PROFIsafe telegram

The following data is displayed in the "Details PROFIsafe telegram" window:

Telegram 30 transfers the following user data:

  • Receive direction

    • S_STW1

  • Send direction

    • S_ZSW1

Telegram 31 transfers the following user data:

  • Receive direction

    • S_STW2

  • Send direction

    • S_ZSW2

Telegram 901 transfers the following user data:

  • Receive direction

    • S_STW2

    • S_SLS_LIMIT_A (variable SLS limit)

  • Send direction

    • S_ZSW2

    • S_SLS_LIMIT_A_ACTIVE (active SLS value of level 1)

    • S_CYCLE_COUNT (counter value)

    • S_XIST16 (safety-relevant position value in the 16-bit format)

Telegram 902 transfers the following user data:

  • Receive direction

    • S_STW2

    • S_SLS_LIMIT_A (variable SLS limit)

  • Send direction

    • S_ZSW2

    • S_SLS_LIMIT_A_ACTIVE (active SLS value of level 1)

    • S_CYCLE_COUNT (counter value)

    • S_XIST32 (safety-relevant position value in the 32-bit format)

Telegram 903 transfers the following user data:

  • Receive direction

    • S_STW2

    • S_SLS_LIMIT_A (variable SLS limit)

  • Send direction

    • S_ZSW2

    • Z_SWC (32 bit Safe Cam)

    • S_SLS_LIMIT_A_ACTIVE (active SLS value of level 1)

Note

Changing the view

In this diagnostics view, you can display the signal information of various drives. You can select the required drive using softkeys "Drive-", "Drive +" or "Select drive".
