Telegram configuration

Precondition

The "Settings - Configuration" window is only available if the commissioning mode is activated for the drive.

Settings

You can make the following settings in the "Settings - Telegram configuration" window:

  • Enable SIC/SCC

    Only active for basic functions and extended functions. The following options can be configured:

    • SIC/SCC module number

      Activates the enable and setting of the SIC/SCC module number.

    • Telegram configuration (p60122)

      Sets the telegram for the Safety Information Channel (SIC)/Safety Control Channel (SCC)

    • Takes into account the stop D/SS2E transition time in the braking response

      Activates the adaptation of the brake ramp for stop D/SS2E.

  • PROFIsafe

    Only active for basic functions and extended functions "via PROFIsafe". The following options can be configured:

    • PROFIsafe address (p9610)

      Sets the PROFIsafe address as hexadecimal value for the Control Unit.

    • Telegram configuration (p60022)

      Sets the telegram number for PROFIsafe.

    • Safety parameter assignment (p9611)

      Sets the PROFIsafe telegram number for the Control Unit.

    • Stop response for PROFIsafe communication failure (p9612)

      Setting the stop response in case of failure of PROFIsafe communication.

Recommended values

As an alternative to the manual input in the displayed input fields, you can go to a dialog by pressing the "Recommended values" softkey. Here, the configured PROFIsafe address and the telegrams for SIC/SCC and PROFIsafe are read out from HW Config and displayed. You can accept these recommended values by pressing the "Yes" softkey. You can exit the dialog with "No" without accepting the values.

If, in HW Config, there is no configuration, then the recommended value fields are not preassigned.

Parameter list

When required, each setting in this dialog window can also be made via a parameter list. You can access the parameter list using the ">>" softkey followed by the "Parameter list" softkey.
