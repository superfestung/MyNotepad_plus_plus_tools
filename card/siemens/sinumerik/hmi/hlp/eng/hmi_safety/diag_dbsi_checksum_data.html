SI checksums overview

You can obtain the following information about the drive system in the "SI checksums overview":

  • Details about global checksums

    • Global Safety Engineering

    • Safety-relevant hardware

    • SafeUserData (SUD)
      Note: Is only displayed if SafeUserData was configured in the PLC project.

  • F-PLC checksums

    • F-collective signature

    • F-SW collective signature

    • F-HW collective signature

    • F-communication address signature

  • Status, drive checksums

    Displays whether for all drive checksums, reference and actual values are identical.

You can call the following diagnostic views from the "SI checksums overview":

  • Global checksums

  • SUD signatures

  • Group signatures

  • Drive checksums

Note

SINUMERIK MC

This diagnostic overview "Group signatures" is not included in the scope of functions of the SINUMERIK MC and can therefore not be called up there.
