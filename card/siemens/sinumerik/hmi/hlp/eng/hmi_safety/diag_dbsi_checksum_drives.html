Drive checksums

Information about the checksums of the basic functions and extended functions is provided in the "Drive checksums" window:

  • Checksums of the extended functions

    Here, for monitoring channel 1 (Control Unit) and monitoring channel 2 (Motor Module), the actual and reference checksums for the following extended functions are compared with one another.

    • Motion monitoring

    • Actual values

    • Hardware

  • Checksums of the basic functions

    Here, for monitoring channel 1 (Control Unit) and monitoring channel 2 (Motor Module), the actual and reference checksums for the parameters are compared with one another.

Based on the status symbol, you can identify as to whether for the particular information, the actual checksum matches the reference checksum.

Symbol

Meaning

 

Checksums match

 

Checksums are different

Note

Changing the view

In this diagnostics view, you can display the signal information of various drives. You can select the required drive using softkeys "Drive-", "Drive +" or "Select drive".
