Safely-Limited Speed (SLS)

The "Safely-Limited Speed" (SLS) function is used to protect a drive against unintentionally high speeds in both directions of rotation. This is achieved by monitoring the actual drive speed up to a speed limit. SLS prevents a parameterized speed limit from being exceeded. Limit values must be specified based on results of the risk analysis.

Precondition

The commissioning mode must be activated so that the parameters of this safety function can be changed. An extended function must be set in the options.

Settings

You can make the following settings for the drive in this window:

  • Delay time for selection of SLS -> SLS active (p9551)

    Sets the delay time for the SLS changeover and for the activation of SOS for the functions "Safely-Limited Speed" (SLS) and "Safe Operating Stop" (SOS) in ms.

    Is effective at all 4 levels.

  • Vmax (mm/min or rpm)

    Sets the limit values for the "Safely-Limited Speed" function (SLS) for up to 4 levels.

    • Level 1 (p9531[0])

    • Level 2 (p9531[1])

    • Level 3 (p9531[2])

    • Level 4 (p9531[3])

  • Stop response

    Sets the SLS-specific stop response for the "Safely-Limited Speed" function (SLS) for up to 4 levels.

    • Level 1 (p9563[0])

    • Level 2 (p9563[1])

    • Level 3 (p9563[2])

    • Level 4 (p9563[3])

  • SLS limit via PROFIsafe (p9501)

    Sets the enable signals for the safety-related motion monitoring functions.

    Is only displayed if the "Extended functions via PROFIsafe" function is activated in the options.

The following LEDs can display the status (blue = active; gray = inactive):

  • Speed level selected (level 1...4)

    For the control signals for the safe motion monitoring functions integrated in the drive, indicates as to whether "Select, SLS bit 0" or "Select SLS bit 1" is set.

  • Speed level active (level 1...4)

    For the status signals for the safe motion monitoring functions integrated in the drive in the monitoring channel, indicates as to whether "SLS level bit 0" or "SLS level bit 1" is set.

Parameter list

When required, each setting in this dialog screen form can also be made via a parameter list. You can access the parameter list using the ">>" softkey followed by the "Parameter list" softkey.
