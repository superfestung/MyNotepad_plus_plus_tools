Additional functions for the acceptance test

Other functions that can be called using softkeys

  • Full screen

    Hides the list of the setting areas and displays the detail settings of the currently selected setting area over the complete screen width.

  • Load template

    Loads a template (*.xml), which can also be externally created.

    The template must be located in directory .../user/sinumerik/hmi/data/safety/atw/templates.

  • Export template

    Exports the current acceptance test as template (* .xml). The template contains a list of all of the possible tests that must be carried out for the acceptance test. Further, all of the texts of the complete acceptance test are included; e.g. details about the machine on the overview page.

  • Save results

    Saves the result file (*.atw) to directory .../user/sinumerik/hmi/data/safety/atw/results.

  • Load results

    Loads the saved results from a directory. This function is useful if the test is exited before completion and you wish to continue at the position where you left off.

  • Start new test

    Starts an additional test in the currently displayed setting area.

  • Repeat test

    Repeats a selected test in the currently displayed setting area.

  • Delete test

    Deletes the currently marked test.

  • Go to overview

    Jumps from the current setting area into the "Overview" setting area.

  • Go to finished

    Skips the following setting area and jumps directly to the "Finished" setting area.

  • Display trace

    Displays the trace that was recorded while the test was being performed.

  • Exit the SI acceptance

    Exits the acceptance test for Safety Integrated
