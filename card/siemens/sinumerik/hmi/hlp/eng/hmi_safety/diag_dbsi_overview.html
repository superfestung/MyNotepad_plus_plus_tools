SI - diagnostics overview

The following signal information is displayed in the "Safety Integrated diagnostics overview" window

  • General

    • Safety operating mode (which operating mode is set)

    • Number of safety NC axes

    • Number of safety PLC axes

  • Status

    • SI alarms (which alarms are active)

    • SI I/O (F-modules signal faults or wait for user acknowledgment)

    • SI checksums (display, if checksums differ)

    • SI telegrams (display for conflicts in the PROFIsafe or the SIC/SCC configuration)

    The status messages are identified using the following symbols:

    Symbol

    Meaning

     

    Fault/error

     

    Note/warning

     

    No problem known

You can call the following diagnostic overviews from the diagnostic overviews:

  • SI drives status (status information about safe drives)

  • SI telegrams (display of PROFIsafe and SIC/SCC telegrams)

  • Integr. F-PCU (settings for safety operation and access protection F-PLC)

  • SI checksum (display of global checksum, checksums F-PLC, drive checksums, folder checksums)

  • SI alarms (display of all safety alarms/faults)

  • SI I/Os overview of F modules (status information of SI I/O modules)

You can call the following diagnostic overview from the menu forward key:

  • SI safe cam (status of the "Safe cam" function)

To do this, use the softkeys in the vertical softkey bar.

Note

SINUMERIK MC

With the SINUMERIK MC, the diagnostic overviews "Integr. F-CPU" and "SI I/Os overview of F modules" are not included in the scope of functions. Both diagnostic overviews can therefore only be called up with SINUMERIK ONE.
