SI I/O details F-module

Detailed diagnostics data of the selected F-module are displayed in the "SI I/O details of F module" window:

  • Device name

  • Device name number

  • Name of the FIO DB

  • Slot number

  • Type 

  • I/O address

  • Number of the F-I/O DB

  • Projected F-monitoring time

  • Diagnostic data

    If the diagnostic data are available, then the hexadecimal value and the significance of the individual bits are displayed in text form.

Note

SINUMERIK MC

This diagnostic overview is not included in the scope of functions of the SINUMERIK MC and can therefore not be called up there.

Note

Changing the view

In this diagnostics view, you can display the diagnostics data of various F-modules. For this purpose, you can select the required module using the "Module-" and "Module+" softkeys. If there are several modules in the system, you can select the desired module with the help of a list.
