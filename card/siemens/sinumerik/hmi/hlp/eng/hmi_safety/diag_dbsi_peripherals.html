SI I/Os overview of F modules

An overview of the configured F-modules is shown in the "SI I/Os overview of F modules" window.

Note

SINUMERIK MC

This diagnostic overview is not included in the scope of functions of the SINUMERIK MC and can therefore not be called up there.

The following information, structured according to modules, is displayed for each module:

  • Name of the FIO DB

  • I/O address

  • Type

  • Status

The current status of the listed F-modules is marked with the following symbols:

Symbol

Meaning

 

Fault

 

Acknowledgment request for reintegration

 

Normal operation

Status details are shown in the status line.

Additional information about the selected module can be displayed using the "Details" softkey.
