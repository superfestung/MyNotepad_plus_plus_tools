SIC/SCC - details PROFIdrive telegram

The following data is displayed in the "SIC/SCC - details PROFIdrive telegram" window:

Telegram 700 transfers the following user data:

  • Receive direction

    • S_ZSW1B

    • S_ZSW2B

    • S_ZSW3B

    • S_V_LIMIT_B

Telegram 701 transfers the following user data:

  • Receive direction

    • S_ZSW1B

    • S_ZSW2B

    • S_ZSW3B

    • S_V_LIMIT_B

  • Send direction

    • S_STW1B

    • S_STW3B

Note

Changing the view

In this diagnostics view, you can display the signal information of various drives. You can select the required drive using softkeys "Drive-", "Drive +" or "Select drive".
