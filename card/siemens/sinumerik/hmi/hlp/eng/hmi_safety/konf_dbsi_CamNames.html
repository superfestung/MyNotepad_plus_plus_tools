Cam identifiers/cam tooltips

You can define names and tooltips for the selected drive (or the selected axes) in the window "Cam identifier/cam tooltips". The names and tooltips are displayed in screen "SI safe cams" for the selected drive (or the selected axis). The names and tooltips entered in this dialog are saved to a language file.

Settings

From the axis/drive field, first select the required axis or drive.

Enter the required name or tooltip.

You can add an axis or a drive using softkey "Add axis/drive".

You can delete an axis or a drive - including the created name and tooltip - using softkey "Delete axis/drive".
