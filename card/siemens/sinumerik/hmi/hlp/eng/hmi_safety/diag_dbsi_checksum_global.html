Details about global checksums

Information about the following global checksums integrated in the drive is available in the "Details on the global checksums" window:

Engineering checksum

  • F-PLC F-collective signature

    • Checksum from the signature of the F program and the hardware checksum F-CPU

  • Extended Safety master checksum

    Comprises the following checksums:

    • Checksum across SI parameters for motion monitoring (r9728[0])

    • Checksum across SI parameters for actual values (r9728[1])

  • Extended Safety MM master checksum

    • Checksum across SI parameters for motion monitoring (r9398[0]) for Motor Modules/Hydraulic Modules

  • Drive-based master checksum

    Comprises the following checksums:

    • Checksum of SI parameters on the Control Unit (r9798)

    • Checksum of SI parameters on the Motor Modules/Hydraulic Modules (r9898)

Hardware checksum

  • Safety-relevant hardware checksum

    Comprises the following checksums:

    • Checksum across SI parameters for hardware (r9728[2])

    • Checksum across SI parameters with hardware reference (r9398[1])

Note

Determining checksums

A progress display is shown in the window, if it takes a long period of time to determine checksums for the display. This indicates which global check sum is presently being calculated.
