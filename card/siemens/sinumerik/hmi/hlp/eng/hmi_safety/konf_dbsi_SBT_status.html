Safe Brake Test (SBT) status

The diagnostic function "Safe Brake Test" function (SBT) checks the holding torque of a brake (operating or holding brake).

For diagnostic purposes, you can check the active status of a brake test in a separate status window.

Requirement

  • For the status display of SBT, the commissioning mode must be deactivated.

  • Safety Integrated Extended Functions with encoder have been enabled and configured.

"Safe Brake Test - SBT - Status"

The following information is displayed in window "Safe Brake Test - SBT - Status":

  • Actual holding torque or force

    Displays the torque actual value or force actual value (unsmoothed).

  • Max. holding torque

    Displays the maximum holding torque that occurred (= highest value from "Act. holding torque").

    Only displayed when the brake test is inactive.

  • Load torque or force

    Displays the load torque or the load force for the Safe Brake Test. This load torque or this load force is available at the drive when initializing the brake test. The displayed value is kept until the brake test is deselected.

  • Test torque or force

    Displays the maximum test force or the maximum test torque available on the motor side. The displayed value remains until the start of the next test sequence.

  • Test progress

    Displays the test progress as bar chart.

  • Test status

    Displays the current status of the Safe Brake Test.

  • Test result

    Status display of the brake test that was last performed.

    • Is only displayed if the screen form was active when performing the test.

    • Is only displayed during an active SINUMERIK session and is not saved.

Other options

In window "Safe Brake Test - SBT- Status", using the "Trace" softkey, you can open a trace dialog and load a drive trace session of the current drive into it.

  • The following functions can be used via softkeys with the same name:

    • Starting the trace

    • Scale

    • Zoom

    • Adapt all

    • Selection/legend

    • Cursors >

Using softkey "<<Back", you can return to dialog "Safe Brake Test - SBT - Status".

Significance of the SBT parameters

Display of the SBT parameters

Depending on the particular drive type, the following terminology is used:

  • "...torque"

    this term is used for rotating drives

  • "...force"

    is used for linear drives
