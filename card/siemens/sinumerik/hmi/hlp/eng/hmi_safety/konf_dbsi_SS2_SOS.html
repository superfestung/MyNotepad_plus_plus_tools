Safe Stop 2 (SS2)/Safe Operating Stop (SOS)

The "Safe Stop 2" (SS2) function is used to brake the motor safely on the OFF3 deceleration ramp (p1135) with subsequent transition to the SOS state after the delay time expires (p9552). The delay time set must allow the drive to brake to a standstill from every speed of the operating process within this time. The standstill tolerance (p9530) may not be violated after this time.

The "Safe Operating Stop" (SOS) function is used for safe monitoring of the standstill position of a drive.

Precondition

The commissioning mode must be activated so that the parameters of this Safety Integrated Function can be changed. The SS2/SOS Safety Integrated Function is only available for the selected drive if, in the setting area options, an extended function has been previously set and parameterized.

Settings

You can make the following settings for the drive in this window:

  • Standstill tolerance SOS (p9530)

    Sets the tolerance for the "Safe operating stop" (SOS) function in mm for a linear axis or in degrees for a rotary axis.

  • Delay time, SS2/STOP C -> SOS active (p9552)

    Sets the transition time from STOP C to "Safe Operating Stop" (SOS) in ms.

  • Delay time, SLS/SOS - SLS/SOS active (p9551)

    Sets the delay time for the SLS changeover and for the activation of SOS for the functions "Safely limited speed"(SLS) and "Safe operating stop" (SOS) in ms.

  • Delay time, STOP D -> SOS active (p9553)

    Sets the transition time from STOP D to "Safe operating stop" (SOS) in ms.

  • Delay time, STOP E -> SOS active (p9554)

    Sets the transition time from STOP E to "Safe operating stop" (SOS) in ms.

  • Switchover speed to SOS/SLS (p9567)

    Sets the switchover speed to SOS/SLS in mm/min.

  • Transition time to SOS after standstill (p9569)

    Sets the transition time to SOS after standstill in ms.

  • Enable deactivation SOS/SLS during an external STOP A (p9501.23)

    "Enable deactivation SOS/SLS during an external STOP A" can be activated using the check box.

For the following parameters, the status symbols indicate the actual state (blue = active; gray = inactive):

  • SS2 active (r9722.2)

    Setting "SS2 active"

  • SOS active (r9722.3)

    Setting "SOS active"

Parameter list

When required, each setting in this dialog window can also be made via a parameter list. You can access the parameter list using the ">>" softkey followed by the "Parameter list" softkey.
