Using test case groups

You can enter test case groups, e.g. for specific machine types in the template editor of SINUMERIK Operate. You can select the test case group(s) in the particular test types. This means that you have the option of using a template for different machine types.

Enter or select test case group(s)

  1. Open the template editor using MENU SELECT > Commissioning > Menu forward key > Safety > Acceptance test >
    Template editor

  2. Attach a test to the template.

  3. In field "Test case group(s)", select already created groups from the list.

  4. If groups have still not been entered, in field "Test case group(s)" enter one or several groups.

    For additionally required tests, you can then assign test case group(s) via the list field.

  5. Save the template.

Acceptance test - loading the template for a specific test case group

  1. Open the acceptance test using MENU SELECT > Commissioning > Menu forward key > Safety > Acceptance test

  2. Load the template using softkey "Load template".

  3. Select the template.

  4. Apply the selection by clicking "OK".

    The following dialog is displayed.

  5. Select whether you wish to use the complete template or a template with valid axis information.

  6. Activate option "Test case group" if you wish to combine the template with a specific test case group.

    Then select the required test case group.

  7. Click "OK" to confirm the selection.
