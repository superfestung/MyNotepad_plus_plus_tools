Safe Stop 2 with external stop (SS2E)

The 'Safe Stop 2 with external stop' (SS2E) safety function is used to brake the motor safely. The drive does not automatically brake the motor, but instead follows the specified speed setpoint.

Precondition

The commissioning mode must be activated so that the parameters of this Safety Integrated Function can be changed. The SS2E Safety Integrated Function is only available for the selected drive if an extended function has been previously set and parameterized in the options setting area.

Settings

You can make the following settings for the drive in this window:

  • Release SS2E (p9501.18)

    Release SS2E is activated via the parameters.

  • Delay time, STOP D -> SOS active (p9553)

    Sets the transition time from STOP D to "Safe Operating Stop" (SOS) in ms.

For the following parameters, the status symbols indicate the actual state (blue = active; gray = inactive):

Status/control signals:

  • Select SS2E (r9720.28)

    Control signal "De-select SS2E"

  • Select SOS (r9720.3)

    Control signal "Deselect SOS"

Status signals:

  • SS2E active (r9722.28)

    Feedback "SS2 active"

  • SOS active (r9722.3)

    Feedback "SOS active"

  • STOP C active (r9721.13)

    Feedback "STOP C active"

  • STOP D active (r9721.14)

    Feedback "STOP D active"

Parameter list

When required, each setting in this dialog window can also be made via a parameter list. You can access the parameter list using the ">>" softkey followed by the "Parameter list" softkey.
