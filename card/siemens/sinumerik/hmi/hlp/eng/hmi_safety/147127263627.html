SafeUserData - Confirm/save changed values

Call

MENU SELECT > Commissioning > Menu forward key > Safety > SafeUserData > Details > Change > Accept > Confirm/Save

Vertical softkey

Name

Description

OK

You exit the change mode using the softkey.

You return to screen form "SafeUserData - Overview - Details".

Result

The SafeUserData has now been configured. The data was saved and is available at the user interface. The block signature was updated.

Note

Acceptance test required

At the end, carry out an acceptance test in SINUMERIK Operate (or in the SINUMERIK ONE Commissioning Tool).
