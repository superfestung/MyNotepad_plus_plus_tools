Control Unit machine data in Safety Integrated

The Control Unit parameters are listed in the "Control Unit MD" window and can be changed.

Activating or deactivating the commissioning mode

Use the "Control Unit +" or "Control Unit -" softkey to select the Control Unit that you want to switch to the startup mode.

Searching for machine data

 

1.

Press the "Search" softkey, enter the machine data number being searched for in the window that opens or a character string that is being searched for an press the "OK" softkey.

 

2.

Press the "Continue search" softkey if the text position found does not correspond to the required machine data.

 

3.

Press the "Go to start" or "Go to end" softkey to move the cursor to the beginning or to the end of the displayed data.

 

4.

Press the "Search" softkey if you wish to change the search term.
