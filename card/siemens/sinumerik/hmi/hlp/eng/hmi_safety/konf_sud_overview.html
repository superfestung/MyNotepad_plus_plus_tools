SafeUserData - Block overview

All of the SafeUserData instances that have been created are listed in screen form "SafeUserData - Overview".

Note

The sort sequence of the SUD instances in the "SafeUserData - Overview" is based on the "instanceid" set in the TIA project.

Call

MENU SELECT > Commissioning > Menu forward key > Safety > SafeUserData

Information

Name

Description

Block name

The "User interfaces" of SafeUserData are listed in this column. This may also include "Nested blocks".

Nested blocks are DBs whose values are managed in the overall DB. Nested blocks have names with dots in the name.

Data type

The data type of the respective "user interface" is displayed in this column.

Signature

The actual signature of the SafeUserData instance is displayed in this column.

Vertical softkeys

Name

Description

Export

This softkey opens the storage location for exporting all SUD current values. Here, you can define an appropriate name for the export file and start the export.

Import

Using this softkey, for the import of all SUD actual values, the storage location for import files is opened. Here, you can select the required import file and start the import.

Details

Using this softkey, you switch to screen form "SafeUserData - Details".
