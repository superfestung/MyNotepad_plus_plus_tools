Safe Brake Ramp (SBR)

The Safe Brake Ramp (SBR) function provides a safe method for monitoring the braking ramp. The SBR is used to monitor braking with the functions "SS1 with/without encoder," "SLS without encoder," SS2 and STOP B/STOP C (for safety with encoder).

Note

Precondition for parameterization

The commissioning mode must be activated so that the parameters of this safety function can be changed. Further, parameter p9507.3 must have been parameterized = 0.

You can make the following settings for the active drive in this window:

  • Delay time (p9582)

    Set the delay time in ms for monitoring the braking ramp.

  • Monitoring time (p9583)

    Sets the monitoring time to determine the braking ramp in s.

  • Reference velocity (p9581)

    Sets the reference value to determine the braking ramp in mm/min for linear axes or rpm for rotary axes.

The following settings are displayed:

  • Ramp-down time (OFF3)

    Setting the ramp down time

  • Maximum velocity (linear axis)/maximum speed (rotary axis/spindle)

    Setting the highest possible speed/velocity.

Parameter list

When required, each setting in this dialog screen form can also be made via a parameter list. You can access the parameter list using the ">>" softkey followed by the "Parameter list" softkey.
