Safety Integrated overview - checksums

In the "Overview-checksums" window, all of the safety reference and actual checksums are displayed for the axes and drives. Status symbols are used to flag differences between reference and actual checksums:

  • Green: Reference and actual checksums are identical

  • Red: Reference and actual checksums are different

The checksums are read from the SI parameters. An overview is provided below:
