Calling an acceptance test

SINUMERIK Operate uses an acceptance test wizard, which uses specific input screens to guide the person authorized by the machine manufacturer through the acceptance test. You can test the correct mode of operation of all Safety Integrated Functions in the "Acceptance test" window. The results are displayed in a tabular form.

After all of the relevant data has been entered and the test cases are concluded, a report template is generated, which must be completely filled in, checked, and signed. A corresponding log is saved as an RTF file.

Note

Need for a new acceptance test

If you carry out certain measures after the acceptance test, this may make a new (full or partial) acceptance test necessary. This includes the following measures:

  • Safety-related function extensions

  • Transfer of the commissioning to other series machines

  • Hardware changes

  • Software upgrades

  • Changes as part of modular machine concepts
Note

Detailed information about the acceptance tests

You can find detailed information about the general requirements, structure and content of the complete acceptance test in the "SINUMERIK Operate Acceptance Test (ONE)" Function Manual.

Calling an acceptance test

 

1.

Select the "Setup" operating area.

 

2.

Press the menu forward key and the "Safety" softkey.

 

3.

Press the "Acceptance test" softkey.

The acceptance test window is then displayed:

  • The test types that are available are displayed in the left-hand column.

  • The setting area is displayed in the right-hand column for every test type. The tests are parameterized in the setting area.

  

Navigation using the following softkeys

  

Press the "First test" softkey to display the first acceptance test.

Note:

The softkey is only displayed if you selected "Overview" in the left-hand column.

  

Press the "Next test" softkey to display the following acceptance test.

  

Press the "Previous test" softkey to display the previous acceptance test.

  

In the "Finish" setting area, click on softkey "Exit the SI acceptance".

This completes and closes the "Acceptance test" function.
