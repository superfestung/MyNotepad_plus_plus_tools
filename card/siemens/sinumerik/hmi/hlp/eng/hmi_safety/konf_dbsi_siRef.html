Safe referencing

The "Safe referencing" function allows a safe absolute position to be defined. This safe position is used for the following functions:

  • SLP (Safely-Limited Position)

  • SP (Safe Position)

  • SCA

Precondition

The commissioning mode must be activated so that the parameters of this safety function can be changed. An extended function must be set in the options.

Note

User agreement can be also be set without SI commissioning mode.

Settings

You can make the following settings for the drive in this window:

  • Enable referencing via SCC (p9501.27)

    Sets the enable signals for the safety-related motion monitoring functions. "Enable referencing via SIC/SCC" can be activated using the checkbox.

  • SI Motion user agreement select/deselect (p9726)

    User agreement Safe Position

    (is only displayed if functions with Safe Position evaluation are activated)

  • Tolerance of the position actual values (p9544)

    For an incremental encoder, the actual values are checked after referencing, for an absolute encoder, when switching on.

The following LEDs can display the status (blue = active; gray = inactive):

  • Position referenced (r9723.17)

  • Safely referenced (r9722.23)

Parameter list

When required, each setting in this dialog screen form can also be made via a parameter list. You can access the parameter list using the ">>" softkey followed by the "Parameter list" softkey.
