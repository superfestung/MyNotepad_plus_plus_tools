SI safe cam

In the "SI safe cams" diagnostic screen, the following information about the Safe Cam (SCA) safety function of the selected drive is displayed:

  • Status of the function (SCA)

    Indicates for the selected drive whether the Safe Cam (SCA) safety function is selected, active and valid. In addition, the actual position is displayed (load-side actual value on the CU).

    Symbol

    Meaning

     

    Function is inactive

     

    Function is active

     

    SCA function is not valid. Safe homing was not performed.

  • Status of the enabled cams

    For each configured and enabled cam, shows the identifier, the minus and plus positions - as well as the validity of the cam position.

    Symbol

    Meaning

     

    Cam position is reached and the cam is invalid.

     

    Cam position is reached and the cam is valid. Safe homing was performed.

Note

Changing the view

In this diagnostics view, you can display the signal information of various drives. You can select the required drive using softkeys "Drive-", "Drive +" or "Select drive".
