Safe Stop 2 Extended Stop and Retract (SS2ESR)

The "Safe Stop 2 Extended Stop and Retract (SS2ESR)" safety function is used to brake the motor safely. The drive does not automatically brake the motor, but instead follows the specified speed setpoint. This can also result in fast retraction motion. During the delay time p9554, the brake ramp (SBR) and the acceleration (SAM) are not monitored, and there is no standstill detection. SOS becomes active after the delay time p9554 has expired.

Precondition

The commissioning mode must be activated so that the parameters of this Safety Integrated Function can be changed. The SS2ESR Safety Integrated Function is only available for the selected drive if an extended function has been previously set and parameterized in the Options setting area.

Settings

You can make the following settings for the drive in this window:

  • SS2ESR enable (p9501.04)

    SS2ESR enable is activated via the parameter.

  • Delay time SS2ESR/STOP E -> SOS active (p9554)

    Sets the transition time from STOP E to "Safe Operating Stop" (SOS) in ms.

For the following parameters, the status symbols indicate the actual state (blue = active; gray = inactive):

Status/control signals:

  • Select SS2ESR (r9720.29)

    Control signal "De-select SS2ESR"

  • Select SOS (r9720.3)

    Control signal "Deselect SOS"

Status signals:

  • SS2ESR active (r9722.27)

    Feedback "SS2 active"

  • STOP E active (r9721.15)

    Checkback signal "STOP E"

  • STOP C active (r9721.13)

    Feedback "STOP C active"

  • STOP D active (r9721.14)

    Feedback "STOP D active"

  • SOS active (r9722.3)

    Checkback signal "SOS active"

Parameter list

When required, each setting in this dialog window can also be made via a parameter list. You can access the parameter list using the ">>" softkey followed by the "Parameter list" softkey.
