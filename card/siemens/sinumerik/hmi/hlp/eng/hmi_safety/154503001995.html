Examples of how the safety functions can be applied

Safety function

Application examples

Possible solution

STO

When an Emergency Stop button is pressed, the drive must be braked as quickly as possible and transitioned into STO

Selecting STO in the inverter via terminal strip using an Emergency Stop button.

A central Emergency Stop button ensures that several drives cannot unintentionally start.

Evaluating the Emergency Stop button in a central control, selecting STO in the inverter viaPROFIsafe.

SS1

It is only permissible to open a protective door if the motor torque has been switched off.

Select SS1 in the inverter via a failsafe input or via PROFIsafe.

After pressing the Emergency Stop button, the drive is braked along the braking ramp and then brought into a torque-free condition.

SOS

The standstill position of the motor must be monitored and ensured.

Select SOS, e.g. using SS2, in order to monitor the standstill position of the motor after braking.

SLS

After opening a protective door, the machine operator must enter the machine and in the hazardous zone, operate a horizontal conveyor belt running at a slow speed using an acknowledgment button.

Selecting SLS in the inverter via terminal block. The inverter limits and monitors the velocity of the horizontal conveyor.

A spindle drive must not exceed a specific maximum speed, depending on the selection of the cutting tool.

Selecting SLS and the corresponding SLS level in the inverter via PROFIsafe.

SSM

It is only permissible to open a protective door when a motor is at standstill.

Selecting SSM in the inverter; the inverter safety monitors the speed and releases the process to proceed via PROFIsafe using the status bit "Status SSM".

SDI

It is only permissible to open a protective door if a drive moves in the safe direction (away from the operator).

Selecting SDIin the inverter; enable the protective door via status bit (PROFIsafe) of the inverter.

When replacing the plates of the pressure cylinders, the drive must only move in the safe direction of rotation.

Selecting SDI in the inverter. Disabling the hazardous direction of rotation.

After a protective device to detect a jammed door responds, a rolling shutter gate may only start to move in one direction.

When a crane trolley is at the operating limit switch then it may only start in the opposite direction.

SLP

The drive may not exit the specified position ranges.

Selection of SLP in the inverter; inhibits the range that is not permitted.

SBT

The effect of a brake is reduced through wear.

The "Safe Brake Test SBT" function determines whether a brake can provide the required braking effect.
