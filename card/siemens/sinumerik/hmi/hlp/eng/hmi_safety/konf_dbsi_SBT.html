Safe Brake Test (SBT) diagnostic function

The "Safe Brake Test" function (SBT) checks the holding torque of a brake (operational or holding brake). The drive purposely generates a configurable torque against the applied brake. If the brake is operating correctly, the axis motion remains within a parameterized tolerance. However, if more significant axis motion is identified from the encoder actual values, the brake is not in a position to provide the specified holding torque. The brake must now be serviced or replaced.

Requirement

The commissioning mode must be activated so that the parameters of this safety function can be changed. The SBT safety function is only available for the selected drive if, in options, an extended function with encoder was previously set and parameterized.

Commissioning screen "Safe Brake Test - SBT"

You can make the following settings for the drive in window "Safe Brake Test - SBT":

Activation area:

  • Enable SBT (p10201.1)

    Setting to enable the safe brake test. "Enable safe brake test" can be activated using the checkbox.

  • Motor type (p10204)

    Selecting the motor type for the safe brake test.

    • 0: Rotary

    • 1: Linear

  • SBT selection (p10203)

    Selecting the control of the safe brake test.

    • 0: SBT via SCC (p10235)

    • 1: SBT via BICO (p10230)

    • 2: SBT for test stop selection (p9705/p10250.8)

    For integrated drives (CU_I, CU_NX), only "SBT via SCC" is permissible.

  • Positioning response (MD36968.2 $MA_SAFE_BRAKETEST_CONTROL, bit 2)

    Selects the positioning response at the end of the brake test.

    • 0: Positioning at the actual axis position

      To conclude the brake test, the actual axis position for the next path motion is used. (MD36968 $MA_SAFE_BRAKETEST_CONTROL bit 2 = 0). This results in motion to the last programmed axis position on the path.

    • 1: Positioning to the last programmed axis position

      Acceptance of the actual axis position is not carried out. After the axis has been pressed against the brake, the closed-loop position controller withdraws the axis to the last programmed position.

      Vertical axes can thus be prevented from sagging as a result of a repeated brake test. It is also possible to briefly exceed the software limit switch while pressing against the brake without initiating an alarm.

Range brake 1/2:

  • Select brake

    Selecting the brake to be tested.

    • Brake 1 (p10202[0])

    • Brake 2 (p10202[1])

    The following values refer to the brake set here.

  • Brake test

    Defines whether and which brake should be tested.

    • 0: Inhibit

    • 1: Test motor holding brake

    • 2: Test external brake

  • Build-up of the test torque

    Setting the time in ms in which the test torque is ramped up against the closed brake. After the safe brake test, the test torque is ramped-down again.

    • Brake 1 (p10208[0])

    • Brake 2 (p10208[1])

  • SBT brake holding torque

    Setting the effective holding torque on the motor side of the brake to be tested Nm.

    • Brake 1 (p10209[0])

    • Brake 2 (p10209[1])

  • Test torque factor

    Sets the factor for the test force of test sequence 1 or 2 for the safe brake test. The factor is referred to the brake holding force (p10209).

    • Test frequency 1, brake 1 (p10210[0])

    • Test frequency 1, brake 2 (p10210[1])

    • Test frequency 2, brake 1 (p10220[0])

    • Test frequency 2, brake 2 (p10220[1])

  • Calculated test torque (Mt)

    Value calculated from "SBT holding torque of the brake" and "Test torque factor".

  • Test duration

    Sets the test duration in ms for test sequence 1 or 2 for the safe brake test.

    • Test frequency 1, brake 1 (p10211[0])

    • Test frequency 1, brake 2 (p10211[1])

    • Test frequency 2, brake 1 (p10221[0])

    • Test frequency 2, brake 2 (p10221[1])

  • Position tolerance

    Sets the tolerated position deviation for test sequence 1 or 2 for the safe brake test in mm for a linear axis or in degrees for a rotary axis.

    • Test frequency 1, brake 1 (p10212[0])

    • Test frequency 1, brake 2 (p10212[1])

    • Test frequency 2, brake 1 (p10222[0])

    • Test frequency 2, brake 2 (p10222[1])

Parameter list

When required, each setting in this dialog screen form can also be made via a parameter list. You can access the parameter list using the ">>" softkey followed by the "Parameter list" softkey.
