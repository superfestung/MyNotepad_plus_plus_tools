Creating a new dressing program

You can create dressing programs and then render G code blocks for them in a directory/workpiece.

Procedure

 

1.

Select the "Program manager" operating area.

 

2.

Select a storage location and place the cursor on the "Dressing programs" folder.

 

3.

Press the "New" softkey.

  

The "New Dressing Program" window opens.

The "DRS" file type is set as default.

 

4.

Enter the desired program name and press the "OK" softkey.
