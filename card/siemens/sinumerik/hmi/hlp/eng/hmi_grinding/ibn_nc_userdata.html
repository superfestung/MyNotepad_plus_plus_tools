NC user memory for programs and data

The "NC user memory for programs and data" window can be used to change, search and display the saved user variables and data.

Set up here the initial memory setting for important areas. Here, you see at a glance the memory allocation, make optimizations of the memory assignments for certain topics and detect the available memory space.

NC user data

You obtain a complete list of the user data that you can filter for the following areas:

  • Global user data (GUD)

  • Local user data (LUD)

  • Macros

  • R parameters

  • Basic tools

  • Tool management

  • Frames

  • Compensations

  • Curve tables

  • Protection zones

  • Collision avoidance

  • Cycles

  • File system

  • REORG

  • Interpolation buffer

  • Execution of external

  • Synchronized actions

  • Channel

  • Axis

  • DRAM

  • SRAM

Procedure

 

1.

Select the "Start-up" operating area.

 

2.

Press the "NC" and "NC memory" softkeys.

 

 

The "NC User Memory for Programs and Data" window opens. A list of all user memory data appears.

 

3.

Press the "Filter/search" softkey to activate the "Filter" selection list.

 

4.

Select the desired data range (e.g. tool management) from the selection list.

 

5.

Press again the "Filter/search" softkey to activate the "Search" selection list.

 

6.

Enter the desired search term.

The hits found in the displayed user memory data matching the selection made for "Filter" are listed.

 

7.

Press the "Delete search" softkey if you want to start a new search.
