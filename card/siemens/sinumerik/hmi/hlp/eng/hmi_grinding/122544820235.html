Select teach in mode

Change to Teach in mode to adapt the current program.

Requirement

"AUTO" mode: The program to be edited is selected.

"MDI" mode The program to be edited is loaded into the MDI buffer.

Procedure

 

1.

Select the "Machine" operating area.

 

2.

Press the <AUTO> or <MDA> key.

 

3.

Press the <TEACH IN> key.

 

4.

Press the "Teach prog." softkey.
