You can use the configuration file to specify the data to be displayed in the "Additional Data" window.

 

Machine manufacturer

Please refer to the machine manufacturer's specifications.

Procedure

 

1.

The tool list is opened.

 

2.

In the list, select an appropriate tool, e.g. an angle head cutter.

 

3.

Press the "Additional data" softkey.

The "Additional Data - ..." window opens.

The "Additional data" softkey is only active if a tool for which the "Additional Data" window is configured is selected.
