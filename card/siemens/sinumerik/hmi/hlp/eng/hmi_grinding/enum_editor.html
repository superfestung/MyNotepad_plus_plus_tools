Editing enum values

For drive parameters, you have the option of editing enum elements (e.g. r2, p3, p4, p97 etc.) using a selection list.

Procedure

 

1.

Select the "Start-up" operating area.

 

2.

Press the "Mach. data" softkey.

 

3.

Select the desired drive parameter area, e.g. "Control unit MD".

 

4.

Select the desired parameter, e.g. "p3" and position the cursor at the entry to be changed, e.g. "[1] Display, Si...".

 

5.

Press the <SELECT> or <INSERT> key.

A window containing a list of all defined enum values and their descriptions opens.

 

6.

You can select a value from the list.

 

 

- OR

 

 

You can also directly enter the value as decimal number into the brackets [ ].
