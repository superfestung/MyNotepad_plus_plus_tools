Creating templates

You can store your own templates to be used for creating part programs and workpieces. These templates provide the basic framework for further editing.

You can use them for any part programs or workpieces you have created.

Storage location for templates

The templates used to create part programs or workpieces are stored in the following directories:

HMI Data/Templates/Manufacturer/Part programs or Workpieces

HMI Data/Templates/User/Part programs or Workpieces

Procedure

 

1.

Select the "Start-up" operating area.

 

2.

Press the "System data" softkey.

 

3.

Position the cursor on the file that you wish to store as a template and press the "Copy" softkey.

 

4.

Select the directory in which you want to store the data - "Part programs" or "Workpieces" - and press the "Paste" softkey.

 

 

Stored templates can be selected when a part program or a workpiece is being created.
