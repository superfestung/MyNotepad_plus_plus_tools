Creating kinematic elements

Offset

 

1.

Mark the "Active data" folder and press the "New element" softkey.

A selection window with kinematic elements opens.

 

2.

Select the "Offset" kinematic element and press the "OK" softkey.

The appropriate input window for the element values opens in the lower part of the tree view.

 

3.

Enter the required values:

  

Name:

ROOT

  

Offset (X):

0

  

Offset (Y):

0

  

Offset (Z):

0

 

4.

Press the "OK" softkey to save the values.

  

- OR -

 

 

Press the "Cancel" softkey to close the input window.
