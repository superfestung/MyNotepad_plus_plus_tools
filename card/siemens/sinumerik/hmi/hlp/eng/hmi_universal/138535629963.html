Work station safety

In order to secure machines against manipulation and protect people from accidents, proceed as follows when leaving the work station:

 

1.

Set the keyswitch to 0 and then remove it.

 

2.

Press the "Delete password" softkey.

Access authorization is then initiated.

You can reset the password when you return to the work station.

You can find more information about access levels and creating passwords in the SINUMERIK Operate Commissioning Manual.
