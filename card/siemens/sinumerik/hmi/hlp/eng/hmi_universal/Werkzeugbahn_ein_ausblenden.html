Showing and hiding the tool path

The path display follows the programmed tool path of the selected program. The path is continuously updated as a function of the tool movement. The tool paths can be shown or hidden as required.

Procedure

 

1.

The simulation or the simultaneous recording is started.

 

2.

Press the ">>" softkey.

The tool paths are displayed in the active view.

 

3.

Press the softkey to hide the tool paths.

The tool paths are still generated in the background and can be shown again by pressing the softkey again.

 

4.

Press the "'Delete tool path" softkey.

All of the tool paths recorded up until now are deleted.
