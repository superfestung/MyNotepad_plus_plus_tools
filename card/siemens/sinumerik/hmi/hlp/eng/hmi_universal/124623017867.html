Displaying the softkey

The softkeys on the user interface for calling the program lists are activated via general configuration machine data.

Configuring a softkey

MD51041 $MNS_ENABLE_PROGLIST_USER

0

No softkey

1

The "Prog. list" softkey is displayed

MD51043 $MNS_ENABLE_PROGLIST_MANUFACT

0

No softkey

1

The "MANUFACTURER" softkey is displayed
