Editing user memory

The list shows the maximum assigned memory data as well as the currently used data for the complete user memory data.

You can change the memory assignments and so, for example, create free memory space for other user data.

Procedure

 

1.

The "NC User Memory Data for Programs and Data" window opens.

 

2.

Position the cursor in the "Assigned" column of the desired user data whose memory assignment you want to change and enter a new value.

The field with the modified value is highlighted in yellow. If you entered an illegal value, the field is highlighted in pink.

In addition, the status bar shows information about the changed assignment.

 

3.

Press the "Use all" softkey to save the values.

 

 

- OR -

 

 

Press the "Cancel all" softkey to discard your input.
