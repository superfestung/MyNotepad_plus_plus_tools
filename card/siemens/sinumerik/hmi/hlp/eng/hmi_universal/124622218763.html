Simulating the program block by block

You have the capability of controlling the program execution during the simulation, i.e. to execute a program, e.g. program block by program block.

Procedure

 

1.

Simulation is started.

 

2.

Press the "Program control" and "Single block" softkeys.

 

3.

Press the "Back" and "Start SBL" softkeys.

The pending program block is simulated and then stops.

 

4.

Press "Start SBL" as many times as you want to simulate a single program block.

 

5.

Press the "Program control" and the "Single block" softkeys to exist the single block mode.

Switching a single block on and off

   

Press the <CTRL> and <S> keys simultaneously to enable and disable the single block mode.
