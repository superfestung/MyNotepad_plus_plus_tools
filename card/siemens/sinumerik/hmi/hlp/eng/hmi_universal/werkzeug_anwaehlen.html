Selecting a tool

Procedure

 

1.

Select the "JOG" operating mode.

 

 

 
 

2.

Press the "T, S, M" softkey.

 

3.

Enter the name or the number of the tool T in the input field.

 

 

- OR -

 

 

Press the "Select tool” softkey.

The tool selection window opens.

Place the cursor on the desired tool and press the "OK" softkey.

The tool is transferred to the "T, S, M... window" and displayed in the field of tool parameter "T".

 

4.

Select tool edge D or enter the number directly in field "D".

 

5.

Select the sister tool ST or enter the number directly into the "ST" field

 

6.

Press the <CYCLE START> key.

The tool is loaded into the spindle.
