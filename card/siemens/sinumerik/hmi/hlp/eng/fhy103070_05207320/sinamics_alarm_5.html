F207815 <location>Drive: Power unit has been changed

Reaction: NONE

Number of the incorrect parameter.
See also: r0200 (Power unit code number actual), p0201 (Power unit code number)

For inverters, the following applies:
If the new power unit is accepted, then if required, the current limit (p0640) can be reduced by a lower maximum current of the power unit (r0209) (torque limits stay the same).
If not only the power unit is changed, but also the motor, then the motor must be re-commissioned (e.g. using p0010 = 1). This is also necessary if motor data is still to be downloaded via DRIVE-CLiQ.
See also: r0200 (Power unit code number actual)
