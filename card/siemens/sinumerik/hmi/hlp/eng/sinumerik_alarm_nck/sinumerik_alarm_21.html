26380[Channel %1: ] Block %2: Incorrect AFIS configuration for axis %3 identifier %4
Parameters:%1 = Channel number
%2 = Block number, label
%3 = Axis
%4 = Error code
Definitions:Automatic filter switching (AFIS) is active, but with an incorrect configuration for an axis.
1: The axis is configured as AFIS axis, but no jerk filter is activated by MD32400 $MA_AX_JERK_ENABLE. The jerk filter must always be active for AFIS axes.
2: The axis is configured with only one filter chain (e.g. MD32402 $MA_AX_JERK_MODE=0x). Two filter chains must always be configured for AFIS axes.
3: The axis has the configuration MD32332 $MA_AFIS_ENABLE=1 and MD32402 $MA_AX_JERK_MODE=5x. This configuration is not allowed for a channel axis in a channel with MD20630 $MC_AFIS_MODE=1.
Reaction:- Alarm display.
- Interface signals are set.
- Interpreter stop
Remedy:1: Activate the jerk filter for the axis with MD32400 $MA_AX_JERK_ENABLE
2: a: Parameterize two filter chains for the axis with MD32402 $MA_AX_JERK_MODE
b: Deactivate AFIS for the axis: MD32332 $MA_AFIS_ENABLE = FALSE
3: Change values of MD32332 $MA_AFIS_ENABLE, MD32402 $MA_AX_JERK_MODE or MD20630 $MC_AFIS_MODE
Program Continuation:Clear alarm with the RESET key. Restart part program
