Block A/SE: Axis-specific setting data

This block contains the axis-specific setting data
 
AA_OFF_LIMIT Compensation value upper limit $AA_OFF with clear SD 43350: $SA_AA_OFF_LIMIT
Column index:
43350 		Line index:
 Max. line index:
Upper limit of compensation value which can be preset by means of synchronized actions via the system variable $AA_OFF.
This limit value acts on the absolutely effective compensation value via $AA_OFF.
It is possible to interrogate the compensation value for limit-range violation via the system variable $AA_OFF_LIMIT.
Type
TYPE_DOUBLE 		Unit
No unit 		Default Value
 Minimum Value
 Maximum Value
