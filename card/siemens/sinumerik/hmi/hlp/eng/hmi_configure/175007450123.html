Configuring start and stop triggers for the NC/PLC trace

Use the start and stop trigger settings to be able to automatically trace the sequence for certain events with defined NC/PLC conditions.

Requirement

An NC/PLC trace has been set up.

Procedure

To open the trigger settings, click on the "Start trigger" or "Stop trigger" softkey.

The dialog for the start trigger or stop trigger settings opens.

You have the following recording options:

Start recording

Manual

Recording is started immediately after pressing the "Start trace" symbol.

Through trigger condition

The trace session is activated using a variable. You obtain a selection of variables by pressing the "Variable list" softkey. Variables are selected using the "Select" softkey.

Select the parameter (e.g. axis, channel, numeric index) for the trigger variable.

After the session has been activated, define under which condition and with which value trace recording should be started.

The following conditions are listed:

=, ≠, >, ≥, <, ≤, ascending, descending, changed

Stop recording

Manual

Recording is immediately stopped after pressing the "Stop trace" symbol.

Through trigger condition

The trace session is activated using a variable. You obtain a selection of variables by pressing the "Variable list" softkey. Variables are selected using the "Select" softkey.

Select the parameter (e.g. axis, channel, numeric index) for the trigger variable.

After the session has been activated, define under which condition and with which value trace recording should be stopped.

The run-on time (post-trigger time) defines in seconds how long recording continues to run after the stop trigger event.

The following conditions are listed:

=, ≠, >, ≥, <, ≤, ascending, descending, changed

Only measurement data keep the following duration

If the trace recording is stopped manually or with a trigger condition, then it is possible to configure a ring buffer. This means, only the latest part of the measured data is retained up to the given time limit and the data recorded earlier is discarded.

The ring buffer is active if the "Retain only measured data of the following duration" check box is selected and the specified time limit is greater than 0.

Time unit: Seconds, minutes, hours

After the specified time

Recording stops automatically when the specified time has elapsed.

Time unit: Seconds, minutes, hours

Default setting: 10 seconds

Note

If the trace configuration contains only PLC variables traced with the non-cyclic "OB1" event, then the run-on time is ineffective, and the recording is immediately stopped when the stop trigger is detected.

Note

If a new trace session file is loaded, time values (ring buffer and "after specified time") is displayed in the time unit which was last selected from the selection list for the time unit. The time unit is not saved in the session file.
