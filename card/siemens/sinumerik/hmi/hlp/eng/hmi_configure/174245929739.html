Troubleshooting

Error cases and possible solutions

The following table contains possible errors that can occur when pairing the NCU with the IPC:

Error message

Cause of the error

Possible solution

No NCU has been configured via SINUMERIK Operate.
OR

No NCU is configured in the "mmc.ini" file.

Error message in SINUMERIK Operate.

  • Configure the NCU via SINUMERIK Operate.
    OR

  • Configure the NCU in file "mmc.ini".

See Chapter Overview and status

Incorrectly configured NCU via SINUMERIK Operate
OR

Incorrectly configured NCU in the file "mmc.ini".

Despite the configuration, no NCU appears in the "Pairing" dialog of SINUMERIK Operate.

  • Check the correct configuration via SINUMERIK Operate
    OR

  • Details of the NCU must be configured correctly in file "mmc.ini".

See Chapter Overview and status

NCU pairing certificates cannot be accessed

Error message in SINUMERIK Operate

  • Check whether the connection with the NCU is correct.

  • If the connection is correct and the error still exists, the NCU may be defective. Reset the NCU to the factory settings.

The NCU cannot create pairing certificates

Pairing status in SINUMERIK Operate shows "Not in pairing mode"

  • Status is displayed when the installed software version of the NCU is too old so that pairing cannot be performed as a function. Update the software version of the NCU.

  • If the software version of the NCU is up to date, the security host may have crashed or not have started. Start the security host manually via the command "sc start sec".

NCU was already paired with a different IPC. This is not the IPC with which it is connected. Pairing cannot be performed.

Pairing status in SINUMERIK Operate shows "Paired to different IPC"

  • Connect the NCU to the originally paired IPC

  • Or reset the NCU to its factory settings and pair this with the currently connected IPC.

The date on the IPC and NCU do not match. Pairing cannot be performed.

Error message in SINUMERIK Operate

  • Synchronize the data of the IPC and NCU. You can set the date of the NCU in SINUMERIK Operate via "Setup > HMI > Date Time".

Connection problems between the IPC and NCU. Pairing or unpairing cannot be performed.

Error message in SINUMERIK Operate

  • Check the correct connection of the IPC and NCU

  • Perform pairing or unpairing again.

The manufacturer password for authentication is not correct.

Error message in SINUMERIK Operate

  • Enter the correct manufacturer password.
