View certificate

Introduction

Use dialog "View certificate" if you wish to view the details of a selected certificate.

Requirement

  • With active user management: You have role "Security Admin" or access level 3 (user).

OR

  • Without active user management: You have at least access level 3 (user).

Procedure

  1. In the "Setup" operating area, call the certificate store via the "Security > Certificate store" softkeys.

  2. Select a certificate that is saved in the certificate store and click on "View certificate". A detailed display of the certificate opens. You can see the following data in the upper part of the display:

    • The certificate meta data, which was assigned when importing.

    • The certificate status: you can see whether the certificate is valid or whether it has already expired.

    • The certificate path shows the hierarchy of the certificate string. The Common Name of the certificate is displayed if it involves an identity certificate. If the Common Name in the certificate is not found or if a hierarchy of the certificate string is empty, then "certificate" is displayed as text. In the hierarchy, you can also select a higher hierarchy level.

    You can see the internal data, which were read out of the certificate itself, in the lower part of the display. These internal data cannot be edited.

By clicking on "Close", you can reclose the detailed display of the certificate.

Reference

You can execute the following actions in dialog "View certificate".

  • Delete certificate Action "Delete certificate" removes the certificate/the certificate string file.
