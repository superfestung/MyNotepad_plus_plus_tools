Disconnecting central end user domains

Requirement

  • User management is activated.

  • The local domain for the manufacturer is set up. The security administrator for the manufacturer is set up.

  • The local domain for the end user is set up. The security administrator for the end user is set up.

  • The end user domain was connected with a central user management.

  • You have the "Security admin end user" role.

Procedure

  1. In the "Setup" operating area, call the user management via the "Security > User management" softkeys. The overview page for user management opens in the "Configuration" softkey.

  2. Click on the "Disconnect" button next to the central end user domain.

  3. Confirm the confirmation prompt with "OK".

Result

You have successfully disconnected the end user domain from the central user management.
