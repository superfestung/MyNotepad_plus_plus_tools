Configuration parameters

Enter the following parameters in SINUMERIK Operate:

Entry

Meaning

Select camera

Select or enter the camera of your choice.

Camera name

Enter the camera name.

You can choose an appropriate name that refers, for example, to the area monitored by the camera.

IP address

Enter the IP address of the camera.

The IP address must be static and in the same subnet as the SINUMERIK NCU.

The camera is connected via the X120 interface of the NCU and uses the plant network.

Profile

Enter the streaming profile of the camera (manufacturer-independent)

Port

Enter the RTSP port number (1 to 65536).

User name

Enter the user name for access to the camera.

Password

Enter the password for access to the camera.

The password is displayed encrypted as "*".
