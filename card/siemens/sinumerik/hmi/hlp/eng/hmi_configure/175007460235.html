Loading the trace session file

You have the option of loading already saved session data, in order to restart them or to display the trace recording from other systems.

Settings

Meaning

Variables, settings

You can restart a trace session that has already been created.

Variables, settings and recorded values

You can view the result of a trace session, change the variable and if required, restart.

Requirement

A session file of recorded traces is available from the following systems in the XML format.

  • SINUMERIK Operate

  • Analyze MyMachine /SINUMERIK Tuning Center

Note

A trace session that was created in another application and loaded in compatibility mode cannot be edited.

If the trace configuration does not match the control, the trace cannot be started. Adapt the trace configuration to the control so that you can then restart the recording.

Procedure

  1. The "Trace" function is selected.

  2. Press the "Load trace" softkey.

    The window "Load trace: Please select file" opens.

  3. Select the required session file from the appropriate directory, e.g. "Trace":

    Default path: .../user/sinumerik/hmi/data/trace/

    - or -

    Press the "Search" softkey to use dialog "Search file".

  4. Once you have selected a session file, press the "OK" softkey.

    The window "Load trace: session..." opens.

    Note

    If the loaded file does not contain any measured data, the session is opened directly.

  5. Select the check box "Variables, settings", if you wish to immediately start the trace session.

    - or -

    Select the check box "Variables, settings and recorded values", if you wish to display the trace session.

  6. Press the "OK" softkey.

    The data of the selected session are loaded and displayed in the "Trace" window.
