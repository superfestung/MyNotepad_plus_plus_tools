Move and Zoom modes

Toggling between "Move" and "Zoom" mode ("hand" and "rectangle" mode) is possible from the context menu or from the toolbar.

If "Move" mode is selected:

The basis mouse cursor has the shape of a hand. Using the hand-shaped cursor, click on an empty background area and drag this with the left-hand mouse key pressed: Shift the whole trace diagram, in the drag direction.

When the CTRL key is pressed, the mouse cursor changes to the "Zoom" mode as long as the key is kept pressed. This way it is possible to draw a "rectangle" in "Move" mode.

If "Zoom" is selected:

The basis mouse cursor is an arrow. Press the left mouse button on an empty background area and drag: Draw a "rectangle". When the mouse key is released, the diagram will be zoomed in to the area enclosed by the rectangle.

In both the "Move" and "Zoom" modes:

The mouse cursor changes to the pointing hand shape over signal curves and vertical scales, which can be dragged. Using the hand-shaped cursor, click on a signal curve and drag it with the left mouse button pressed: The signal curve is individually shifted. The other signals are not shifted and the active signal is not changed.

If the selected signal is part of a signal group, then all signals in the group are shifted together.
