Expert Trace features

The following functions are only available with an Expert Trace license:

  • The cursor table makes it possible to create up to ten individual cursors, assign cursors to different signals and create cursor groups.

  • FFT (Fast Fourier Transform) diagram, to display the frequency spectrum of signals.

  • Mathematical operations: Perform calculations on measured signals and create new signals based on mathematical formulas.

  • Saving measured data in a CSV file, in addition to the XML session file.

 

Software option

To be able to use this function, you need software option

"SINUMERIK Operate Expert Trace 6FC5800-0BS30-0YB0".
