The following table shows the different zoom options in the trace diagram:

Note

All zoom actions are only carried out in the selected direction(s). The setting as to whether all signals or only the selected signal / signal group should be zoomed is also taken into account.

Zooming with keyboard shortcuts that use the <Ctrl> key works not only when the trace diagram is in focus, but also when the associated legend widget or the signal table is in focus. Zooming with the single <+> or <-> keys works only when the trace diagram is in focus.

On a touch screen, you can zoom in or out with pinch gestures.

The "Zoom out to maximum signal range" function scales the signal groups so that the vertical minimum and maximum values for the groups remain uniform.
