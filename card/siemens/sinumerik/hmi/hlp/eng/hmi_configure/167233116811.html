Delete user groups

Introduction

If you want to delete a new user group, you can use the "Delete user groups" dialog.

Requirement

  • You have the role "Security admin manufacturer" or "Security admin end user".

Procedure

  1. In the "Setup" operating area, call the security rules of the user management via the "Security > User management" softkeys.

  2. Click the "User groups" softkey.

  3. In the overview page of user groups, select a user group you want to delete.

  4. Click "Delete".

  5. Confirm the confirmation prompt with "OK". The user group is removed from the list of all user groups.
