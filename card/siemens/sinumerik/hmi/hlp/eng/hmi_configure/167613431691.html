User logon

Introduction

You must log in as user if user management is activated.

Requirement

  • User management is activated.

  • The domain for the manufacturer is set up and the security administrator for the manufacturer is set up.

  • One or more machine roles have been created.

  • One or more user groups have been created and assigned to the machine roles.

  • One or more users have been created and assigned to the user groups.

  • You have any local or central user role in the user management.

Procedure

  1. Click on the login button at the top right in SINUMERIK Operate and select "Log in" in the context menu. The login dialog opens.

  2. Select the manufacturer domain or the end user domain, depending on which domain your account belongs to.

  3. Enter your defined user name.

  4. Enter your defined password.

    Notice

    Hiding/seeing passwords

    You can hide the password or make it visible using the eye icon.

    • Carefully ensure that no other person can see your personal password if you switch to the readable password view.

  5. Confirm with "OK".

Result

You are logged on to SINUMERIK Operate as a user and have the rights defined according to your user group/machine role.

The login name displayed below the login icon is the selected user name, which may be abbreviated to save space.

System use notification

You have the option of displaying a system use notification in the upper part of the login dialog of the user management. The system use notification can include additional notes, restrictions of use or declaration of consent for all users that log onto SINUMERIK Operate via the user management. You can activate/deactivate the system use notification via the guidelines dialog.
