Overview of machine roles

Machine roles are intended to combine roles from different parts of the system (NCK, PLC, SINAMICS, OA applications, WinCC Unified, etc.) at machine level into one machine-global role. You can assign a SINUMERIK access level (role) to a machine role.

Requirement

  • You have the "Security admin manufacturer" role.

List of machine roles

Machine roles

You can see all of the defined machine roles of the user management for the manufacturer domain in the list of machine roles.

The names and description of the machine role are listed as well as the assigned SINUMERIK access level.

Some standard machine roles are already predefined. You can use and/or change these:

  • Manufacturer machine role

  • Service machine role

  • Customer machine role

  • Key switch 0-3 machine role

  • Security Admin Manufacturer

Further, check box "May be used by end user" displays which machine roles are also released for users. The end user can only assign this role if the checkmark is set. Thus, the manufacturer can decide which rights the user can utilize on the machine.
