Key transfer

In normal operation, SINUMERIK Operate evaluates the keys of the operator panel directly and calls the associated functions. When you set up a VNC Viewer, you can define whether a key combination will be sent to the external system instead.

Configuring key transfer

Press the "Details" softkey to show the "Key Transfer" area and select the relevant check boxes.

When you press "Menu Select" an additional window opens in which you can select the key combination that you want to use.

Note

Be aware that, depending on the operator panel that you are using, not all keys will be available to you.
