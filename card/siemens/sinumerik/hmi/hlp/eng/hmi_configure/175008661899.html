Using the signal table

In the signal table, you can view or edit a trace recording configuration, or configure the display of signal curves in the trace widget.

Arrange window

  • The signal table can be opened and closed via the context menu, the toolbar or with the "Signal table" softkey, and can also be closed via the symbol on the window frame.

  • The signal table can be docked to the trace widget using the symbol or displayed on a PC/IPC system as a freely movable window. If the content of the grid cannot be displayed within the available height or width, then vertical and/or horizontal scrollbars are displayed automatically.

Arranging signals

  • The active signal in the trace diagram can be changed by:

    • Double-clicking on any cell in a line, if the cell contains textual data, for example signal name or signal address.

      The active signal is not changed when double-clicking a cell containing a dropdown list or a symbol.

    • Pressing the toggle key (Numpad 5, without NumLock, on a PC keyboard) on any cell in a line.

  • The lines in the grid can be rearranged by dragging the signal number. The order of signals in the configuration is saved in the trace session file and is restored when the file is reloaded.

  • The columns can be rearranged by dragging the column header. However, changes to the column arrangement are not saved with the trace configuration and are reset when the user interface is restarted.

  • If the application detects an error in the trace configuration, the grid cells involved turn red. These cells may also have a tooltip to show the cause of the error.
