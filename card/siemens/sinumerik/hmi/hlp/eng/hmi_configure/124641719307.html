Overview

You can integrate a camera of your choice into SINUMERIK Operate.

In this way, a camera image can be integrated without an additional display device.

  • The camera image allows you to track remote processes and monitor difficult-to-access areas.

  • You can also document and store machine states.

  • You can create and configure up to two cameras.

 

Machine manufacturer

Follow the machine manufacturer's instructions.

Note

A camera cannot be used for operating SINUMERIK Operate on NCU ("Embedded HMI") when using an NCU 710.

Requirement

The cameras used must fulfill the following criteria:

  • Frame rate: 12 fps

  • Supported network protocols: TCP/IP, UDP RTSP/RTP

  • Compression rates: H.264, H.265

  • Resolution, 1280 x 720 pixels

Before configuring the camera, you must appropriately take into account the following criteria:

  • Number of cameras in the system

  • SINUMERIK Operate on NCU or PCU/IPC

  • NCU type being used

Note

  • We recommend that a resolution of 1280 x 720 pixels and a frame rate of 12 fps are set. However, you can vary the video streaming quality depending on the application scenario.

  • Reduce the frame rate in order to achieve a better resolution.

  • Two connected cameras generate two streams, therefore occupying twice the network resources. To compensate for this, you can appropriately reduce the resolution and the frame rate.

Sequence of steps

  1. Connect the camera to your SINUMERIK device via the X120 interface.

  2. Configure the camera via the web interface as follows:

    • Enter a name for the camera profile, which is applied when the camera is used.

    • Define the appropriate parameters, e.g. resolution, frame rate, compression etc. - and assign these parameters to the camera profile.

    • Specify the profile names in all of the additional setting screen forms required.

  3. Enter the configuration parameters in SINUMERIK Operate.

Note

To ensure that the camera is listed or displayed in the hardware version image and in the TCP/IP diagnosis of SINUMERIK Operate, the camera must support the Simple Network Management Protocol "SNMPv1".
