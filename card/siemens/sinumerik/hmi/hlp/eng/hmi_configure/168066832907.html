Overview of user groups

Users are combined to form user groups. You can assign these user groups to machine roles and the associated SINUMERIK access rights.

Note

Only domain-specific editing possible

Please note that editing user groups is always domain-specific:

  • Log in as Security Admin Manufacturer: You can edit user groups of the manufacturer domain.

  • Log in as Security admin end user: You can edit the user groups of the end user domain.

Requirement

  • You have the role "Security admin manufacturer" or "Security admin end user".

List of user groups

User groups

In the list of user groups, you can see all of the defined user groups of the user management for the active domain:

  • Local user groups

  • Central user groups

The names and descriptions of the user groups are listed as well as the machine role that is assigned to each particular user group.

You have the option of starting the following actions using buttons:

  • Create local user group

  • Create central user group

  • Change user group

  • Delete user group
