Overview

With SINUMERIK Operate it is possible to access and operate external controllers or PCs via the VNC Viewer.

You start the VNC Viewer via a configured softkey in the operating area menu. If the sidescreen is activated, you can also call up the VNC Viewer via the navigation bar of the sidescreen.

Parameters which can be configured include the following:

  • Details about the connection

  • Display of the VNC Viewer in SINUMERIK Operate

  • Assignment to an operating area

  • Using the keys of the operator panel

  • Size and position of the picture detail

Note

The VNC Viewer only supports hextile encoding with color depths of 16 or 32 bits.

Requirement

A VNC server is set up in at least one system in the plant or corporate network.
