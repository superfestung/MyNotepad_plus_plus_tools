Defining channel groups in the "netnames.ini" file

Defining channel groups

All channel groups that are used at the system are defined in the "netnames.ini" file. Independent of the specific operator stations.

A separate section [CH_GRP_x] (1 <= x <= 32) should be created for every channel group.

The entries required for a channel group are as follows:

  • The text to be displayed in the channel for the channel group at the corresponding horizontal softkey.

  • The list of jump destinations summarized in this channel group, i.e. NC channels. The jump destinations are defined in a consecutively numbered list with the following entries:

    • Member y .Target: Jump destination

    • Member y .Text: softkey text to be displayed in the channel menu (1 <= y <= 8).

"netnames.ini" file

A channel group, e.g. channel group 1 is therefore specified as follows:

Value

Description

[chan HMI]

ShowChanMenu=true

Section to activate the channel menu function of the current user interface:

true: Activated

false: Deactivated

[T2M2N]

Channel groups horizontal softkeys (optional)

Correlation between channel group and horizontal softkeys in file "config.ini"

Section [T2M2N] can also be created in file "netnames.ini" instead of in file "config.ini".

This configuration is applicable for all user interfaces that are not configured via their "config.ini".

SK1=CH_GRP_1

SK2=CH_GRP_2

Assigning the channel group to a horizontal softkey

[CH_GRP_1]

Name of the channel group

This schematic must be applied for the notation. The groups are consecutively numbered at the last position.

Text=My_ONE_1

Softkey text My_ONE_1 for the channel group for display in the channel menu (horizontal softkey)

Switchover target 1 of channel group

Member1.Target= <HMI>.<NCU>.#<CHN_NR>

Example:

Member1.Target=
DESKTOP-CT0N.NCU_Berlin.#1

Computer name (host name) or the IP address of the computer on which the user interface to be selected runs (NCU or IPC):

<HMI> = IP address or name of the host (PC/IPC); <HMI> is not required if the user interface runs locally on the NCU

<NCU> = IP address or name of the NCU; alternatively, the machine name from "mmc.ini" can also be specified.

Optionally, using .#<CHN_NR>, the NC channel that should be taken when switching over can be specified.

Member<NR>.Text = NCU_ONE_CH1

Softkey text NCU_ONE_CH1 for the display in the channel menu (vertical softkey)
