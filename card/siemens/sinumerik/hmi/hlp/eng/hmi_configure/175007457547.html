Trace events

If a trace configuration is loaded from a session file, then also the trace events are loaded from the file.

If a new trace configuration is created in this application, the variables are traced with the following default settings:

  • Servo variables: Position control cycle (servo cycle)

  • NC variables: Interpolation cycle (IPO cycle)

  • PLC variables from the SINUMERIK ONE system: PLC cycle variable with OB1

  • PLC variables on the SINUMERIK 828D system: Interpolation cycle (IPO cycle)
