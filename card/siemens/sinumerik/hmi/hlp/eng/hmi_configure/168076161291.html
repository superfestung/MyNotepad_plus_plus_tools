Overview of users

You need users for personalized rights assignment. Users are combined to form user groups. You can assign these user groups to machine roles and the associated SINUMERIK access rights.

Only explicitly defined users are allowed to log on to SINUMERIK Operate when user management is active.

Note

Only domain-specific editing possible

Please note that editing users is always domain-specific:

  • Login as Security Admin Manufacturer: You can edit users of the manufacturer domain.

  • Login as Security Admin User: You can edit the users of the end user domain.

Requirement

  • You have the role "Security admin manufacturer" or "Security admin end user".

List of users

Users

You can see all of the defined users of the user management for the active domains in the list of users.

The user name and name are listed as well as the 1-n user groups, to which the respective users are assigned. It is also displayed whether Security Admin has locked a user. A locked user cannot log into the system.
