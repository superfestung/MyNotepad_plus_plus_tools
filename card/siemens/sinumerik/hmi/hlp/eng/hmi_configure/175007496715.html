Starting trace recording

Procedure

 

1.

You have selected the "Trace" function.

 

2.

Create a new trace configuration and select the parameters for each variable.

You can also load an existing Configuration.

 

3.

Click on the "Start trace" symbol in the toolbar.

If the recording has successfully started, then the recording time is displayed at the bottom of the trace window.

If there are any errors, the application displays a pop-up with an error message.

 

4.

You can wait for the recording to end or click on the "Stop trace" symbol in the toolbar.

The data that have been recorded are displayed in the trace diagram.

 

 

Note

Recorded data will not be saved if you stop a drive trace recording with the "Cancel trace" toolbar symbol.

You can also use the corresponding SINUMERIK Operate softkeys to start and stop a trace.

When a drive trace recording is started, the recording time displayed in the status bar includes not only the recording itself, but also the time needed to download the recorded data from the control.
