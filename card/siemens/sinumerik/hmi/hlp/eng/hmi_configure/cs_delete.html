Deleting a certificate

Introduction

Use dialog "Delete certificate" if you wish to delete a certificate from the certificate store. You can delete a certificate if it has expired, if it was incorrectly/inadvertently imported or if you no longer require the certificate.

Requirement

  • With active user management: You have role "Security Admin" or access level 3 (user).

OR

  • Without active user management: You have at least access level 3 (user).

Procedure

  1. In the "Setup" operating area, call the certificate store via the "Security > Certificate store" softkeys.

  2. Select a certificate that is saved in the certificate store and click on "View certificate". A detailed display of the certificate opens.

  3. Click on "Delete".

  4. Confirm the confirmation prompt again with "Delete".

Result

The certificate has now been deleted from the certificate store.
