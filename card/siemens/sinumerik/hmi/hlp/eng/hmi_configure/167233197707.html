Changing a user

Introduction

You can use the "Change user" dialog if you wish to change the user name, password, real name or the assignment to a user group for a user.

Note

User password forgotten?

If a user forgot his or her password, the Security Admin can also replace the old password with a new password in this dialog.

Requirement

  • You have the role "Security admin manufacturer" or "Security admin end user".

Procedure

  1. In the "Setup" operating area, call the user management via the "Security > User management" softkeys.

  2. Click the "User" softkey. On the overview page, click on "Change" to change a user.

  3. Make your changes regarding user name, name, password, assignment to machine roles or user groups or lock status. Click "OK" to apply the changes and close the dialog.

Result

The changes can be seen in the list of users.
