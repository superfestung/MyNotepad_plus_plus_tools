Changing the display area from left to right

In addition to the standard display on the left side, the sidescreen window can also be displayed on the right side of the operating area.

Procedure

  1. Copy the sample file "slsidescreen.ini" from directory
    /siemens/sinumerik/hmi/template/sidescreen.

  2. Place the copy in the directory
    /oem/sinumerik/hmi/cfg or
    /user/sinumerik/hmi/cfg.

  3. Open the copy of file "slsidescreen.ini", and insert the following line under the Section [sidescreen]:

    PROPERTY003=name:=mirrorMode, type:=bool, value:="true"

    Note

    If you use a copy of the configuration file from directory siemens/sinumerik/hmi/cfg under Section [Sidescreen], in the line with the keyword PROPERTY003 change value:="false" to value:="true".

  4. Save the setting.
    After run-up, the sidescreen view is located on the right-hand side.
