Using the toolbar

The toolbar provides quick access to the functions required to work with the trace diagram. Each symbol has a tooltip that indicates its function. When symbols are deactivated, the tooltip text changes and displays the reason why the function is inactive.

Symbols are functionally grouped within the toolbar, separated by a bigger space. Mutually exclusive functions within a group are separated by vertical lines.

Note

Most toolbar functions can also be accessed through the right mouse button's context menu.

The trace diagram and the FFT diagram have different toolbars. In the FFT toolbar certain symbols that are relevant only for the trace diagram are not displayed, e.g., start/stop trace, or symbols to open other widgets.
