Setting up an NX server

You enter the access data for your NX server in window "Set up NX server".

The connection to the NX server is encrypted.

Settings

Server name

Logical name of the NX server or the IP address

User name

Access data to the NX server: The email address that was specified under "Email" when the user was created for the NX server

Password

Password characters are displayed with "●".

Port

Default setting, port "443"

Procedure

 

1.

Select the "Setup" operating area.

 

2.

Press softkeys "HMI" and ">>".

 

3.

Press softkey "Set up NX server".

Window "Set up NX server" opens.

 

4.

Enter the server name, port (443) and the access data.

 

5.

Click "OK" to confirm your entries.

Note

When opening a CAD file with the "Import from CAD" function, a dialog is displayed with the certificate verification. Check the keys displayed and verify the match using the "Trust" softkey.
