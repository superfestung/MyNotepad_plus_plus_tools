Displaying and editing the logbook

Procedure

 

1.

Select the "Diagnostics" operating area.

 

2.

Press the "Version" softkey.

 

3.

Press the "Logbook" softkey.

The "Machine logbook" window opens.

Editing end customer data

 

4.

You have the option of changing the address data of the end customer using the "Change" softkey.

 

 

- OR -

 

 

Using the "Clear" softkey, you can delete all logbook entries.

 

 

All entries, except the date of the first commissioning, are deleted. Softkey "Clear" is deactivated.

Note

Deleting logbook entries

As soon as the 2nd commissioning has been completed, the "Clear" softkey to delete the logbook data is no longer available.
