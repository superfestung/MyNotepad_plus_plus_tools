Change user group

Introduction

If you want to change a new user group, you can use the "Change user groups" dialog.

Requirement

  • You have the role "Security admin manufacturer" or "Security admin end user".

Procedure

  1. In the "Setup" operating area, call the user management via the "Security > User management" softkeys.

  2. Click the "User groups" softkey. On the overview page, click on "Change" to change an existing user group.

  3. Implement your changes regarding name, description, assignment to machine roles, or assignment of users. If you change a central user group, using "Select", then you can change the name for the central user group from the central UMC server.

  4. Click "OK" to apply the changes and close the dialog.
