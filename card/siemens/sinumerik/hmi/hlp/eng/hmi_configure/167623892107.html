User logout

Introduction

If you want to log out as a user or change the user, you can use the logout dialog.

Requirement

  • You are logged into the system as a user.

Procedure

  1. Click on the login button at the top right in SINUMERIK Operate and select "Log out" in the context menu. The logout dialog opens.
    Alternatively, you can use the <Ctrl-I> shortcut to open the logout dialog.


    Login button

  2. Confirm the confirmation prompt with "OK".

Result

You are logged out as a user in SINUMERIK Operate. When logging out, your user rights revert to "Access level 7 / Key switch 0". At the instant you log out, you jump back to the root screen of the machine.

You can now log in again or change the user.
