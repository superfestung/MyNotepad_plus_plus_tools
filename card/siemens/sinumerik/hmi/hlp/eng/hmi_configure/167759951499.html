Delete users

Introduction

If you want to delete a user, you can use the "Delete user" dialog.

Requirement

  • You have the role "Security admin manufacturer" or "Security admin end user".

Procedure

  1. In the "Setup" operating area, call the user management via the "Security > User management" softkeys.

  2. Click the "User" softkey.

  3. In the overview page of users, select a user you want to delete.

  4. Click "Delete".

  5. Confirm the confirmation prompt with "OK". The user is removed from the list of all users.

Result

The user is deleted from the list of users.
