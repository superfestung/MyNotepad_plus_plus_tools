Deleting machine roles

Introduction

If you want to delete a new machine role, you can use the "Delete machine roles" dialog.

Requirement

  • You have the "Security admin manufacturer" role.

Notice

Possible effects on the end user domain

Before deleting a machine role, keep in mind that deleting shared machine roles (i.e., machine roles that can be used by both manufacturer and end user) will also affect the end user domain.

  • Therefore, carefully consider whether you really want to delete machine roles.

Procedure

  1. In the "Setup" operating area, call the user management via the "Security > User management" softkeys.

  2. Click the "Machine roles" softkey.

  3. In the machine roles overview page, select a machine role that you want to delete.

  4. Click "Delete".

  5. Confirm the confirmation prompt with "OK". The machine role is removed from the list of all machine roles.

Result

The selected machine role is deleted.
