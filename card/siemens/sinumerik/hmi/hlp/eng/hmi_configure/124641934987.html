Transferring the data to the EUNA database

Preconditions

You have generated an XML file with the configuration files of the machine.

You have installed the "PridaNet:identSNAPSHOT" software on your PC/PG (download address: www.siemens.com/sinumerik/register).

You have registered as a user once.

Procedure

Note

"PridaNet:identSNAPSHOT" user manual

For further information on transferring data to the database, please refer to the notes in the online help.

1.

Start the "PridaNet:identSNAPSHOT" software on your PC/PG.

2.

Load the XML file with the configuration files.

3.

Transfer the data to the database.
