Editing a PLC symbol or address

In an editable trace configuration, you can change the PLC addresses or PLC symbolic names that are already in the signal table.

Procedure

  1. To switch to the edit mode, click twice on the field in column "Address/symbol" or select using the keyboard and press "Insert".

  2. Enter a new PLC address (or a new symbol) or change a part of the existing address or the existing symbol.

    An existing PLC variable can only be replaced by another valid PLC variable. If the edited address is invalid or is not a PLC variable, an error message is displayed and the trace configuration is not changed.

If the PLC address is changed, then the description and data type of the variable are automatically updated. All measured values related to the changed variable are deleted.

Note

OPI addresses of other variable types, such as servo variables, cannot be edited. If you wish to change a variable such as this, then delete it from the configuration and add a new variable.
