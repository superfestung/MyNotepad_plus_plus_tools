Changing the password

Introduction

If you want to change your personal user password, you can use the "Change password" dialog.

Requirement

  • You have any user role in the user management and are logged into the system with user name and password.

Procedure

  1. Click on the login button at the top right in SINUMERIK Operate and select "Change password" in the context menu. 


    Changing the password

  2. First, enter your current password for authorization.

  3. Then assign a new password. Observe the password rules specified in the dialog.

  4. Repeat the new password.

  5. Confirm the dialog with "OK".

Result

The personal password of the user is now changed. The user can log in with the new password from now on.
