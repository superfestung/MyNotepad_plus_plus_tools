Using the time server

You can connect a time server to the control system to ensure that the correct time information is always automatically available to the control system. Make your settings in the "Set Date and Time" window.

Requirement

You require as a minimum access level 3 (user) to use time servers.

Checking the connection status

The icon on the right in field "Name / IP address" shows the status of the server connection.

Symbol

Meaning

 

The settings have just been configured, but have still not been processed by the system. The connection to the time server is established.

 

There is an active connection to the selected time server. The system is successfully synchronizing times. The time server specifies the time.

 

Synchronization with the selected time server is not active. The PLC specifies the time (default setting) (no connection to the time server).

Note

The system checks the time server connection cyclically. If a change is made, it may take a few seconds for the setting to take effect.

Procedure

 

1.

Select the "Setup" operating area.

 

2.

Press the "HMI" softkey.

 

3.

Press the "Date/Time" softkey.

The "Set Date and Time" window opens.

 

4.

For SINUMERIK Operate in the NCU:

Activate check box "Use time server (UTC time)".

For SINUMERIK Operate on the IPC/PC:

Activate check box "Use time server for NCU / PU time (UTC time)".

Field "Name / IP address" is displayed.

 

5.

For SINUMERIK Operate in the NCU:

In field "Name / IP address", enter the DNS name or IP address of the time server.

You can enter up to three time servers separated by blanks.

For SINUMERIK Operate on the IPC/PC:

Select an option in field "Name / IP address".

For IPC, if available:

  • pool.ntp.org - standard time server

  • ETH1 - IP address company network

  • ETH2 - IP address system network

You can also enter your own time server (DNS name or IP address).

For IPC: One of the local Ethernet adapters

 

6.

Press the "Accept" softkey.

If a connection to a time server exists, the settings will be synchronized with the system immediately. After that, a cyclic check is made as to whether the settings are effective. The "Set date and time" window remains open.

Softkey "Accept" is only active if you have made changes.

 

7.

Confirm the entry with the "OK" softkey.

The time server settings and the current times are taken over from the control.

Note

If check box "Use time server (UTC time)" is deactivated, then you must manually set the date and the time.
