SMB security settings

SMB is a network protocol for file, print and other services. The NCU uses this to access files on Windows shares You can configure several SMB protocol options

  • Allow SMB V1 in auto-negotiation. If this option is enabled, then it is possible that the NCU uses outdated/insecure protocol version V1

  • "Use SMB encryption" facilitates the encryption of transferred files, supported since V3

You have the option of setting the security-relevant SMB protocol settings via the SINUMERIK ONE user interface on the control system.

Requirement

  • With active user management: You have the role "Security Admin" or access level 2 (service).
    OR

  • Without active user management: You have at least access level 3 (user).

Procedure

  1. In the "Setup" operating area, switch to the "Security settings" display via
    "MENU SELECT > Setup > Security > HMI security".

  2. Click the "Change" softkey under "SMB on NCU" and select the required security settings for the SMB protocol:

    • Permit SMB V1 in auto negotiation (permit SMB V1 in the negotiation of the protocol version)

    • Use SMB encryption (only from SMB V3)

  3. Confirm with "OK".

  4. Restart the system.

    The SMB security settings are active.
