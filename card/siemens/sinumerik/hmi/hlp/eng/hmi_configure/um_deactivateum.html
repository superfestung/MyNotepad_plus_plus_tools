Deactivating user management

Introduction

If you want to disable the user management function, you can do this through the user management settings.

Notice

Data loss

Please note that you will irrevocably delete all configured user management data when you deactivate it. This includes:

  • Domains

  • Settings in the "Policies" dialog

  • Machine roles

  • Groups

  • Users

Therefore, carefully consider whether you really want to deactivate the user management or whether you have saved the data in the security archive.
Notice

Old access level passwords active again

After deactivating the user management, the access level passwords that were valid before activating the user management in the NCU are active again.

  • Therefore, disable the user management only if you are sure that you know the old access level passwords. If you no longer know the old access level passwords, you must reset the NCU to the factory default state!
Notice

Automatic deactivation of user management by unpairing

As soon as you remove the relationship of trust between the IPC and NCU by unpairing, secure communication between the two devices is no longer possible.

  • The user management on the NCU is automatically deactivated as soon as you actively terminate the pairing. The user management on the IPC is still active.

  • After unpairing, the access level passwords that were valid before activating the user management in the NCU are active again.

  • Therefore only unpair the NCU from the IPC if you are sure that you know the old access level passwords. If you no longer know the old access level passwords, you must reset the NCU to the factory default state!

Requirement

  • You have the "Security admin manufacturer" role.

Procedure

Deactivating user management

  1. In the "Setup" operating area, call the user management via the "Security > User management" softkeys.

  2. Click the "Configuration" softkey to call the user management settings.

  3. Click on "Deactivate user management". The confirmation prompt opens.

  4. If you are sure that you want to deactivate the user management, confirm your selection with "OK".

Result

The user management function is now disabled. The manufacturer domain is deleted. The classic form of access level management is now active again. The passwords of the access level that were valid before user management was activated are restored.
