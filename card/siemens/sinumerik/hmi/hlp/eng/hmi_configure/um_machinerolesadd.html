Add new machine role

Introduction

Use dialog "Add new machine role" if you wish to create a new machine role.

Requirement

  • You have the "Security admin manufacturer" role.

Procedure

  1. In the "Setup" operating area, call the user management via the "Security > User management" softkeys.

    Note

    Alternatively: Using standard machine roles

    If you do not want to create a completely new machine role, machine roles have already been precreated by default for each access level and security admin. With the role "Security admin manufacturer", you can alternatively take over, modify, or delete these roles.

  2. Click the "Machine roles" softkey. Click on "+ Machine role" on the overview page to create a new machine role. A pop-up dialog box opens.


    Creating a new machine role

  3. Assign any name for the machine role and a description that is as fitting as possible and describes the rights of the machine role.

    Note

    Note that special characters are not allowed in machine role names.

  4. Use the "May be used by end user" check box to select whether the user may also use/assign this machine role. This applies to all machine roles with the exception of machine role "Security Admin". This role cannot be shared.

    Note

    As the manufacturer, bear in mind that the Security Admin User can only assign those roles for which the checkmark is set here.

    • Therefore, plan carefully in advance which roles/rights should be reserved for the manufacturer only.

  5. Select a SINUMERIK role from the list.

  6. If you have already created user groups, you can define which user group(s) should be assigned the machine role.

    • If several machine roles with different SINUMERIK access levels are assigned to a user group, the user always receives the rights of the machine role with the highest access level.

    • It therefore only makes sense to assign several machine roles to a user group if they supplement the SINUMERIK access level, e.g. through the SINUMERIK role "Security Admin" or by giving the machine roles other rights in other parts of the system (e.g. OA applications, PLC, etc.).

  7. Click on "OK" to add the new machine role to the list of machine roles. If necessary, create additional machine roles by repeating steps 2-5.

Result

One or more machine roles have now been created and are displayed in the list of machine roles.
