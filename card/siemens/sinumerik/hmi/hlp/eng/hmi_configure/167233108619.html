Changing machine roles

Introduction

If you want to change a machine role, you can use the "Change machine roles" dialog.

Requirement

  • You have the "Security admin manufacturer" role.

Procedure

  1. In the "Setup" operating area, call the user management via the "Security > User management" softkeys.

  2. Click the "Machine roles" softkey. On the overview page, click on "Change" to change an existing machine role.

  3. Make your changes regarding name, description, sharing the role with the end user or assignment to SINUMERIK access rights or assigning to user groups.

  4. Click "OK" to apply the changes.

Result

The selected machine role is changed.
