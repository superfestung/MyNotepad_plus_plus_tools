HMI on PCU 50.3 cannot establish a network connection to the NC

Error

Error cause 1

Remedy

HMI displays alarm 120202 "Wait for connection to NC/PLC" and/or no values (only '#').

The IP address of the target NCK/PLC
set on the PCU is incorrect.

From the HMI, correct the target address for NCK/PLC and restart the PCU.
