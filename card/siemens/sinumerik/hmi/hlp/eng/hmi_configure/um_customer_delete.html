Deleting a local end user domain

Introduction

You can use dialog "Delete end user domain" to delete a local end user domain.

Notice

Data loss

Please note that if you delete the domain, then you will irrevocably lose all configured guidelines, machine roles, user groups and users of the domain.

If the local end user domain is connected with a central user management, then this connection is automatically disconnected when deleting the domain.

Therefore, carefully consider whether you really want to delete the domain or if you have backed up the data in the security archive.

Requirement

  • User management is activated.

  • A local domain for the manufacturer is defined and the security administrator for the manufacturer is set up.

  • A local domain for the end user is defined and the security administrator for the end user is set up.

  • You have the "Security admin manufacturer" role.

Procedure

Deleting a local end user domain

  1. In the "Setup" operating area, call the user management via the "Security > User management" softkeys. The overview page for user management opens in the "Configuration" softkey.

  2. Click on "Delete end user domain".

  3. Confirm the confirmation prompt with "OK".

Result

The local end user domain is deleted. All users and settings of the domain were also deleted, unless you previously backed up the data in the security archive.
