Selecting the active signal

The following table shows the different possibilities to select the active signal:

Keyboard

Mouse

  • Pressing up/down arrows in the legend widget

  • Pressing "numpad 5" in a row of the signal table (when NumLock is not pressed)

  • Only with SINUMERIK Operate on the controller: Press the shift key in a row of the signal table

  • Clicking the signal curve in the diagram

  • Clicking the signal name in legend widget

  • Double-clicking a cell containing text in a row of the signal table

Changing the active signal displays the corresponding vertical axis (if the "scale of selected signal" option is active in the context menu). It can also affect the appearance of the grid, as the grid lines are attached to the vertical scale of the active signal.

When the measurement cursors are operating in "active signal mode", they will be automatically assigned to the new active signal. When the cursors are in the "individual mode", then changing the active signal has no effect on which signals the cursors are assigned to.

In the signal table, double-clicking will only change the active signal if the selected grid cell contains text. The active signal is not changed when double-clicking a cell containing a dropdown list or a symbol.
