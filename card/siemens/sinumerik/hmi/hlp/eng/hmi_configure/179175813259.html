Security archives / Security restore password

Requirement

  • With active user management: You have the role "Security Admin" or access level 2 (service).

OR

  • Without active user management: You have at least access level 3 (user).

Enter password

SINUMERIK ONE has a "Security" archive component to back up and restore confidential data in a DSF file.

This data is encrypted before an archive is created. An individual key is generated to do this. To do this, a password for security archives or the security restore password must be entered under "Setup > Security > Security settings > HMI security".

The security restoration password is also required for restoring the SD card encryption if a part has to be replaced.

Caution

Know-how loss

If the password for security archive/security restore password is guessed, there is a threat of know-how loss and data misuse as all important security data is stored in the security archive.

Therefore always use the most secure password possible, which corresponds to the high standards for using secure passwords.

  • You can obtain tips for the assignment of secure passwords, for example, from the Federal Office for Information Security ().
Notice

Security archives / Security restore password

Keep the password in a secure place as it is required for restoring the security archive. It will not be possible to restore the security archive and the SD card encryption if the password is not known!

If you have not assigned the security restore password, then SD card encryption cannot be activated.

If you delete the security restore password, when hardware is replaced, SD card encryption cannot be re-established.

Observe the following password policies for the security archives / security restore password:

  • Minimum password length: 8 characters

  • Minimum number of digits: 1

  • At least one uppercase letter and one lowercase letter

This password must be entered when restoring the security archive.

Note

Access level "User" or role "Security admin manufacturer" or "Security admin end user" is required to change the security archives / security restore password.
