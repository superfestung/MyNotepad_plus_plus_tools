Using the legend widget

The legend widget can be opened and closed via the toolbar or the context menu. It can also be closed using the symbol on the window frame.

The legend can be displayed as a dockable window in the trace widget. On a user interface with a PC/IPC system, the window can also be moved freely.

The legend widget contains the following information:

  • Labels of signals that are currently visible in the trace diagram (when the visibility check boxes are deactivated).

  • Labels of signals that have measured data, whose visibility can be switched on or off (when the visibility check boxes are activated).

The signal label consists of the name, axis identifier or OPI index and physical unit of the signals, in the same format as displayed in the vertical trace scale.

Clicking on the name of a visible signal in the legend widget selects it as the active signal. This procedure is similar to clicking on the signal curve.

The check boxes in front of the signal names can be used to show or hide the signals in the trace diagram. The colored squares indicate the color of the signal curve.

Symbols

Functionality of the symbols on the window frame:

 

Close the legend widget.

 

Toggle between docked and freely movable window (symbol with horizontal or vertical pin).

 

Toggle between displaying signal labels and signal comments.

 

Display or hide the visibility check boxes. The next two symbols are visible only when check boxes are activated.

 

Display all signals.

 

Hide all signals.

If the "Display signal comments" is activated, instead of the signal label, the user comment from the signal table is displayed in the legend widget. (OPI indexes and physical units are not displayed after a comment.) Comments for a signal can be entered in the signal table.

When the mouse cursor hovers over a signal label, a tooltip is displayed with the full signal label. If the signal has a comment, the tooltip displays both the signal label and the comment, irrespective of the status of the "Display signal comments" symbol.

When the "Arrangement in tracks" symbol on the toolbar is pressed, the signal names in the legend widget are automatically re-arranged, so that the order of names matches the new order of signal curves in the trace diagram. If the visibility check boxes are activated, any invisible signals are set to the end of the list.
